# 🍴 현대 레시피 북 - 요리 커뮤니티

## 📝 기능 소개

현대 레시피 북은 누구나 요리법을 공유하고 배우는 열린 커뮤니티 공간입니다.
초보자부터 전문가까지, 일상 속 요리부터 특별한 날의 레시피까지 모두 모여있어요.

## 📚 현재 사용 가능한 기능들

### 레시피 열람 및 검색
- 최신순으로 레시피 목록 자동 정렬
- 난이도별 필터링 (⭐~⭐⭐⭐⭐⭐)
- 조리시간별 필터링 (15분, 30분, 1시간 등)
- 재료명으로 검색 가능

### 레시피 상세 정보
- **기본 정보**: 제목, 설명, 썸네일, 난이도, 조리시간
- **영양 정보**: 칼로리, 탄수화물, 단백질, 지방, 나트륨
- **재료 목록**: 구조화된 재료명, 수량, 단위
- **조리 단계**: 번호순으로 정리된 단계별 설명
- **별점 평가**: 1-5점 평점 시스템
- **댓글**: 사용자들 간 의견 공유

### 레시피 업로드
- 사진과 함께 레시피 작성
- 난이도 및 조리시간 설정
- 재료 목록 구조화 입력
- 조리 단계별 설명 작성

### 요리 모드
- 타이머 기능으로 단계별 요리 지원
- 재료 체크리스트로 진행도 추적
- 단계별 사진 및 설명 표시

## 💡 가상 사용 시나리오

### 시나리오 1: 새로운 레시피 배우기
```
1. 홈페이지 → "레시피 북" 메뉴 클릭
2. "김치찌개" 검색 또는 스크롤로 최신 레시피 탐색
3. 마음에 드는 레시피 클릭 → 상세 페이지 진입
4. 재료 목록 확인 → 영양 정보 체크 (칼로리, 영양소)
5. "요리 시작하기" 버튼 클릭 → 요리 모드 진입
6. 단계별 타이머와 체크리스트로 요리 진행
7. 완성 후 별점 평가와 댓글 남기기
```

### 시나리오 2: 나만의 레시피 공유하기
```
1. "새 레시피 만들기" 버튼 클릭
2. 레시피 기본 정보 입력:
   - 제목: "초간단 두부조림"
   - 난이도: ⭐⭐ (쉬움)
   - 조리시간: 15분
   - 설명: "아이들도 좋아하는 단짠단짠 맛"
3. 재료 추가:
   - 연두부 1모 (300g)
   - 간장 3큰술, 설탕 1큰술
   - 참기름 1작은술
4. 조리 단계 작성:
   1. 두부를 먹기 좋은 크기로 썰기
   2. 간장, 설탕, 물 섞어 양념 만들기
   3. 두부에 양념 부어 약불로 끓이기
5. 사진 업로드 후 "게시하기" 클릭
6. 다른 사용자들의 평가 기다리기
```

### 시나리오 3: 건강식 레시피 찾기
```
1. 검색창에 "두부" 입력
2. 난이도 "쉬움" 필터 적용
3. 조리시간 "15분 이내" 필터 추가
4. 영양 정보 확인 (저칼로리, 고단백)
5. 비슷한 재료의 다른 레시피 추천 받기
```

## 🔍 검색 및 필터링 기능

### 검색 옵션
- **키워드 검색**: 레시피 제목, 설명, 재료명
- **난이도 필터**: 1점(⭐) ~ 5점(⭐⭐⭐⭐⭐)
- **조리시간 필터**: 15분, 30분, 1시간, 2시간+
- **최신순 정렬**: 기본 정렬 (추천순은 추후 구현 예정)

### 검색 예시
```
"두부" 검색 → 두부 관련 모든 레시피
"쉬움" + "15분" 필터 → 초보자용 빠른 요리
"김치찌개" → 정확한 레시피명 검색
```

## 🤔 사용자 피드백 포인트

### 👍 좋은 점
- 단계별로 요리가 쉬워서 초보자도 따라하기 좋음
- 사진이 있어서 시각적으로 이해하기 쉬움
- 타이머와 체크리스트로 요리 과정이 체계적임

### 😅 개선 희망사항
- 영양 정보가 더 자세했으면 좋겠음 (GI 지수, 비타민 등)
- 재료를 클릭하면 구매 링크로 연결되면 편할 듯
- 비슷한 레시피 추천 기능이 있었으면...
- 전문가 검증 마크가 있었으면 신뢰성이 높아질 듯

### 💡 추가 기능 제안
- **상업화 인증 시스템**: 전문가가 검증한 레시피에 인증 마크
- **UGC 마켓**: 인증 레시피를 밀키트로 판매
- **레시피 챌린지**: 월간 테마별 요리 대회
- **AI 재료 대체**: 냉장고 속 재료로 가능한 요리 추천
- **식단 연동**: 식단 추천과 레시피 북 연계
