# 🧠 AI 맞춤 식단 큐레이션 - 건강한 식생활 파트너

## 💊 기능 소개

AI 맞춤 식단은 개인의 건강 상태와 취향에 맞는 최적의 식단을 추천해주는 지능형 서비스입니다.
당신의 건강 데이터를 분석하여 칼로리, 영양소, 질병 제한사항까지 고려한 맞춤 식단을 제공합니다.

## 🏥 현재 사용 가능한 기능들

### 건강 정보 입력
- **기본 신체 정보**: 나이, 키, 몸무게, 성별, 활동량
- **질병 정보**: 당뇨, 고혈압, 고지혈증, 신장질환, 통풍 등
- **알레르기 정보**: 우유, 계란, 땅콩, 견과류, 생선, 조개류 등
- **식재료 선호도**: 좋아하는 음식과 싫어하는 음식 설정

### 가족 구성원 관리
- 가족 구성원별 개별 건강 정보 관리
- 어린이(0-18세) 자동 감지 및 성장기 식단 고려
- 각 구성원의 질병과 알레르기 별도 저장
- 가족 전체 통합 식단 + 개인별 식단 제공

### AI 식단 추천
- 건강 정보 기반 일일 식단 자동 생성
- 아침/점심/저녁/간식별 추천 레시피
- 영양소 정보 표시 (칼로리, 탄단지, 지방, 나트륨)
- 질병별 제외 음식 자동 필터링

### 질병별 음식 제한 자동화
```
당뇨병 환자 → 설탕, 꿀, 탄산음료, 과자 자동 제외
고혈압 환자 → 소금, 간장, 햄, 치즈 자동 제외
신장질환 환자 → 바나나, 오렌지, 고구마 자동 제외
```

## 💡 가상 사용 시나리오

### 시나리오 1: 당뇨 환자의 맞춤 식단
```
1. "건강 관리" → "프로필 관리" 클릭
2. 개인 정보 입력:
   - 나이: 55세, 키: 170cm, 몸무게: 75kg
   - 질병: 당뇨병 체크
   - 알레르기: 땅콩 알레르기
   - 선호도: 채식 위주, 매운 음식 좋아함
3. "식단 추천" 페이지 이동
4. "오늘의 식단 생성" 버튼 클릭
5. AI가 분석 후 추천:
   아침: 현미밥 + 나물 + 두부조림 (저GI, 저칼로리)
   점심: 불고기 (설탕 대신 올리고당 사용)
   저녁: 김치찌개 (저염 조리)
   간식: 견과류 (땅콩 제외)
6. 각 식단의 영양 정보와 칼로리 확인
```

### 시나리오 2: 가족 맞춤 식단 (다둥이 부모)
```
1. "가족 관리" 메뉴 클릭
2. 가족 구성원 추가:
   - 아버지: 50세, 고혈압, 소금 싫어함
   - 어머니: 48세, 건강함, 채식 선호
   - 첫째: 15세, 성장기, 알레르기 없음
   - 둘째: 12세, 아토피, 우유 알레르기
3. "가족 식단 생성" 선택
4. AI가 각 구성원 고려하여 식단 생성:
   - 가족 공통 식단: 저염 김치찌개 + 밥 + 나물
   - 아버지 특화: 별도 저염 반찬 추가
   - 어머니 특화: 채식 옵션 제공
   - 아이들 특화: 성장기 영양소 강화
5. 가족 전체 식단표 + 개인별 식단표 함께 제공
```

### 시나리오 3: 다이어트 식단
```
1. 건강 정보에 체중 감량 목표 설정
2. 일일 칼로리 목표: 1,800kcal 설정
3. 활동량: "적당한 운동" 선택
4. AI가 저칼로리 식단 추천:
   - 고단백 식품 위주 (두부, 닭가슴살, 생선)
   - 채소 비율 높임
   - 간식으로 과일 대신 견과류
5. 주간 식단 패턴 제공으로 식단 지루함 방지
```

## 📊 영양소 분석 및 제한

### 자동 필터링 질병 목록
- **당뇨병**: 고당류 음식 자동 제외 (설탕, 꿀, 시럽, 과자 등)
- **고혈압**: 고염분 음식 자동 제외 (소금, 간장, 햄, 치즈 등)
- **고지혈증**: 고지방 음식 자동 제외 (버터, 마요네즈, 튀김 등)
- **신장질환**: 칼륨/인 제한 음식 자동 제외 (바나나, 고구마, 우유 등)
- **통풍**: 퓨린 함량 높은 음식 자동 제외 (등푸른생선, 내장, 맥주 등)

### 영양소 균형 목표
```
칼로리: 개인 목표치에 맞춤 (예: 2,000kcal)
탄수화물: 50-60% (주 에너지원)
단백질: 15-20% (근육 유지)
지방: 20-30% (필수 영양소)
나트륨: 2,000mg 이하 (권장)
```

### 어린이 성장기 특화
```
탄수화물: 50% (에너지 공급)
단백질: 20% (성장 촉진)
지방: 30% (두뇌 발달)
칼로리 분배: 아침 25%, 점심 35%, 저녁 30%, 간식 10%
```

## 👨‍👩‍👧‍👦 가족 관리 상세

### 구성원별 프로필 관리
- **이름과 관계**: 아버지, 어머니, 첫째, 둘째 등
- **생년월일**: 나이 자동 계산, 어린이 감지
- **건강 정보**: 질병, 알레르기, 키/몸무게, 활동량
- **식습관**: 채식, 육식, 매운 음식 선호도 등

### 식단 생성 옵션
- **개인 식단**: 각 구성원별 맞춤 식단
- **통합 식단**: 가족이 함께 먹을 수 있는 식단
- **하이브리드**: 공통 식단 + 개인별 추가 반찬

## 🤔 사용자 피드백 포인트

### 👍 좋은 점
- 건강 상태를 입력하면 자동으로 위험 음식 제외됨
- 가족 구성원별로 개별 관리 가능
- 영양소 정보가 자세하게 나와서 신뢰감 있음
- 어린이 성장기 식단 고려되는 점 좋음

### 😅 개선 희망사항
- 식단이 매일 자동으로 생성되지 않고 수동으로 눌러야 함
- 생성된 식단의 영양 목표 달성도를 분석해주면 좋겠음
- 추천 식단의 재료를 바로 구매할 수 있는 기능이 없음
- 같은 식단이 반복되어 지루할 때가 있음
- 영양사 상담 기능이 있었으면...

### 💡 추가 기능 제안
- **실시간 알림**: 매일 아침 5시에 자동 식단 생성 및 팝업
- **영양 리포트**: 주간/월간 영양 섭취 분석 및 목표 달성도
- **식자재 연동**: 추천 재료를 바로 장바구니에 담기
- **GI 지수 필터링**: 혈당 관리용 저GI 식단 특화
- **식단 커스터마이징**: 추천 식단을 개인 취향에 맞게 조정
- **식단 일기**: 실제 섭취한 음식 기록 및 건강 데이터 분석

- 가족 구성원 관리 (최대 5명)

- 전문 영양사 1:1 상담 (월 1회)
- 식자재 원클릭 구매 연동
- GI 지수 기반 정밀 식단
- 무제한 가족 구성원 관리

## 📈 건강 데이터 분석

### 현재 추적 가능한 건강 지표
- 일일 칼로리 섭취량

- 나트륨 섭취량
- 식단의 질병 적합성 점수
