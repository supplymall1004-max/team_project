# 🍴 맛의 아카이브 (Flavor Archive) - 사용 설명서

## 📖 소개

**맛의 아카이브**는 요리의 과거와 현재, 미래를 연결하는 디지털 플랫폼입니다.

- **레거시 아카이브**: 잊혀져가는 전통 요리 지식을 보존하고 전수
- **현대 레시피 북**: 누구나 참여하는 요리 커뮤니티
- **AI 맞춤 식단**: 건강과 취향에 맞는 개인화된 식단 추천

---

## 🎯 주요 페르소나별 사용 가이드

### 1. 전통 요리 학습자 페르소나
*"어려운 시대에 구전으로만 전해지던 할머니의 손맛을 배우고 싶어요"*

#### 현재 사용 가능한 기능들

**🏛️ 레거시 아카이브 탐방**
- **명인 인터뷰 영상 시청**: 지역별, 시대별로 분류된 전통 명인의 조리 시연 영상
- **전문 문서화 자료 열람**: 영상 내용을 전문가가 정리한 레시피와 조리법
- **대체 재료 가이드**: 전통 재료를 현대 마트에서 쉽게 구할 수 있는 재료로 대체하는 방법

**💡 가상 사용 시나리오**
```
1. 홈페이지 접속 → "레거시 아카이브" 섹션 클릭
2. "경북 안동" 지역 선택 → "전통 장 담그기" 영상 재생
3. 영상 하단의 "문서화 자료" 버튼 클릭 → 자세한 레시피 확인
4. "대체 재료" 탭에서 현대 재료로 변환된 조리법 확인
```

#### 불편한 점이나 개선 희망사항
- 프리미엄 콘텐츠가 있어 무료로 모든 전통 지식을 볼 수 없음
- 영상이 많아 찾고 싶은 내용 검색이 어려움
- 지역별/재료별 필터링이 더 세밀했으면 좋겠음

---

### 2. 건강 관리 식단 사용자 페르소나
*"당뇨가 있는데 맛있는 음식을 먹으면서 건강도 챙기고 싶어요"*

#### 현재 사용 가능한 기능들

**🏥 건강 정보 입력**
- 기본 신체 정보 (나이, 키, 몸무게, 활동량)
- 질병 정보 입력 (당뇨, 고혈압, 고지혈증 등)
- 알레르기 및 선호/비선호 식재료 설정

**👨‍👩‍👧‍👦 가족 구성원 관리**
- 가족 구성원별 건강 정보 개별 관리
- 각 구성원의 질병과 알레르기 별도 저장
- 어린이(0-18세) 자동 감지 및 성장기 식단 고려

**🍽️ AI 맞춤 식단 추천**
- 건강 정보 기반 일일 식단 자동 생성
- 아침/점심/저녁 별 추천 레시피
- 칼로리, 탄수화물, 단백질, 지방 영양소 정보 표시

**💡 가상 사용 시나리오**
```
1. "건강 관리" 메뉴 → "프로필 관리" 클릭
2. 본인 정보 입력: 나이 45세, 당뇨병 체크, 키/몸무게 입력
3. "가족 관리"에서 자녀 정보 추가: 12세 어린이, 알레르기 없음
4. "식단 추천" 페이지에서 "오늘의 식단 생성" 버튼 클릭
5. 가족 전체 식단 + 개인별 식단 함께 확인
```

#### 불편한 점이나 개선 희망사항
- 식단 추천이 매일 자동으로 생성되지 않음 (수동으로 생성해야 함)
- 추천받은 식단의 영양소 목표 달성도를 확인하기 어려움
- 식재료를 바로 구매할 수 있는 기능이 없음

---

### 3. 일반 요리 창작자 페르소나
*"내가 만든 요리법을 사람들과 공유하고, 수익도 내고 싶어요"*

#### 현재 사용 가능한 기능들

**📝 레시피 업로드**
- 사진과 함께 단계별 조리법 작성
- 난이도(1-5점), 조리시간, 재료량 설정
- 재료 목록 구조화 입력

**🔍 레시피 검색 및 열람**
- 최신순으로 레시피 목록 조회
- 재료명, 난이도, 조리시간으로 필터링
- 별점 평가 및 댓글 기능

**👨‍🍳 요리 모드**
- 업로드된 레시피를 요리할 때 타이머 기능
- 단계별 체크리스트로 요리 진행도 추적

**💡 가상 사용 시나리오**
```
1. 회원가입 후 "레시피 북" → "새 레시피 만들기" 클릭
2. "김치찌개" 제목 입력, 난이도 3점, 조리시간 30분 설정
3. 재료 추가: 돼지고기 200g, 김치 300g, 두부 1모 등
4. 조리 단계 작성: "1. 냄비에 기름 두르고 돼지고기 볶기..."
5. 사진 업로드 후 "게시하기" 클릭
6. 다른 사람의 평가와 댓글 확인
```

#### 불편한 점이나 개선 희망사항
- 내가 만든 레시피로 수익을 낼 수 있는 방법이 없음
- 전문가 검증을 받을 수 없어서 신뢰성이 떨어짐
- 인기 레시피를 상품화할 수 있는 기능이 부족

## 💰 가격 정책 및 무료 사용 범위

### 현재 무료로 사용 가능한 기능들

| 기능 카테고리 | 무료 범위 | 제한사항 |
|---|---|---|
| **레거시 아카이브** | 일부 무료 영상 시청 | 프리미엄 영상은 유료 구독 필요 |
| **레시피 북** | 모든 레시피 열람, 업로드, 평가 | 제한 없음 |
| **건강 관리** | 기본 건강 정보 입력 | 제한 없음 |
| **식단 추천** | 수동 식단 생성 (하루 1회) | 자동 생성 없음 |
| **가족 관리** | 가족 구성원 정보 입력 | 제한 없음 |

### 향후 계획된 유료 기능들

**프리미엄 구독 (월 9,900원)**
- 모든 레거시 아카이브 영상 무제한 시청
- 매일 자동 식단 추천 및 알림
- 영양소 분석 리포트
- 우선 고객 지원

**상업화 기능 (레시피별 수수료)**
- 전문가 검증된 '인증 마크' 부여
- 레시피를 밀키트나 손맛 딜리버리로 판매
- 수익의 20% 플랫폼 수수료

## 🔮 구현 예정 기능들 (피드백 수집용)

### A. 레거시 아카이브 고도화
- **프리미엄 구독 시스템**: 모든 전통 지식 무제한 접근
- **AI 검색 엔진**: 재료나 조리법으로 관련 영상 자동 추천
- **증강 현실(AR) 데모**: 스마트폰으로 3D 조리 시연

### B. 현대 레시피 북 확장
- **상업화 인증 시스템**: 셰프/영양사 검증을 통한 공식 인증
- **UGC 마켓플레이스**: 인증 레시피를 상품으로 판매
- **요리 클래스 플랫폼**: 실시간 화상 강의 기능

### C. AI 식단 큐레이션 강화
- **실시간 식단 알림**: 매일 아침 5시에 자동 생성 및 팝업 알림
- **영양 코칭 리포트**: 주간/월간 영양소 섭취 분석
- **식자재 원클릭 구매**: 추천 식단 재료를 바로 장바구니에 담기
- **GI 지수 필터링**: 혈당 관리용 저GI 식단 특화 기능

### D. 커뮤니티 및 소셜 기능
- **요리 챌린지**: 월간 테마별 레시피 대회
- **팔로우 시스템**: 관심 요리사 팔로우 및 피드
- **요리 일기**: 자신의 식단 기록 및 건강 데이터 추적
