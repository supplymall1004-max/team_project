# 🏛️ 레거시 아카이브 - 전통 요리 지식 보존

## 📜 기능 소개

레거시 아카이브는 사라져가는 전통 요리 지식을 디지털로 보존하고 전수하는 공간입니다.
100년 장독대를 지켜온 명인들의 손맛부터 지역별 특색 있는 조리법까지, 시간과 공간을 초월한 요리 유산을 만나보세요.

## 🎬 현재 사용 가능한 콘텐츠

### 명인 인터뷰 영상
- **김연자 명인**: 안동 전통 장 담그기 (32분)
  - 100년 장독대 비법 공개
  - 계절별 메주 관리 방법
- **박정희 장인**: 전라도 한정식 상차림 (27분)
  - 산지 직송 식재료 활용
  - 전통 한정식 코스 구성

### 전문 문서화 자료
- 영상 내용을 전문 작가가 정리한 상세 레시피
- 재료 고증 및 출처 정보
- 시대별 조리 도구 사용법

### 대체 재료 가이드
```
전통 재료 → 현대 대체품
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
안동 메주   → 국산 메주 블록 (전통시장/온라인 구입)
옹기 장독   → 스테인리스 발효통
죽부인 체   → 스테인리스 체
```

## 💡 가상 사용 시나리오

### 시나리오 1: 전통 장 담그기 배우기
```
1. 홈페이지 → "레거시 아카이브" 섹션
2. 지역: "경북 안동" 선택
3. "전통 장 담그기" 영상 클릭
4. 32분 영상 시청하면서 명인의 손기술 관찰
5. 영상 하단 "문서화 자료" 탭 클릭
6. 메주 띄우기부터 장독 숙성까지 4계절 프로세스 확인
7. "대체 재료" 탭에서 현대 재료 변환 팁 확인
   → "전통 메주보다 향이 약하므로 소금량 5% 감소 권장"
```

### 시나리오 2: 지역별 요리 문화 탐방
```
1. 검색창에 "전라도" 입력
2. "전라도 한정식 상차림" 영상 발견
3. 지역 특산품(표고버섯, 들기름, 대파) 활용법 학습
4. 비슷한 지역의 다른 콘텐츠 추천 받기
```

## 🔍 검색 및 필터링

### 현재 사용 가능한 필터
- **지역별**: 경북 안동, 전남 담양 등
- **시대별**: 조선 후기, 근현대 등
- **태그별**: 발효, 장류, 명인, 한정식 등

### 검색 키워드 예시
- "장 담그기", "한정식", "발효"
- 지역명: "안동", "담양"
- 재료명: "메주", "된장", "표고버섯"

## 🎯 이 기능이 필요한 사람들

- **전통 음식 계승자**: 가족의 손맛을 잊지 않으려는 사람들
- **지역 문화 연구자**: 특정 지역의 요리 문화에 관심 있는 사람들
- **발효 음식 마니아**: 장류, 김치 등 발효 식품에 관심 있는 사람들
- **역사적 요리 탐구자**: 과거 시대의 식문화에 관심 있는 사람들
