## 1. 챕터 구조 개요

### 1.1 챕터 분류

```
홈페이지 (/)
│
├─ 📚 챕터 1: 레시피 & 식단 아카이브
│   ├─ 현대 레시피 (RecipeSection)
│   ├─ 궁중 레시피 (RoyalRecipesQuickAccess)
│   ├─ 건강 맞춤 식단 (DietSection)
│   ├─ 주간 식단 (LazyWeeklyDietSummary)
│   └─ 마카의 음식 동화 (StorybookSection)
│
└─ 💚 챕터 2: 건강 관리 현황
    ├─ 가족 건강 대시보드

    ├─ 건강 인사이트
    └─ 건강 목표 추적
```

## 3. 챕터 1: 레시피 & 식단 아카이브

### 3.1 건강 맞춤 식단 시각화 컴포넌트

건강 맞춤 식단 상세페이지에 있는 시각화 자료들을 메인페이지에 통합합니다:

#### 시각화 컴포넌트 목록
1. **HealthMetricsCard**: 현재 건강 상태 (BMI, 체지방률, 근육량, 기초대사율)
2. **NutritionBalanceChart**: 영양 균형 도넛 차트 (탄수화물/단백질/지방 비율)
3. **MealImpactPredictor**: 식단 효과 예측 (식사 전/후 건강 상태 비교)
4. **DiseaseRiskGauge**: 질병 위험도 게이지 (심혈관, 당뇨, 신장 등)

#### 메인페이지 배치 전략
- **챕터 1의 "건강 맞춤 식단" 섹션**: 시각화 미리보기 추가
  - 현재 건강 상태 요약 카드
  - 영양 균형 차트 미리보기
  - 식단 효과 예측 미리보기
- **챕터 2의 "건강 관리 현황"**: 건강 시각화 대시보드 통합
  - 건강 메트릭스 카드
  - 질병 위험도 게이지
  - 영양 균형 차트

## 11. 건강 시각화 컴포넌트 상세 사양

### 11.1 챕터 1: 건강 맞춤 식단 시각화 미리보기

#### HealthMetricsCard 미리보기
- **표시 항목**: BMI, 건강점수, 상태 등급 (⭐)
- **크기**: 컴팩트 버전 (전체 정보는 상세페이지에서 확인)
- **인터랙션**: 클릭 시 건강 맞춤 식단 상세페이지로 이동

#### NutritionBalanceChart 미리보기
- **표시 항목**: 탄수화물/단백질/지방 비율 (도넛 차트)
- **크기**: 작은 도넛 차트 (약 150px)
- **인터랙션**: 호버 시 상세 비율 표시

#### MealImpactPredictor 미리보기
- **표시 항목**: 목표 달성률, 주요 개선 포인트 1-2개
- **크기**: 컴팩트 버전
- **인터랙션**: 클릭 시 식단 상세페이지로 이동

### 11.2 챕터 2: 건강 관리 현황 시각화 대시보드

#### HealthDashboard 통합
- **표시 항목**: 가족 전체 건강 종합 정보
- **레이아웃**: 3열 그리드 (건강 메트릭스, 질병 위험도, 영양 균형)
- **인터랙션**: 각 카드 클릭 시 상세 페이지로 이동

#### DiseaseRiskGauge 통합
- **표시 항목**: 심혈관, 당뇨, 신장 질병 위험도
- **레이아웃**: 가로 배치 (3개 게이지)
- **인터랙션**: 게이지 클릭 시 상세 설명 모달

### 11.3 데이터 로딩 전략

#### 메인페이지 미리보기
- **로딩 방식**: 지연 로딩 (Lazy Loading)
- **데이터 소스**: `/api/health/metrics` (현재 건강 상태)
- **캐싱**: 5분간 캐시 (사용자 건강 정보는 자주 변경되지 않음)

#### 챕터 페이지 전체 보기
- **로딩 방식**: 즉시 로딩
- **데이터 소스**: 
  - `/api/health/metrics` (건강 메트릭스)
  - `/api/health/meal-impact` (식단 효과 예측)
- **캐싱**: 1분간 캐시 (실시간성 중요)
