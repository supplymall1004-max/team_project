## 1. 전체 네비게이션 구조

### 1.1 상단 네비게이션 (Navbar)

```
┌─────────────────────────────────────────────────────────────────┐
│ [로고] Flavor Archive  [검색창]  레시피북  AI식단  [설정] [사용자] │
└─────────────────────────────────────────────────────────────────┘
```

**현재 구조 유지** + 건강 메뉴 추가 (선택적)

### 1.2 건강 섹션 네비게이션

```
/health (건강 관리 메인)
│
├─ /health/dashboard (가족 건강 대시보드) ← 신규
│   ├─ /health/dashboard (가족 전체 개요)
│   └─ /health/dashboard/[memberId] (개인 상세)
│
├─ /health/profile (건강 프로필) ← 기존
├─ /health/family (가족 관리) ← 기존
│
├─ /health/records (건강 기록) ← 신규
│   ├─ /health/records/checkups (건강검진 기록)
│   ├─ /health/records/medications (약물 복용 기록)
│   ├─ /health/records/activities (활동량 기록)
│   └─ /health/records/nutrition (영양 섭취 기록)
│
├─ /health/premium (프리미엄 기능) ← 기존
│
└─ /health/settings (건강 설정) ← 신규
    ├─ /health/settings/privacy (개인정보 공유 설정)
    ├─ /health/settings/notifications (알림 설정)
    └─ /health/settings/integrations (API 연동 설정)
```

### 1.3 하단 네비게이션 (모바일)

```
┌─────────────────────────────────────┐
│  홈  레시피  찜  식단  마이          │
└─────────────────────────────────────┘
```

**현재 구조 유지** (건강은 "마이" 메뉴 내에서 접근)

## 7. 인터랙션 패턴

### 7.1 호버 효과

- 카드 호버 시 살짝 올라감 (transform: translateY(-2px))
- 그림자 증가

### 7.2 클릭 피드백

- 버튼 클릭 시 스케일 다운 효과
- 로딩 상태 표시

### 7.3 스크롤 애니메이션

- 스크롤 시 카드가 순차적으로 나타남 (fade-in)
- 차트는 스크롤 시 애니메이션으로 표시

## 8. 접근성 고려사항

### 8.1 키보드 네비게이션

- 모든 인터랙티브 요소는 Tab 키로 접근 가능
- Enter/Space로 활성화

### 8.2 스크린 리더

- 모든 이미지에 alt 텍스트
- 차트에 aria-label 제공
- 버튼에 명확한 라벨

### 8.3 색상 대비

- WCAG AA 기준 준수 (4.5:1 이상)
- 색상만으로 정보 전달하지 않음 (아이콘, 텍스트 병행)
