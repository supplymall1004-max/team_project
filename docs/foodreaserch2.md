## 🚀 최적의 이미지 검색 로직 (Search Logic)

Unsplash나 Pixabay 같은 해외 사이트는 한국 음식을 찾을 때 다음의 **3단계 검색 우선순위**를 따르는 것이 가장 많은 고화질 이미지를 얻을 수 있는 방법입니다.

| 순위 | 검색 방법 | 특징 및 효과 |
| :--- | :--- | :--- |
| **🥇 1순위** | **로마자 표기 (고유명사)** | **`Bibimbap`, `Jjigae`, `Banchan`** 등. 해외에서 가장 널리 쓰이는 표기이며, 전문 푸드 포토그래퍼의 사진이 많습니다. |
| **🥈 2순위** | **영어 설명 (Descriptive)** | **`Kimchi Stew`, `Spicy Beef Soup`** 등. 외국인들이 쉽게 검색하는 단어로, 일상적인 사진이나 다양한 각도의 사진을 찾을 수 있습니다. |
| **🥉 3순위** | **핵심 재료명 + 분류** | **`Spinach side dish`, `Radish kimchi`** 등. 특정 음식이 없을 때 재료 자체의 사진이나 비슷한 요리를 찾는 데 유용합니다. |

## 1. 🍚 현대 레시피 북 - 반찬 및 밥류 검색 로직

흰쌀밥, 나물, 볶음류 등 다양한 **반찬(Banchan)** 사진을 찾는 데 중점을 둔 검색어와 로직입니다.

### 📝 주요 반찬 및 밥류 검색어 목록 (활용 예시)

| 한글 메뉴명 | 1순위 (로마자 표기) | 2순위 (영어 설명) | 3순위 (재료 + 분류) |
| :--- | :--- | :--- | :--- |
| **흰쌀밥** | `Bap` | `Cooked Rice` | `Rice` |
| **현미밥** | `Hyeonmi Bap` | `Brown Rice` | `Brown Rice grain` |
| **잡곡밥** | `Japgokbap` | `Multigrain Rice` | `Grain mix` |
| **시금치나물** | `Sigeumchi Namul` | `Seasoned Spinach` | `Spinach side dish` |
| **콩나물무침** | `Kongnamul Muchim` | `Seasoned Bean Sprouts` | `Bean sprout side dish` |
| **고구마줄기볶음** | `Goguma-julgi Bokkeum` | `Stir-fried Sweet Potato Stems` | `Sweet potato stems` |
| **멸치볶음** | `Myeolchi Bokkeum` | `Stir-fried Anchovies` | `Anchovy side dish` |

### 🔍 이미지 검색 로직 구조 (사용자님 시스템에 적용할 규칙)

1.  **반찬명 로마자 검색:**
    * **예시:** `Kongnamul Muchim` (콩나물무침) 검색
    * **결과:** 이미지가 있으면 **이미지 사용 후 종료**.
2.  **반찬명 영어 설명 검색:**
    * **예시:** `Seasoned Bean Sprouts` 검색
    * **결과:** 이미지가 있으면 **이미지 사용 후 종료**.
3.  **광범위한 반찬 카테고리 검색:**
    * **예시:** `Banchan` 또는 `Korean Side Dish` 검색
    * **결과:** 콩나물무침은 아니지만, 다양한 반찬이 함께 담긴 **'반찬 세트'** 이미지를 대표 이미지로 사용할 수 있습니다.

## 2. 🍲 AI 맞춤 식단 큐레이션 - 국/찌개류 검색 로직

아침, 점심, 저녁의 **대표 이미지 (국이나 찌개)**를 찾기 위한 검색 로직입니다. 이 분류는 고유명사(로마자)의 성공률이 특히 높습니다.

### 📝 주요 국/찌개류 검색어 목록 (활용 예시)

| 한글 메뉴명 | 1순위 (로마자 표기) | 2순위 (영어 설명) | 3순위 (재료 + 분류) |
| :--- | :--- | :--- | :--- |
| **김치찌개** | **`Kimchi Jjigae`** | `Kimchi Stew` | `Kimchi soup` |
| **된장찌개** | **`Doenjang Jjigae`** | `Soybean Paste Stew` | `Doenjang soup` |
| **미역국** | **`Miyeok Guk`** | `Seaweed Soup` | `Seaweed broth` |
| **갈비탕** | **`Galbitang`** | `Beef Rib Soup` | `Beef ribs soup` |
| **육개장** | **`Yukgaejang`** | `Spicy Beef Soup` | `Beef soup` |
| **떡국** | **`Tteok Guk`** | `Sliced Rice Cake Soup` | `Rice cake soup` |

### 🔍 이미지 검색 로직 구조 (사용자님 시스템에 적용할 규칙)

1.  **국/찌개명 로마자 검색 (가장 강력):**

    * **결과:** 이미지가 있으면 **이미지 사용 후 종료**.
2.  **국/찌개명 영어 설명 검색:**
    * **예시:** `Kimchi Stew` 검색
    * **결과:** 이미지가 있으면 **이미지 사용 후 종료**.
3.  **광범위한 분류 검색:**
    * **예시:** `Jjigae` 또는 `Korean Stew` (찌개인 경우), `Guk` 또는 `Korean Soup` (국인 경우) 검색
    * **결과:** 해당 메뉴는 아니지만, **'오늘의 국/찌개'**를 대표할 수 있는 **일반적인 찌개 또는 국 이미지**를 사용할 수 있습니다.

## 💡 한글 음식명을 영어로 검색할 때의 핵심 요령

사용자님의 요청에 따라, 한글 음식명을 영어로 검색할 때 **정확한 단어**를 찾고 **검색 성공률**을 높이는 요령을 추가로 알려드립니다.

### 1. 고유명사 (로마자)를 가장 우선 사용하세요.
* **김치찌개 (O)** → **`Kimchi Jjigae`**
* **비빔밥 (O)** → **`Bibimbap`**
* **이유:** 해외 스톡 사이트 작가들은 사진을 올릴 때 표준화된 로마자 표기를 **태그(tag)**로 가장 많이 사용합니다. 이것이 이미지를 불러오는 가장 정확한 방법입니다.

### 2. 검색 시 불필요한 관형사/형용사는 빼고 명사만 검색하세요.
* **X:** `delicious kimchi jjigae` (맛있는 김치찌개)
* **O:** `Kimchi Jjigae`
* **이유:** 검색어는 간결할수록 좋습니다. 음식 이름 외의 수식어는 오히려 검색 결과를 방해합니다.

### 3. 음식의 재료가 복잡하면 재료의 메인 이름을 활용하세요.
* 예를 들어, **`고구마줄기볶음`**처럼 길고 복잡한 이름은 정확한 사진이 부족할 수 있습니다.
    * 1순위: `Goguma-julgi Bokkeum`
    * 2순위: `Sweet Potato Stems`
    * 3순위: `Stir-fried vegetables`
* **이유:** 1순위, 2순위에서 실패하면, 그 음식의 핵심인 `Sweet Potato Stems` 사진을 찾아 **'음식 재료'** 사진으로 대체하는 것도 좋은 방법입니다.
