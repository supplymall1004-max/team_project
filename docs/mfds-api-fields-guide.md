# 식약처 API 필드 가이드

이 문서는 식약처(식품의약품안전처) 레시피 API(COOKRCP01)에서 제공하는 모든 필드에 대한 상세 설명입니다.

## 📋 기본 정보 필드

### RCP_SEQ (레시피 순번)
- **타입**: String
- **설명**: 각 레시피를 고유하게 식별하는 순번(ID)
- **예시**: "1", "2", "100"
- **용도**: 레시피 상세 조회, 데이터베이스 외래키

### RCP_NM (레시피명)
- **타입**: String
- **설명**: 레시피의 이름
- **예시**: "된장찌개", "김치볶음밥"
- **용도**: 레시피 검색, 카드 제목 표시

### RCP_WAY2 (조리방법)
- **타입**: String
- **설명**: 음식을 만드는 방법 (볶음, 끓이기, 굽기, 찜, 튀김 등)
- **예시**: "볶음", "끓이기", "굽기"
- **용도**: 레시피 필터링, 카테고리 분류

### RCP_PAT2 (요리종류)
- **타입**: String
- **설명**: 요리의 종류 (밥, 국, 찌개, 반찬, 후식 등)
- **예시**: "밥", "국", "찌개", "반찬"
- **용도**: 레시피 필터링, 카테고리 분류

## 🥗 영양 정보 필드

### INFO_ENG (칼로리)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 칼로리 (kcal)
- **예시**: "250", "350.5"
- **용도**: 칼로리 계산, 식단 추천

### INFO_CAR (탄수화물)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 탄수화물 함량 (g)
- **예시**: "45.2", "30"
- **용도**: 영양소 분석, 다이어트 식단 추천

### INFO_PRO (단백질)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 단백질 함량 (g)
- **예시**: "15.3", "20"
- **용도**: 영양소 분석, 헬스 식단 추천

### INFO_FAT (지방)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 지방 함량 (g)
- **예시**: "8.5", "12"
- **용도**: 영양소 분석, 저지방 식단 추천

### INFO_NA (나트륨)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 나트륨 함량 (mg)
- **예시**: "850", "1200"
- **용도**: 고혈압 환자 식단 추천, 저염식 식단

### INFO_FIBER (식이섬유)
- **타입**: String (숫자 문자열)
- **설명**: 1인분 기준 식이섬유 함량 (g)
- **예시**: "5.2", "8"
- **용도**: 건강 식단 추천, 변비 예방

### INFO_K (칼륨) - 옵셔널
- **타입**: String (숫자 문자열) | null
- **설명**: 1인분 기준 칼륨 함량 (mg)
- **예시**: "350", "500"
- **용도**: 신장질환 환자 식단 추천 (칼륨 제한)

### INFO_P (인) - 옵셔널
- **타입**: String (숫자 문자열) | null
- **설명**: 1인분 기준 인 함량 (mg)
- **예시**: "200", "300"
- **용도**: 신장질환 환자 식단 추천 (인 제한)

### INFO_GI (GI 지수) - 옵셔널
- **타입**: String (숫자 문자열) | null
- **설명**: 혈당지수 (Glycemic Index)
- **예시**: "55", "70"
- **용도**: 당뇨 환자 식단 추천

## 🥘 재료 및 조리법 필드

### RCP_PARTS_DTLS (재료 정보)
- **타입**: String
- **설명**: 레시피에 사용되는 재료 목록 (쉼표 또는 줄바꿈으로 구분)
- **예시**: "된장 1큰술, 두부 1/2모, 대파 1대, 물 2컵"
- **용도**: 재료 표시, 장보기 리스트 생성

### MANUAL01 ~ MANUAL20 (조리 방법)
- **타입**: String | null
- **설명**: 단계별 조리 방법 설명 (최대 20단계)
- **예시**: 
  - MANUAL01: "된장을 물에 풀어주세요"
  - MANUAL02: "두부를 깍둑썰기로 자릅니다"
- **용도**: 단계별 조리 가이드, 요리 모드

### MANUAL_IMG01 ~ MANUAL_IMG20 (조리 방법 이미지)
- **타입**: String (URL) | null
- **설명**: 각 조리 단계에 해당하는 이미지 URL
- **예시**: "http://www.foodsafetykorea.go.kr/uploadimg/cook/..."
- **용도**: 시각적 조리 가이드

## 🖼️ 이미지 필드

### ATT_FILE_NO_MAIN (대표 이미지)
- **타입**: String (URL) | null
- **설명**: 레시피의 대표 이미지 URL
- **예시**: "http://www.foodsafetykorea.go.kr/uploadimg/cook/..."
- **용도**: 레시피 카드 썸네일, 상세 페이지 메인 이미지

### ATT_FILE_NO_MK (만드는 법 이미지)
- **타입**: String (URL) | null
- **설명**: 레시피 만드는 법 전체 이미지 URL
- **예시**: "http://www.foodsafetykorea.go.kr/uploadimg/cook/..."
- **용도**: 레시피 상세 페이지, 인쇄용 이미지

## 📊 필드 그룹별 분류

### 1. 기본 식별 정보
- RCP_SEQ, RCP_NM

### 2. 분류 정보
- RCP_WAY2, RCP_PAT2

### 3. 영양 정보 (필수)
- INFO_ENG, INFO_CAR, INFO_PRO, INFO_FAT, INFO_NA, INFO_FIBER

### 4. 영양 정보 (선택)
- INFO_K, INFO_P, INFO_GI

### 5. 재료 정보
- RCP_PARTS_DTLS

### 6. 조리 방법 (텍스트)
- MANUAL01 ~ MANUAL20

### 7. 조리 방법 (이미지)
- MANUAL_IMG01 ~ MANUAL_IMG20

### 8. 이미지
- ATT_FILE_NO_MAIN, ATT_FILE_NO_MK

## 💡 활용 예시

### 건강 맞춤 식단 생성
- INFO_ENG: 칼로리 목표 달성 여부 확인
- INFO_CAR, INFO_PRO, INFO_FAT: 3대 영양소 균형 확인
- INFO_NA: 고혈압 환자 나트륨 제한
- INFO_K, INFO_P: 신장질환 환자 미네랄 제한
- INFO_GI: 당뇨 환자 혈당 관리

### 주간 식단 생성
- RCP_PAT2, RCP_WAY2: 다양한 요리 종류와 조리 방법 선택
- RCP_PARTS_DTLS: 장보기 리스트 자동 생성
- INFO_ENG: 주간 칼로리 목표 달성 확인

### 레시피 상세 페이지
- MANUAL01~20: 단계별 조리 가이드
- MANUAL_IMG01~20: 시각적 조리 가이드
- ATT_FILE_NO_MAIN: 메인 이미지 표시

## 🔄 정적 파일 시스템과의 연동

이 필드들은 `docs/mfds-recipes/recipes/*.md` 마크다운 파일에 저장되어 있으며, 정적 파일 로더(`lib/mfds/static-recipe-loader.ts`)를 통해 읽어옵니다.

API 호출 없이 빠르게 데이터를 불러올 수 있어 로딩 시간이 크게 단축됩니다.
