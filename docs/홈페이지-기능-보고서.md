## 1. 홈페이지 개요

**Flavor Archive**는 가족의 건강과 식단을 통합적으로 관리할 수 있는 웹 애플리케이션입니다. 

### 핵심 가치

- **가족 건강 통합 관리**: 가족 구성원의 건강 정보를 한 곳에서 관리
- **맞춤형 식단 추천**: 개인 건강 상태에 맞는 식단 자동 생성
- **건강 정보 시각화**: 복잡한 건강 데이터를 쉽게 이해할 수 있는 차트와 그래프로 제공
- **응급 상황 대응**: 알레르기 반응, 의료기관 찾기 등 긴급 상황 지원

## 2. 주요 기능 소개

### 2.1 홈 화면 구성

홈 화면에는 다음과 같은 주요 섹션들이 있습니다:

#### 📍 빠른 접근 카드 (응급조치 안내)
- **응급조치 안내**: 알레르기 반응 시 즉시 대처 방법 안내
- **예방접종 안내**: 나이별 맞춤 예방접종 일정 확인
- **주변 의료기관 찾기**: 병원, 약국, 동물병원 위치 확인
- **건강 맞춤 식단**: 개인 맞춤 식단 상세 정보 확인
- **Recipe Genie**: 냉장고 재료 확인하고 레시피 추천

#### 🌤️ 날씨 위젯
- 현재 위치의 날씨 정보 표시
- 날씨에 맞는 요리 추천 메시지 제공

#### 🎯 빠른 시작 메뉴
홈 화면에서 바로 접근할 수 있는 주요 기능들:
- **레시피**: 최신 레시피 모음
- **궁중요리**: 전통 궁중 레시피
- **식단관리**: 맞춤 식단 추천
- **주간식단**: 7일 식단 계획
- **건강관리**: 건강 정보 확인
- **식재료**: 신선한 채소 정보
- **음식안전**: 안전한 식생활
- **요리이야기**: 맛있는 이야기들

#### 👥 커뮤니티 미리보기
- 인기 그룹 목록
- 건강하고 맛있는 삶을 나누는 커뮤니티

## 3. 가족 건강 관리 시스템

### 3.1 가족 구성원 등록 및 관리

#### 가족 구성원 추가하기
1. **건강 관리** 메뉴 → **가족 건강** 탭 클릭
2. **가족 관리** 버튼 클릭
3. **가족 구성원 추가** 버튼 클릭
4. 다음 정보를 입력:
   - 이름
   - 생년월일 (나이 자동 계산)
   - 성별
   - 관계 (본인, 배우자, 자녀, 부모 등)
   - 키, 체중
   - 질병 정보 (있는 경우)
   - 알레르기 정보 (있는 경우)
   - 활동 수준 (거의 안 함, 가벼운 활동, 보통 활동, 활동적, 매우 활동적)

#### 가족 구성원 관리
- 최대 10명까지 등록 가능 (구독 플랜에 따라 다름)
- 각 구성원별로 건강 정보를 개별 관리
- 가족 구성원별 건강 상태를 한눈에 비교 가능

### 3.2 가족 건강 대시보드

#### 가족 건강 개요
**위치**: 건강 관리 → 대시보드 탭

**표시되는 정보**:
- **가족 구성원별 건강 상태 요약 카드**
  - 최근 건강검진 결과 요약
  - 현재 복용 중인 약물
  - 예방접종 상태
  - 건강 점수 (0-100점)

- **가족 전체 건강 트렌드**
  - 체중 변화 추이 (라인 차트)
  - 혈압/혈당 추이 (라인 차트)
  - 활동량 추이 (바 차트)

- **건강 알림 및 권장사항**
  - 예방접종 예정일
  - 건강검진 권장일
  - 약물 복용 알림

#### 개인 건강 상세 대시보드
**위치**: 건강 관리 → 가족 건강 → 구성원 선택

**표시되는 정보**:
- **건강 지표 카드**
  - 체중, 키, BMI
  - 혈압, 혈당, 콜레스테롤
  - 최근 건강검진 결과

- **활동량 및 운동**
  - 일일 걸음 수
  - 주간 운동 시간
  - 소모 칼로리

- **영양 섭취 분석**
  - 일일 칼로리 섭취량
  - 3대 영양소 비율 (탄수화물/단백질/지방)
  - 부족/과다 영양소 알림

- **수면 패턴** (연동 시)
  - 평균 수면 시간
  - 수면 품질 점수
  - 수면 패턴 그래프

### 3.3 건강 시각화 대시보드

**위치**: 건강 관리 → 대시보드 탭

건강 데이터를 직관적인 차트와 그래프로 시각화하여 제공합니다:

#### 건강 메트릭스 카드
- **BMI**: 체질량지수 표시
- **체지방률**: 체지방 비율
- **근육량**: 근육량 정보
- **기초대사량**: 기초 대사량

#### 질병 위험도 게이지
- **심혈관 질환 위험도**: 낮음/보통/높음
- **당뇨병 위험도**: 낮음/보통/높음
- **신장 질환 위험도**: 낮음/보통/높음

#### 영양 균형 도넛 차트
- 탄수화물, 단백질, 지방의 비율을 시각화
- 목표 대비 섭취량 표시

#### 건강 트렌드
- 시간에 따른 건강 지표 변화 추이
- 체중, 혈압, 혈당 등의 변화 그래프

## 4. 건강 관리 기능 상세

### 4.1 건강 프로필 관리

**위치**: 건강 관리 → 건강 프로필 탭

#### 기본 건강 정보 입력
- 나이, 성별, 키, 체중
- 활동 수준
- 질병 정보
- 알레르기 정보
- 식이 선호도 (도시락, 헬스인, 저탄수, 비건 등)

#### 건강 목표 설정
- 일일 칼로리 목표
- 체중 목표
- 운동 목표

### 4.2 예방접종 안내

**위치**: 홈 화면 → 예방접종 안내 카드 클릭

#### 생애주기별 예방접종
- **영유아기** (출생 ~ 6세): B형 간염, 5가 혼합백신, MMR, 수두 등
- **청소년기** (11세 ~ 18세): HPV, Tdap, 수막구균 등
- **성인기** (19세 ~ 64세): 파상풍, A형/B형 간염, HPV 등
- **노년기** (65세 이상): 폐렴구균, 대상포진 등

#### 가족 구성원별 예방접종 권장사항
- 가족 구성원의 나이를 자동 계산하여 필요한 예방접종 추천
- 접종 완료 여부 표시
- 다음 접종 예정일 알림

### 4.3 건강 기록 관리

**위치**: 건강 관리 → 건강 기록 탭

#### 기록 가능한 항목
- **건강검진 결과**: 혈압, 혈당, 콜레스테롤 등
- **약물 복용 기록**: 복용 중인 약물 정보
- **활동량 기록**: 걸음 수, 운동 시간 등
- **영양 기록**: 섭취한 음식 및 칼로리

### 4.4 생애주기별 건강 알림

**위치**: 건강 관리 → 대시보드 탭

나이와 성별에 따라 필요한 건강 관리 항목을 자동으로 알려줍니다:
- 예방접종 일정
- 건강검진 권장일
- 약물 복용 알림
- 건강 개선 권장사항

### 4.5 반려동물 건강 관리

**위치**: 건강 관리 → 반려동물 건강 탭

#### 반려동물 프로필 관리
- 반려동물 종류 (강아지/고양이)
- 이름, 생년월일
- 체중 정보

#### 생애주기별 건강 이벤트
- AVMA/AAHA 기준에 따른 생애주기별 건강 이벤트 자동 매칭
- 백신 D-Day 카운트다운
- 체중 변화 그래프 시각화
- 건강 검진 일정 관리

## 5. 식단 관리 기능

### 5.1 건강 맞춤 식단

**위치**: 홈 화면 → 건강 맞춤 식단 카드 클릭 또는 식단관리 메뉴

#### 개인 맞춤 식단 생성
시스템이 다음 정보를 바탕으로 자동으로 식단을 생성합니다:
- 건강 프로필 정보 (나이, 성별, 키, 체중, 활동 수준)
- 질병 정보 (당뇨, 고혈압, 신장 질환 등)
- 알레르기 정보
- 식이 선호도

#### 식단 구성
- **아침**: 목표 칼로리의 25%
- **점심**: 목표 칼로리의 35%
- **저녁**: 목표 칼로리의 30%
- **간식**: 목표 칼로리의 10%

#### 영양소 균형
- 탄수화물, 단백질, 지방의 균형 잡힌 비율
- 질병별 영양소 제한 반영 (예: 당뇨 환자의 저탄수화물, 신장 질환 환자의 저단백질)

### 5.2 주간 식단 계획

**위치**: 식단관리 → 주간식단 메뉴

#### 7일 식단 자동 생성
- 매주 일요일 오후 6시에 다음 주 식단 자동 생성
- 레시피 다양성 확보 (같은 레시피가 너무 자주 반복되지 않도록)
- 주간 영양 밸런스 고려

#### 장보기 리스트
- 7일치 식단의 재료를 자동으로 집계
- 카테고리별로 정리 (채소, 육류, 유제품 등)
- 필요한 재료 수량 표시

### 5.3 가족 통합 식단

**위치**: 건강 관리 → 가족 건강 → 가족 식단

#### 가족 식단 생성
- 가족 구성원들의 건강 정보를 종합하여 통합 식단 생성
- 각 구성원의 식이 제한사항을 모두 반영
- 가족 구성원별 개별 식단도 제공

## 6. 응급 상황 대응 기능

### 6.1 응급조치 안내

**위치**: 홈 화면 → 응급조치 안내 카드 클릭

#### 알레르기 반응 대처
- 알레르기 반응 증상 확인
- 즉시 대처 방법 안내
- 응급실 방문 필요 시점 안내

### 6.2 주변 의료기관 찾기

**위치**: 홈 화면 → 주변 의료기관 찾기 카드 클릭

#### 의료기관 검색
- **병원**: 일반 병원, 종합병원, 대학병원
- **약국**: 일반 약국, 24시간 약국
- **동물병원**: 반려동물 병원
- **동물약국**: 반려동물 약국

#### 검색 기능
- 현재 위치 기반 검색
- 주소 입력 검색
- 거리순 정렬
- 24시간 영업 여부 표시
- 현재 영업 중 여부 표시

#### 지도 표시
- 네이버 지도를 통한 의료기관 위치 표시
- 거리 정보 표시
- 전화번호, 주소 정보 제공

## 7. 사용 방법 요약

### 7.1 처음 시작하기

1. **회원가입 및 로그인**
   - 이메일 또는 Google 계정으로 가입
   - 로그인 후 자동으로 Supabase에 사용자 정보 동기화

2. **건강 프로필 설정**
   - 건강 관리 → 건강 프로필 탭
   - 기본 건강 정보 입력 (나이, 성별, 키, 체중 등)
   - 질병 및 알레르기 정보 입력

3. **가족 구성원 추가**
   - 건강 관리 → 가족 건강 탭 → 가족 관리
   - 가족 구성원 정보 입력

4. **식단 생성**
   - 식단관리 메뉴에서 건강 맞춤 식단 확인
   - 주간 식단 계획 확인

### 7.2 일상적인 사용

#### 매일 확인할 항목
- **홈 화면**: 날씨 정보, 건강 맞춤 식단 추천
- **건강 대시보드**: 가족 건강 상태 요약
- **식단 관리**: 오늘의 식단 확인

#### 주간 확인할 항목
- **주간 식단**: 다음 주 식단 계획 확인 및 장보기 리스트 작성
- **건강 기록**: 건강검진 결과, 약물 복용 기록 입력

#### 필요 시 확인할 항목
- **예방접종 안내**: 가족 구성원별 예방접종 일정 확인
- **의료기관 찾기**: 병원, 약국 찾기
- **응급조치 안내**: 알레르기 반응 등 응급 상황 대처

### 7.3 건강 관리 팁

1. **정기적인 건강 정보 업데이트**
   - 체중, 혈압 등 건강 지표를 정기적으로 입력
   - 건강검진 결과를 기록

2. **식단 기록**
   - 섭취한 음식을 기록하여 영양 분석 정확도 향상

3. **활동량 기록**
   - 걸음 수, 운동 시간을 기록하여 활동량 추적

4. **가족 건강 비교**
   - 가족 구성원들의 건강 상태를 비교하여 전체적인 건강 관리

## 8. 게임 기능

### 8.1 냉장고 디펜스 게임

**위치**: `/game/fridge-defense` 경로 또는 홈 화면에서 게임 메뉴 클릭

냉장고 디펜스는 타워를 배치하여 세균이 냉장고에 도달하는 것을 막는 **타워 디펜스 게임**입니다.

#### 게임 목표
- 세균들이 경로를 따라 냉장고에 도달하기 전에 타워를 배치하여 막아내기
- 최대한 많은 웨이브를 버티며 높은 점수 달성하기

#### 게임 시작하기

1. **게임 접속**
   - 홈 화면에서 게임 메뉴를 클릭하거나 `/game/fridge-defense` 경로로 직접 이동
   - 게임 화면이 표시됩니다

2. **게임 시작**
   - 왼쪽 사이드바 하단의 **"START GAME"** 버튼 클릭
   - 게임이 시작되면 세균들이 자동으로 등장하기 시작합니다

#### 게임 플레이 방법

##### 1. 타워 배치하기
- **타워 선택**: 왼쪽 사이드바에서 원하는 타워를 클릭
  - **🍗 PROTEIN 타워** (150G): 강력한 공격력, 중간 사거리
  - **🥦 VITAMIN 타워** (120G): 빠른 공격 속도, 짧은 사거리
  - **🥑 SUGAR 타워** (200G): 매우 강력한 공격력, 긴 사거리
- **배치 위치**: 게임 보드(중앙 영역)에서 원하는 위치를 클릭하여 타워 배치
- **골드 확인**: 타워를 배치하려면 충분한 골드가 필요합니다 (화면 상단의 골드 표시 확인)

##### 2. 적(세균) 종류
게임에는 3가지 종류의 적이 등장합니다:
- **🦠 일반 세균 (GERM)**: 기본 적, 체력 100, 속도 보통, 처치 시 40G 획득
- **🍭 설탕 스파이크 (SUGAR_SPIKE)**: 빠른 적, 체력 60, 속도 빠름, 처치 시 60G 획득
- **🍟 지방 폭탄 (FATTY_BOMB)**: 강한 적, 체력 400, 속도 느림, 처치 시 120G 획득

##### 3. 게임 진행
- **적 처치**: 타워가 자동으로 사거리 내의 적을 공격합니다
- **골드 획득**: 적을 처치하면 골드를 획득하여 더 많은 타워를 배치할 수 있습니다
- **웨이브 진행**: 시간이 지나면서 더 많은 적이 등장하고, 웨이브가 올라갈수록 적의 체력이 증가합니다
- **생명력 관리**: 적이 냉장고에 도달하면 생명력이 1 감소합니다 (최대 5개)

##### 4. 특수 스킬 사용
- **VITAMIN SHOCKWAVE**: 모든 적에게 150 데미지를 입히는 특수 스킬
- **쿨타임**: 30초 (사용 후 30초 동안 재사용 불가)
- **사용 방법**: 왼쪽 사이드바의 보라색 버튼 클릭

##### 5. 식단 연동 버프
- 오늘 실제로 먹은 영양소에 해당하는 타워는 **공격 속도가 30% 빨라집니다**
- 버프가 적용된 타워 위에는 **✨ 반짝이 효과**가 표시됩니다
- 예: 오늘 단백질을 먹었다면 PROTEIN 타워가 버프를 받습니다

#### 게임 종료 및 점수 저장

1. **게임 오버**
   - 생명력이 0이 되면 게임이 종료됩니다
   - 최종 웨이브가 점수로 기록됩니다

2. **랭킹 등록**
   - 게임 오버 후 랭킹 보드가 자동으로 표시됩니다
   - 이름을 입력하고 점수를 등록할 수 있습니다
   - 상위 5명의 랭킹을 확인할 수 있습니다

3. **다시 플레이**
   - **"다시하기"** 버튼을 클릭하여 새 게임 시작
   - **"닫기"** 버튼을 클릭하여 랭킹 보드 닫기

#### 게임 팁

1. **전략적 타워 배치**
   - 경로의 굽어지는 지점에 타워를 배치하면 적을 더 오래 공격할 수 있습니다
   - 다양한 타입의 타워를 조합하여 효율적인 방어선을 구축하세요

2. **골드 관리**
   - 초반에는 저렴한 VITAMIN 타워로 시작하여 골드를 모으세요
   - 골드가 충분해지면 강력한 SUGAR 타워를 배치하세요

3. **특수 스킬 활용**
   - 많은 적이 몰려올 때 VITAMIN SHOCKWAVE를 사용하세요
   - 쿨타임을 고려하여 적절한 타이밍에 사용하세요

4. **식단 연동 활용**
   - 오늘 먹은 영양소에 맞는 타워를 우선적으로 배치하면 더 강력한 방어가 가능합니다

#### 게임 화면 구성

- **왼쪽 사이드바**: 게임 정보 및 타워 선택
  - 생명력 (LIVES): 남은 생명력 표시
  - 골드 (GOLD): 현재 보유 골드 표시
  - 웨이브 (WAVE): 현재 웨이브 표시
  - 특수 스킬 버튼
  - 타워 선택 버튼들
  - 게임 시작/일시정지 버튼

- **중앙 게임 보드**: 실제 게임이 진행되는 영역
  - 경로: 세균이 이동하는 경로 표시
  - 타워: 배치한 타워 표시
  - 적: 이동하는 세균 표시
  - 투사체: 타워가 발사하는 공격 표시
  - 데미지 숫자: 적에게 입힌 데미지 표시

## 9. 주요 특징 요약

### ✅ 통합 관리
- 가족 구성원의 건강 정보를 한 곳에서 관리
- 건강, 식단, 예방접종 등 모든 정보 통합

### ✅ 자동화
- 건강 정보 기반 자동 식단 생성
- 예방접종 일정 자동 추천
- 건강 알림 자동 발송

### ✅ 시각화
- 복잡한 건강 데이터를 차트와 그래프로 쉽게 이해
- 건강 트렌드 한눈에 파악

### ✅ 맞춤형 서비스
- 개인 건강 상태에 맞는 식단 추천
- 나이별 예방접종 일정 추천
- 질병별 영양소 제한 반영

### ✅ 응급 상황 대응
- 알레르기 반응 대처 방법 안내
- 주변 의료기관 빠른 검색
