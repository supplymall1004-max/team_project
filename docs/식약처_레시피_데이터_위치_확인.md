# 식약처 레시피 데이터 위치 확인

## 현재 상황

### 1. 정적 파일 위치
- **경로**: `docs/mfds-recipes/recipes/`
- **상태**: 디렉토리만 존재, 파일 없음 (0개)
- **형식**: 마크다운 파일 (`.md`)

### 2. API 폴백
- **경로**: `/api/mfds-recipes`
- **상태**: 정적 파일이 없어 API 호출 시도
- **문제**: API 타임아웃으로 데이터 수집 실패

### 3. 페이지 위치
- **목록 페이지**: `/recipes/mfds` (`app/recipes/mfds/page.tsx`)
- **상세 페이지**: `/recipes/mfds/[recipeId]` (`app/recipes/mfds/[recipeId]/page.tsx`)

## 데이터 흐름

```
/recipes/mfds 페이지
  ↓
/api/mfds-recipes API 호출
  ↓
[1] 정적 파일 로드 시도 (docs/mfds-recipes/recipes/*.md)
  ↓ (실패 - 파일 없음)
[2] 식약처 API 호출 폴백
  ↓ (실패 - 타임아웃)
빈 배열 반환
```

## 해결 방법

### 방법 1: 데이터 수집 스크립트 실행 (권장)
```bash
pnpm tsx scripts/collect-mfds-recipes.ts
```

이 스크립트가 실행되면:
- 식약처 API에서 레시피 데이터 수집
- `docs/mfds-recipes/recipes/{RCP_SEQ}.md` 형식으로 저장
- 각 파일에 레시피 정보와 참고사항 섹션 포함

### 방법 2: API 연결 문제 해결
- 네트워크 연결 확인
- API 키 유효성 확인
- 타임아웃 시간 조정

### 방법 3: 수동 샘플 데이터 생성 (테스트용)
테스트를 위해 몇 개의 샘플 레시피 파일을 수동으로 생성할 수 있습니다.

## 예상되는 파일 구조

데이터 수집 후:
```
docs/mfds-recipes/
```

## 현재 문제점

1. **정적 파일 없음**: `docs/mfds-recipes/recipes/` 디렉토리가 비어있음
2. **API 폴백 실패**: 식약처 API 호출이 타임아웃됨
3. **결과**: `/recipes/mfds` 페이지에 레시피가 표시되지 않음
