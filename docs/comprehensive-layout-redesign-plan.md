## 1. 현재 기능 분석

### 1.1 레시피 관련 기능
- ✅ **현대 레시피** (`/recipes`): 별점, 난이도 기반 레시피 아카이브
- ✅ **궁중 레시피** (`/royal-recipes`): 삼국시대~조선시대 전통 궁중 음식
- ✅ **식약처 레시피** (`/recipes/mfds`): 식약처 공식 레시피 API 연동
- ✅ **레시피 상세** (`/recipes/[slug]`): 단계별 조리법, 타이머, 체크리스트
- ✅ **레시피 업로드** (`/recipes/new`): 사용자 레시피 등록

### 1.2 식단 관련 기능
- ✅ **건강 맞춤 식단** (`/diet`): AI 기반 개인 맞춤 식단 추천
- ✅ **주간 식단** (`/diet/weekly`): 7일간 식단 캘린더, 장보기 리스트
- ✅ **식단 상세** (`/diet/breakfast|lunch|dinner/[date]`): 식사별 상세 정보
- ✅ **식단 시각화**: 건강 상태, 영양 균형, 식단 효과 예측
- ✅ **가족 식단** (`/health/family/diet/[date]`): 가족 통합 식단 관리

### 1.3 건강 관리 기능
- ✅ **건강 프로필** (`/health/profile`): 기본 건강 정보 입력/관리
- ✅ **가족 관리** (`/health/manage`): 가족 구성원 추가/관리
- ✅ **건강 시각화**: BMI, 체지방률, 영양 균형, 질병 위험도
- ✅ **건강 대시보드** (`/chapters/health`): 가족 건강 종합 현황
- ✅ **응급조치** (`/health/emergency`): 알레르기 응급 대응 가이드

### 1.4 스토리/콘텐츠 기능
- ✅ **마카의 음식 동화** (`/storybook`): 전통 음식 스토리 (인터랙티브 방)
- ✅ **음식 스토리** (`/food-stories`): 음식 관련 동영상 콘텐츠

### 1.5 유틸리티 기능
- ✅ **검색** (`/search`): 통합 검색 기능
- ✅ **자주 구매하는 식자재**: 장보기 편의 기능
- ✅ **설정** (`/settings`): 프로필, 건강, 가족, 알림, 결제 설정

## 3. 새로운 네비게이션 구조

### 3.1 상단 네비게이션 (Navbar)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ [로고] Flavor Archive  [검색창]  레시피  식단  건강  스토리  [설정] [사용자] │
└─────────────────────────────────────────────────────────────────────────────┘
```

**메뉴 구조:**
- **레시피**: 레시피 아카이브 페이지 (`/archive/recipes`)
- **식단**: 식단 관리 페이지 (`/diet`)
- **건강**: 건강 관리 페이지 (`/health`)
- **스토리**: 스토리 & 학습 페이지 (`/stories`)

### 3.2 하단 네비게이션 (모바일)

```
┌─────────────────────────────────────┐
│  홈  레시피  식단  건강  마이          │
└─────────────────────────────────────┘
```

**메뉴 설명:**
- **홈**: 메인 화면 (모든 카테고리 미리보기)
- **레시피**: 레시피 아카이브
- **식단**: 식단 관리
- **건강**: 건강 관리
- **마이**: 설정 및 프로필
