## 9. 인터랙션 패턴

### 9.1 카드 호버 효과
- **데스크톱**: 카드가 위로 4px 이동, 그림자 증가
- **태블릿**: 카드가 위로 2px 이동, 그림자 증가
- **모바일**: 터치 피드백 (스케일 다운)

### 9.2 탭 전환
- **애니메이션**: fade-in/out 효과 (300ms)
- **URL 업데이트**: 쿼리 파라미터로 탭 상태 관리 (`?tab=modern`)

### 9.3 스크롤 애니메이션
- **섹션 진입**: fade-in 효과 (500ms)
- **카드 순차 표시**: stagger 애니메이션 (각 100ms 지연)

## 10. 접근성 고려사항

### 10.1 키보드 네비게이션
- 모든 인터랙티브 요소는 Tab 키로 접근 가능
- Enter/Space로 활성화
- 탭 전환은 화살표 키로 가능

### 10.2 스크린 리더
- 모든 이미지에 alt 텍스트
- 차트에 aria-label 제공
- 버튼에 명확한 라벨
- 섹션에 aria-labelledby

### 10.3 색상 대비
- WCAG AA 기준 준수 (4.5:1 이상)
- 색상만으로 정보 전달하지 않음 (아이콘, 텍스트 병행)
