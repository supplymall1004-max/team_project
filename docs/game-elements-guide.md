# 게임 요소 사용 가이드

## 개요

캐릭터창에 통합된 게임 요소들을 통해 건강 관리를 재미있고 동기부여가 되는 경험으로 만들었습니다. 이 가이드는 각 게임 요소의 사용 방법과 기능을 설명합니다.

## 게임 요소 목록

### 1. 퀘스트 시스템

#### 일일 퀘스트
- **위치**: 캐릭터 페이지 → 퀘스트 탭 → 일일 퀘스트
- **기능**: 매일 새로운 건강 관련 퀘스트를 완료하여 포인트를 획득합니다.
- **퀘스트 유형**:
  - 운동 퀘스트: 만보 걷기, 운동 시간 달성 등
  - 건강 관리: 약물 복용, 건강검진 예약 등
  - 영양 관리: 식사 기록, 영양 균형 맞추기 등

#### 주간 퀘스트
- **위치**: 캐릭터 페이지 → 퀘스트 탭 → 주간 퀘스트
- **기능**: 일주일 동안 지속되는 장기 퀘스트를 완료하여 더 많은 포인트를 획득합니다.
- **보상**: 일일 퀘스트보다 높은 포인트 보상

#### 퀘스트 완료 방법
1. 퀘스트 목록에서 원하는 퀘스트 선택
2. "진행 업데이트" 버튼으로 진행 상황 업데이트
3. 목표 달성 시 "완료하기" 버튼 클릭
4. 포인트 자동 적립 및 배지 획득 가능

### 2. 레벨 시스템

#### 레벨업 조건
- 건강 점수 기반으로 경험치 획득
- 경험치가 일정량 쌓이면 자동 레벨업
- 레벨업 시 포인트 및 스킨 보상

#### 레벨 보상
- 레벨 5: 기본 스킨 해제
- 레벨 10: 고급 스킨 해제
- 레벨 20: 희귀 스킨 해제
- 레벨 50: 전설 스킨 해제

### 3. 컬렉션 시스템

#### 스킨 컬렉션
- **위치**: 캐릭터 페이지 → 컬렉션 탭 → 스킨 컬렉션
- **기능**: 레벨업, 퀘스트 완료, 이벤트 참여 등을 통해 다양한 스킨을 수집합니다.
- **스킨 등급**:
  - 일반 (Common): 기본 스킨
  - 고급 (Rare): 레벨업 보상
  - 희귀 (Epic): 특별 이벤트 보상
  - 전설 (Legendary): 최고 등급 스킨

#### 배지 갤러리
- **위치**: 캐릭터 페이지 → 컬렉션 탭 → 배지 갤러리
- **기능**: 건강 관리 목표 달성 시 배지를 획득합니다.
- **배지 종류**:
  - 첫 걸음: 첫 알림 완료
  - 연속 달성: 3일, 7일, 30일 연속 완료
  - 포인트 달성: 100점, 500점, 1000점 달성

### 4. 랜덤 이벤트

#### 이벤트 유형
- **일일 이벤트**: 매일 랜덤하게 발생하는 작은 이벤트
- **가족 이벤트**: 가족 구성원과 함께 참여하는 이벤트
- **특별 이벤트**: 계절별, 기념일 이벤트
- **시즌 이벤트**: 계절에 맞는 특별 이벤트

#### 이벤트 참여 방법
1. 이벤트 팝업이 나타나면 확인
2. 이벤트 내용 읽기
3. 이벤트 완료 조건 달성
4. 보상 포인트 획득

### 5. 가족 상호작용

#### 가족 챌린지
- **위치**: 캐릭터 페이지 → 커뮤니티 탭 → 가족 챌린지
- **기능**: 가족 구성원들과 함께 건강 목표를 달성합니다.
- **챌린지 유형**:
  - 주간 챌린지: 일주일 동안 진행
  - 월간 챌린지: 한 달 동안 진행
  - 특별 챌린지: 기간 한정 챌린지

#### 가족 친밀도
- 가족 구성원과의 상호작용으로 친밀도 증가
- 친밀도가 높을수록 특별 보상 획득 가능

#### 가족 랭킹
- 가족 구성원들의 건강 관리 점수 비교
- 주간/월간 랭킹 확인 가능

### 6. 건강 퀴즈

#### 퀴즈 유형
- **건강 관리**: 질병 예방, 건강 관리 방법
- **영양**: 영양소, 식단 관리
- **운동**: 운동 방법, 운동 효과
- **약물**: 약물 복용, 부작용 관리

#### 퀴즈 참여 방법
1. 건강 퀴즈 섹션에서 퀴즈 선택
2. 문제 읽고 정답 선택
3. 정답 시 포인트 획득
4. 오답 시 해설 확인

### 7. 미니게임

#### 게임 종류
- **약물 기억력 게임**: 약물 복용 시간을 기억하는 게임
- **운동 타이밍 게임**: 운동 동작을 정확한 타이밍에 맞추는 게임
- **영양 퍼즐 게임**: 영양소를 조합하여 균형잡힌 식단을 만드는 게임

#### 게임 플레이 방법
1. 미니게임 섹션에서 게임 선택
2. 게임 규칙 확인
3. 게임 플레이
4. 점수에 따라 포인트 획득

### 8. 리더보드

#### 주간 리더보드
- **위치**: 캐릭터 페이지 → 커뮤니티 탭 → 주간 리더보드
- **기능**: 다른 사용자들과 건강 관리 점수를 비교합니다.
- **랭킹 기준**: 주간 포인트, 퀘스트 완료 수, 건강 점수 등

## 포인트 시스템

### 포인트 획득 방법
1. **퀘스트 완료**: 일일/주간 퀘스트 완료 시 포인트 획득
2. **레벨업**: 레벨업 시 포인트 보상
3. **이벤트 참여**: 랜덤 이벤트 완료 시 포인트 획득
4. **퀴즈 정답**: 건강 퀴즈 정답 시 포인트 획득
5. **미니게임**: 미니게임 점수에 따라 포인트 획득

### 포인트 사용
- 포인트는 배지 획득 조건으로 사용됩니다.
- 높은 포인트 달성 시 특별 배지 획득 가능

## 배지 시스템

### 배지 획득 조건
- **첫 걸음**: 총 포인트 10점 이상
- **3일 연속**: 3일 연속 알림 완료
- **일주일 마스터**: 7일 연속 알림 완료
- **한 달 도전자**: 30일 연속 알림 완료
- **백점 클럽**: 총 포인트 100점 이상
- **오백점 마스터**: 총 포인트 500점 이상
- **천점 레전드**: 총 포인트 1000점 이상

## 팁과 트릭

### 효율적인 포인트 획득
1. 매일 일일 퀘스트 완료하기
2. 주간 퀘스트 꾸준히 진행하기
3. 랜덤 이벤트 놓치지 않기
4. 건강 퀴즈 정기적으로 참여하기

### 레벨업 빠르게 하기
1. 건강 점수 관리하기
2. 건강 활동 꾸준히 하기
3. 건강검진, 백신 접종 등 건강 관리 실천하기

### 컬렉션 완성하기
1. 다양한 퀘스트 완료하기
2. 레벨업 보상 받기
3. 특별 이벤트 참여하기
4. 가족 챌린지 완료하기

## 문제 해결

### 퀘스트가 완료되지 않아요
- 퀘스트 목표 수치를 정확히 달성했는지 확인하세요.
- "완료하기" 버튼을 클릭했는지 확인하세요.
- 페이지를 새로고침해보세요.

### 포인트가 적립되지 않아요
- 퀘스트 완료 후 잠시 기다려보세요.
- 포인트는 자동으로 적립됩니다.
- 문제가 계속되면 페이지를 새로고침해보세요.

### 레벨업이 안 돼요
- 건강 점수가 충분한지 확인하세요.
- 경험치는 건강 점수 기반으로 계산됩니다.
- 건강 활동을 더 많이 하면 레벨업이 빨라집니다.

## 문의 및 피드백

게임 요소에 대한 문의나 피드백이 있으시면 개발팀에 연락해주세요.
