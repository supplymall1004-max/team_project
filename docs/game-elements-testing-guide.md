# 게임 요소 테스트 가이드

## 테스트 목적

게임 요소들이 올바르게 작동하고 사용자 경험이 원활한지 확인하기 위한 테스트 가이드입니다.

## 테스트 환경

- **브라우저**: Chrome, Firefox, Safari, Edge 최신 버전
- **디바이스**: 데스크톱, 태블릿, 모바일
- **인터넷 연결**: 안정적인 인터넷 연결 필요

## 테스트 시나리오

### 시나리오 1: 퀘스트 완료 → 포인트 획득 → 배지 획득
1. 일일 퀘스트 중 하나를 완료합니다.
2. 포인트가 적립되는지 확인합니다.
3. 포인트 합계가 증가하는지 확인합니다.
4. 배지 획득 조건을 만족하면 배지가 획득되는지 확인합니다.

### 시나리오 2: 레벨업 → 스킨 해제
1. 건강 점수를 높여 경험치를 획득합니다.
2. 레벨업이 발생하는지 확인합니다.
3. 레벨업 애니메이션이 표시되는지 확인합니다.
4. 레벨업 보상으로 스킨이 해제되는지 확인합니다.
5. 스킨 컬렉션에서 해제된 스킨을 확인합니다.

### 시나리오 3: 가족 챌린지 완료 → 친밀도 증가
1. 가족 챌린지를 시작합니다.
2. 챌린지 목표를 달성합니다.
3. 챌린지 완료 시 포인트가 적립되는지 확인합니다.
4. 가족 친밀도가 증가하는지 확인합니다.

### 시나리오 4: 여러 게임 요소 동시 사용
1. 퀘스트를 완료하면서 동시에 미니게임을 플레이합니다.
2. 포인트가 정확히 적립되는지 확인합니다.
3. 모든 게임 요소가 정상적으로 작동하는지 확인합니다.

## 버그 리포트 양식

버그를 발견한 경우 다음 정보를 포함하여 리포트해주세요:

1. **버그 설명**: 무엇이 잘못되었는지 설명
2. **재현 단계**: 버그를 재현하는 단계
3. **예상 동작**: 어떻게 작동해야 하는지
4. **실제 동작**: 실제로 어떻게 작동하는지
5. **스크린샷**: 가능한 경우 스크린샷 첨부
6. **환경 정보**: 브라우저, 디바이스, OS 정보

테스트를 완료한 후 다음 정보를 기록하세요:
