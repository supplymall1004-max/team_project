# 추가 구현 가능한 기능 제안

## 🎯 우선순위 높은 기능

### 1. GLTF 모델 로더 및 샘플 모델
**우선순위: 높음**
- 더 상세한 캐릭터 모델
- 더 많은 가구/오브젝트
- 애니메이션 포함 모델

**구현 방법:**
```tsx
import { useGLTF } from "@react-three/drei";

function CharacterModel() {
  const { scene } = useGLTF("/models/character.glb");
  return <primitive object={scene} />;
}
```

**필요한 작업:**
- GLTF 모델 파일 준비 (Blender 등으로 제작)
- 모델 최적화
- 로딩 상태 처리

---

### 2. 물리 엔진 통합
**우선순위: 중간**
- 오브젝트 간 충돌 감지
- 중력 시뮬레이션
- 더 현실적인 움직임

**추천 라이브러리:**
- `@react-three/cannon` (Cannon.js)
- `@react-three/rapier` (Rapier - 더 현대적)

**사용 사례:**
- 장난감 떨어뜨리기
- 공 튕기기
- 캐릭터와 오브젝트 상호작용
