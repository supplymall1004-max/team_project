# 콩고기샐러드

콩고기샐러드은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 콩고기(30g)
- 아보카도(10g)
- 바나나(10g)
- 가지(10g)
- 연근(10g)
- 방울토마토(10g)
- 만두피(20g)
- 저염간장양념 저염간장(2.5g)
- 물(25g)
- 설탕(3g)
- 허브오일드레싱 저염소금(1g)
- 식초(5g)
- 로즈마리(1g)
- 통후추(1g)
- 올리브유(15g)
- 꿀(5g)

## 조리 방법

1. 1. 콩고기를 15분~20분 정도 물에
불려 물기를 완전히 제거한 뒤
저염간장양념으로 버무려 볶는다.
2. 2. 아보카도, 바나나, 가지, 연근은
깍둑 썰고, 방울토마토는 반을
가른다.
3. 3. 팬에 식용유(5g)를 둘러가며
방울토마토, 가지, 연근을 각각
볶아 건진다.
4. 4. 만두피를 작은 체에 넣고
바구니 모양으로 만들어 튀긴다.
5. 5. 콩고기와 아보카도, 바나나,
방울토마토, 가지, 연근을 튀긴
만두피에 담은 뒤 허브오일드레싱을
곁들여 마무리한다.

## 영양 정보

- 칼로리: 333.2 kcal
- 탄수화물: 19.26g
- 단백질: 13.13g
- 지방: 6.88g
- 나트륨: 426.2mg
