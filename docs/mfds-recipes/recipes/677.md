# 민어매운탕

민어매운탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- - 주재료 : 민어 50g
- 쇠고기 양지15g
- 다진 마늘 1.25g
- 후추 0.25g
- 참기름 0.5g
- 무 40g
- 애호박 20g
- 애느타리버섯 20g
- 풋 고추 5g
- 홍고추 2.5g
- 쑥갓 10g
- 대파 5g
- 생강즙 0.5g
- 물 300g
- 소금 0.5g - 양념장 : 간 홍고추 10g
- 간 양파 10g
- 다진 마늘 2g
- 국간장 2.5g
- 간장 2.5g
- 맛술 5g
- 굵은 고춧가루 5g
- 고운 고춧가루 1.25g

## 조리 방법

1. 1. 양념장 재료를 모두 섞어 냉장고에서 하루 숙성시킨다.
2. 2. 민어는 비늘을 긁고 지느러미, 내장을 제거하여 흐르는 물에 씻어 5cm 길이로 토막 낸다.
3. 3. 소고기는 채 썰어 다진 마늘, 후추로 밑간한다.
4. 4. 무는 나박 썰기, 애호박은 반달 썰기, 애느타리는 한입 크기로 찢고 고추와 대파는 어슷 썬다.
5. 5. 냄비에 참기름을 두르고 소고기를 볶다가 물 1컵 반과 무를 넣고 양념장을 풀어 끓인다.
6. 6. 무가 반 정도 익으면 민어와 채소를 넣고 한소끔 끓인 뒤 어슷 썬 대파와 생강즙을 넣어 살짝 끓이고 소금 간하고 쑥갓을 넣는다.

## 영양 정보

- 탄수화물: 11.7g
- 단백질: 16.2g
- 지방: 4.6g
- 나트륨: 532.1mg
