# 들깨곤약수제비

들깨곤약수제비은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 곤약(150g)
- 표고버섯(1개)
- 새송이버섯(1개)
- 들깨가루(200g)
- 사골육수(400g)
- 마늘(5g)
- 식초(0.3g)

## 조리 방법

1. 1. 곤약은 얇게 썬다.
2. 2. 끓는 물에 식초를 한두 방울 넣고, 썰어
놓은 곤약을 데친다.
3. 3. 표고는 불려서 채를 썬다.
4. 4. 마늘을 다져서 냄비에 볶는다.
5. 5. 냄비에 표고버섯과 새송이버섯을 넣고
함께 볶는다.
6. 6. 냄비에 들깨가루와 사골육수를 넣고
끓인 뒤 데친 곤약을 넣은 뒤 한소끔 더
끓인다.

## 영양 정보

- 칼로리: 137.7 kcal
- 탄수화물: 7.5g
- 단백질: 6.1g
- 지방: 9.3g
- 나트륨: 51.4mg
