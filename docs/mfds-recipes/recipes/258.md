# 두부곤약조림

두부곤약조림은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 곤약(100g)
- 두부(100g)
- 가지(20g)
- 애호박(20g)
- 꽈리고추(15g)
- 조림장 감자전분(6g)
- 저염간장(3g)
- 물(165g)
- 다진 마늘(2g)
- 유자청(10g)
- 고추기름(10g)

## 조리 방법

1. 1. 곤약과 두부는 사방 1㎝ 크기로
깍둑 썰고, 가지와 애호박은 조금
더 작은 크기로 썰고, 꽈리고추는
얇게 송송 썬다.
2. 2. 곤약을 끓는 물에 식초를 한두 방울
떨어뜨린 후 삶아 냄새를 제거한 후
찬물에 식힌다.
3. 3. 올리브유(10g)를 두른 팬에 두부를
올려 살짝 굽는다.
4. 4. 팬에 조림장을 넣은 후 약한 불에서
저어가며 끓이다가 조림장이 고루
섞이면 곤약과 두부, 가지, 애호박을
모두 넣고 조린다.
5. 5. 조림장이 반 이상 졸아들면
꽈리고추를 넣고 조금 더 조려
마무리한다.

## 영양 정보

- 칼로리: 183.4 kcal
- 탄수화물: 4.61g
- 단백질: 3.15g
- 지방: 3.13g
- 나트륨: 97.2mg
