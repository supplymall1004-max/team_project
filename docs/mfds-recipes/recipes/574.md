# 원소병

원소병은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 찹쌀가루(200g)
- 치자(10g)
- 오미자(10g)
- 대추(5알)
- 귤병(20g)
- 유자청(30g)
- 꿀(50g)
- 녹말(50g)
- 잣(10알)

## 조리 방법

1. 1. 찹쌀가루는 체에 내려 2등분한다.
2. 2. 찹쌀가루는 오미자와 치자를 우린
뜨거운 물로 익반죽하여 비닐봉투에
담아 숙성시킨다
3. 3. 대추는 씨를 발라내 곱게 다지고 귤병,
유자청, 꿀을 섞어 원소병 소를 만든다.
4. 4. 굴병과 꿀을 섞어 원소병 소를
준비한다.
5. 5. 익반죽을 지름 3cm 크기로 동그랗게
만들고 안에 소를 넣어 다시 동그랗게
만든다
6. 6. 녹말가루를 입혀 끓는 물에 넣은 뒤
떠오르면 건져 찬물에 헹궈 물기를
빼고 그릇에 담아 꿀물을 붓고 잣을
올린다.

## 영양 정보

- 칼로리: 320.6 kcal
- 탄수화물: 73g
- 단백질: 6.1g
- 지방: 0.5g
- 나트륨: 47.2mg
