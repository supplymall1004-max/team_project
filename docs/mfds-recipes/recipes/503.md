# 코코넛밀크카레

코코넛밀크카레은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 두부(100g)
- 소금(0.3g)
- 코코넛밀크(20g)
- 당근(20g)
- 양파(20g)
- 브로컬리(20g)
- 돼지고기(50g)
- 감자(20g)
- 버터(10g)
- 카레가루(30g)
- 물(250g)

## 조리 방법

1. 1. 두부는 소창에 넣고, 짜서 물기를
제거하고, 으깨어 놓는다.
2. 2. 으깬 두부에 소금과 코코넛 밀크를
넣고 골고루 섞는다.
3. 3. 섞여진 두부를 김이 오른 찜통에 넣고
약 5분 정도 찐다.
4. 4. 당근과 양파는 사각지게 썰고,
브로컬리는 데쳐 당근 크기로 썰어
놓는다.
5. 5. 돼지고기와 감자는 당근 크기로 썰어
주고, 냄비에 버터를 녹여 돼지고기를
먼저 볶다가 양파와 당근, 감자,
브로컬리를 넣고 다시 볶아 준 뒤,
쪄놓은 두부를 넣어 한 번 더 볶아준다.
6. 6. 볶아지는 냄비(⑤)에 카레 가루를 넣고
살짝 더 볶아준 뒤, 물을 넣고 자작하게
끓여 완성한다.

## 영양 정보

- 칼로리: 330.2 kcal
- 탄수화물: 27g
- 단백질: 24g
- 지방: 14g
- 나트륨: 389.6mg
