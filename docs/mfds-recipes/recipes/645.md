# 겨자소스로 구운 산마샐러드

겨자소스로 구운 산마샐러드은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 산마(100g)
- 참나물(20g)
- 돈나물(20g)
- 토마토(100g)
- 레디쉬(1개)
- 어린잎(10g)
- 저염간장(20g)
- 식초(10g)
- 설탕(20g)
- 겨자가루(10g).

## 조리 방법

1. 1. 어린잎은 찬물에 담근다.
2. 2. 마는 껍질을 벗겨 동글동글하게 썬다
3. 3. 썰어 놓은 마는 팬에 기름을 살짝
두르고 앞뒤로 굽는다.
4. 4. 참나물, 돈나물, 토마토, 레디쉬는
먹기 좋은 크기로 썬다.
5. 5. 겨자가루는 30℃ 물을 섞어 발효
시킨다.
6. 6. 저염간장에 식초, 설탕, 발효겨자를
넣고 골고루 섞어 소스를 만들고,
접시에 구운 마와 손질한 야채를 넣고
만들어 놓은 소스와 어린잎을 올린다.

## 영양 정보

- 칼로리: 112 kcal
- 탄수화물: 16g
- 단백질: 3g
- 지방: 4g
- 나트륨: 144.8mg
