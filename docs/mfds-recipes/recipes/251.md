# 버섯콩불고기

버섯콩불고기은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 재료 콩고기(150g)
- 양파(20g)
- 당근(5g)
- 양배추(30g)
- 깻잎(10g)
- 느타리버섯(15g)
- 표고버섯(15g)
- 새송이버섯(15g)
- 부추(10g)
- 미나리(10g)
- 청양고추(3g)
- 양념장 설탕(1g)
- 저염진간장(10g)
- 맛술(3g)
- 배(5g)
- 다진 양파(5g)
- 매실액(10g)
- 참기름(0.5g)
- 양념 들기름(5g)
- 다진 파(5g)
- 다진 마늘(2g)
- 다진 생강(2g)

## 조리 방법

1. 1. 콩고기는 물에 담가 20분 정도 불린
뒤 양념장 ½과 버무린다.
2. 2. 양파, 당근은 채 썰고, 양배추와
깻잎, 버섯은 한입 크기로 썬다.
3. 3. 부추와 미나리는 양파와 비슷한
길이로 썰고, 청양고추는 어슷 썬다.
4. 4. 들기름을 두른 팬에 다진 파와 마늘,
생강을 넣고 약한 불에 볶다가
콩고기를 넣고 나머지 양념장을
뿌려 볶는다.
5. 5. 어느 정도 볶아지면 양파, 당근을
넣고 볶다가 양배추를 넣고 볶는다.
6. 6. 양배추가 익으면 먹기 좋게 손질한
버섯을 넣고 살짝 볶아 마무리한다.

## 영양 정보

- 칼로리: 323.8 kcal
- 탄수화물: 13.5g
- 단백질: 11.2g
- 지방: 8.51g
- 나트륨: 494.7mg
