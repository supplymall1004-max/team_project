# 맛간장삼치구이

맛간장삼치구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 삼치 200g
- 생강 50g
- 대파 100g
- 통깨 1g
- 버터 10g
- 소스 : 맛간장 30g
- 생강청 25g
- 커피 1g

## 조리 방법

1. 1. 소스 재료를 이용해 맛 간장으로 만든 데리
야끼소스를 만들어준다.
2. 2. 참치는 손질해 수분을 빼준 후 흰 후추로
밑간을 해준다.
3. 3. 생강은 아주 곱게 채 썰어 냉수에 헹군다.
4. 4. 대파도 아주 가늘게 채 썰어서 냉수에 담가
매운맛을 빼준다.
5. 5. 버터를 올린 팬에 삼치를 약불에서 구워준다.
6. 6. 구워진 참치를 소스에 졸여주고 대파채를
올린다. 삼치에 소스를 뿌리고 생강채를 올려
완성한다.

## 영양 정보

- 칼로리: 220.9 kcal
- 탄수화물: 5.2g
- 단백질: 30.2g
- 지방: 8.8g
- 나트륨: 570.8mg
