# 비트양파김치

비트양파김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 양파 150g
- 절임물 : 물 200g
- 볶은소금 2g
- 식초 15g
- 비트 2g
- 국물 : 절임물 100g
- 생강청 30g
- 식초 15g
- 청양고추 7g

## 조리 방법

1. 1. 양파는 네 번 정도 칼집을 내어 준다.
2. 2. 절임물을 만들고 절임 물에 칼집 낸 양파를
하루(24시간) 정도 절여준다.
3. 3. 절임물을 버리지 말고 절임물 반에 생강청,
식초, 청양고추를 잘라서 섞어주고 국물을
만든다.
4. 4. 만들어진 국물에 절여진 양파를 담가준다.
5. 5. 24시간 후부터 먹을 수가 있다.

## 영양 정보

- 칼로리: 28 kcal
- 탄수화물: 4.9g
- 단백질: 1.8g
- 지방: 0.1g
- 나트륨: 42.5mg
