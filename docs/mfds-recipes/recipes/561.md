# 머스터드튤립치킨

머스터드튤립치킨은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 닭고기(날개
- 8개)
- 어린잎(20g)
- 파슬리(5g)
- 머스터드(50g)
- 양파(50g)
- 마늘(20g)
- 레몬(1/4개)
- 튀김가루(100g)
- 소금(0.1g)
- 후추(0.01g)
- 식용유(400g)

## 조리 방법

1. 1. 닭날개에 칼집을 넣고 뒤집어
튤립모양의 봉을 만든다.
2. 2. 닭날개에 소금, 후춧가루로 밑간을
한다.
3. 3. 파슬리는 곱게 다져 가루를 만든다
4. 4. 양파, 마늘은 즙을 내서 튀김가루,
파슬리 가루와 함께 골고루 섞어
튀김반죽을 만든다.
5. 5. 어린잎은 찬물에 담궈 건져 놓는다.
6. 6. 튀긴 닭날개의 기름을 빼고 레몬과
어린잎으로 장식한다.

## 영양 정보

- 칼로리: 342.4 kcal
- 탄수화물: 5.1g
- 단백질: 21.5g
- 지방: 26.2g
- 나트륨: 290mg
