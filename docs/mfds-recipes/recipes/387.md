# 오징어구이

오징어구이은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 오징어 300g
- 미니파프리카 40g
- 양파 50g
- 청고추 15g
- 홍고추 15g
- 양념장 : 다진마늘 15g
- 고춧가루 15g
- 홀토마토 50g
- 고추장 15g
- 설탕 15g
- 올리고당 15g
- 후추 1g
- 참기름 15g
- 고명 : 볶은 현미 15g

## 조리 방법

1. 1. 오징어는 솔방울 모양으로 잘라준다.
2. 2. 미니 파프리카, 양파, 청홍고추는 적당한
크기로 잘라준다.
3. 3. 다진마늘을 볶다가 오징어를 넣어 볶는다.
4. 4. 홀토마토와 양념장 재료를 넣어 볶는다.
5. 5. 팬을 약간 태우는 듯 해서 불맛과 매운맛을
더해준다.
6. 6. 접시에 담아주고 볶은 현미를 뿌려 완성한다.

## 영양 정보

- 칼로리: 278.1 kcal
- 탄수화물: 24.8g
- 단백질: 22.9g
- 지방: 9.7g
- 나트륨: 611.4mg
