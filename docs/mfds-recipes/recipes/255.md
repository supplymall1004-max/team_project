# 검은콩피시볼조림

검은콩피시볼조림은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 피시볼반죽 검은콩(15g)
- 쪽파(5g)
- 동태포(30g)
- 감자전분(10g)
- 밀가루(7g)
- 청양고추(1g)
- 달걀흰자(7g)
- 소스 마늘(5g)
- 녹말가루(2g)
- 가다랑어장국(3g)
- 간장(3g)
- 물(100ml)
- 유자청(3g)
- 대파(3g)

## 조리 방법

1. 1. 검은콩을 불려 삶는다.
2. 2. 마늘은 편으로 썰고, 쪽파는 송송
썬다.
3. 3. 동태포는 물기를 제거한다.
4. 4. 믹서에 피시볼반죽 재료를 넣어 간
뒤 동그랗게 빚고 콩식용유(5g)를
두른 팬에 굴려가며 익힌다.
5. 5. 팬에 녹말가루를 제외한 소스
재료를 넣고 끓이다가 피시볼을
넣고 조리고, 녹말가루로 농도를
맞춘 뒤 쪽파를 뿌려 마무리한다.

## 영양 정보

- 칼로리: 166.7 kcal
- 탄수화물: 16.1g
- 단백질: 13.8g
- 지방: 4.28g
- 나트륨: 193.3mg
