# 물파래콩전

물파래콩전은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 물파래 5g
- 노란콩 25g(불린것 52g)
- 우리밀가루 10g
- 현미유 5g
- 홍고추 2g
- 물 120g 간장 2g
- 파 1g
- 식초 0.5g
- 통깨 0.2g

## 조리 방법

1. 1. 물파래는 씻어 체에 걸러 물기를 제거하고 잘게 썬다.
2. 2. 물에 5시간정도 불린 노란콩은 물 반컵을 넣고 약 10분가량 삶아 건져 믹서에 곱게 간다.
3. 3. 파는 다지고 홍고추는 어슷썰기 한다.
4. 4. 밀가루에 1,2와 물 20g을 넣어 반죽을 한다.
5. 5. 팬에 현미유를 두르고 반죽을 떠서 지름 8cm 가량의 크기로 동그랗게 부치면서 윗면에 홍고추로 장식한다.
6. 6. 간장, 파, 식초, 통깨를 넣고 초간장을 만들어 물파래콩전과 곁들인다.

## 영양 정보

- 칼로리: 181.5 kcal
- 탄수화물: 15g
- 단백질: 10g
- 지방: 9.9g
- 나트륨: 124.1mg
