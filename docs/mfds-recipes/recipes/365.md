# 블루베리양갱

블루베리양갱은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 흰앙금 100g
- 블루베리 30g
- 설탕 30g
- 한천 20g
- 물100g

## 조리 방법

1. 1. 블루베리는 믹서에 갈아서 체에 내려준다.
2. 2. 물에 끓여주면서 흰앙금과 설탕을 풀어준다.
3. 3. 여기에 다시 한천가루를 넣어 끓여준다.
4. 4. 여기에 블루베리를 넣어 끓인다.
5. 5. 몰드에 넣어서 시원한 곳에서 식혀준다.
6. 6. 접시에 완성한다.

## 영양 정보

- 칼로리: 142.4 kcal
- 탄수화물: 20.3g
- 단백질: 8g
- 지방: 3.3g
- 나트륨: 32.2mg
