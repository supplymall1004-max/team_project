# 삼색딤섬

삼색딤섬은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 밀가루(200g)
- 백년초가루(20g)
- 뽕잎가루(20g)
- 치자가루(10g)
- 미나리(20g)
- 두부(30g)
- 오징어(30g)
- 다진 소고기(60g)
- 양파(20g)
- 부추(30g)
- 숙주(30g)
- 대파(10g)
- 마늘(10g)
- 참기름(10g)
- 후춧가루(0.05g)
- 통깨(5g)

## 조리 방법

1. 1. 밀가루를 세등분하여 각각 백년초,
뽕잎, 치자 가루를 넣고 물을 넣고
반죽을 한 뒤 비닐봉투에 넣어 숙성을
시킨다.
2. 2. 두부는 소창에 물기를 짜고, 양파, 파,
마늘은 다지고, 부추는 송송 썰어 준비
하고, 숙주는 뜨거운 물에 데쳐 곱게
다진다.
3. 3. 오징어는 껍질을 벗겨 입자있게 다져
놓는다.
4. 4. 다진고기에 준비한 두부와 양파, 파,
마늘, 부추, 숙주(②)와 오징어(③)를
넣어 섞은 후 참기름과 후춧가루, 통깨를
넣어 딤섬 속을 만든다.
5. 5. 숙성된 밀가루 반죽을 꺼내 밀대를
이용하여 넓게 밀어 준다.
6. 6. 밀어 놓은 반죽을 사각으로 자른 후
그 안에 딤섬 속을 넣어 미나리를 데쳐
끈을 만들어 딤섬 위쪽을 묶은 후
찜통에서 김이 오르면 약 10분 정도
쪄서 완성한다.

## 영양 정보

- 칼로리: 350.4 kcal
- 탄수화물: 49.5g
- 단백질: 20.2g
- 지방: 7.9g
- 나트륨: 235.1mg
