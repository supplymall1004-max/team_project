# 연어오븐구이

연어오븐구이은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 연어(150g)
- 양파(50g)
- 토마토(100g)
- 파프리카(50g)
- 당근(30g)
- 마늘(20g)
- 청피망(30g)
- 레몬(1/4개)
- 소금(0.3g)
- 후춧가루(0.01g)
- 생크림(20g)
- 화이트크림(50g)
- 버터(20g)

## 조리 방법

1. 1. 연어는 소금, 후춧가루, 레몬으로
마리네이드 한다.
2. 2. 양파, 마늘은 입자있게 다진다.
3. 3. 토마토는 씨를 제거하고 입자있게
썰고, 파프리카, 피망은 채 썬 뒤
볶는다.
4. 4. 화이트크림, 생크림을 골고루 섞어
크림소스를 만든다.
5. 5. 팬에 버터를 넣고 양파, 마늘을 볶다가
만들어 놓은 크림소스를 넣고 졸인다.
6. 6. 접시에 볶은 야채를 담고, 200℃
오븐에서 10분 정도 구운 연어를 담은
뒤 크림소스를 올린다.

## 영양 정보

- 칼로리: 285.9 kcal
- 탄수화물: 8.1g
- 단백질: 28.7g
- 지방: 15.4g
- 나트륨: 222.1mg
