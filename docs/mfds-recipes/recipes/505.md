# 둥지튀김

둥지튀김은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 닭고기살(150g)
- 소금(0.3g)
- 후춧가루(0.2g)
- 달걀(50g)
- 녹말가루(20g)
- 백년초국수(20g)
- 뽕잎국수(20g)
- 치자국수(20g)
- 튀김기름(400g)
- - 소스 : 레몬(10g)
- 설탕(20g)
- 연유(10g)
- 물(50g)
- 다진 마늘(10g)
- 생강(5g)
- 식초(20g)
- 녹말가루(20g)

## 조리 방법

1. 1. 닭고기살은 작게 썰어 소금과 후춧
가루를 살짝 뿌려 밑간을 하고 달걀을
넣어 섞는다.
2. 2. 달걀을 넣어 섞은 닭고기 살에 녹말
가루를 넣어 골고루 묻힌다.
3. 3. 국수는 각각 끓는 물에 살짝 데쳐 물기를
빼놓는다.
4. 4. 물기가 빠진 국수는 튀김기름 170℃
온도에서 각각 작은 채에 그대로 살짝
튀겨 둥지를 만든 후 기름을 냅킨에
빼 놓는다.
5. 5. 녹말가루를 묻힌 닭고기살도 튀김
기름에 튀겨 기름을 빼 놓은 둥지국수
위에 올린다.
6. 6. 냄비에 레몬과 설탕을 넣고 물을 넣어
끓이다가 레몬은 건져 내고, 마늘과
생강, 연유, 식초, 녹말을 풀어 넣고
살짝 한 번 더 끓여 녹말소스를 만들어,
튀겨진 닭고기 위에 올려 완성한다.

## 영양 정보

- 탄수화물: 149.1g
- 단백질: 242.2g
- 지방: 37.3g
- 나트륨: 74.9mg
