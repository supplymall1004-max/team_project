# 봄나물주먹밥

봄나물주먹밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 쌀 70g
- 취나물 20g
- 고사리 20g
- 빨강파프리카 5g
- 노랑파프리카 5g
- 참기름 3g
- 다진 마늘 1g
- 소금 0.7g
- 통깨 0.5g

## 조리 방법

1. 1. 쌀은 불린 후 밥을 짓는다.
2. 2. 고사리는 끓는 물에 삶고, 잘게 썬 파프리카는 고사리 삶는 물에 살짝 데친다.
3. 3. 고사리 삶는 중간에 취나물을 넣어 함께 데친다.
4. 4. 고사리와 취나물은 잘게 썬다 .
5. 5. 파프리카, 고사리, 취나물에 참기름, 마늘, 소금, 통깨를 넣고 양념을 한다.
6. 6. 밥에 참기름, 통깨, 소금을 넣어 양념한 후 4의 재료를 섞어 주먹밥을 만든다.

## 영양 정보

- 칼로리: 295.8 kcal
- 탄수화물: 57.9g
- 단백질: 6g
- 지방: 3.9g
- 나트륨: 240.3mg
