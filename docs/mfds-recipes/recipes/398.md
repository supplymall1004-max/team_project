# 라이스버거떡갈비

라이스버거떡갈비은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 라이스버거 : 밥 200g
- 맛간장 15g
- 전분가루 15g
- 팽이버섯 20g
- 흰후추 1g
- 달걀 60g
- 떡갈비 : 다진소고기 50g
- 다진돼지고기 50g
- 다진양파 30g
- 다진쪽파 15g
- 맛간장 10g
- 후추 1g
- 토마토겉절이 : 토마토 80g
- 미나리 20g
- 맛간장 5g
- 청고추 15g
- 홍고추 15g

## 조리 방법

1. 1. 팽이버섯을 송송 다진 후 라이스버거 재료로
라이스버거를 만들어준다.
2. 2. 만들어준 버거는 팬에 구워준다.
3. 3. 토마토와 미나리를 먹기 좋게 잘라서 겉절이를
만들어준다.
4. 4. 떡갈비 재료로 떡 갈비를 만들어준다.
5. 5. 만들어진 떡갈비를 구워준다.
6. 6. 접시에 토마토 겉절이를 담고 라이스버거와
떡갈비를 올려 완성한다.

## 영양 정보

- 칼로리: 206.2 kcal
- 탄수화물: 28.3g
- 단백질: 12.4g
- 지방: 4.8g
- 나트륨: 206.1mg
