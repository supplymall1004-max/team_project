# 과일갈비찜

과일갈비찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 돼지갈비(150g)
- 홍고추(5g)
- 참기름(5g)
- 통깨(2g)
- 배(20g)
- 사과(20g)
- 대추(20g)
- 밤(10g)
- 오이(20g)
- 양파(20g)
- 당근(20g)
- 무(20g)
- - 양념장 : 저염간장(20g)
- 올리고당(20g)
- 설탕(10g)
- 다진 마늘(20g)
- 다진 대파(10g)
- 다진 생강(5g)
- 후춧가루(0.03g)
- 소주(10g)

## 조리 방법

1. 1. 돼지갈비는 물에 담그어 핏물을 충분히
제거한다.
2. 2. 홍고추는 어슷썰어 냄비에 볶는다.
3. 3. 핏물을 제거한 돼지갈비를 홍고추와
함께 넣어 볶아준다.
4. 4. 배는 강판에 갈고, 파, 마늘, 생강은
다진 뒤 간장과 올리고당, 설탕,
후춧가루, 소주와 함께 넣고 골고루
섞어서 양념장을 만든다.
5. 5. 사과, 대추, 밤, 오이, 양파, 당근, 무는
사각 깍둑 썰기로 준비한다.
6. 6. 볶아진 갈비(③)에 양념장(④)과 물을
넣고 은근히 갈비를 졸이다가, 국물이
자작하게 남으면, 손질한 과일과 채소
등(⑤)을 참기름과 함께 넣고, 한 번 더
졸여 완성한다.

## 영양 정보

- 칼로리: 669.2 kcal
- 탄수화물: 29.9g
- 단백질: 67.1g
- 지방: 31.3g
- 나트륨: 903mg
