# 묵은지고등어와 새우말이꼬치

묵은지고등어와 새우말이꼬치은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 고등어(1마리)
- 새우(5마리)
- 새송이버섯(1개)
- 당근(30g)
- 미나리(20g)
- 깻잎(2장)
- 가지(20g)
- 오이(30g)
- 아몬드(20g)
- 마늘(20g)
- 파프리카(50g)
- 어린잎(10g)
- 양상추(30g)
- 땅콩(10g)
- 묵은지(200g)

## 조리 방법

1. 1. 고등어는 길게 포를 뜬다.
2. 2. 새우는 껍질을 벗겨 손질한다.
3. 3. 새송이와 당근은 채썬다
4. 4. 미나리는 끓는 물에 살짝 데친다.
5. 5. 묵은지는 흐르는 물에 헹군 뒤,
묵은지에 고등어를 넣고 새우, 새송이,
당근, 가지를 넣고 돌돌 말아 미나리로
묶은 뒤 찐다.
6. 6. 양상추, 오이, 깻잎, 파프리카를
채썰고 땅콩가루를 버무려 함께
담아낸다.

## 영양 정보

- 칼로리: 403 kcal
- 탄수화물: 22.3g
- 단백질: 41.5g
- 지방: 16.4g
- 나트륨: 1199.1mg
