# 닭가슴살말이

닭가슴살말이은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 재료 닭가슴살(150g)
- 당근(10g)
- 시금치(10g)
- 마른 표고버섯(10g)
- 간장(5g)
- 참기름(3g)
- 견과류(16g)
- 단촛물 설탕(3g)
- 식초(10g)
- 겨자소스 설탕(28g)
- 발효겨자(28g)
- 식초(28g)
- 다진 잣(4g)
- 다진 땅콩(4g)
- 참깨(4g)

## 조리 방법

1. 1. 닭가슴살은 곱게 다진다.
2. 2. 당근은 채 썰어서 단촛물에 5분 정도 절이고, 시금치는 데쳐서 물기를 제거하고, 마른 표고버섯은 불린 뒤 곱게 채를 썰어 간장과 참기름에 버무린다.
3. 3. 종이포일에 다진 가슴살을 올리고 그 위에 시금치, 당근, 표고버섯과 견과류를 올려 말아준다
4. 4. 김 오른 찜기에 넣어서 25분간 찐다.
5. 5. 겨자소스를 만든다.
6. 6. 닭가슴살말이에 겨자소스를 곁들여
마무리한다.

## 영양 정보

- 칼로리: 233.1 kcal
- 탄수화물: 3.98g
- 단백질: 2.22g
- 지방: 3.59g
- 나트륨: 113.6mg
