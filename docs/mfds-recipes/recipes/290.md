# 샤브된장국

샤브된장국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 우삼겹(50g)
- 깻잎(7.5g)
- 느타리버섯(20g)
- 팽이버섯(20g)
- 쪽파(10g)
- 송송 썬 청양고추(3g)
- 송송 썬 붉은 고추(3g)
- 육수 닭가슴살(50g)
- 대파(20g)
- 마늘(10g)
- 양념 저염된장(5g)
- 다진 마늘(5g)

## 조리 방법

1. 1. 물(600g)에 육수 재료를 넣고 15분간
센 불에 끓인 후 10분간 약한 불로
끓여 육수를 만든다.
2. 2. 우삼겹을 펼쳐 깔고 그 위에 깻잎,
느타리버섯, 팽이버섯 순으로
올린다.
3. 3. 돌돌 말고 쪽파로 묶어 고정시킨다.
4. 4. 육수에 저염된장을 풀고 다진
마늘을 넣은 뒤 청양고추와 돌돌 만
고기를 넣고 약한 불에 살짝 익힌다.
5. 5. 그릇에 담은 뒤 붉은 고추를 올려
마무리한다.

## 영양 정보

- 칼로리: 218.9 kcal
- 탄수화물: 2.34g
- 단백질: 3.13g
- 지방: 5.03g
- 나트륨: 299.9mg
