# 양배추롤

양배추롤은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 닭고기살(100g)
- 천일염(0.5g)
- 후춧가루(0.3g)
- 식용유(10g)
- 정종(10g)
- 양배추(80g)
- 부추(30g)
- 깻잎(10g)
- 홍파프리카(20g)
- 노랑파프리카(20g)
- 당근(20g)
- 양파(20g)
- - 소스 : 요거트(20g)
- 머스터드(20g)
- 꿀(10g)
- 레몬즙(10g)
- 설탕(5g)

## 조리 방법

1. 1. 닭고기살은 천일염과 후춧가루를 뿌려
마리네이드를 해놓는다.
2. 2. 양배추와 부추는 김이 오른 찜통에
쪄서 식혀 놓는다.
3. 3. 깻잎은 채를 썰고, 파프리카는 속을 제거
하고 5cm 길이로 채를 썰고, 당근과
양파도 같은 길이로 채를 썰어 놓는다.
4. 4. 마리네이드를 해 놓은 닭고기살은 팬에
기름을 두르고, 팬이 따끈해지면 앞뒤를
돌려가며 정종을 살살 뿌려 닭고기살을
굽고, 닭고기살이 익으면 살을 찢어 준비
한다.
5. 5. 요거트와 머스터드, 꿀 ,레몬즙, 설탕을
넣고 고루 섞어 소스를 만들어 놓는다.
6. 6. 쪄 놓은 양배추에 준비한 채소(③)와
닭고기살(④)을 넣고, 돌돌 말아 쪄
놓은 부추로 양배추를 묶어 완성하고,
만들어 놓은 요거트 소스를 함께 제공
한다.

## 영양 정보

- 칼로리: 164.8 kcal
- 탄수화물: 14.8g
- 단백질: 17.5g
- 지방: 3.9g
- 나트륨: 111.5mg
