# 닭강정

닭강정은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 닭고기(200g)
- 카레가루(20g)
- 녹말가루(10g)
- 튀김기름(400g)
- 저염간장(20g)
- 레몬즙(10g)
- 설탕(10g)
- 꿀(10g)
- 다진마늘(10g)
- - 소스 : 요거트(20g)
- 양파(20g)
- 당근(20g)
- 파프리카(20g)
- 오이(20g)
- 케첩(10g)

## 조리 방법

1. 1. 닭은 깨끗이 씻어 여러 번 토막을 낸다.
2. 2. 토막 낸 닭을 카레가루와 녹말가루에
버무려 놓는다.
3. 3. 양파와 당근, 파프리카, 오이는 크기를
작은 깍두기 모양으로 썰어 준비한다.
4. 4. 팬에 식용유를 두르고 다진마늘를 살짝
볶은 뒤 저염간장에 레몬즙과 설탕, 꿀,
볶은 마늘을 넣어 섞는다.
5. 5. 요거트에 썰어 놓은 양파와 당근,
파프리카, 오이, 케첩을 넣어 섞어준다.
6. 6. 튀김기름 온도가 170℃가 되면 버무려
놓은 닭(②)을 넣고 두 번 튀겨, 기름을
제거하고, ④번 소스에 살짝 버무려
접시에 담아, 준비한 요거트소스(⑤)를
올려 완성한다.

## 영양 정보

- 칼로리: 573.1 kcal
- 탄수화물: 22.4g
- 단백질: 29.1g
- 지방: 40.8g
- 나트륨: 377.9mg
