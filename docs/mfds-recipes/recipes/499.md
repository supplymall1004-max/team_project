# 황태리조또

황태리조또은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 두부(100g)
- 소금(0.2g)
- 홍합(50g)
- 버터(5g)
- 황태(100g)
- 육수(100g)
- 생크림(20g)
- 달걀(10g)
- 참기름(5g)
- 통깨(5g)

## 조리 방법

1. 1. 두부는 김이 오르는 찜통에 소금을
넣고 약 5분 정도 쪄 놓는다.
2. 2. 황태는 강판에 갈아 보프리를 만들어
놓는다.
3. 3. 홍합은 끓는 육수에 데쳐 육수는 따로
걸러 놓고, 홍합은 살만 꺼내 따로 준비
한다.
4. 4. 냄비에 버터를 녹이고, 두부와 홍합을
으깨어 볶는다.
5. 5. 볶아지는 냄비에 만들어 놓은 육수와
생크림을 넣고, 골고루 섞어 가면서
볶는다.
6. 6. 두부가 육수에 볶아지면, 마지막에
황태보프라기를 넣고, 달걀과 참기름
통깨를 넣어 살짝 익혀 완성한다.

## 영양 정보

- 칼로리: 292.3 kcal
- 탄수화물: 5.4g
- 단백질: 29.1g
- 지방: 17.1g
- 나트륨: 504.9mg
