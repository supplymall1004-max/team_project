# 민들레 샐러드

민들레 샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●주재료 : 민들레 잎 40g
- 비트 5g(1작은술)
- 오렌지(껍질) 40g(1/5개)
- ●드레싱 : 간장 20g(1⅓큰술)
- 다진 홍고추 10g(1개)
- 레몬즙 60g(4큰술)
- 설탕 6g(1작은술)

## 조리 방법

1. 1. 간장, 레몬즙, 설탕, 다진 홍고추를 섞어 드레싱을 만든다.
2. 2. 민들레 잎은 4×4cm로 썰고, 비트는 채 썰고 물에 담가 색을 뺀다.
3. 3. 오렌지는 씻어 껍질만 발라 채 썰고 민들레 잎, 비트, 오렌지 껍질을 섞어 접시에 담고 드레싱을 곁들인다.

## 영양 정보

- 칼로리: 53.9 kcal
- 탄수화물: 10.9g
- 단백질: 2.4g
- 지방: 0g
- 나트륨: 874.2mg
