# 우무오미자냉화채

우무오미자냉화채은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 오미자 10g
- 물 200g
- 올리고당 14g
- 우무묵 20g

## 조리 방법

1. 1. 오미자를 헹궈 따뜻한 물에 담가 12시간 우린다.
2. 2. 우려낸 오미자물을 면보에 거른다.
3. 3. 오미자 국물 1컵에 올리고당을 넣어 감미한다.
4. 4. 우무를 곱게 채 썰거나 모양 틀로 찍어낸다.
5. 5. 오미자 국물을 살짝 얼려 믹서에 갈아 슬러쉬를 만든다.
6. 6. 오미자 슬러쉬에 우무를 올린다.

## 영양 정보

- 칼로리: 71.3 kcal
- 탄수화물: 14.4g
- 단백질: 1.65g
- 지방: 0.88g
- 나트륨: 17mg
