# 미역줄기두부무침

미역줄기두부무침은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 건조미역줄기 10g
- 두부 70g
- 로즈마리 1g
- 혼합견과류 25g
- 소스 : 맛간장 15g
- 다진양파 15g
- 다진마늘 5g
- 흰후추 1g
- 통깨 3g
- 식초 5g

## 조리 방법

1. 1. 두부는 물기를 꼭 짜준 후 으깬다.
2. 2. 미역줄기는 불린 후 먹기 좋게 자른다.
3. 3. 양념장을 만들어준다.
4. 4. 준비된 견과류를 송송 다져준다.
5. 5. 불린 미역줄기에 소스를 섞어 재워준다.
6. 6. 불린 미역줄기에 두부를 넣어 섞어준다.

## 영양 정보

- 칼로리: 178.3 kcal
- 탄수화물: 4.6g
- 단백질: 17.5g
- 지방: 10g
- 나트륨: 446.4mg
