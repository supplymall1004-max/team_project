# 토마토김치

토마토김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 방울토마토 200g
- 영양부추 20g
- 양파 40g
- 청고추 15g
- 홍고추 15g
- 양념 : 바나나 40g
- 배 40g
- 고춧가루 15g
- 매실청 10g
- 간편 어간장 15g
- 통깨 2g
- 다진마늘 15g
- 참기름 5g
- 젓갈 5g

## 조리 방법

1. 1. 토마토는 먹기 좋은 크기로 잘라준다.
2. 2. 청고추, 홍고추, 양파는 채 썰고 부추는 먹기
좋게 잘라준다.
3. 3. 바나나, 배, 젓갈을 갈아 준 뒤 고춧가루와
섞어준다.
4. 4. 만들어진 국물에 나머지 재료를 넣고 최종
김치 국물을 만들어준다.
5. 5. 국물에 영양부추만 뺀 재료를 넣고 버무려준다.
6. 6. 마지막에 영양부추를 넣어 김치를 완성한다.

## 영양 정보

- 칼로리: 80.2 kcal
- 탄수화물: 13.8g
- 단백질: 3.8g
- 지방: 1.1g
- 나트륨: 109.5mg
