# 대하마늘볶음밥

대하마늘볶음밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- •필수 재료 : 대하(3마리)
- 쌀(90g)
- 당근(20g)
- 깻잎(2장)
- 양파(30g)
- 숙주(50g)
- 대추(2알)
- 청고추(1개)
- 파(10g)
- 마늘(10g)
- 생강(5g)
- 식용유(5g)
- •양념 : 저염간장(5g)

## 조리 방법

1. 1. 새우는 내장을 제거하고 껍질을 벗겨 깨끗이 씻는다.
2. 2. 찜기에 새우와 생강을 넣어 약 5분 정도 찐다.
3. 3. 마늘은 납작 썰고, 당근과 양파, 파, 깻잎, 청고추, 대추는 굵게 다진다.
4. 4. 숙주는 꼬리를 떼어 다듬고, 끓는 물에 살짝 데친다.
5. 5. 팬에 식용유를 두른 뒤 손질한 채소를 넣어 볶는다.
6. 6. 밥과 새우를 넣어 볶고, 저염간장을 넣어 볶아 마무리한다.

## 영양 정보

- 칼로리: 286.6 kcal
- 탄수화물: 32.5g
- 단백질: 14g
- 지방: 11.2g
- 나트륨: 191.4mg
