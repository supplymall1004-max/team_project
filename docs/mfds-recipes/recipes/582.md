# 해물아란치니

해물아란치니은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 오징어(100g)
- 새우(3마리)
- 토마토(50g)
- 양파(30g)
- 마늘(20g)
- 샐러리(20g)
- 밥(100g)
- 달걀(1개)
- 어린잎(10g)
- 소금(0.2g)
- 레몬(1/4개)
- 빵가루(20g)

## 조리 방법

1. 1. 새우는 껍질을 벗겨 다진다.
2. 2. 오징어는 썰어서 다진다.
3. 3. 양파, 샐러리, 마늘은 다져 팬에 볶아
식힌다.
4. 4. 토마토는 껍질을 벗겨 다진다.
5. 5. 다진 토마토를 팬에 볶다가 다진
새우와 오징어, 밥을 넣어 볶고, 3번을
넣고 볶아 동그랗게 모양을 만든다
6. 6. 동그란 밥에 레몬을 섞은 달걀노른자를
묻히고 빵가루 옷을 입혀 팬에 기름을
둘러 굴려가며 익힌 후 접시에 담고
어린잎을 올린다.

## 영양 정보

- 칼로리: 283.8 kcal
- 탄수화물: 41g
- 단백질: 7.4g
- 지방: 10g
- 나트륨: 393.2mg
