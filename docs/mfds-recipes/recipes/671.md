# 표고버섯 감자찜

표고버섯 감자찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 표고버섯(3개)
- 감자(200g)
- 고수(10g)
- 당근(30g)
- 양파(30g)
- 홍고추(1개)
- 녹말(20g)

## 조리 방법

1. 1. 표고버섯은 따뜻한 물에 불린다.
2. 2. 감자는 강판에 간다.
3. 3. 고수는 잘게 다진다.
4. 4. 당근은 곱게 다지고, 양파와 홍고추도
다진다.
5. 5. 감자에 다진 당근과 양파, 홍고추를
섞는다.
6. 6. 불린 표고버섯은 기둥을 자르고 바닥에
녹말가루를 묻혀 ?번을 넣고 김이
오른 찜통에 약 10분 정도 찐다.

## 영양 정보

- 칼로리: 176.8 kcal
- 탄수화물: 37.6g
- 단백질: 5.4g
- 지방: 0.5g
- 나트륨: 108.3mg
