# 냉파스타

냉파스타은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 스파게티(100g)
- 올리브오일(10g)
- 소고기(50g)
- 저염간장(10g)
- 설탕(10g)
- 마늘(10g)
- 깻잎(10g)
- 양파(20g)
- 파프리카(20g)
- 육수(400g)
- 설탕(10g)
- 식초(10g)
- 저염간장(10g)

## 조리 방법

1. 1. 스파게티는 약 10분 정도 삶아 건져
찬물에 헹궈 물기를 빼고 올리브오일을
발라 준비한다.
2. 2. 소고기는 넓게 편으로 펴서 간장, 설탕,
마늘을 넣어 양념을 한다.
3. 3. 양념된 소고기를 팬에 구워 냅킨위에
올려 기름을 빼놓는다.
4. 4. 깻잎과 양파, 파프리카는 길게 채를
썰어 준비한다.
5. 5. 팬에 기름을 넣고 팬이 뜨거울 때
양파를 넣어 재빠르게 볶아낸다.
6. 6. 준비한 육수에 설탕과 식초, 저염간장을
넣어 차갑게 냉장보관하고, 그릇에
스파게티를 담고, 그 위에 볶은 양파와
깻잎, 파프리카와 구운 고기를 올린 후
차가운 육수를 부어 완성한다.

## 영양 정보

- 칼로리: 289.1 kcal
- 탄수화물: 57.8g
- 단백질: 9g
- 지방: 2.4g
- 나트륨: 81.9mg
