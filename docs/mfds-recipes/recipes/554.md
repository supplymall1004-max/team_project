# 겨자소스를 곁들인 새우롤과 전복샐러드

겨자소스를 곁들인 새우롤과 전복샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 새우(3마리)
- 전복(2개)
- 연어(100g)
- 겨자가루(20g)
- 어린잎(30g)
- 요구르트(20g)
- 고추장(30g)
- 설탕(10g)
- 식초(10g)
- 소금(1g)

## 조리 방법

1. 1. 새우는 껍질을 제거하고 포를 뜬 뒤 삶아 익힌다.
2. 2. 전복은 수저로 떼어내 소금으로 씻고
달군 팬에 굽는다.
3. 3. 연어는 포를 뜬다.
4. 4. 겨자가루에 30℃ 정도의 물을 넣고
골고루 섞어 발효시키고, 설탕, 식초를
넣고 겨자소스를 만든다.
5. 5. 고추장에 요구르트를 넣고
고추장소스를 만든다
6. 6. 접시에 익힌 새우, 전복, 연어를 담은 뒤
겨자소스와 고추장소스를 올리고
어린잎으로 장식한다.

## 영양 정보

- 칼로리: 379.2 kcal
- 탄수화물: 1.5g
- 단백질: 45.9g
- 지방: 21.1g
- 나트륨: 330.4mg
