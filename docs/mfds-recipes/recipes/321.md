# 계란숙샐러드

계란숙샐러드은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 달걀 120g
- 쇠고기 50g
- 게맛살 20g
- 감자 1개
- 천연조미료 5g
- 물 200g
- 브로콜리 50g
- 방울토마토 60g
- 미니파프리카 25g
- 영양부추 5g
- 소스 : 매실청 30g
- 발사믹식초 15g
- 올리브오일 15g
- 후추 1g
- 식초 15g

## 조리 방법

1. 1. 소고기에 물을 넣어 육수를 뽑아준 후 천연
조미료로 간을 해준다.
2. 2. 소고기는 곱게 다져준 뒤 볶아준다.
3. 3. 감자를 삶아서 체에 내려준다.
4. 4. 계란을 풀고 소고기 다진 것과 게맛살, 물을
넣고 섞은 뒤 찜기에 쪄준다.
5. 5. 부추와 브로콜리는 뜨거운 물에 데쳐 먹기
좋게 잘라주고, 방울토마토와 미니파프리카도
먹기 좋게 잘라준다.
6. 6. 매실청과 올리브오일, 후추로 드레싱을 만들어
준비된 채소를 섞어준 뒤 그릇에 담는다.

## 영양 정보

- 칼로리: 96.2 kcal
- 탄수화물: 6.5g
- 단백질: 53.9g
- 지방: 5.2g
- 나트륨: 112.5mg
