# 아삭

아삭은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 아스파라거스(5개)
- 당근(100g)
- 무(100g)
- 식초(200g)
- 설탕(200g)
- 월계수잎(3장)
- 통후추(20알)
- 소금(0.5g)

## 조리 방법

1. 1. 아스파라거스는 깨끗이 씻어 5cm
길이로 썬다.
2. 2. 당근을 아스파라거스 크기로 썬다.
3. 3. 무도 아스파라거스 크기로 썬다.
4. 4. 식초, 설탕을 동량으로 섞는다.
5. 5. 식초 설탕물이 끓으면 통에 뜨거울 때
붓고, 떠오르지 않도록 눌러 담는다.
6. 6. 약 7일 정도 숙성한다.

## 영양 정보

- 칼로리: 96.5 kcal
- 탄수화물: 20.7g
- 단백질: 2.3g
- 지방: 0.5g
- 나트륨: 1016.3mg
