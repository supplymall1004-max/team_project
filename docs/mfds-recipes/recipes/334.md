# 주꾸미연포탕

주꾸미연포탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 주꾸미 200g
- 무 30g
- 양파 50g
- 대파 20g
- 미나리 20g
- 단호박 25g
- 청고추 25g
- 국물 : 해물육수 400g
- 청양고추 20g
- 마늘 10g
- 흰후추 1g

## 조리 방법

1. 1. 주꾸미는 세척해 물기를 빼준다.
2. 2. 무와 단호박은 먹기 좋게 썰어준다.
3. 3. 청고추, 미나리, 양파를 먹기 좋게 썰어준다.
4. 4. 청양 고추를 어슷하게 썰고 마늘은 편으로 썰어
국물을 끓여준다.
5. 5. 국물이 만들어지면 무, 단호박을 넣고 끓여준다.
6. 6. 국물에 대파를 넣어주고 모든 재료를 넣어
완성한다.

## 영양 정보

- 칼로리: 90.3 kcal
- 탄수화물: 6.9g
- 단백질: 12.3g
- 지방: 1.5g
- 나트륨: 280.6mg
