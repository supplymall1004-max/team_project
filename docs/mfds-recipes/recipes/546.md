# 오징어토마토초무침

오징어토마토초무침은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 오징어(100g)
- 미나리(20g)
- 소금(0.3g)
- 토마토(50g)
- 양파(20g)
- 청고추(10g)
- 홍고추(10g)
- 대파(10g)
- 당근(20g)
- 통깨(5g)
- - 양념장 : 토마토(50g)
- 고추장(20g)
- 식초(10g)
- 생강즙(10g)
- 올리고당(20g)

## 조리 방법

1. 1. 오징어는 껍질을 벗겨 안 쪽으로 칼집을
넣어 무늬를 만들어 놓는다.
2. 2. 끓는 물에 소금을 넣고 미나리를 데쳐
찬물에 헹구고, 바로 오징어를 데쳐
준비한다.
3. 3. 준비한 토마토 반은 굵게 다지고, 반은
썰어서 접시에 돌려 담아놓는다.
4. 4. 다진 토마토와 고추장, 식초, 생강즙,
올리고당을 골고루 섞어 양념장을
만든다.
5. 5. 당근, 양파, 대파, 청 ․ 홍고추를 썰어
놓는다.
6. 6. 만들어 놓은 양념장에 데친 오징어와
채소를 넣어 무친 후 토마토를 돌려
담은 접시에 담아 통깨를 올려 완성한다.

## 영양 정보

- 칼로리: 155.7 kcal
- 탄수화물: 27.7g
- 단백질: 7.4g
- 지방: 1.7g
- 나트륨: 410.1mg
