# 애호박장떡말이밥

애호박장떡말이밥은(는) 밥 요리로, 굽기 조리법을 사용합니다.

## 재료

- 밥 100g
- 애호박 25g
- 양파 10g
- 풋고추 5g
- 고추장 15g
- 우리밀가루 50g
- 물 50g
- 현미유 1.5g
- 흑임자 1g
- 참기름 1g
- 소금 0.2g

## 조리 방법

1. 1. 양파는 갈고 애호박과 풋고추는 잘게 다진다.
2. 2. 물 1/2컵에 고추장을 풀고 우리밀가루를 넣어 골고루 섞는다.
3. 3. 1의 재료를 2에 넣어 반죽한다.
4. 4. 팬에 현미유를 두르고 3의 재료를 얇게 부친다.
5. 5. 밥에 참기름과 흑임자, 소금을 넣고 양념을 한다.
6. 6. 애호박장떡에 양념한 밥을 얇게 펴놓고 돌돌 말아 2~3cm 가량의 길이로 자른다.

## 영양 정보

- 칼로리: 400 kcal
- 탄수화물: 82.8g
- 단백질: 9.3g
- 지방: 4.5g
- 나트륨: 447.6mg
