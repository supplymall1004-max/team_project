# 녹차귀리라떼

녹차귀리라떼은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 브로콜리 60g
- 귀리 30g
- 닭육수 250ml
- 우유150ml
- 녹차가루 10g
- 아몬드슬라이스 20g
- 무염버터 5g
- 흰후추 3g
- 소금 0.5g
- 우유 50ml
- 생크림 50ml
- 다진양파 15g

## 조리 방법

1. 1. 귀리는 불려준 뒤 닭육수에 삶아준다.
2. 2. 브로콜리는 잘 손질하여 잘게 썰어 놓는다.
3. 3. 양파는 채 썰어 버터로 볶다가 1의 브로콜리와
귀리, 아몬드슬라이스를 넣고 같이 볶아준다.
4. 4. 어느 정도 볶아지면 우유를 넣고 다시 한 번
볶아준다.
5. 5. 닭육수를 넣고 귀리가 익을 때 까지 푹 익힌 후
믹서에 갈아 체에 걸러준다.
6. 6. 마지막으로 생크림을 넣고 농도를 조절하고
녹차가루를 올려 완성한다.

## 영양 정보

- 칼로리: 150.8 kcal
- 탄수화물: 10.4g
- 단백질: 5.3g
- 지방: 9.8g
- 나트륨: 62.4mg
