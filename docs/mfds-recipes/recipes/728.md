# 과일겨자채

과일겨자채은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 사과 25g
- 배 50g
- 밤 10g
- 대추 6g
- 곶감 10g
- 호두 5g
- 땅콩 5g
- 양상추 15g
- 연겨자 1.25g
- 연유 2.5g
- 사과식초 2.5g
- 꿀 3.5g
- 배즙 2.5g

## 조리 방법

1. 1. 사과와 배는 사방 2cm 네모로 썰고, 밤도 얇게 썰어준다.
2. 2. 1을 설탕물(물 1컵 + 설탕 2.5g)에 담근다.
3. 3. 대추, 곶감은 굵게 채 썰고 호두, 땅콩은 굵게 다진다.
4. 4. 분량의 재료를 섞어 겨자소스 만든다.
5. 5. 과일을 체에 받쳐 물기를 빼고 소스에 버무린다.
6. 6. 한입 크기로 자른 양상추에 샐러드를 올려낸다.

## 영양 정보

- 칼로리: 123.6 kcal
- 탄수화물: 16.1g
- 단백질: 3.1g
- 지방: 6.2g
- 나트륨: 109.2mg
