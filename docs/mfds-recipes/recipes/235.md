# 고구마두부스테이크

고구마두부스테이크은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 스테이크재료 두부(200g)
- 고구마(130g)
- 마늘(15g)
- 대파(20g)
- 양파(60g)
- 다진 견과류(20g)
- 찹쌀가루(30g)
- 청국장소스 양송이버섯(20g)
- 다시마육수(80g)
- 청국장(15g)
- 유 자청(10g)
- 녹말가루(5g)

## 조리 방법

1. 1. 두부는 물기를 없앤 뒤 체에 밭쳐
으깬 뒤 칼등으로 한 번 더 으깬다.
2. 2. 고구마는 삶아 껍질을 벗겨 으깬다.
3. 3. 마늘, 대파, 양파는 잘게 다진 뒤
식용유(5g)를 두른 팬에서 살짝 볶아
키친타월에 밭쳐 물기를 제거하며
식힌다.
4. 4. 볶은 채소, 두부, 고구마,
다진 견과류, 찹쌀가루를 섞은 뒤
반죽해 둥글 납작하게 모양을
빚고, 식용유(15g)를 두른 팬에서
앞뒤로 노릇하게 굽는다.
5. 5. 양송이버섯은 채 썬다.
6. 6. 다시마육수에 청국장, 유자청,
녹말가루, 양송이버섯을 넣고 조린
청국장소스를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 741.4 kcal
- 탄수화물: 20.8g
- 단백질: 7.8g
- 지방: 11g
- 나트륨: 216.6mg
