# 누룽지새우튀김

누룽지새우튀김은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 새우(100g)
- 소금(0.3g)
- 후춧가루(0.02g)
- 튀김가루(100g)
- 달걀(50g)
- 누룽지(50g)
- 튀김기름(200g)
- - 소스 : 오렌지주스(100g)
- 설탕(10g)
- 식초(10g)
- - 곁들임채소 : 어린잎(20g)

## 조리 방법

1. 1. 새우는 껍질을 제거하고 소금과
후춧가루를 넣어 밑간을 해놓는다.
2. 2. 어린잎은 찬물에 담그어 놓는다.
3. 3. 튀김가루에 달걀을 넣어 잘 섞어
튀김반죽을 만들어 놓는다.
4. 4. 밑간을 해 놓은 새우를 튀김반죽에
넣어 옷을 입힌다.
5. 5. 누룽지는 170℃의 온도 튀김 기름에
튀겨내고, 튀김옷 입힌 새우도 튀겨낸다.
6. 6. 냄비에 오렌지주스와 설탕, 식초를
넣어 은근히 졸여 소스를 만들고,
어린잎은 찬물에 건져 물기를 제거하고
접시에 깔고 튀긴 누룽지와 튀긴 새우를
올려 완성한다.

## 영양 정보

- 칼로리: 558 kcal
- 탄수화물: 63.5g
- 단백질: 23.5g
- 지방: 23.3g
- 나트륨: 156.7mg
