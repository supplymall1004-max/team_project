# 닭가슴살 스테이크

닭가슴살 스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 닭고기(가슴살
- 150g)
- 토마토(100g)
- 가지(1개)
- 치즈(1장)
- 오레가노(2g)
- 마늘(20g)
- 소금(0.2g)
- 후춧가루(0.01g)
- 올리브오일(20g)

## 조리 방법

1. 1. 닭가슴살을 소금과 후추로 밑간한다.
2. 2. 토마토와 마늘은 입자있게 다진다
3. 3. 다진 토마토와 마늘을 냄비에 담아
볶다가 물과 오레가노를 넣고 끓여
소스를 만든다.
4. 4. 팬에 올리브오일을 두르고 밑간 한
닭가슴살을 노릇노릇하게 굽는다.
5. 5. 가지는 길게 썬다.
6. 6. 가지에 치즈를 감싸 익히고, 구운 닭
스테이크를 옆에 담고 소스를 옮긴다.

## 영양 정보

- 칼로리: 242.4 kcal
- 탄수화물: 4.4g
- 단백질: 22g
- 지방: 15.2g
- 나트륨: 549.7mg
