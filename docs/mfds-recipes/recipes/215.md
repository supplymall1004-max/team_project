# 가지볶음밥

가지볶음밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 재료 단호박(100g)
- 양파(30g)
- 석박지(35g)
- 파프리카(40g)
- 가지(90g)
- 두부(50g)
- 밥(180g)
- 세발나물(10g)
- 양념 다진 마늘(5g)
- 저염된장(12g)
- 후춧가루(0.3g)
- 들기름(4g)

## 조리 방법

1. 1. 단호박은 삶아 으깬다.
2. 2. 양파, 석박지, 파프리카, 가지는 깍둑
썰고, 두부는 으깬다.
3. 3. 팬에 다진 마늘, 양파, 석박지를 넣어
볶다가 두부를 넣고 수분이 없어질
때까지 볶다가 가지, 파프리카도
넣어 볶는다
4. 4. 밥과 저염된장, 후춧가루를 넣고
볶다가 들기름을 넣어 한 번 더
볶는다.
5. 5. 그릇에 담고 으깬 단호박과
세발나물을 곁들여 마무리한다.

## 영양 정보

- 칼로리: 644.9 kcal
- 탄수화물: 20.15g
- 단백질: 3.33g
- 지방: 1.14g
- 나트륨: 505.1mg
