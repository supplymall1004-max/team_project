# 족발수육

족발수육은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- ●주재료 : 양념족발 200g
- 고구마 150g
- 가지 150g
- 맥주 400g
- 백김치 50g
- ●백김치 무침양념 : 참기름 15g
- 통깨 3g

## 조리 방법

1. 1. 조미된 족발은 뜨거운 맥주에 데쳐내 기름기를
제거한다.
2. 2. 고구마는 편으로 썰어 냉수에 헹궈 기름
없는 팬에 구워준다.
3. 3. 가지는 편으로 썰어서 수분을 제거한다.
4. 4. 가지는 편으로 썰어 살짝 구워준다.
5. 5. 백김치는 물에 헹궈서 손으로 꼭 짠 후 송송
썰어서 양념을 해 무쳐준다.
6. 6. 무쳐준 김치와 함께 구워진 채소와 족발수육을
담아 완성한다.

## 영양 정보

- 칼로리: 282.2 kcal
- 탄수화물: 16.2g
- 단백질: 22.2g
- 지방: 14.3g
- 나트륨: 471.7mg
