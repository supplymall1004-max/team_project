# 함초타락죽

함초타락죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 찹쌀 25g
- 우유 200g
- 함초 15g
- 크렌베리 10g
- 다진양파 15g
- 파마산치즈 15g

## 조리 방법

1. 1. 함초는 세척 후 송송 잘라준다. 이때 물이
안 나오게 잘라주는 것이 중요하다.
2. 2. 찹쌀은 미리 30분전에 불려준다.
3. 3. 불린 찹쌀의 물기를 빼고 물 50g을 넣고 죽을
끓이다가 함초를 넣고 더 끓인다.
4. 4. 여기에 우유를 넣고 약불에서 은근히 끓여준다.
5. 5. 죽이 어느정도 완성되면 치즈가루를 뿌려준다.
6. 6. 죽이 완성되면 그릇에 담아 완성한다.

## 영양 정보

- 칼로리: 354.3 kcal
- 탄수화물: 63.6g
- 단백질: 10.5g
- 지방: 6.5g
- 나트륨: 251.6mg
