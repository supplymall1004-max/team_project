# 도라지 유자청생채

도라지 유자청생채은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 도라지(120g)
- 풋고추(1개)
- 홍고추(1개)
- 양파(30g)
- 라임주스(20g)
- 흑임자(20g)
- 유자청(30g)
- 소금(0.2g)

## 조리 방법

1. 1. 통 도라지는 껍질을 벗긴다
2. 2. 껍질을 벗긴 도라지는 잘라 소금에
살짝 절여 쓴맛을 뺀다.
3. 3. 청·홍고추와 양파는 채썬다.
4. 4. 썰어 놓은 청. 홍고추와 양파는 차가운
물에 담군 뒤 체에 밭쳐 물기를 뺀다.
5. 5. 라임쥬스, 흑임자와 유자청을 골고루
섞어 소스를 만들고 도라지와 야채를
넣어 버물려 접시에 담는다
6. 6. 홍고추는 반으로 잘라 장식한다.

## 영양 정보

- 칼로리: 133.3 kcal
- 탄수화물: 30.6g
- 단백질: 2.5g
- 지방: 0.1g
- 나트륨: 217.8mg
