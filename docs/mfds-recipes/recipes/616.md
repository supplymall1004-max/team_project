# 다마고도후

다마고도후은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 달걀(1개)
- 다시마(10g)
- 요리술(10g)
- 저염간장(5g)
- 날치알(20g)
- 말린 매생이(10g)
- 무순(2g)

## 조리 방법

1. 1. 물에 다시마를 넣고 약 10분 정도 끓인
뒤 걸러 육수를 만든다.
2. 2. 달걀과 다시마 육수를 1:1로 섞고 체에
거른다.
3. 3. ?를 체에 내린다.
4. 4. 달걀물에 매생이를 풀어둔다.
5. 5. 매생이를 풀어 놓은 달걀물에 요리술을
넣어 섞는다.
6. 6. 쪄낸 다마고도후에 날치알을 올려
완성한다.

## 영양 정보

- 칼로리: 107.6 kcal
- 탄수화물: 0.8g
- 단백질: 9g
- 지방: 7.6g
- 나트륨: 178.3mg
