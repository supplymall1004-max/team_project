# 들깨된장육개장

들깨된장육개장은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 숙주(30g)
- 느타리버섯(25g)
- 양파(25g)
- 대파(10g)
- 청양고추(2g)
- 얼갈이배추(25g)
- 육수 무(50g)
- 돼지고기(사태
- 25g)
- 다시마(2g)
- 청주(1g)
- 보리새우(3g)
- 물(300g)
- 양념장 고춧가루(1g)
- 들깻가루(5g)
- 다진 마늘(3g)
- 된장(10g)

## 조리 방법

1. 1. 냄비에 무, 돼지고기, 다시마를 넣고
끓이다가 끓어오르면 다시마를
건지고 청주와 보리새우를 넣어
약한 불로 끓여 고기가 익으면 체에
거른다.
2. 2. 숙주는 끝을 다듬고, 느타리버섯은
결대로 찢고, 양파는 채 썰고, 대파와
청양고추는 어슷 썰고, 무는
납작하게 썬다.
3. 3. 얼갈이배추는 밑동을 잘라 3cm
간격으로 썰어 데친 뒤 찬물로 헹궈
물기를 짠다.
4. 4. 양념장을 만든 뒤 찢은 고기와
얼갈이배추를 넣어 버무린다.
5. 5. 육수에 무, 양념에 버무린 고기와
얼갈이배추, 숙주, 느타리버섯, 대파,
청양고추 순으로 넣어가며 푹 끓여
마무리한다.

## 영양 정보

- 칼로리: 123.4 kcal
- 탄수화물: 2.37g
- 단백질: 2.41g
- 지방: 0.26g
- 나트륨: 525.7mg
