# 숙주라면

숙주라면은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 홍합(20g)
- 숙주(50g)
- 청경채(20g)
- 청고추(5g)
- 홍고추(5g)
- 고춧가루(10g)
- 굴소스(10g)
- 라면(100g)
- 대파(10g)
- 달걀(50g)
- - 육수 : 멸치(20g)
- 건새우(10g)
- 무(20g)
- 양파(20g)
- 마늘(10g)

## 조리 방법

1. 1. 멸치, 건새우, 무, 양파, 마늘을 넣고
육수를 만들어 소창에 걸러 준비한다.
2. 2. 홍합은 끓는 물에 살짝 데쳐 놓는다.
3. 3. 숙주는 씻어 놓고 청경채는 반으로
자르고 청 ․ 홍고추는 어슷썰어 준비
한다.
4. 4. 만들어 놓은 육수에 고춧가루와
굴소스를 넣어 끓인다.
5. 5. 달걀은 풀어 준비한다.
6. 6. 육수(④)가 끓으면 라면을 넣고, 다시
한 번 더 끓으면 홍합을 넣어 끓이다가,
불을 줄인 뒤 달걀을 넣어 줄알을 치고,
청경채와 숙주, 홍고추와 대파를 넣고
살짝 끓은 뒤 그릇에 담아 완성한다.

## 영양 정보

- 칼로리: 342.8 kcal
- 탄수화물: 51g
- 단백질: 12.8g
- 지방: 9.7g
- 나트륨: 218.4mg
