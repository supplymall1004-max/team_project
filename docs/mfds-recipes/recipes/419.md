# 연근초무침

연근초무침은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연근 200g
- 양파 60g
- 오이 40g
- 밤 60g
- 소스 : 유자청 25g
- 강황가루 2g
- 잣 2g
- 설탕 5g
- 식초 15g

## 조리 방법

1. 1. 양파는 채 썰어 물에 담가주고, 오이는 돌려
깎기 해서 물에 담가준다.
2. 2. 연근은 껍질을 까고 얇게 썰어서 식초 2방울을
넣은 물에 데쳐준다.
3. 3. 밤도 아주 얇게 썰어서 물에 담가준다.
4. 4. 소스 재료를 모두 넣고 믹서에 간다.
5. 5. 물에 데쳐낸 연근은 키친타월에 올려서 물기를
완전히 제거한다.
6. 6. 모든 재료를 넣고 소스에 버무려준다.

## 영양 정보

- 칼로리: 119.1 kcal
- 탄수화물: 20.4g
- 단백질: 1.6g
- 지방: 3.5g
- 나트륨: 81.4mg
