# 된장시금치옹심이

된장시금치옹심이은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 시금치 150g
- 감자 80g
- 애호박 30g
- 당근 20g
- 양파 50g
- 청고추 15g
- 홍고추 15g
- 밀가루 30g
- 국물 : 바지락 100g
- 저염된장 25g
- 볶은소금 1g

## 조리 방법

1. 1. 바지락은 볶은 소금에 담가 해감을 시켜준다.
2. 2. 물 400g에 저염 된장과 바지락을 넣어 국물을
만들어준다.
3. 3. 시금치 는 데 친 후 물기를 제거해 송송
다져준다.
4. 4. 당근, 양파와 청홍고추는 채 썰고 애호박과
감자는 반달모양으로 썰어준다.
5. 5. 다진 시금치와 밀가루를 이용해 완자를
만들어준다.
6. 6. 만들어진 국물에 옹심이를 넣어 끓여주고
나머지 채소를 넣어 완성한다.

## 영양 정보

- 칼로리: 167.3 kcal
- 탄수화물: 25.1g
- 단백질: 14.9g
- 지방: 0.8g
- 나트륨: 771mg
