# 두부청국장죽

두부청국장죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 청국장가루 20g
- 두부 60g
- 밥 150g
- 표고버섯 12g
- 당근 20g
- 애호박 25g
- 양념장 : 다진쇠고기 30g
- 참기름 15g
- 맛간장 10g
- 버섯마늘소금 1g
- 흰후추 1g
- 다진마늘 10g
- 다진대파 7g
- 흑임자 1g

## 조리 방법

1. 1. 다진 쇠고기는 양념장에 버무려주고 두부는
으깨준다.
2. 2. 애호박, 당근은 송송 다져준다.
3. 3. 표고버섯은 불려준 뒤 채 썰어 참기름과
버섯마늘소금에 살짝 무쳐서 볶아준다.
4. 4. 양념된 소고기는 팬에 볶아준다.
5. 5. 소고기를 볶아주던 팬에 밥과 표고를 넣고
끓여주다 두부를 넣고 죽을 해준다.
6. 6. 다진 애호박, 당근을 넣어서 끓여주고 불을
끄기 전에 청국장과 흑임자를 넣어 죽을
완성한다.

## 영양 정보

- 칼로리: 255.1 kcal
- 탄수화물: 37.6g
- 단백질: 16.3g
- 지방: 4.4g
- 나트륨: 118.2mg
