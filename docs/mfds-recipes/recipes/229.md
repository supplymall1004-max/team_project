# 오징어불고기김밥

오징어불고기김밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 오징어 몸통(130g)
- 파프리카(30g)
- 당근(20g)
- 피망(30g)
- 밥(200g)
- 채 썬 쇠고기(우둔살
- 60g)
- 달걀(50g)
- 김(2g)
- 깻잎(20g)
- 새송이버섯(30g)
- 양념 참기름(15g)
- 고추장(3g)
- 매실청(15g)

## 조리 방법

1. 1. 오징어 몸통 안쪽에 칼집을 내고
끓는 물에 데친다.
2. 2. 파프리카와 당근, 피망은 작게 다져
볶은 뒤 밥에 넣어 섞는다.
3. 3. 쇠고기는 참기름에 볶는다.
4. 4. 달걀을 풀어 약한 불로 달군 팬에
부은 뒤 김가루를 뿌려 지단을
부치고, 바닥이 익으면 깻잎과
밥을 올린다.
5. 5. 데친 오징어, 볶은 쇠고기 순으로
올려 만 뒤 랩으로 싼다.
6. 6. 고추장과 매실청을 섞은 뒤 팬에
넣고 약한 불에서 끓여 소스를
만들고 한입 크기로 썬 김밥에
구운 버섯을 함께 곁들여
마무리한다.

## 영양 정보

- 칼로리: 452.2 kcal
- 탄수화물: 14.5g
- 단백질: 13.9g
- 지방: 3.42g
- 나트륨: 244.4mg
