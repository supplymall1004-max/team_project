# 감닭떡갈비

감닭떡갈비은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 : 닭고기(120g)
- 감자(200g)
- 양파(50g)
- 달걀(1개)
- 마늘(10g)
- 찹쌀가루(50g)
- 매실청(5g)
- 소금(0.3g)
- 후춧가루(0.5g)
- 식용유(20g)

## 조리 방법

1. 1. 닭 살을 곱게 다진다.
2. 2. 다진 닭살에 소금, 후춧가루, 매실청 및
다진 마늘을 넣고 골고루 양념한다.
3. 3. 믹서에 양파, 마늘, 감자, 찹쌀가루를
넣고 간다.
4. 4. 갈아 놓은 감자에 양념된 닭살과
달걀을 섞는다.
5. 5. 4번을 잘 치대 떡갈비 모양으로 만든다.
6. 6. 팬에 기름을 살짝 두르고 떡갈비를
앞뒤로 구워 익힌다.

## 영양 정보

- 칼로리: 348.3 kcal
- 탄수화물: 55.7g
- 단백질: 17.2g
- 지방: 6.3g
- 나트륨: 128.8mg
