# 해산물 훈제연어롤

해산물 훈제연어롤은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연어(훈제연어
- 120g)
- 주꾸미(100g)
- 소라(50g)
- 완두콩(20g)
- 샐러리(30g)
- 파프리카(1/2개)
- 오이(20g)
- 당근(20g)
- 양파(20g)
- 고추장(20g)
- 식초(20g)
- 설탕(20g)
- 소금(0.3g)

## 조리 방법

1. 1. 훈제 연어는 넓게 썬다.
2. 2. 당근, 오이, 파프리카, 양파, 샐러리는
길게 썬다.
3. 3. 주꾸미는 소금에 주물러 씻는다.
4. 4. 손질한 주꾸미, 소라는 끓는 물에
데치고 썰어준다.
5. 5. 고추장에 식초, 설탕을 넣고
초고추장소스를 만든다.
6. 6. 훈제 연어에 야채와 주꾸미, 소라를
넣고 돌돌 말아 접시에 담고
초고추장소스를 올리고 완두콩으로
장식한다.

## 영양 정보

- 칼로리: 271.8 kcal
- 탄수화물: 8.3g
- 단백질: 33g
- 지방: 11.8g
- 나트륨: 566.2mg
