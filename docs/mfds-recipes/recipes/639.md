# 호박잎 삼계탕

호박잎 삼계탕은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 호박잎(5장)
- 닭고기(가슴살
- 120g)
- 찹쌀(100g)
- 대추(3알)
- 미나리(20g)
- 인삼(1뿌리)
- 소금(0.2g)
- 후춧가루(0.01g)

## 조리 방법

1. 1. 찹쌀은 깨끗이 씻어 30분 정도 불린다
2. 2. 불린 찹쌀은 냄비에 넣고 질게 밥을
한다.
3. 3. 닭가슴살은 넓게 펴서 소금,
후춧가루를 뿌린다.
4. 4. 호박잎과 미나리는 끓는 물에 살짝
데치고, 대추는 돌려 깎아 씨를
제거하고 인삼은 뇌두를 제거한 뒤
채를 썰어둔다.
5. 5. 데친 호박잎에 닭 가슴살을 올린다.
6. 6. 닭 가슴살위에 찹쌀밥과 준비한 야채와
대추를 올려 돌돌 말아 질게 된
찹쌀밥에 넣고 약 20분 정도 더 쪄낸다.

## 영양 정보

- 칼로리: 314.1 kcal
- 탄수화물: 53.7g
- 단백질: 16g
- 지방: 3.9g
- 나트륨: 55.9mg
