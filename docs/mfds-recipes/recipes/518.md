# 삼색소고기만두

삼색소고기만두은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 애호박(30g)
- 소금(0.5g)
- 부추(30g)
- 숙주(30g)
- 두부(30g)
- 달걀(50g)
- 소고기(50g)
- 다진 파(20g)
- 다진 마늘(10g)
- 참기름(5g)
- 후춧가루(0.03g)
- - 만두피 : 시금치(30g)
- 소금(0.5g)
- 당근(30g)
- 들깨가루(30g)
- 밀가루(200g)
- 녹말가루(20g)
- - 육수 : 양파(20g)
- 다시마(10g)
- 건새우(10g)
- 건멸치(20g)

## 조리 방법

1. 1. 시금치는 끓는 물에 소금을 넣고 데쳐
찬물에 식힌 후 곱게 갈아 준비한다.
2. 2. 당근즙, 시금치즙, 들깨가루를 각각
준비한 뒤, 밀가루와 녹말가루를
세 개로 나눠 각각의 즙을 넣어 반죽 후
숙성시킨다.
3. 3. 호박은 채를 썰어 소금에 살짝 절여
물기를 제거하고, 부추는 송송 썰고,
숙주는 데쳐 물기를 제거하고 다지고,
두부는 면포에 물기를 꼭 짜서 준비한다.
4. 4. 달걀은 노란자와 흰자를 나눠 각각
지단을 붙여 준비한다. 냄비에 멸치와
다시마, 건새우, 양파를 넣고 약30분
정도 끓여 체에 걸러 육수를 만들어
놓는다.
5. 5. 소고기는 곱게 다져, 파 , 마늘, 참기름,
후춧가루를 넣어 양념을 해 놓는다.
6. 6. 준비한 채소(③)와 고기(⑤)를 섞어
만두속을 만들고, 숙성된 밀가루
반죽(②)을 밀어 만든 만두피에 만두
속을 넣어 만두를 만든다. 준비한 육수
(④)에 삼색 만두를 넣어 끓여 담고 그
위에 달걀지단을 올려 완성한다.
