# 토마토두루치기

토마토두루치기은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 재료 양배추(40g)
- 양파(15g)
- 표고버섯(10g)
- 양송이버섯(30g)
- 파프리카(10g)
- 피망(10g)
- 대파(5g)
- 마늘(2g)
- 토마토(30g)
- 비트(0.5g)
- 식용유(5g)
- 돼지고기(100g)
- 양념 소금(0.5g)
- 설탕(1g)
- 후춧가루(0.2g)
- 간장(1g)
- 생강(0.1g)
- 고추장(5g)
- 물엿(3g)

## 조리 방법

1. 1. 양배추, 양파, 표고버섯, 양송이버섯,
파프리카, 피망은 먹기 좋은 크기로
깍둑 썰고, 대파는 어슷 썬다.
2. 2. 마늘은 편 썰어 기름 두른 팬에
투명해질 때까지 4~5분 정도
볶는다.
3. 3. 토마토는 김 오른 찜기에 찐 후
껍질을 벗겨 양파와 함께 으깨가며
볶는다.
4. 4. 비트도 곱게 갈아 넣고 함께 볶는다.
5. 5. 식용유를 두른 팬에 돼지고기를
넣고 토마토비트소스와 함께 볶는다.
6. 6. 양념을 넣어 간을 맞춘 후 양배추,
표고버섯, 양송이버섯, 파프리카,
피망, 대파를 넣고 살짝 볶아
마무리한다.

- 탄수화물: 9.64g
- 단백질: 11.5g
- 지방: 9.99g
