# 간장아귀찜

간장아귀찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 아귀(150g)
- 새우(20g)
- 미나리(15g)
- 풋고추(5g)
- 붉은 고추(5g)
- 콩나물무침 콩나물(30g)
- 참기름(3g)
- 참깨(1g)
- 간장소스 설탕(1g)
- 저염간장(3g)
- 청주(15g)
- 다시마물(50g)
- 아귀찜양념 다진 양파(50g)
- 다진 생강(5g)
- 다진 마늘(10g)

## 조리 방법

1. 1. 아귀는 얇게 포를 뜬다.
2. 2. 새우를 다져 아귀포 안에 채워 넣은
뒤 데친 미나리로 묶는다.
3. 3. 냄비에 간장소스, 아귀,
아귀찜양념을 넣고 약한 불에서
끓인다.
4. 4. 아귀가 익으면 고추를 어슷 썰어
올린 뒤 살짝 더 익힌다.
5. 5. 콩나물은 끓는 물에 데친 뒤 참기름,
참깨를 넣고 무친다.
6. 6. 아귀찜에 콩나물무침을 곁들여
마무리한다.

## 영양 정보

- 칼로리: 100.5 kcal
- 탄수화물: 1.15g
- 단백질: 15.1g
- 지방: 0.76g
- 나트륨: 119.3mg
