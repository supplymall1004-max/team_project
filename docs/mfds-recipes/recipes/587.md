# 두부 카프리제

두부 카프리제은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 소고기(100g)
- 돼지고기(100g)
- 표고버섯(2개)
- 마(100g)
- 연근(20g)
- 감자(60g)
- 깻잎(3장)
- 애호박(1/3개)
- 마늘(20g)
- 두유(100g)
- 설탕(20g)
- 소금(0.2g)
- 저염간장(20g)

## 조리 방법

1. 1. 토마토는 깨끗이 씻어 슬라이스한다.
2. 2. 두부는 원형틀을 이용해서 토마토와
같은 크기로 만들고 소금을 살짝 뿌린
뒤 물기를 제거한다.
3. 3. 팬에 올리브오일을 두르고 두부를 살짝
굽는다.
4. 4. 어린잎은 찬물에 담근다.
5. 5. 발사믹소스에 설탕과 레몬즙을 넣고
골고루 섞어 소스를 만든다.
6. 6. 두부와 준비한 토마토를 접시에 돌려
담고 어린잎을 위에 올린 뒤 먹기
직전에 발사믹소스와 파마산치즈를
뿌려 완성한다.

## 영양 정보

- 칼로리: 140.2 kcal
- 탄수화물: 6.8g
- 단백질: 11.1g
- 지방: 7.6g
- 나트륨: 229.6mg
