# 단호박크림파스타

단호박크림파스타은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 단호박(150g)
- 파스타(80g)
- 소금(0.2g)
- 다진 마늘(10g)
- 버터(10g)
- 베이컨(20g)
- 양송이(20g)
- 양파(20g)
- 육수(100g)
- 우유(50g)
- 생크림(20g)

## 조리 방법

1. 1. 단호박은 잘라 씨를 제거하고 껍질을
벗겨 김이 오른 찜통에서 약 20분 정도
찐다.
2. 2. 찐 단호박이 한 김 식으면, 채에 곱게
내려 준비하고, 스파게티는 소금을
넣고 삶아 건져놓는다.
3. 3. 베이컨은 입자있게 썰고, 양송이버섯은
껍질을 벗겨 다지고, 양파도 입자있게
다진다.
4. 4. 팬에 버터를 녹인 뒤 다진 마늘을 먼저
넣고 은근히 볶아 향을 만든다.
5. 5. 볶은 마늘(④)에 베이컨과 양송이,
양파를 넣고 다시 볶는다.
6. 6. 볶은 채소(⑤)에 육수를 붓고 끓이다가
으깬 단호박(②), 생크림, 우유를 넣어
한소끔 끓인 뒤 삶아놓은 스파게티면
(②)을 넣고 다시 한 번 더 끓여 완성
한다.

## 영양 정보

- 칼로리: 236.6 kcal
- 탄수화물: 40.9g
- 단백질: 8.8g
- 지방: 4.2g
- 나트륨: 205.1mg
