# 두부된장무침

두부된장무침은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 두부(100g)
- 된장(20g)
- 쌈채(100g)
- 생크림(20g)

## 조리 방법

1. 1. 쌈채는 먹기 좋게 손으로 뜯어 놓는다.
2. 2. 뜯어 놓은 쌈채를 물에 담그어 싱싱하게
준비해 놓는다.
3. 3. 두부는 면보에 넣어 손으로 짜서 물기를
제거하고 준비한다.
4. 4. 된장에 생크림을 넣어 골고루 섞어
짠 맛을 줄여준다.
5. 5. 으깬 두부에 된장을 넣어 섞어준다.
6. 6. 물에 담그어 놓은 채소를 체에 건져,
물기를 털어내고, 만들어 놓은 된장
소스에 넣어 무친다.

## 영양 정보

- 칼로리: 217.7 kcal
- 탄수화물: 14.4g
- 단백질: 15.5g
- 지방: 10.9g
- 나트륨: 587.3mg
