# 새우보리죽 고등어스테이크

새우보리죽 고등어스테이크은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 고등어(1/2마리)
- 새우(3마리)
- 보리(100g)
- 생강(10g)
- 레몬(1/4개)
- 막걸리(50g)
- 비트(50g)
- 설탕(20g)
- 식초(20g)
- 소금(0.2g)

## 조리 방법

1. 1. 새우는 껍질을 벗겨 손질한다.
2. 2. 보리는 깨끗이 씻어 불린다.
3. 3. 불린 보리와 새우에 5배 정도 물을 넣고
죽을 쑨다.
4. 4. 고등어는 포를 뜬다.
5. 5. 포 뜬 고등어에 생강, 레몬, 막걸리를
넣고 비린맛을 제거한다.
6. 6. 비트에 설탕, 식초, 소금을 넣고 피클을
만들고, 보리죽이 끓으면 포 뜬
고등어를 넣고 은근히 익힌다.

## 영양 정보

- 칼로리: 541.5 kcal
- 탄수화물: 88.4g
- 단백질: 25.3g
- 지방: 9.6g
- 나트륨: 140.7mg
