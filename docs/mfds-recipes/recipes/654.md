# 먹물소스를 곁들인 토마토피클

먹물소스를 곁들인 토마토피클은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 마(50g)
- 우엉(50g)
- 토마토(200g)
- 배(1/2개)
- 새싹(10g)
- 피클링스파이스(10g)
- 와인(20g)
- 먹물(10g)
- 설탕(50g)
- 식초(30g)
- 소금(0.2g)

## 조리 방법

1. 1. 물 한 컵에 설탕, 식초, 소금,
피클링스파이스를 넣고 식촛물을
만든다.
2. 2. 배는 스쿱으로 동그랗게 판다.
3. 3. 토마토는 1/6등분으로 썬다.
4. 4. 마와 우엉은 슬라이스 한다.
5. 5. 썰어 놓은 마와 우엉을 식촛물에
담근다.
6. 6. 와인에 먹물, 설탕을 넣고 섞어
식촛물에 넣고 배, 토마토, 새싹을
넣는다.

## 영양 정보

- 칼로리: 140.9 kcal
- 탄수화물: 33.2g
- 단백질: 1.6g
- 지방: 0.2g
- 나트륨: 73.2mg
