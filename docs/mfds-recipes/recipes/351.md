# 통삼겹맥적구이

통삼겹맥적구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 통삼겹 250g
- 닭육수 400g
- 구이소스 : 저염된장 15g
- 꿀 30g
- 청주 15g
- 참기름 15g
- 다진마늘 10g
- 다진대파 5g
- 통깨 10g
- 샐러드 : 수삼 1개(20g)
- 알배추 35g
- 당근 30g
- 샐러드소스 : 식초 30g
- 다진마늘 15g
- 참기름 15g
- 맛간장 15g

## 조리 방법

1. 1. 구이소스 재료를 이용해 소스를 만든다.
2. 2. 통삼겹은 닭육수에 삶아 냉수에 헹궈 물기를
빼준다.
3. 3. 통삼겹에 만들어진 소스를 발라 준 뒤 10분
정도 숙성시켜준다.
4. 4. 샐러드재료는 곱게 채 썰고 냉수에 담가
각각의 맛을 중화시켜준다.
5. 5. 샐러드소스를 만들어 샐러드를 버무려준다.
6. 6. 양념에 재워진 통삼겹을 구워주고 샐러드를
담아 완성한다.

## 영양 정보

- 칼로리: 397.2 kcal
- 탄수화물: 4.1g
- 단백질: 14.8g
- 지방: 35.7g
- 나트륨: 120.6mg
