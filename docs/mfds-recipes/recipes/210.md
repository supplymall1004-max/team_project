# 두릅 샐러드

두릅 샐러드은(는) 기타 요리로, 굽기 조리법을 사용합니다.

## 재료

- ●주재료 : 두릅 20g(2개)
- 가지 20g(1/5개)
- 레몬(껍질) 12g(1큰술)
- 청경채 25g(2개)
- 슬라이스연어 30g(2장)
- 크림치즈 5g(1작은술)
- 레몬 5g(1작은술)
- ●드레싱 : 레몬즙15g(1큰술)
- 레드와인 30g(2큰술)
- 설탕 5g(1작은술)

## 조리 방법

1. 1. 팬에 레드와인, 레몬즙, 설탕을 넣고 졸여 드레싱을 만든다.
2. 2. 청경채와 두릅은 끓는 물에 데쳐 식히고 레몬은 부채꼴 모양으로 썰고 가지는 어슷하게 썰어 석쇠에 굽는다.
3. 3. 슬라이스연어에 크림치즈를 넣고 말아준 후 접시에 연어, 가지, 레몬, 두릅, 청경채를 담고 드레싱을 뿌려 완성한다.

## 영양 정보

- 칼로리: 123.9 kcal
- 탄수화물: 2.2g
- 단백질: 8.2g
- 지방: 9.1g
- 나트륨: 58.7mg
