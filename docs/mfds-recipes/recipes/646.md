# 깻잎애호박찜

깻잎애호박찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 애호박(1개)
- 깻잎(3장)
- 양파(30g)
- 피망(30g)
- 파프리카(50g)
- 저염간장(50g)
- 매실액(20g)

## 조리 방법

1. 1. 애호박은 길게 반으로 자른다.
2. 2. 자른 애호박에 칼집을 넣는다.
3. 3. 깻잎은 잘게 썰고, 양파는 곱게 다지고
피망, 파프리카는 입자있게 썬다.
4. 4. 저염간장에 다진 양파와 매실액을 넣고
잘 섞어 양념장을 만든다.
5. 5. 양념장에 피망, 파프리카, 깻잎을
섞는다.
6. 6. ?번을 칼집낸 호박 가운데 넣고 찜통
에서 5분 정도 쪄 낸 뒤 그릇에 담아 저
염간장소스를 한번 더 올린다.

## 영양 정보

- 칼로리: 38.5 kcal
- 탄수화물: 6.2g
- 단백질: 2g
- 지방: 0.6g
- 나트륨: 329.2mg
