# 크림치즈 망고무스

크림치즈 망고무스은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 망고퓨레(50g)
- 라즈베리(10g)
- 피스타치오(30g)
- 크림치즈(50g)
- 요거트(50g)
- 생크림(50g)
- 판젤라틴(1장)
- 레몬(1/4개)
- 애플민트(10g)
- 설탕(20g)

## 조리 방법

1. 1. 냄비에 망고퓨레, 레몬, 설탕을 넣고
은근히 끓인다.
2. 2. 젤라틴은 찬물에 불린 뒤 중탕으로
녹여 망고퓨레에 넣고 틀에 굳힌다.
3. 3. 생크림은 차갑게 휘핑하여 요거트와
섞는다.
4. 4. 생크림을 섞은 요거트에 설탕,
크림치즈를 섞는다
5. 5. 냄비에 라즈베리, 설탕을 넣고 은근히
끓인다.
6. 6. 접시에 굳힌 망고무스에 담고 그 위에
라즈베리 소스와 크림치즈 소스, 다진
파스타치오를 올리고 애플민트로
장식한다.

## 영양 정보

- 칼로리: 333.2 kcal
- 탄수화물: 55.6g
- 단백질: 7.3g
- 지방: 9.1g
- 나트륨: 69.7mg
