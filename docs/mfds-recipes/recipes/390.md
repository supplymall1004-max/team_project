# 오징어콩순대

오징어콩순대은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 오징어 1마리(220g)
- 옥수수 30g
- 귀리밥 50g
- 강낭콩 20g
- 당근 20g
- 미니파프리카 20g
- 피망 20g
- 대파 10g
- 양파 25g
- 카레가루 25g
- 흰후추 1g
- 다진마늘 5g
- 참기름 15g
- 밀가루 15g
- 레몬즙 15g

## 조리 방법

1. 1. 오징어는 내장을 빼서 흐르는 물에 깨끗이
씻은 후, 다리는 송송 다져주고 레몬즙을
살짝 뿌려준다.
2. 2. 강낭콩은 먼저 삶아준다.
3. 3. 피망, 당근, 양파, 대파, 파프리카는 송송
다져준다.
4. 4. 잘라준 오징어다리는 다진마늘과 살짝
볶아준다.
5. 5. 볼에 귀리밥을 넣고 카레가루를 넣고 모든
재료를 섞어 소 재료를 만든다.
6. 6. 오징어 안쪽에 밀가루를 묻혀주고 소를 채워
팬에 구워준다.

## 영양 정보

- 칼로리: 213.5 kcal
- 탄수화물: 25g
- 단백질: 17.9g
- 지방: 4.7g
- 나트륨: 295.4mg
