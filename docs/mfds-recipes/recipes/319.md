# 닭가슴살리조또

닭가슴살리조또은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭가슴살 100g
- 찰보리 50g
- 찹쌀 50g
- 청피망 50g
- 표고버섯 30g
- 청양고추 10g
- 양파 60g
- 다진마늘 15g
- 홍피망 50g
- 생크림 30g
- 우유 30g
- 올리브오일 50g
- 함초소금 2g
- 고명 : 건조토마토 10g

## 조리 방법

1. 1. 찹쌀은 30분 정도, 찰보리는 1시간 이상
불려주고 보리쌀은 한번 쪄 준다.
2. 2. 표고버섯은 물에 데친다. 이때 표고버섯 불린
물은 버리지 않고 짜투리 채소와 함께 채소
국물을 만들어준다.
3. 3. 불린 표고버섯, 청피망, 홍피망, 양파, 오리
가슴살은 작은 정사각형으로 잘라주고 청양
고추는 다져준다.
4. 4. 다진 청양고추와 다진마늘은 볶아주고 다시
여기에 다진채소와 가슴살을 볶아준다.
5. 5. 찹쌀과 보리쌀을 넣어 만들어진 채소국물에
끓여주다 생크림과 우유를 넣어 죽 형태로
만든다.
6. 6. 여기에 볶아진 재료를 넣고 소금으로 간을
하고 말린 토마토를 올려 완성한다.

## 영양 정보

- 칼로리: 388.5 kcal
- 탄수화물: 46.7g
- 단백질: 17.1g
- 지방: 14.8g
- 나트륨: 73.1mg
