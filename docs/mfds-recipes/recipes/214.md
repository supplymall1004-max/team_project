# 과일무스 테린

과일무스 테린은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- ●주재료 : 딸기 100g(5개)
- 키위 100g(1개)
- 매실원액 15g(1큰술)
- 젤라틴 20g
- ●소스 : 망고 20g(1/5개)
- 우유 45g(3큰술)

## 조리 방법

1. 1. 망고와 우유를 섞어 믹서에 갈아 팬에 넣고 끓인다.
2. 2. 딸기와 키위는 0.5×0.5×0.5cm로 썰고 젤라틴은 녹여서 매실원액과 섞는다.
3. 3. 딸기, 키위를 젤라틴과 섞어서 틀에 넣어 굳히고 위에 망고소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 67.5 kcal
- 탄수화물: 9.7g
- 단백질: 7.1g
- 지방: 0g
- 나트륨: 12.4mg
