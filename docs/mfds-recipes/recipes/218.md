# 청국장볶음밥

청국장볶음밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 재료 쌀(100g)
- 현미(30g)
- 감자(30g)
- 고구마(15g)
- 양파(15g)
- 연근(15g)
- 당근(15g)
- 애호박(15g)
- 우엉(15g)
- 마늘(3g)
- 시금치(5g)
- 참나물(10g)
- 깻잎(5g)
- 미나리(5g)
- 육수 다시마(2g)
- 양념 생청국장(30g)
- 들깻가루(2g)
- 올리브유(10g)
- 들기름(8g)

## 조리 방법

1. 1. 쌀과 현미는 물에 깨끗이 씻어 불린
후 밥을 고슬고슬하게 짓는다.
2. 2. 감자, 고구마, 우엉, 양파, 당근,
애호박, 연근은 작게 깍둑 썰고,
마늘은 편 썬다.
3. 3. 시금치와 참나물, 깻잎은 굵게 채
썰고, 미나리는 2.5㎝ 길이로 썬다.
4. 4. 물(50g)에 다시마를 우려내 육수를
내고 청국장을 넣어 잘 푼 뒤
들깻가루를 넣고 잘 저어주며 졸여
청국장소스를 만든다.
5. 5. 팬에 올리브유와 들기름을 두르고
양파, 마늘을 약한 불에 볶다가
감자, 고구마, 우엉, 연근, 당근,
애호박을 넣고 같이 볶는다.
6. 6. 밥과 청국장소스를 부어 80% 정도
볶다가 시금치, 참나물, 미나리를
넣고 한 번 더 볶고 채 썬 깻잎을
올려 마무리한다.

## 영양 정보

- 칼로리: 467.9 kcal
- 탄수화물: 25.05g
- 단백질: 4.27g
- 지방: 4.9g
- 나트륨: 234.1mg
