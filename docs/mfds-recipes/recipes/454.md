# 닭고기채소스파게티

닭고기채소스파게티은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 닭고기살(100g)
- 후춧가루(0.03g)
- 스파게티(60g)
- 올리브유(3g)
- 브로컬리(20g)
- 양송이(20g)
- 표고버섯(20g)
- 파프리카(20g)
- 당근(20g)
- 양파(20g)
- 다진마늘(10g)
- 토마토(20g)
- 토마토페스트(20g)
- 육수(50g)
- 토마토소스(20g)
- 케첩(20g)
- 생크림(20g)

## 조리 방법

1. 1. 닭살은 한 번 정도 펴서 후춧가루를
뿌려 사각지게 썰어 준비한다.
2. 2. 브로컬리는 살짝 데쳐 썰고, 양송이
버섯은 껍질을 벗겨 두툼하게 썰어
준비한다.
3. 3. 파프리카와 토마토는 속씨를 제거하고
사각으로 썰고, 당근, 양파, 표고버섯도
사각지게 썰어 준비한다.
4. 4. 스파게티는 끓는 물에 소금을 넣고 약
10분 정도 삶아 건져 올리브 오일을
살짝 발라 준비한다.
5. 5. 팬에 올리브 오일을 두르고, 따끈해지
면, 닭살을 먼저 볶는다.
6. 6. 닭고기살(⑤)에 준비한 채소(②,③)를
넣고 살짝 볶다가 토마토 페이스트를
넣어 다시 볶은 후 육수와 토마토소스,
케첩, 생크림을 넣고 끓인 후 삶아놓은
스파게티(④)를 넣고 한 번 더 끓여
완성한다.

## 영양 정보

- 칼로리: 277.8 kcal
- 탄수화물: 42g
- 단백질: 14.7g
- 지방: 5.7g
- 나트륨: 174.1mg
