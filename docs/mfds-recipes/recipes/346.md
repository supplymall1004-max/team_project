# 토마토제철나물 샐러드

토마토제철나물 샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 영양부추 20g
- 치커리 20g
- 적양배추 50g
- 알배추 30g
- 방울토마토 100g
- 양파70g
- 두부크루통 : 두부 60g
- 식용유 20g
- 소스 : 마늘 7g
- 참깨 5g
- 흑임자 5g
- 꿀 15g
- 참기름 15g
- 맛간장 45g
- 레몬즙 15g
- 식초 5g

## 조리 방법

1. 1. 마늘과 흑임자 참깨를 다져서 소스를 만든다.
2. 2. 영양부추는 5cm 길이로 잘라주고 알배추는
채 썰어 냉수에 담가둔다.
3. 3. 방울토마토는 먹기 좋게 자르고 적양배추,
양파를 채 썬다.
4. 4. 두부는 기름에 구워서 크루통을 만든다.
5. 5. 준비된 채소를 물기를 빼고 접시에 담아
두부크루통을 올린 뒤 소스와 함께 셋팅한다.

## 영양 정보

- 칼로리: 80 kcal
- 탄수화물: 9g
- 단백질: 0.5g
- 지방: 4.3g
- 나트륨: 72.3mg
