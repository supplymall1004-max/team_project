# 강황고구마밥

강황고구마밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 쌀 100g
- 강황가루 15g
- 고구마 50g
- 닭육수 300g
- 양념장 : 영양부추 10g
- 다진청고추 15g
- 다진홍고추 15g
- 다진양파 15g
- 간편 어간장 15g

## 조리 방법

1. 1. 먼저 영양부추를 뺀 재료로 양념장을 30분
전에 만들어준다.
2. 2. 쌀은 30분전에 불려주고 고구마는 껍질째
작은 깍두기 모양으로 잘라 냉수에 담가준다.
3. 3. 닭육수에 강황가루를 풀어주고 쌀과 고구마를
고루 섞어 밥을 해준다.
4. 4. 만들어진 양념장에 영양부추를 한입 크기로
잘라 최종 양념장을 만들어준다.
5. 5. 그릇에 강황고구마밥을 담아주고 양념장과
함께 완성한다.

## 영양 정보

- 칼로리: 251.1 kcal
- 탄수화물: 54.1g
- 단백질: 4.3g
- 지방: 1.9g
- 나트륨: 164.7mg
