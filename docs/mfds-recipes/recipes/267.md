# 양지해장국

양지해장국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 쇠고기육수(30g)
- 느타리버섯(10g)
- 애호박(20g)
- 청양고추(1g)
- 대파(10g)
- 방울토마토(80g)
- 콩나물(50g)
- 양념 다진 마늘(8g)
- 저염국간장(2.5g)
- 소금(0.5g)

## 조리 방법

1. 1. 육수를 끓인 양지는 건져 얇게 썬다.
2. 2. 버섯은 잘게 찢고, 애호박은
반달모양으로 납작 썰고, 고추와
대파는 어슷하게 썬다.
3. 3. 방울토마토에 열십자로 칼집을 넣고
끓는 물에 데쳐 껍질을 벗긴다.
4. 4. 냄비에 식용유(10g)를 두른 뒤
다진 마늘을 넣고 볶아 향을 낸 뒤
방울토마토를 넣고 볶는다.
5. 5. 육수(300g), 콩나물, 쇠고기,
느타리버섯, 애호박을 넣고 국간장과
소금으로 간해 끓인다.
6. 6. 어슷 썬 청양고추와 대파를 넣어
마무리한다.

## 영양 정보

- 칼로리: 103.8 kcal
- 탄수화물: 2.47g
- 단백질: 2.89g
- 지방: 1.32g
- 나트륨: 79.7mg
