# 새콤한연어샐러드

새콤한연어샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연어 60g
- 어린잎 20g
- 말린 토마토 15g
- 양파 50g
- 2가지색 미니파프리카 70g
- 소스 : 유자청 30g
- 감식초 15g
- 정종 100ml
- 볶은소금 0.5g

## 조리 방법

1. 1. 연어는 얇게 모양을 살려 슬라이스 한 후
정종에 소금을 섞어 분무기에 넣어 냉장고에
넣어둔다.
2. 2. 건조된 방울토마토는 올리브오일과 후추로
양념하여 사용한다.
3. 3. 양상추는 먹기 좋은 크기로 뜯어 어린잎과
함께 찬물에 담가 물기를 제거한 후 사용한다.
4. 4. 유자청과 감식초는 함께 섞어 믹서에 갈아
드레싱소스를 완성한다.
5. 5. 양파는 모양을 살려 얇게 슬라이스 하여
찬물에 담가 물기를 제거한 뒤 사용한다.
6. 6. 샐러드 접시에 준비된 모든 채소를 담고,
연어는 장미 모양으로 접어서 보기좋게 담아
드레싱 소스를 곁들여 완성한다.

## 영양 정보

- 칼로리: 118.3 kcal
- 탄수화물: 5.4g
- 단백질: 8.6g
- 지방: 6.9g
- 나트륨: 48.1mg
