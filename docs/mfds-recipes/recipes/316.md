# 비타민이유식

비타민이유식은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 멜론 300g
- 브로콜리 30g
- 감자 60g
- 요거트 15g
- 설탕 5g

## 조리 방법

1. 1. 브로콜리는 데쳐낸 후 송송 다져준다.
2. 2. 감자는 삶아준 후 체에 내려준다.
3. 3. 멜론은 씨를 제거하고 껍질을 벗긴 후 믹서에
갈아 체에 내려준다.
4. 4. 체에 내린 감자와 브로콜리 그리고 멜론즙을
함께 섞어준다.
5. 5. 설탕으로 간을 해주고 볼에 담고 요거트로
장식을 해 완성한다.

## 영양 정보

- 칼로리: 115.9 kcal
- 탄수화물: 21.9g
- 단백질: 2.8g
- 지방: 1.9g
- 나트륨: 64.1mg
