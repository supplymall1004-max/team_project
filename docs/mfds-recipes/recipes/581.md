# 파프리카더덕소고기깻잎롤

파프리카더덕소고기깻잎롤은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 소고기(살치살
- 120g)
- 파프리카(50g)
- 더덕(100g)
- 깻잎(3장)
- 파인애플(50g)
- 아스파라거스(3개)
- 양파(50g)
- 꿀(20g)
- 레몬(1/4개)
- 식초(10g)
- 소금(0.2g)
- 올리브오일(3g)
- 후춧가루(0.01g)

## 조리 방법

1. 1. 소고기는 후춧가루, 올리브오일로
밑간을 한다.
2. 2. 더덕은 칼로 껍질을 돌려 깎고,
파프리카, 아스파라거스와 함께
가늘게 채썬다.
3. 3. 파인애플, 양파, 당근, 대파는 채썬다.
4. 4. 밑간한 고기를 팬에 굽는다.
5. 5. 따로 갈아 놓은 파인애플과 레몬즙
꿀을 섞어 드레싱을 만는다
6. 6. 구워 놓은 고기에 더덕과 파인애플
아스파라거스 깻잎을 넣어 돌돌 말고,
팬에 발사믹 소스를 끓인 뒤 말아 놓은
소고기롤을 넣어 졸인다.

## 영양 정보

- 칼로리: 230.7 kcal
- 탄수화물: 16.9g
- 단백질: 28.8g
- 지방: 5.3g
- 나트륨: 97.8mg
