# 두부까나페

두부까나페은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 단단한 두부 150g
- 슬라이스햄 25g
- 올리브오일 15g
- 로즈마리 1g
- 볶은소금 1g
- 샐러드 : 마요네즈 30g
- 당근 20g
- 양파 50g
- 오이 25g
- 소스 : 맛간장 15g
- 양파 15g
- 다진마늘 5g
- 후추 0.5g

## 조리 방법

1. 1. 두부는 1cm 두께로 잘라 볶은소금을 뿌려
수분을 제거한다.
2. 2. 오이는 껍질 부분만 송송 썰어준다.
3. 3. 당근, 양파는 송송 썰어준다.
4. 4. 두부는 노릇하게 구워준다.
5. 5. 오이, 당근, 양파를 소스에 버무려 샐러드를
만들고 소스도 함께 만든다.
6. 6. 두부를 한입 크기로 잘라준 후 슬라이스햄과
샐러드를 올려 소스랑 함께 완성한다.

## 영양 정보

- 칼로리: 236 kcal
- 탄수화물: 0.1g
- 단백질: 13.3g
- 지방: 20.2g
- 나트륨: 287.6mg
