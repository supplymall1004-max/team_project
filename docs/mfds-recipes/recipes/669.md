# 콜라비오미자 물김치

콜라비오미자 물김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 콜라비(1/2개)
- 오미자(20g)
- 무화과(20g)
- 파프리카(50g)
- 피망(30g)
- 쪽파(10g)
- 양파(30g)
- 건새우(10g)
- 다시마(10g)
- 소금(0.3g)

## 조리 방법

1. 1. 다시마와 건새우를 넣고 은근히 끓여
차갑게 식힌 뒤 육수를 만든다.
2. 2. 차갑게 식힌 육수는 체에 걸려내고
오미자를 넣고 소금으로 간을 한다.
3. 3. 콜라비, 피망, 파프리카는 납작하게
썬다.
4. 4. 육수는 체에 거른다
5. 5. 오미자를 넣고 소금으로 간을 한다.
6. 6. 통에 썰어놓은 야채와 무화과를 담고
?를 넣는다.

## 영양 정보

- 칼로리: 71.6 kcal
- 탄수화물: 15.6g
- 단백질: 1.7g
- 지방: 0.2g
- 나트륨: 151.4mg
