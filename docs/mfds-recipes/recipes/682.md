# 비름나물겉절이

비름나물겉절이은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 주재료 : 비름나물 30g
- 소금물(물 100g
- 소금 3g) - 양념장 : 까나리액젓 1.5g
- 국간장 1.5g
- 물 15g
- 고춧가루 1.7g
- 매실액 6g
- 통깨 1g
- 다진 파 2.5g
- 다진 마늘 1g

## 조리 방법

1. 1. 쇠비름을 씻는다.
2. 2. 씻은 쇠비름을 소금물에 담가 살짝 절인다.
3. 3. 절여논 쇠비름을 헹구어 체에 건진다.
4. 4. 양념 재료를 모두 섞어 양념장을 만든다.
5. 5. 쇠비름에 양념장을 넣어 살짝 무쳐낸다.
6. 6. 그릇에 담아 완성한다.

## 영양 정보

- 칼로리: 29.4 kcal
- 탄수화물: 2.5g
- 단백질: 3.6g
- 지방: 1g
- 나트륨: 381.3mg
