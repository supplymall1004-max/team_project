# 치자연근물김치

치자연근물김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연근 200g
- 당근 40g
- 미나리 20g
- 청고추 15g
- 홍고추 18g
- 알마늘 12g
- 치자가루 15g
- 생강청 30g
- 식초 15g
- 볶은소금 5g
- 설탕 7g

## 조리 방법

1. 1. 당근은 모양틀로 잘라서 편으로 썰어준다.
2. 2. 연 근 은 껍 질 을 벗 겨 서 식 초 물 에 살 짝
담가준다.
3. 3. 청고추, 홍고추, 미나리는 먹기 좋게 썰어준다.
4. 4. 마늘은 편으로 썰어서 물 400g에 끓여주다
불을 끄고 치자가루를 넣어 국물을 만들어준다.
5. 5. 국물에 연근을 담가 색을 들인다.
6. 6. 여기에 생강청, 식초, 설탕, 볶은소금과 준비된
채소를 넣어 치자 연근물김치를 완성한다.

## 영양 정보

- 칼로리: 80.9 kcal
- 탄수화물: 17.3g
- 단백질: 1.6g
- 지방: 0.6g
- 나트륨: 106mg
