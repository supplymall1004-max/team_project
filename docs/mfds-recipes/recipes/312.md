# 토마토샐러드라면

토마토샐러드라면은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 양상추(80g)
- 치커리(15g)
- 굵은 면발 라면(55g)
- 삶은 닭가슴살(50g)
- 방울토마토(15g)
- 파슬리가루(0.1g)
- 소스 설탕(2g)
- 식초(1.5g)
- 레몬즙(1g)
- 저염간장(3g)
- 다진 마늘(1g)
- 다진 토마토(50g)
- 다진 양파(4g)
- 올리브유(1.5g)

## 조리 방법

1. 1. 소스를 만든다.
2. 2. 양상추와 치커리는 굵은 줄기
부분을 빼고 먹기 좋은 크기로 잘게
찢는다.
3. 3. 라면 끓는 물에서 4분간 삶고 체에
건져 찬물에 헹군다.
4. 4. 닭가슴살을 먹기 좋은 크기로 찢어
면과 함께 섞는다.
5. 5. 그릇에 채소와 면을 담고,
방울토마토도 먹기 좋게 잘라
담는다.
6. 6. 소스와 파슬리가루를 뿌려
마무리한다.

## 영양 정보

- 칼로리: 360.6 kcal
- 탄수화물: 17.23g
- 단백질: 6.55g
- 지방: 3.59g
- 나트륨: 241.7mg
