# 효도강정

효도강정은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 볶은현미 25g
- 볶은율무 25g
- 볶은수수 25g
- 혼합견과 50g
- 흑임자 10g
- 소스 : 올리고당 25g
- 설탕 25g
- 기름 10g

## 조리 방법

1. 1. 모든 재료를 한데 섞어준다.
2. 2. 소스를 만들어준다.
3. 3. 준비된 틀에 비닐을 깔고 기름을 발라준다.
4. 4. 소스가 끓어오르면 모든 재료를 넣어 버무려
준다.
5. 5. 준비된 용기에 만들어진 재료를 쏟아 다져준다.
6. 6. 식으면 잘라서 완성한다.

## 영양 정보

- 칼로리: 370 kcal
- 탄수화물: 53.2g
- 단백질: 7.5g
- 지방: 14.1g
- 나트륨: 45.5mg
