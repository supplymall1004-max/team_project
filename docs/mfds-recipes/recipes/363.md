# 미역 닭가슴살전

미역 닭가슴살전은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 닭가슴살 100g
- 미역 10g
- 두부 50g
- 청양고추 10g
- 대파 10g
- 당근 10g
- 계란 60g
- 소스 : 저염간장 30g
- 식초 15g
- 레몬즙 10g
- 고추냉이 5g
- 통깨 1g

## 조리 방법

1. 1. 닭가슴살은 송송 다져준 후 소금, 후추로
밑간을 해준다.
2. 2. 미역은 물에 불려준 후 송송 잘라준다.
3. 3. 두부는 물기를 꼭 짜준다.
4. 4. 청양고추, 대파, 당근은 송송 다져준다.
5. 5. 소스를 만들어준다.
6. 6. 준비된 재료에 계란으로 반죽해 한입 크기로
전을 부쳐준다.

## 영양 정보

- 칼로리: 234.1 kcal
- 탄수화물: 14g
- 단백질: 17.7g
- 지방: 11.9g
- 나트륨: 284.5mg
