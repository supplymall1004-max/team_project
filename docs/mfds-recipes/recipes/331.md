# 해초갈비찜

해초갈비찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 찜갈비 200g
- 마른미역줄기 10g
- 감자 20g
- 양파 25g
- 청고추 15g
- 홍고추 15g
- 대추 4g
- 당근 20g
- 메추리알 25g
- 양념장 : 맛간장 30g
- 다진마늘 5g
- 다진대파 5g
- 생강청 15g
- 후추 1g
- 참기름 3g
- 올리고당 15g
- 통깨 1g

## 조리 방법

1. 1. 갈비는 핏물을 빼주고 뜨거운 물에 80% 이상
데쳐준 뒤 갈비와 감자를 물을 자작하게 넣고
청주, 생강청을 넣고 푹 삶아준다.
2. 2. 양파, 청고추, 홍고추는 먹기 좋게 썰어준다.
3. 3. 마른미역줄기는 냉수에 행군 뒤 물기를 빼준다.
4. 4. 양념장을 바글바글 끓여준다.
5. 5. 끓고 있는 양념장에 삶아진 갈비, 감자, 당근
순으로 끓이다 메추리알을 넣고 졸여준다.
6. 6. 마지막에 미역줄기를 넣고 볶아주는 형식으로
갈비찜을 완성한다.

## 영양 정보

- 칼로리: 288.6 kcal
- 탄수화물: 5.2g
- 단백질: 31.1g
- 지방: 16g
- 나트륨: 604.7mg
