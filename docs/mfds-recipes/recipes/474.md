# 주꾸미보리죽

주꾸미보리죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 주꾸미(100g)
- 밀가루(10g)
- 보리(100g)
- 미나리(10g)
- 당근(20g)
- 양파(20g)
- 참기름(5g)
- 육수(400g)
- 저염치즈(20g)
- 우유(100g)
- 저염간장(5g)

## 조리 방법

1. 1. 보리는 깨끗이 씻어 물을 충분히 붓고
약 30분 정도 불려 준비한다.
2. 2. 주꾸미는 밀가루를 넣고 조물조물
주물러 다시 깨끗이 씻어 준비한다.
3. 3. 씻어놓은 주꾸미는 살짝 데쳐 입자
있게 다져 놓는다.
4. 4. 미나리와 당근, 양파는 입자있게
다져 놓는다.
5. 5. 냄비에 참기름을 두르고, 주꾸미를
먼저 볶다가 불린 보리쌀을 넣고 다시
볶은 뒤 썰어놓은 채소(④)를 넣는다.
6. 6. 냄비에 육수를 넣고 나무주걱으로
은근히 저어가면 끓이다가 죽이 어우러
지면, 우유와 치즈, 간장을 살짝 넣고
한 번 더 끓여 완성한다.

## 영양 정보

- 칼로리: 327.4 kcal
- 탄수화물: 48.7g
- 단백질: 21.8g
- 지방: 5g
- 나트륨: 172.7mg
