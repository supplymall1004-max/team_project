# 구운 바나나

구운 바나나은(는) 후식 요리로, 굽기 조리법을 사용합니다.

## 재료

- ●주재료 : 바나나 100g(1개)
- 설탕 5g(1/2작은술)
- ●장식 : 다진 호두 15g(1큰술)

## 조리 방법

1. 1. 바나나는 껍질을 벗겨 2등분한다.
2. 2. 가열된 팬에 기름을 두르지 않고 바나나를 굽는다.
3. 3. 구워진 바나나의 겉면에 설탕을 뿌리고 가열된 팬에 구운 후 완성된 바나나를 접시에 담고, 다진 호두를 뿌린다.

## 영양 정보

- 칼로리: 158 kcal
- 탄수화물: 27.7g
- 단백질: 2.8g
- 지방: 4g
- 나트륨: 5.2mg
