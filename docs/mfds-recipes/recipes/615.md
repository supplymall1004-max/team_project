# 겨자아욱쌈밥

겨자아욱쌈밥은(는) 밥 요리로, 찌기 조리법을 사용합니다.

## 재료

- 현미(200g)
- 두부(100g)
- 아욱(100g)
- 날치알(50g)
- 단호박(1/6개)
- 새싹(20g)
- 된장(20g)
- 생크림(20g)
- 매실청(20g)
- 참기름(5g)

## 조리 방법

1. 1. 현미는 깨끗이 씻어 30분 이상 충분히
불린 뒤 밥을 짓는다
2. 2. 두부는 끓는 물에 데쳐 물기를 빼고
곱게 다져 된장, 생크림, 매실청을 넣고
골고루 섞는다.
3. 3. 데친 두부에 된장, 생크림 및 매실청을
골고루 섞는다.
4. 4. 아욱과 단호박은 각각 찐 후 식힌다.
5. 5. 찐 아욱 위에 현미밥을 올리고
두부소스, 날치알, 새싹 단호박을
올린다.
6. 6. 아욱에 올린 밥을 동그랗게 말아
접시에 담는다.

## 영양 정보

- 칼로리: 359.5 kcal
- 탄수화물: 64.9g
- 단백질: 13.6g
- 지방: 5g
- 나트륨: 264.4mg
