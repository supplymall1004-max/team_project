# 초계탕

초계탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기(다리
- 2개)
- 양파(50g)
- 파프리카(50g)
- 적양배추(20g)
- 오이(50g)
- 마늘(30g)
- 겨자가루(20g)
- 대파(20g)
- 소금(0.3g)
- 설탕(20g)
- 식초(10g)

## 조리 방법

1. 1. 냄비에 마늘과 대파를 넣고 끓이다가
닭다리를 넣고 삶는다
2. 2. 닭을 삶아 놓은 육수는 걸러 기름을
제거 하고 차갑게 식힌다.
3. 3. 삶아 놓은 닭고기는 결대로 찢는다.
4. 4. 양파, 오이와 파프리카는 채썬다
5. 5. 적채는 채를 썰어 찬물에 담궈 건져
물기를 빼 놓는다.
6. 6. 차갑게 걸러 놓은 닭 육수에 발효된
겨자와 소금 .설탕, 식초를 넣어 육수
간을 하고, 그릇에 닭살과 오이 적채,
파프리카 양파를 올려 차가운 육수를
넣어 완성한다.

## 영양 정보

- 칼로리: 103 kcal
- 탄수화물: 7.1g
- 단백질: 16.2g
- 지방: 1.1g
- 나트륨: 90.1mg
