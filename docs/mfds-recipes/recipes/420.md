# 코다리맑은찜

코다리맑은찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 코다리 250g
- 콩나물 50g
- 미나리 20g
- 함초 25g
- 무 30g
- 함초소금 1g
- 양념 : 간편어간장 15g
- 함초소금 1g
- 생강청 15g
- 흰후추 1g
- 청주 15g
- 레몬즙 5g
- 다진마늘 15g
- 연겨자 7g
- 통깨 10g

## 조리 방법

1. 1. 코다리는 어간장, 청주, 흰후추, 생강청에
살짝 재워준다.
2. 2. 함초는 손질해 잘라준다.
3. 3. 무는 소금에 살짝 절여서 수분과 염분을
제거해준다.
4. 4. 무와 함초를 고루 섞어준다.
5. 5. 미나리와 콩나물은 손질해준 후 찜기에 깔고
코다리를 쪄준다.
6. 6. 소스를 만든 뒤 채소를 깔고 코다리를 올리고
미나리, 콩나물을 올려 완성한다.

## 영양 정보

- 칼로리: 96.7 kcal
- 탄수화물: 1.1g
- 단백질: 19.1g
- 지방: 1.8g
- 나트륨: 431mg
