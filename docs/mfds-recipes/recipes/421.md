# 뿌리채소두부선

뿌리채소두부선은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 두부 300g
- 전분 50g
- 계란 60g
- 흰후추 2g
- 소 : 우엉 30g
- 인삼 15g
- 감자 30g
- 청고추 10g
- 홍고추 15g
- 마늘기름 15g
- 흰후추 1g
- 소스 : 일본된장 15g
- 마늘기름 15g
- 들기름 15g
- 식초 30g
- 올리고당 5g
- 마요네즈 15g
- 땅콩잼 30g
- 참깨 2g

## 조리 방법

1. 1. 두부는 물기를 꼭 짠 뒤 으깨어 전분, 계란과
섞어준다.
2. 2. 우엉, 인삼, 감자는 아주 곱게 채 썰어서 냉수에
담가준다.
3. 3. 볶아진 모든 채소를 각각 볶아서 한데 섞어
준다.
4. 4. 땅콩잼으로 소스를 만들어준다.
5. 5. 김발을 깔고 비닐을 놓아 준 후 두부를 펴준다.
6. 6. 볶아진 채소를 두부에 넣어주고 찜기에 쪄서
식힌 뒤 한입 크기로 잘라준다.

## 영양 정보

- 칼로리: 184.5 kcal
- 탄수화물: 19.7g
- 단백질: 9.4g
- 지방: 7.6g
- 나트륨: 135mg
