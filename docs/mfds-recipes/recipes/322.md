# 한식풍닭가슴살스테이크

한식풍닭가슴살스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 닭가슴살 280g
- 미나리 50g
- 쌀 50g
- 청경채 60g
- 무조림 : 무 150g
- 맛간장 15g
- 고춧가루 15g
- 다진마늘 5g
- 물 200g
- 소스 : 된장 15g
- 매실청 15g
- 들기름 10g
- 다진양파 10g
- 검은깨 5g
- 다진마늘 5g

## 조리 방법

1. 1. 미나리는 데친 후 먹기 좋은 크기로 잘라서
밥을 해준다.
2. 2. 닭가슴살에 밥을 넣고 말아서 오븐에 굽는다.
3. 3. 무는 물에 익히고 고춧가루와 무조림 재료를
넣어 졸여준다.
4. 4. 청경채는 데친 후 팬에 살짝 볶아준다.
5. 5. 소스를 만들어준다.
6. 6. 접시에 무조림을 깔고 닭가슴살 구이를 올리고
청경채와 함께 소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 191.5 kcal
- 탄수화물: 10g
- 단백질: 24.4g
- 지방: 6g
- 나트륨: 294mg
