# 양배추감자전

양배추감자전은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- ●주재료 : 감자 100g(1개)
- 양배추 150g(1/2개)
- 당근 15g(1/10개)
- 두부 20g(1/20모)
- 돼지고기 30g
- 청양고추 5g(1개)
- 부침가루 45g(3큰술)
- 달걀 60g(1개)
- 식용유 15g(1큰술)
- ●소스 : 오렌지즙 15g(1큰술)
- 간장 2g(1/2작은술)
- 식초 10g(2작은술)

## 조리 방법

1. 1. 감자는 믹서에 갈고 양배추는 채 썰고 고기, 당근, 청양고추, 두부는 다진다.
2. 2. 준비해둔 재료를 모두 섞고 부침가루와 계란을 넣어 반죽한다.
3. 3. 가열된 팬에 기름을 두르고 반죽을 부어 굽고 소스를 함께 곁들인다.

## 영양 정보

- 칼로리: 148.9 kcal
- 탄수화물: 14.7g
- 단백질: 7.7g
- 지방: 6.6g
- 나트륨: 107.1mg
