# 불고기덮밥

불고기덮밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 쌀밥 200g
- 소불고기 150g
- 양파 60g
- 청고추 15g
- 홍고추 18g
- 팽이버섯 20g
- 계란 60g
- 쪽파 15g
- 마늘기름 30g
- 통깨 5g
- 양념장 : 맛간장 15g
- 생강청 15g
- 다진마늘 10g
- 다진대파 10g
- 흰후추 1g
- 참기름 5g

## 조리 방법

1. 1. 소 불고기는 양념장을 버무려 팬에 볶아준다.
2. 2. 청고추, 홍고추는 가늘게 채 썰어준다.
3. 3. 양파는 채 썰고 쪽파도 한입 크기로 잘라준다.
4. 4. 팽이도 먹기 좋은 크기로 잘라준다.
5. 5. 계란물을 풀어준다.
6. 6. 채소를 각각 볶아주다가 볶은 불고기에 섞어
주고 계란물을 풀어서 밥에 올려서 통깨를
뿌려 완성한다.

## 영양 정보

- 칼로리: 332.5 kcal
- 탄수화물: 58.3g
- 단백질: 12.6g
- 지방: 5.4g
- 나트륨: 192.5mg
