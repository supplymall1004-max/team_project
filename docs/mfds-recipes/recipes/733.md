# 옥수수감자샐러드

옥수수감자샐러드은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 감자 70g
- 달걀 60g
- 옥수수알 10g
- 완두콩알 5g
- 빨강파프리카 5g
- 마요네즈 12g
- 호상요구르트 10g
- 소금 0.15g

## 조리 방법

1. 1. 감자는 익기 쉽도록 토막 내고 완두콩과 옥수수알과 함께 김이 오른 찜통에 10~15분간 찐다.
2. 2. 찐 감자를 으깬다.
3. 3. 달걀은 삶아 흰자는 굵게 다지고 노른자는 체에 곱게 내린다.
4. 4. 파프리카는 옥수수알 크기로 썬다.
5. 5. 2에 완두콩, 옥수수, 파프리카, 달걀 흰자를 넣어 섞는다.
6. 6. 마요네즈, 호상요구르트, 달걀노른자, 소금을 넣어 샐러드드레싱을 만들어 5와 곁들인다.

## 영양 정보

- 칼로리: 248.6 kcal
- 탄수화물: 14.8g
- 단백질: 10.9g
- 지방: 16.1g
- 나트륨: 212.7mg
