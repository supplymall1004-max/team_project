# 파프리카 물김치

파프리카 물김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●주재료 : 자색고구마 10g(3cm)
- 무 10g(2cm)
- 배 80g(1/4개)
- 배추 200g(1/3개)
- 부추 10g
- 사과 80g(1/3개)
- 쪽파 10g(10cm)
- 소금 20g(1⅓큰술)
- ●양념 : 멥쌀 10g(2작은술)
- 물엿 50g(1/4컵)
- 황파프리카 140g(1½개)

## 조리 방법

1. 1. 황파프리카를 갈아 체에 내리고 멥쌀로 쌀풀을 만들어 황파프리카 즙을 섞고 물엿을 첨가하여 양념을 만든다.
2. 2. 배추를 소금물에 절이고 절인 배추를 찬물로 씻어 소금기를 뺀다.
3. 3. 소금기를 뺀 배추에 채썬 자색고구마, 무, 쪽파, 부추, 사과, 배를 놓고 양념을 부은 후 3일 동안 숙성시켜 완성한다.

## 영양 정보

- 칼로리: 44.9 kcal
- 탄수화물: 10.1g
- 단백질: 1.1g
- 지방: 0g
- 나트륨: 272mg
