# 누룽닭

누룽닭은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭가슴살(120g)
- 생강즙(5g)
- 정종(10g)
- 시금치(20g)
- 당근(20g)
- 양파(20g)
- 샐러리(20g)
- 파프리카(20g)
- 느타리(20g)
- 주꾸미(30g)
- 새우(20g)
- 청경채(20g)
- 누룽지(50g)
- 튀김기름(200g)
- 육수(200g)
- 저염간장(10g)
- 녹말가루(10g)
- 마늘(10g)
- 소금(0.2g)
- 후춧가루(0.1g)

## 조리 방법

1. 1. 닭은 포를 떠서 생강즙과 정종을 섞어
발라 놓는다.
2. 2. 시금치는 살짝 데쳐 썰고, 당근, 양파,
샐러리, 파프리카는 약 6Cm 길이로
채를 썰어 절반은 닭고기말이에 사용
하고, 절반은 곁들임채소로 사용한다.
3. 3. 느타리는 끓는 물에 소금을 넣고 데쳐
손으로 찢어놓고, 포를 뜬 닭가슴살에
느타리와 썰어 놓은 채소를 넣어 말아
주고, 육수를 끓여서 말아놓은 닭을
넣고 익혀낸다.
4. 4. 주꾸미와 새우는 끓는 물에 살짝 데치고,
청경채는 반으로 썰어 살짝 데쳐 준비
한다.
5. 5. 튀김기름 온도가 180℃ 정도 되면
누룽지를 넣고 튀겨 기름을 밭쳐 놓는다.
6. 6. 닭을 넣은 육수(③)에 간장과 녹말,
마늘, 소금, 후춧가루를 넣고 끓이다가,
곁들임채소로 남겨 놓은 채소(②)와
데쳐 놓은 해물 및 청경채를 넣고 한 번 더
끓인 뒤 튀겨진 누룽지를 넣어 완성한다.

## 영양 정보

- 칼로리: 415.5 kcal
- 탄수화물: 47.5g
- 단백질: 23.8g
- 지방: 14.5g
- 나트륨: 290.9mg
