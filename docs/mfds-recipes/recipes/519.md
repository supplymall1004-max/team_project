# 콩가루비빔밥

콩가루비빔밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 청포묵(20g)
- 참기름(5g)
- 소금(0.5g)
- 달걀(50g)
- 소금(0.5g)
- 오이(20g)
- 당근(20g)
- 고사리(20g)
- 표고버섯(20g)
- 식용유(10g)
- 쌀밥(150g)
- - 소고기 곁들임채소 : 소고기(15g)
- 대파(5g)
- 마늘(10g)
- 참기름(5g)
- 후춧가루(0.03g)
- - 고추장소스 : 고추장(20g)
- 콩가루(20g)
- 다진 소고기(15g)

## 조리 방법

1. 1. 청포묵은 4cm 길이로 썰어 참기름과
소금에 살짝 무쳐 준비한다.
2. 2. 달걀은 황백으로 나누어 각각 소금을
조금씩 넣어 섞어 지단을 부친다.
3. 3. 오이는 돌려 깎아 채 썰고 당근도 오이
길이로 채 썰고 고사리는 길에 맞게
다듬고, 표고버섯도 채를 썰어 준비한다.
4. 4. 소고기는 채를 썰어 파, 마늘, 참기름,
후춧가루를 넣어 양념한다.
5. 5. 팬에 기름을 두르고, 준비한 곁들임
채소(③)와 다진 소고기를 재료별로
따로 볶아 준비한다.
6. 6. 고추장에 콩가루를 섞고, 볶아진 소고기를
넣어 다시 볶아 고추장 소스를 준비하고,
쌀밥을 그릇에 담은 후 청포묵(①), 볶아
놓은 채소와 소고기(⑤)를 올리고,
마지막에 고추장소스를 올려 완성한다.

## 영양 정보

- 칼로리: 449.4 kcal
- 탄수화물: 85.2g
- 단백질: 13g
- 지방: 6.3g
- 나트륨: 575.8mg
