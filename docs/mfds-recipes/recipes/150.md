# 감귤 김치

감귤 김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●주재료 : 배추 10g(1/60개)
- 미나리 5g
- 홍고추 5g(1/2개)
- 소금 5g(1작은술)
- ●김치국물 : 귤 100g(1개)
- 고춧가루 5g(1작은술)
- 다진 마늘 10g(2작은술)
- 생강즙 5g(1작은술)

## 조리 방법

1. 1. 귤은 껍질을 까서 믹서에 갈아 체에 내린다.
2. 2. 배추는 3×3cm로 썰어 소금에 절인 후 체에 받치고 미나리, 홍고추는 길이 3cm로 썬다.
3. 3. 귤즙에 고춧가루, 생강즙, 다진 마늘을 넣고 김치 국물을 만들고 배추, 미나리, 홍고추에 김치 국물을 부어 실온에 2일 숙성시킨다.

## 영양 정보

- 칼로리: 21.2 kcal
- 탄수화물: 4.6g
- 단백질: 0.6g
- 지방: 0g
- 나트륨: 64.9mg
