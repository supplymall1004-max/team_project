# 훈제오리가슴살 샐러드

훈제오리가슴살 샐러드은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 오리고기(훈제오리 가슴살
- 150g)
- 발사믹소스(50g)
- 양파(30g)
- 파슬리(5g)
- 블루베리(20g)
- 레디쉬(1개)

## 조리 방법

1. 1. 훈제오리 가슴살을 슬라이스한다.
2. 2. 슬라이스한 훈제오리는 팬에 굽는다.
3. 3. 양파는 채썰고 찬물에 담근 뒤 건진다.
4. 4. 레디쉬는 채썰고 찬물에 담근 뒤
건진다.
5. 5. 먹기 직전에 발사믹소스를 만든다.
6. 6. 접시에 훈제오리와 양파채, 레디쉬,
블루베리를 담고 발사믹소스를
올린다.

## 영양 정보

- 칼로리: 261.8 kcal
- 탄수화물: 11.5g
- 단백질: 21g
- 지방: 14.6g
- 나트륨: 636mg
