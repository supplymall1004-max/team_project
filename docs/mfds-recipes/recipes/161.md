# 유자 대구조림

유자 대구조림은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- ●주재료 : 대구살 120g
- 홍파프리카 10g(1/10개)
- 황파프리카 10g(1/10개)
- 청파프리카 10g(1/10개)
- 황금팽이버섯 10g
- 미나리 20g
- 밀가루 25g(2큰술)
- ●소스 : 유자청 20g(1⅓큰술)
- 발사믹식초 10g(2작은술)
- 화이트와인 10g(2작은술)
- ●장식 : 대파 30g(1개)

## 조리 방법

1. 1. 발사믹 식초, 화이트와인, 다진 유자청 건더기를 넣고 졸여 소스를 만들고 대구를 손질하여 포를 뜬다.
2. 2. 대파, 청파프리카, 홍파프리카, 황파프리카를 채 썬다.
3. 3. 대구포에 밀가루를 묻혀 청파프리카, 홍파프리카, 황파프리카, 황금팽이버섯을 넣고 말아 미나리로 묶어 찐 후 소스를 뿌리고 썬 대파를 올려 완성한다.

## 영양 정보

- 칼로리: 84 kcal
- 탄수화물: 10.6g
- 단백질: 9.6g
- 지방: 0.4g
- 나트륨: 109.7mg
