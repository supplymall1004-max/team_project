# 가지연어구이

가지연어구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 가지(40g)
- 토마토(20g)
- 연어(200g)
- 파슬리가루(1g)
- 올리브유(5g)
- 마늘(10g)
- 소스 사과(20g)
- 유자청(30g)
- 레몬즙(5g)

## 조리 방법

1. 1. 가지는 어슷 썰고, 토마토는 반
자른다.
2. 2. 연어는 파슬리와 올리브유에 재운다.
3. 3. 마늘은 노릇하게 굽는다.
4. 4. 재운 연어와 가지, 토마토는 180℃로
예열한 오븐에 10분 정도 굽는다.
5. 5. 사과는 잘게 다져 유자청과
레몬즙을 넣고 살짝 볶아 소스를
만든다.
6. 6. 연어, 가지, 토마토, 구운 마늘을
담고 소스를 얹어 마무리한다.

## 영양 정보

- 칼로리: 318.2 kcal
- 탄수화물: 9.85g
- 단백질: 15.8g
- 지방: 6.19g
- 나트륨: 83.4mg
