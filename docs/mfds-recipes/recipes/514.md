# 연어차우더스프

연어차우더스프은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 모시조개(50g)
- 연어(100g)
- 청피망(20g)
- 당근(20g)
- 양파(20g)
- 감자(50g)
- 밀가루(10g)
- 버터(5g)
- 육수(200g)
- 월계수잎(1장)
- 통후추(3알)
- 생크림(30g)
- 통마늘(10g)

## 조리 방법

1. 1. 모시조개는 물에 소금을 넣고, 담그어
해감을 한다.
2. 2. 연어는 사각지게 썰어 준비한다.
3. 3. 당근, 양파, 감자는 사각크기로 썰어
놓고, 청피망은 채를 썰어 당근과
비슷한 길이로 잘라둔다.
4. 4. 냄비에 버터와 밀가루를 넣고 볶다가
채소를 넣어 같이 볶는다.
5. 5. 볶아지는 냄비에 모시조개와 육수를
넣고 끓인다.
6. 6. 거품을 걷어내고, 월계수 잎과 통후추,
통마늘을 넣고 끓이다가 연어와
생크림을 넣어 한 소큼 끓으면 완성한다.

## 영양 정보

- 칼로리: 217.6 kcal
- 탄수화물: 23.2g
- 단백질: 11.7g
- 지방: 8.7g
- 나트륨: 215.4mg
