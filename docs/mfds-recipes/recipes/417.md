# 돼지머리수육맑은전골

돼지머리수육맑은전골은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 돼지머리 200g
- 우동면 100g
- 미나리 20g
- 청고추 15g
- 소스 : 일본 된장 30g
- 다진마늘 10g
- 다진대파 10g
- 청주 15g
- 무 25g
- 육수 : 해물육수 300g
- 청양고추 20g
- 맛간장 15g

## 조리 방법

1. 1. 무, 청고추, 미나리, 대파는 먹기 좋게 썰어준다.
2. 2. 돼지머리는 냉수에 데친다. 냉수일 때부터
돼지머리를 넣어야 물이 끓어오르면 불순물이
제거된다.
3. 3. 일본 된장을 풀어 체에 걸러준다.
4. 4. 돼지머리를 살짝 졸여준다.
5. 5. 냄비에 국물을 끓여주다가 채소를 넣어 끓인다.
6. 6. 국물에 우동면을 데쳐낸 뒤 완성한다.

## 영양 정보

- 칼로리: 233.4 kcal
- 탄수화물: 20.7g
- 단백질: 103.8g
- 지방: 10.6g
- 나트륨: 552.7mg
