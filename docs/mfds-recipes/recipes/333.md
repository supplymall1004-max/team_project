# 바다향이 풍부한 전복죽

바다향이 풍부한 전복죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 전복 100g
- 당근 15g
- 건조미역 10g
- 찹쌀 50g
- 흑임자 3g
- 참기름 15g
- 청주 15g
- 흰후추 2g

## 조리 방법

1. 1. 찹쌀은 불려주고 전복은 손질해 내장과 분리해
준 뒤 청주를 넣어서 전복 내장을 믹서에
갈아준다.
2. 2. 전복 몸에 칼집을 내준 후 참기름을 두른 팬에
구워준다.
3. 3. 당근과 미역은 송송 잘라준다.
4. 4. 전복내장을 먼저 참기름에 충분히 볶아준다.
5. 5. 내장을 볶아주다가 불린 찹쌀을 넣고 충분히
볶아준다.
6. 6. 죽이 완성될 때 당근, 미역을 넣고 죽을 끓여
주고 전복은 고명으로 올려준다.

## 영양 정보

- 칼로리: 220.4 kcal
- 탄수화물: 34.1g
- 단백질: 8.7g
- 지방: 5.5g
- 나트륨: 95.8mg
