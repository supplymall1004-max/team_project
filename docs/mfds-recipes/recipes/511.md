# 닭고기또띠아

닭고기또띠아은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 또띠아(50g)
- 닭고기살(100g)
- 느타리버섯(20g)
- 사과(20g)
- 오이(20g)
- 파프리카(20g)
- 애호박(20g)
- 양파(20g)
- 식초(10g)
- - 소스 : 아몬드(10g)
- 우유(20g)
- 머스터드(20g)
- 올리고당(20g)

## 조리 방법

1. 1. 닭고기살은 끓는 물에 데쳐 손으로
찢고, 느타리버섯도 데쳐 손으로 찢어
준비한다.
2. 2. 사과는 씨를 제거하고 약 6cm 길이로
채를 썰고, 오이와 파프리카도 같은
길이로 채를 썰어 놓는다.
3. 3. 애호박은 반달로 썰어 식초에 살짝
절여서 볶고, 양파는 채를 썰어 물에
잠시 담가 놓는다.
4. 4. 아몬드는 우유를 넣어 갈아 준비한다.
5. 5. 머스터드에 올리고당을 넣어 잘 섞어
준다.
6. 6. 또띠아를 팬에 기름 없이 굽고, 그 안에
닭고기 살과 느타리버섯, 오이, 사과,
파프리카와 호박을 넣어 속이 보이도록
말아놓고, ④번과 ⑤번은 섞어서 소스를
만들어 완성한다.

## 영양 정보

- 칼로리: 516.4 kcal
- 탄수화물: 82.8g
- 단백질: 21.6g
- 지방: 11g
- 나트륨: 831.7mg
