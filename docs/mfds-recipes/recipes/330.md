# 열무김치파스타

열무김치파스타은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 파스타면 50g
- 열무김치 100g
- 오이 20g
- 청홍고추 각각 15g
- 노란 파프리카 30g
- 양파 30g
- 당근 25g
- 생들기름 30g
- 통깨 2g
- 청양고추 30g

## 조리 방법

1. 1. 뜨거운 물에 파스타면을 삶아낸 뒤 물기를
빼준다.
2. 2. 송송 썬 청양고추를 생들기름에 볶아주다
면을 볶는다.
3. 3. 볶아준 청양고추에 삶아진 면을 볶아준다.
4. 4. 열무김치는 송송 썰어서 들기름에 살짝
볶아준다.
5. 5. 오이는 돌려깎기 해서 채 썰어주고 청고추,
2가지색 파프리카, 당근도 채 썰어 각각 볶아
준다.
6. 6. 청양고추에 볶아진 파스타면을 볶은 채소와
한 번 더 볶아 완성한다.

## 영양 정보

- 칼로리: 270.4 kcal
- 탄수화물: 42.1g
- 단백질: 6.6g
- 지방: 8.4g
- 나트륨: 371mg
