# 황태탕수육

황태탕수육은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 황태채(100g)
- 밀가루(30g)
- 찹쌀가루(10g)
- 녹말가루(10g)
- 삼색분말(각 10g)
- 달걀(50g)
- 튀김기름(200g)
- - 소스 : 파프리카(30g)
- 양파(30g)
- 당근(30g)
- 오이(30g)
- 파인애플(30g)
- 저염간장(20g)
- 설탕(20g)
- 식초(10g)
- 물녹말(녹말가루 10g
- 물 10g)

## 조리 방법

1. 1. 황태는 가위로 약 3cm 길이로 잘라
스프레이로 물기를 준다.
2. 2. 밀가루를 세 군데로 나눠 각각 찹쌀
가루와 녹말가루, 삼색 가루를 넣어
세 가지의 색을 만든다.
3. 3. 만들어 놓은 밀가루에 달걀을 넣어
골고루 섞는다.
4. 4. 튀김 반죽(③)에 황태(①)를 넣어
섞은 후, 튀김 기름 170℃ 의 온도에서
약 1분 정도 튀겨 기름을 빼 놓는다.
5. 5. 파프리카는 속씨를 제거하고, 양파와
당근, 오이, 파인애플과 함께 썰어 준비
한다.
6. 6. 저염간장에 설탕과 식초를 넣어 골고루
섞은 후 냄비에서 바글바글 끓이다가
물녹말을 넣고, 썰어놓은 채소를 넣어
한 번 더 끓인 후 황태 탕수육(④)에
곁들여 완성한다.

## 영양 정보

- 칼로리: 274.4 kcal
- 탄수화물: 13.2g
- 단백질: 17.1g
- 지방: 17g
- 나트륨: 365.3mg
