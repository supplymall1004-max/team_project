# 머위향 통밀푸실리

머위향 통밀푸실리은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 머윗대(100g)
- 푸실리(50g)
- 올리브오일(20g)
- 마늘(10g)
- 양파(20g)
- 생크림(100g)
- 화이트와인(30g)
- 소금(0.2g)

## 조리 방법

1. 1. 푸실리는 뜨거운 물에 약 10분 정도
삶아 건져 올리브오일을 살짝 바른다.
2. 2. 머윗대는 살짝 데친 후 찬물에 헹궈
껍질을 벗긴다.
3. 3. 껍질 벗긴 머윗대는 푸실리 크기로
자른다.
4. 4. 양파, 마늘은 입자있게 다진다.
5. 5. 팬에 올리브오일을 두르고 다진 마늘과
양파를 볶다가 화이트와인을 넣고
볶는다.
6. 6. 5번에 생크림, 소금을 넣고 믹서에 간
뒤 푸실리와 머윗대를 넣고 한소끔
끓인다.

## 영양 정보

- 칼로리: 275.9 kcal
- 탄수화물: 58g
- 단백질: 9.7g
- 지방: 0.6g
- 나트륨: 202.1mg
