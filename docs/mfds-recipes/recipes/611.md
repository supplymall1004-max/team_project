# 토마토주스를곁들인컵토마토

토마토주스를곁들인컵토마토은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 줄기토마토(3개)
- 토마토(100g)
- 오이(50g)
- 양파(30g)
- 올리브오일(10g)
- 설탕(30g)

## 조리 방법

1. 1. 토마토는 끓는 물에 살짝 데친다.
2. 2. 데친 토마토를 믹서에 곱게 갈고
설탕을 넣고 섞는다.
3. 3. 줄기토마토는 윗부분을 잘라내고 속을
파낸다.
4. 4. 오이와 양파는 입자있게 썬다.
5. 5. 팬에 올리브오일을 두르고 썰어 놓은
오이와 양파를 살짝 볶는다.
6. 6. 을 파낸 줄기토마토에 볶아 놓은
야채와 ?번을 넣는다.

## 영양 정보

- 칼로리: 53.7 kcal
- 탄수화물: 12.2g
- 단백질: 1g
- 지방: 0.1g
- 나트륨: 226.9mg
