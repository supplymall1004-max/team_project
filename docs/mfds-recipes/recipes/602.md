# 세 가지 미니 에피타이저

세 가지 미니 에피타이저은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 연어(훈제연어
- 120g)
- 새우(3마리)
- 애호박(1/2개)
- 사과(1/4개)
- 오렌지(1/4개)
- 레몬(1/4개)
- 석류주스(200g)
- 판젤라틴(1장)

## 조리 방법

1. 1. 훈제 연어는 슬라이스로 썰어서
굽는다.
2. 2. 사과는 채썬다.
3. 3. 판젤라티은 찬물에 담궈 불린 후
중탕해서 녹인다.
4. 4. 석류주스를 끓인 뒤 중탕한 젤라틴을
넣고 굳혀 접시에 담는다.
5. 5. 애호박은 껍질을 벗겨 끓는 물에
소금을 넣고 살짝 데치고, 새우는 살짝
데쳐 껍질을 벗겨 포를 뜬 뒤 다시
굽는다.
6. 6. 석류주스 젤리가 담긴 접시에 새우와
애호박을 담고 소스를 올린 뒤 연어와
사과, 호박을 올려 담는다.

## 영양 정보

- 칼로리: 230.2 kcal
- 탄수화물: 18.7g
- 단백질: 20.7g
- 지방: 8g
- 나트륨: 168.8mg
