# 꽃밥

꽃밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 라이스페퍼(50g)
- 당근(20g)
- 달걀(30g)
- 시금치(20g)
- 소금(0.5g)
- 돼지고기(50g)
- 마늘(10g)
- 대파(10g)
- 맛술(10g)
- 저염간장(20g)
- 설탕(10g)
- 통깨(5g)
- 참기름(5g)
- 후춧가루(0.02g)
- 고구마(30g)
- 두부(20g)
- 토마토(20g)
- 양파(20g)
- 견과류(20g)

## 조리 방법

1. 1. 당근은 채를 곱게 썰어 준비한다.
2. 2. 달걀은 노란자와 흰자로 나눠 각각
지단을 부쳐 약 5cm 길이로 채를
썰어놓는다.
3. 3. 시금치는 끓는 물에 소금을 넣고 파랗게
데쳐 물기를 제거하고 무쳐놓는다.
4. 4. 돼지고기는 마늘, 파, 설탕, 맛술, 간장,
후춧가루, 통깨, 참기름을 넣고 30분간
숙성시킨 뒤 팬에 볶아서 준비한다.
5. 5. 고구마는 삶아 으깨고, 두부는 소창에
물기를 짜고, 토마토와 양파, 견과류는
입자있게 다져섞는다.
6. 6. 라이스 페이퍼를 따뜻한 물에 담궜
다가 바로 꺼내어 으깬 고구마(⑤)를
먼저 깔고, 그 위에 준비한 재료(②,
③, ④)를 올려 돌돌 말은 뒤 먹기 좋은
크기로 썰어 완성한다.

## 영양 정보

- 칼로리: 480.7 kcal
- 탄수화물: 56.8g
- 단백질: 23.2g
- 지방: 173.9g
- 나트륨: 272.6mg
