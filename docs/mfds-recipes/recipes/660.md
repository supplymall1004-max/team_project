# 삼색채소 냉파스타

삼색채소 냉파스타은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기(가슴살
- 100g)
- 파스타(100g)
- 쥬키니호박(50g)
- 사과(1/4개)
- 비트(50g)
- 레몬(1/4개)
- 마늘(20g)
- 시금치(30g)
- 소금(0.2g)
- 올리브오일(20g)
- 꿀(20g)

## 조리 방법

1. 1. 끓는 물에 파스타를 넣고 약 10분 정도
삶아 건져 올리브오일을 살짝 바른다.
2. 2. 쥬키니 호박은 채썰어 소금에 절인 뒤
볶는다.
3. 3. 닭가슴살은 마늘을 넣고 삶은 뒤 먹기
좋게 찢는다.
4. 4. 사과와 비트, 레몬을 믹서에 갈아
소스를 만든다.
5. 5. 소스에 꿀을 넣어 맛을 내고 차갑게
보관한다.
6. 6. 팬에 삶아 놓은 파스타를 넣고 볶다가
닭살과 호박채, 시금치를 넣고 볶은 뒤
접시에 담고 차갑게 보관한 소스를
올린다.

## 영양 정보

- 칼로리: 323.2 kcal
- 탄수화물: 51.1g
- 단백질: 13.6g
- 지방: 7.2g
- 나트륨: 50.9mg
