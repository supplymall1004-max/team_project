# 묵은지가지말이

묵은지가지말이은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 돼지고기(80g)
- 가지(30g)
- 김치(40g)
- 청피망(15g)
- 붉은피망(15g)
- 양파(40g)
- 생강(2g)
- 마늘(10g)
- 올리브유(5g)
- 레몬(50g)
- 레드와인(120g)
- 올리고당(15g)
- 고기 삶는 재료 양파(15g)
- 월계수잎(2g)
- 정향(1g)
- 후춧가루(2g)
- 마늘(3g)
- 생강(5g)
- 물(300g)

## 조리 방법

1. 1. 냄비에 고기 삶는 재료를 넣고
끓으면 돼지고기를 넣어 익힌다.
2. 2. 가지는 길이로 납작하게 썰고,
김치는 씻어 5㎝ 길이로 채 썰고,
피망, 양파도 같은 길이로 채 썰고,
생강과 마늘은 편으로 썰어 준비한다.
3. 3. 팬에 올리브유를 약간 두르고
가지를 살짝 굽는다.
4. 4. 김치, 피망을 가지에 넣고 돌돌 만다.
5. 5. 팬에 올리브유, 마늘, 생강,
삶은 고기를 넣고 육즙이 나오지
않도록 고기 겉면을 익힌 뒤 채 썬
양파, 마늘, 레몬, 레드와인,
올리고당을 넣고 조린다.
6. 6. 고기에 간이 배면 건져내 먹기
좋은 크기로 썰어 마무리한다.

## 영양 정보

- 칼로리: 312.2 kcal
- 탄수화물: 7.35g
- 단백질: 9.66g
- 지방: 17g
- 나트륨: 141mg
