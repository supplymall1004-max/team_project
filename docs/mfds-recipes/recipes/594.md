# 미역소스와 타르타르

미역소스와 타르타르은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 관자(2개)
- 새우(3마리)
- 미역(20g)
- 새송이버섯(1개)
- 시금치(30g)
- 토마토(50g)
- 마늘(10g)
- 블랙올리브(3알)
- 메추리알(5개)
- 레몬(1/4개)
- 두유(100g)
- 생크림(50g)
- 우유(200g)

## 조리 방법

1. 1. 미역은 찬물에 충분히 불린다.
2. 2. 새우는 껍질을 벗겨 관자 한 개와 다져
완자를 만든다.
3. 3. 나머지 관자 한 개는 썰어 팬에 굽는다
4. 4. 버섯, 토마토, 블랙올리브는 곱게
다진다.
5. 5. 시금치와 미역은 끓는 물에 데쳐 잘게
썬다.
6. 6. 마늘을 다져 우유와 함께 끓이다가
썰어 놓은 미역과 시금치, 새우완자를
넣고 한소끔 끓인다.

## 영양 정보

- 칼로리: 202.3 kcal
- 탄수화물: 16.1g
- 단백질: 13.1g
- 지방: 9.5g
- 나트륨: 306.6mg
