# 해산물리조또

해산물리조또은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 보리쌀(50g)
- 당근(20g)
- 양파(20g)
- 마늘(10g)
- 대파(10g)
- 버터(10g)
- 오징어(50g)
- 새우(50g)
- 시금치(20g)
- 소금(0.2g)
- 생크림(20g)
- 우유(100g)
- 치즈(50g)
- 파슬리가루(0.3g)
- - 육수(100g) : 조개(30g)
- 다시마(10g)

## 조리 방법

1. 1. 보리는 깨끗이 씻어 물에 약 30분 이상
불려 놓고, 당근, 양파, 마늘, 대파는
입자있게 다져 준비한다.
2. 2. 시금치는 끓는 물에 소금을 넣고 데친 후
생크림을 넣고 함께 갈아서 준비한다.
3. 3. 오징어는 껍질을 벗겨 안쪽에 칼집을
내어 새우와 함께 데치고, 조개는 해감
후 다시마를 넣고 삶은 뒤 소창에 걸러
육수를 만들어 놓는다.
4. 4. 냄비에 버터를 넣고, 다진 마늘과 양파
당근을 볶다가 불린 보리를 넣어 다시
볶는다.
5. 5. 볶아지는 냄비에 조개 삶은 육수를
넣어 끓인다.
6. 6. 끓여지는 냄비에 데친 조개와 해물,
치즈, 우유를 넣어 어우러지게 저어
가며 졸이듯 끓이다가 마지막에
시금치와 생크림 갈은 것(②)을
넣고 살짝 끓인 후 파슬리 가루를
살짝 뿌려 완성한다.

## 영양 정보

- 칼로리: 306.8 kcal
- 탄수화물: 47.7g
- 단백질: 16.7g
- 지방: 5.4g
- 나트륨: 237.7mg
