# 코다리맑은탕

코다리맑은탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 코다리 200g
- 무 60g
- 감자 60g
- 양파 15g
- 청양고추 30g
- 미나리 25g
- 백일송이버섯 25g
- 참기름 15g
- 국물 : 다진마늘 15g
- 저염간장 15g
- 흰후추 1g

## 조리 방법

1. 1. 코다리는 먹기 좋게 잘라서 물에 헹궈 물기를
빼준다.
2. 2. 감자와 무, 백일송이버섯, 미나리는 먹기 좋게
썰어준다.
3. 3. 양파, 청양고추도 먹기 좋게 썰어준다.
4. 4. 냄비에 참기름을 넣어 코다리를 살짝 볶아준다.
5. 5. 코다리를 볶은 냄비에 무, 감자, 국물 재료를
넣고 충분히 끓인다.
6. 6. 끓고 있는 국에 양파, 청양고추, 백일송이
버섯을 올려 완성한다.

## 영양 정보

- 칼로리: 162.3 kcal
- 탄수화물: 7.7g
- 단백질: 24.2g
- 지방: 3.9g
- 나트륨: 110.8mg
