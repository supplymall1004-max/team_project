# 오렌지삼겹찜

오렌지삼겹찜은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 삼겹살(100g)
- 로즈마리(5g)
- 고구마(30g)
- 생크림(10g)
- 감자(20g)
- 튀김기름(200g)
- 브로컬리(30g)
- 올리브오일(10g)
- - 소스 : 오렌지(50g)
- 설탕(10g)
- 정종(10g)
- 저염소금(5g)
- 와사비(5g)

## 조리 방법

1. 1. 삼겹살은 로즈마리를 뿌려
숙성시킨다.
2. 2. 고구마는 삶아 준비하고, 감자는 웨지
감자형으로 썰어 익힌 후 기름에 튀겨
놓는다.
3. 3. 삶아진 고구마를 으깨어 생크림을 넣어
골고루 섞는다.
4. 4. 오렌지는 즙을 짜 놓는다.
5. 5. 숙성된 삼겹살은 오븐에 굽고, 브로컬리는
살짝 데쳐 올리브오일에 볶아놓는다.
6. 6. 팬에 오렌지즙과 설탕, 정종, 소금을
넣고 바글바글 졸인 후 와사비를 넣어
소스를 만들고, 오븐에 구어진 삼겹살을
잘라 접시에 담고 그 위에 소스를
살짝 올리고, 한쪽에는 감자튀김(②),
고구마무스(③), 브로컬리(⑤)를
곁들여 완성한다.

## 영양 정보

- 칼로리: 638.1 kcal
- 탄수화물: 22g
- 단백질: 26.7g
- 지방: 49.2g
- 나트륨: 206mg
