# 청국장소스 연어 스테이크

청국장소스 연어 스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- ●주재료 : 마늘종 60g(10개)
- 연어 200g
- 식용유 5g(1/2작은술)
- ●소스 : 검정깨 5g(1작은술)
- 청국장가루 5g(1작은술)
- 마요네즈 20g(4작은술)
- ●장식 : 대추 5g(1개)

## 조리 방법

1. 1. 검정깨, 마요네즈, 청국장 가루를 섞어 소스를 만든다.
2. 2. 팬에 식용유를 둘러 연어와 마늘종을 굽는다.
3. 3. 마늘종을 접시에 깔고 연어를 올린 뒤 소스를 뿌리고 채썬 대추채를 얹어 장식한다.
대추채를 얹어 장식한다.

## 영양 정보

- 칼로리: 209.3 kcal
- 탄수화물: 3.9g
- 단백질: 17.6g
- 지방: 13.7g
- 나트륨: 56.6mg
