# 호박잎다슬기된장국

호박잎다슬기된장국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 조리 방법

1. 1. 물 1컵 반에 국멸치와 건다시마를 넣고 멸치다시마국물 1컵을 낸다.
2. 2. 호박잎은 줄기를 꺾으며 잡아당겨 실 같은 껍질을 벗기고 씻는다.
3. 3. 멸치다시마국물 1컵에 된장을 풀고 씻은 다슬기살을 함께 넣어 중불에서 5분정도 국물을 우려낸다.
4. 4. 국물낸 다슬기는 건져 달걀물과 함께 섞는다.
5. 5. 부추는 1cm 가량의 크기로 썰고 대파는 어슷썰기 한다.
6. 6. 국물이 끓어 오르면 호박잎과 부추, 파, 다진 마늘을 넣고 마지막으로 달걀물을 입힌 다슬기를 살포시 넣는다.

## 영양 정보

- 칼로리: 86.7 kcal
- 탄수화물: 6.2g
- 단백질: 9.5g
- 지방: 5.9g
- 나트륨: 577.2mg
