# 세가지샐러드

세가지샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연두부(100g)
- 닭고기살(50g)
- 통후추(1g)
- 단호박(20g)
- 바나나(20g)
- 파프리카(20g)
- 토마토(20g)
- 어린잎(10g)
- - 소스 : 마(20g)
- 생크림(20g)
- 요거트(20g)
- 식초(5g)
- 올리브오일(10g)
- 설탕(10g)
- 흑임자(5g)
- 잣(10g)

## 조리 방법

1. 1. 연두부는 동글게 썰어 준비한다.
2. 2. 호박은 갈라 씨를 제거 하고 껍질을
벗겨 쪄서 작은 깍둑썰기로 썰어 준비
한다.
3. 3. 바나나와 파프리카 토마토는 호박과
같은 크기의 작은 깍두기로 썰어 놓는다.
4. 4. 닭고기살은 통후추를 넣고 약 20분 정도
삶아 건져 작은 깍두기로 썰어 준비한다.
5. 5. 마와 잣, 흑임자를 갈아 놓은 뒤 생크림,
요거트, 식초, 올리브오일, 설탕을
섞어 샐러드 소스를 만들어 놓는다.
6. 6. 썰어 놓은 연두부 위에 호박 , 닭고기,
토마토, 파프리카를 각각 올린 뒤 어린
잎을 올리고 소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 202.4 kcal
- 탄수화물: 12.6g
- 단백질: 16.1g
- 지방: 9.8g
- 나트륨: 209.2mg
