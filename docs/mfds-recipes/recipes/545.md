# 전복죽

전복죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 쌀(50g)
- 전복(50g)
- 새우살(20g)
- 양송이(20g)
- 당근(10g)
- 참기름(10g)
- 잣(10g)
- - 소스 : 저염간장(5g)

## 조리 방법

1. 1. 깨끗이 씻은 쌀에 물을 2배 정도 넣고
약 30분 정도 충분히 불려 준비한다.
2. 2. 전복은 수저를 이용하여 껍질에서
떼어내어 내장을 분리해 놓는다.
3. 3. 내장을 제거한 전복은 깨끗이 씻어,
썰어 놓는다.
4. 4. 새우살은 꼬리를 제거하고 입자있게
썰어 놓는다.
5. 5. 양송이는 껍질을 벗겨 입자있게
썰고, 당근도 양송이 크기에 맞게
썰어 놓는다.
6. 6. 냄비에 참기름을 두르고, 전복을 은근히
볶다가, 다시 불린 쌀을 넣고 볶은 뒤
물을 넣고 약 20분 정도 끓인다. 죽이
끓으면 새우와 양송이, 당근을 넣고
은근히 저어가면서 다시 20분 정도
더 끓여, 죽이 어우러지면 잣을 올리고
저염간장을 함께 제공한다.

## 영양 정보

- 칼로리: 225.9 kcal
- 탄수화물: 36g
- 단백질: 17.6g
- 지방: 1.3g
- 나트륨: 109.9mg
