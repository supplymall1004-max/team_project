# 두부스테이크

두부스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 두부(200g)
- 양파(30g)
- 새송이버섯(1개)
- 달걀(1개)
- 케일(1장)
- 당근(20g)
- 토마토(100g)
- 칵테일새우(30g)
- 레몬(1/4개)
- 파인애플(50g)
- 마늘(10g)
- 올리브오일(20g)
- 소금(0.2g)

## 조리 방법

1. 1. 새우, 양파, 당근, 새송이버섯은
입자있게 다진다.
2. 2. 두부는 곱게 으깬다.
3. 3. ?번과 으깬 두부, 소금, 달걀을 골고루
섞는다.
4. 4. 파인애플을 넣어 곱게 갈아
올리브오일을 섞어 소스를 만든다.
5. 5. 으깬 두부를 원형으로 만든다.
6. 6. 팬에 기름을 두르고 두부와 토마토를
굽고, 접시에 케일을 깔고 그 위에
두부와 토마토를 올린 뒤
파인애플소스를 올린다.

## 영양 정보

- 칼로리: 308.8 kcal
- 탄수화물: 8.7g
- 단백질: 19g
- 지방: 22g
- 나트륨: 60.7mg
