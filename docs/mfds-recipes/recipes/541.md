# 닭가슴살현미스테이크

닭가슴살현미스테이크은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 닭가슴살(150g)
- 정종(10g)
- 카레가루(20g)
- 현미쌀(50g)
- 튀김기름(200g)
- - 소스 : 토마토소스(50g)
- 생크림(20g)
- - 곁들임채소 : 적양배추(20g)
- 양배추(20g)

## 조리 방법

1. 1. 닭가슴살에 칼을 1/3 쯤 넣고 양쪽으로
포를 뜨고, 정종을 살짝 발라놓는다.
2. 2. 포뜬 닭가슴살 위에 카레가루를 솔솔
뿌려 숙성을 시킨다.
3. 3. 현미쌀은 물에 충분히 불려, 물기를
빼 놓고, 양배추와 적 양배추는 곱게
채를 썰어 각각 찬물에 담그어 싱싱
해 지면 건져 물기를 빼놓는다.
4. 4. 냄비에 토마토소스를 끓이다가
생크림을 넣고 살짝 한 번 더
끓여놓는다.
5. 5. 튀김기름에 온도가 약 170℃ 정도가
되면, 불려 놓은 현미쌀의 물기를
제거한 뒤 튀겨 기름을 제거해 놓는다.
6. 6. 숙성된 닭가슴살(①)에 튀긴 현미쌀
(⑤)을 앞뒤로 골고루 바르고 다시
170℃ 온도의 튀김 기름에 약 3~5분
정도 튀겨내고, 양배추와 적채를 깔고
그 위에 튀겨진 닭가슴살을 올리고,
토마토소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 400 kcal
- 탄수화물: 33.2g
- 단백질: 34.8g
- 지방: 14.2g
- 나트륨: 194.3mg
