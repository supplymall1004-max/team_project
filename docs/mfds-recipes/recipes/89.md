# 시금치 우유 소스와 그린매쉬드포테이토

시금치 우유 소스와 그린매쉬드포테이토은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- ●그린매쉬드포테이토 : 감자 80g(1/2개)
- 시금치우유 소스 5g(1작은술)
- 아몬드 2g(1알)
- 설탕 2g(1/3작은술)
- 크랜베리 3g
- 치커리 약간
- ●시금치우유 소스 : 시금치 10g
- 우유 10g(2작은술)

## 조리 방법

1. 1. 시금치는 끓는 소금물에 데쳐 찬물에 헹구어 물기를 짜고 우유를 넣고 블렌더에 곱게 갈아 체에 거른다.
2. 2. 껍질 벗긴 감자는 찜기에 넣어 20분 정도 삶고 꺼내서 으깨고, 아몬드는 잘게 다지며, 치커리는 곱게 다진다.
3. 3. 으깬 감자, 다진 치커리, 시금치 우유소스, 설탕을 넣고 섞어 접시에 담고 아몬드, 크랜베리를 올리고 시금치우유 소스를 곁들인다.

## 영양 정보

- 칼로리: 155 kcal
- 탄수화물: 32g
- 단백질: 5g
- 지방: 1g
- 나트륨: 24mg
