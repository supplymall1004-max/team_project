# 황태미역 곤약스프

황태미역 곤약스프은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 황태채(100g)
- 곤약(100g)
- 건미역(20g)
- 들깨가루(20g)
- 두부(50g)
- 된장(10g)
- 액젓(10g)
- 표고버섯(1개)
- 마늘(10g)
- 참기름(5g)

## 조리 방법

1. 1. 된장에 두부를 넣고 골고루 섞는다.
2. 2. 표고버섯과 마늘은 함께 갈아둔다.
3. 3. 미역은 충분히 불린다.
4. 4. 냄비에 황태채와 갈아둔 표고버섯,
마늘을 넣고 은근히 끓여 육수를
만든다.
5. 5. 곤약은 납작납작하게 썰어 끓는 물에
데친다.
6. 6. 육수에서 황태를 건져 두부된장과 함께
갈아 육수에 붓고, 미역, 곤약,
들깨가루를 넣고 약 20분 정도 끓인 후
액젓으로 간을 한다.

## 영양 정보

- 칼로리: 124.9 kcal
- 탄수화물: 3.2g
- 단백질: 14.3g
- 지방: 6.1g
- 나트륨: 790.3mg
