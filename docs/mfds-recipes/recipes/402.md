# 닭가슴살스튜

닭가슴살스튜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭가슴살 200g
- 당근 30g
- 양파 30g
- 브로콜리 30g
- 감자 40g
- 방울토마토 100g
- 우유 100g
- 올리브유 15g
- 소스 : 토마토페이스트 100g
- 다진마늘 15g
- 밀가루 7g
- 흰후추 3g

## 조리 방법

1. 1. 닭가슴살은 먹기 좋은 크기로 잘라서 우유에
재워준 뒤 뜨거운 물에 데쳐준다.
2. 2. 방울토마토는 반으로 자르고, 양파는 먹기 좋은
크기로 잘라준다.
3. 3. 감자, 당근, 브로콜리는 먹기 좋은 크기로
잘라 뜨거운 물에 순서대로 데쳐준다.
4. 4. 모든 재료를 올리브유에 볶아준다.
5. 5. 토마토페이스트와 닭가슴살을 데쳐낸 물을
섞어 끓여준다.
6. 6. 끓고 있는 국물 50g에 밀가루를 섞어 농도를
맞추어 준다.

## 영양 정보

- 칼로리: 148.7 kcal
- 탄수화물: 10.6g
- 단백질: 19.4g
- 지방: 3.2g
- 나트륨: 223.6mg
