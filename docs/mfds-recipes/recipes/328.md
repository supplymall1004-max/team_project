# 돼지수육맑은전골

돼지수육맑은전골은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 통삼겹 250g
- 된장 10g
- 통후추 2g
- 다진대파 5g
- 팽이버섯 10g
- 당근 20g
- 배추 30g
- 청경채 20g
- 애호박 25g
- 숙주 25g
- 청양고추 20g
- 육수 : 천연다시다 15g
- 물 300g
- 다진마늘 5g
- 곁들이장 : 어간장 15g
- 겨자 5g
- 식초 15g
- 깨 5g

## 조리 방법

1. 1. 당근, 청경채, 배추, 애호박, 팽이버섯은 먹기
좋은 크기로 잘라준다.
2. 2. 흑돼지고기, 저염된장, 후추를 넣어서 불순물을
제거하면서 고기를 삶아준다.
3. 3. 천연다시다로 육수를 만들어 체에 걸러준다.
4. 4. 손질한 야채들과 청양고추를 잘라 함께 육수에
데쳐 국물에 매운 맛을 낸다.
5. 5. 곁들이장을 만들어 준다.
6. 6. 냄비에 데쳐준 채소와 국물을 붓고 고기를
먹기좋게 썰어 소스랑 완성한다.

## 영양 정보

- 칼로리: 120.3 kcal
- 탄수화물: 9.6g
- 단백질: 0.6g
- 지방: 8.8g
- 나트륨: 147.7mg
