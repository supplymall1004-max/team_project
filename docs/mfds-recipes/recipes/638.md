# 초계탕과 사색곤약

초계탕과 사색곤약은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기(1마리)
- 가시오가피(10g)
- 대파(20g)
- 다시마(10g)
- 건새우(20g)
- 실곤약(100g)
- 비트(30g)
- 치자가루(10g)
- 마늘(20g)
- 소금(0.3g)
- 후춧가루(0.01g)
- 양파(50g)
- 오이(50g)
- 겨자가루(10g)
- 식초(20g)
- 설탕(20g)

## 조리 방법

1. 1. 닭을 깨끗이 손질하고, 부위별로
자른다.
2. 2. 냄비에 가시오가피, 대파, 다시마,
마늘, 건새우를 넣고 은근히 끓인 후
닭을 넣고 약 20분 정도 삶아 건진 뒤
살을 찢어둔다.
3. 3. 양파, 오이는 채썰고 식초, 설탕,
소금을 넣고 초절임한다
4. 4. 겨자가루에 30℃ 정도의 물을 넣고
골고루 섞어 발효시키고, 설탕, 식초,
소금을 넣고 겨자소스를 만든다
5. 5. 흰곤약과 비트, 치자가루를 섞은 곤약,
채썬 다시마를 접시에 담는다.
6. 6. 양파, 오이 초절임과 닭살을 한쪽에
담아 먹기 직전에 겨자소스를 올린다.

## 영양 정보

- 칼로리: 100.1 kcal
- 탄수화물: 7.7g
- 단백질: 15.4g
- 지방: 0.9g
- 나트륨: 54.4mg
