# 토마토콩조림

토마토콩조림은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 콩(대두
- 100g)
- 양파(50g)
- 양송이버섯(30g)
- 토마토(200g)
- 양념 설탕(4g)
- 후춧가루(2g)
- 파슬리가루(2g)

## 조리 방법

1. 1. 콩을 충분히 불린 후 냄비에 물과
함께 넣고 삶는다.
2. 2. 양파와 버섯은 곱게 다진다.
3. 3. 토마토를 끓는 물에 데쳐 껍질을
벗긴 뒤 으깨 팬에서 끓인다.
4. 4. 양파와 버섯을 넣고 끓여 숨이
죽으면 설탕과 후춧가루로 간을
한다.
5. 5. 삶은 콩을 넣고 조린 뒤
파슬리가루를 뿌려 마무리한다.

## 영양 정보

- 칼로리: 452.5 kcal
- 탄수화물: 16.1g
- 단백질: 12.9g
- 지방: 4.68g
- 나트륨: 26.8mg
