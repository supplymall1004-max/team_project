# 미역볶음밥

미역볶음밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 재료 쌀(80g)
- 완두콩(10g)
- 닭고기(100g)
- 대추(10g)
- 수삼(20g)
- 양파(20g)
- 당근(10g)
- 양념 강황가루(3g)
- 맛술(15g)

## 조리 방법

1. 1. 쌀과 다시마를 넣고 쌀과 동량의
물을 부어 밥을 고슬고슬하게
짓는다.
2. 2. 양파는 굵게 다지고, 쪽파와 고추는
송송 썰고, 오징어와 주꾸미는 먹기
좋은 크기로 썬다.
3. 3. 달군 팬에 들기름을 두르고 다진
마늘을 볶아 향을 낸다.
4. 4. 미역을 넣고 볶다가 손질해둔
오징어, 주꾸미, 새우살, 양파를 넣어
볶는다.
5. 5. 밥을 넣고 소금을 넣어 볶다가
들깻가루를 넣어 볶는다.
6. 6. 송송 썬 쪽파와 청양고추, 붉은
고추를 넣어 조금 더 볶아
마무리한다.

## 영양 정보

- 칼로리: 515.2 kcal
- 탄수화물: 25.5g
- 단백질: 5.85g
- 지방: 5.43g
- 나트륨: 488.5mg
