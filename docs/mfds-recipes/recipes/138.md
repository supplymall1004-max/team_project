# 부대된장찌개

부대된장찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 다시마 1g
- 두부 10g
- 떡국 떡 10g
- 스팸(마일드) 10g
- 다진 마늘 5g
- 무 20g
- 김치 15g
- 소시지 10g(1/2개)
- 우민찌 5g(1작은술)
- 양파 5g
- 저염된장 15g(1큰술)
- 베이컨 5g
- 대파 5g
- 청양고추 5g
- 홍고추 1g

## 조리 방법

1. 1. 김치, 베이컨, 스팸, 소시지, 양파, 두부, 무는 두께 0.5cm로 썬다.
2. 2. 다시마와 물을 끓여 다시마물을 만든다.
3. 3. 냄비에 소시지, 베이컨, 두부, 스팸, 무, 우민찌, 김치, 다시마물 300g을 넣어 끓인 후 저염된장, 양파, 대파, 다진 마늘, 떡국 떡을 넣고 재료가 다 익으면 홍고추와 청양고추를 넣어 완성한다.

## 영양 정보

- 칼로리: 72.6 kcal
- 탄수화물: 4.2g
- 단백질: 5.3g
- 지방: 3.9g
- 나트륨: 191.9mg
