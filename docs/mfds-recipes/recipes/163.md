# 꼬막 달래 된장 무침

꼬막 달래 된장 무침은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- ●주재료 : 꼬막 1000g
- 달래 100g(2단)
- 깻잎 10g(10장)
- 파프리카 100g(1개)
- ●소스 : 된장 15g(1큰술)
- 고추장 5g(1작은술)
- 사과즙 300g(1/2컵)
- 참기름 20g(1⅓큰술)
- 깨 15g(1큰술)
- 다진 마늘 10g(2작은술)

## 조리 방법

1. 1. 된장, 고추장, 사과즙, 깨, 다진 마늘, 참기름을 넣어 소스를 만든다.
2. 2. 꼬막을 30분간 소금물에 해감시킨 후 끓는 물에 10분 데친다.
3. 3. 달래, 깻잎, 파프리카를 5cm로 썰고 꼬막과 달래, 깻잎, 파프리카에 소스를 넣고 버무린다.

## 영양 정보

- 칼로리: 70.1 kcal
- 탄수화물: 6.9g
- 단백질: 5.1g
- 지방: 2.4g
- 나트륨: 204mg
