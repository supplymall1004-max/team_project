# 떡갈비와 미니잡곡밥

떡갈비와 미니잡곡밥은(는) 밥 요리로, 굽기 조리법을 사용합니다.

## 재료

- 소고기(120g)
- 돼지고기(120g)
- 떡볶이떡(100g)
- 흑미(20g)
- 현미(20g)
- 백미(20g)
- 조(10g)
- 애호박(1/4개)
- 당근(20g)
- 토마토(150g)
- 양송이버섯(20g)
- 양파(30g)
- 마늘(10g)
- 달걀(1개)
- 시금치(30g)
- 설탕(20g)
- 소금(0.2g)
- 저염간장(20g)
- 참기름(5g)

## 조리 방법

1. 1. 떡볶이떡은 잘게 다진다.
2. 2. 현미, 흑미, 백미 및 조는 깨끗이 씻어
물에 불린다.
3. 3. 소고기와 돼지고기에 소금, 다진 마늘,
설탕 및 참기름을 넣고 양념한다
4. 4. 당근과 호박은 잘게 다진다
5. 5. 달걀은 황백으로 나눠 지단을 부친다.
6. 6. 불린 쌀로 밥을 짓고, 밥과 3번과 4번을
섞어 미니 떡갈비를 만들어 팬에
기름을 두르고 익힌 뒤 황백지단으로
장식한다.

## 영양 정보

- 칼로리: 357.4 kcal
- 탄수화물: 29.3g
- 단백질: 32.7g
- 지방: 12.2g
- 나트륨: 408.9mg
