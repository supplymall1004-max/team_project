# 냉머위들깨나물

냉머위들깨나물은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 주재료 : 머위대 70g
- 들기름 2g
- 쪽파 5g
- 물 100g - 양념 : 거피들깨가루 6g
- 국간장 3g
- 다진 파 2.5g
- 다진 마늘 1g
- 소금 0.5g

## 조리 방법

1. 1. 머위대를 잘라 데쳐서 껍질을 벗긴다.
2. 2. 데친 머위대를 찬물에 1~2 시간 담가 쓴맛을 제거한다.
3. 3. 양념을 넣어 머위대를 무친다.
4. 4. 팬에 들기름을 두르고 양념한 머위대를 볶는다.
5. 5. 물 반컵을 넣고 국물이 자박하도록 푹 끓여낸다.
6. 6. 완성된 머위나물을 식혀서 냉장고에 차게 보관하였다가 먹는다.

## 영양 정보

- 칼로리: 73.1 kcal
- 탄수화물: 4.9g
- 단백질: 3.8g
- 지방: 4.7g
- 나트륨: 395.8mg
