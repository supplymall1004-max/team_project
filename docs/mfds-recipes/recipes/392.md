# 토마토소고기찜

토마토소고기찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 소고기우둔살 200g
- 샐러리 40g
- 당근 30g
- 양파 50g
- 화이트와인 25g
- 홀토마토 100g
- 버터 20g
- 레몬 25g
- 흰후추 1g
- 파슬리가루 2g
- 함초소금 1g

## 조리 방법

1. 1. 당근, 양파, 샐러리는 사방 2cm 정도로
잘라준다.
2. 2. 레몬은 껍질을 아주 곱게 채 썰어준다.
3. 3. 소고기는 사방 3cm 정도로 자르고 흰 후추를
뿌려준다.
4. 4. 홀토마토는 먹기 좋게 부수어 준 후 볶아준다.
5. 5. 팬에 버터를 넣고 소고기부터 볶기 시작하다
채소를 넣고 볶아준다. 이 때 두꺼운 채소부터
볶아주는 것이 고기랑 익는 시간이 같아진다.
6. 6. 마지막에 다져 놓은 홀토마토를 넣고 함초소금
으로 간을 한 후 레몬껍질을 올려서 마무리
한다.

## 영양 정보

- 칼로리: 302.9 kcal
- 탄수화물: 11.2g
- 단백질: 26.6g
- 지방: 16.8g
- 나트륨: 289.3mg
