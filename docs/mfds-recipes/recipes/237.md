# 삼색꼬치구이

삼색꼬치구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 돼지고기(목살
- 120g)
- 파인애플(30g)
- 브로콜리(20g)
- 방울토마토(40g)
- 배사과소스 배(30g)
- 사과(50g)
- 레몬즙(5g)
- 꿀(5g)

## 조리 방법

1. 1. 껍질을 벗긴 배, 사과를 믹서기에
갈아 레몬즙과 꿀을 넣고
배사과소스를 만든다.
2. 2. 돼지고기와 파인애플은 깍둑 썬다.
3. 3. 브로콜리는 약 2~3cm 크기로 자른
후 끓는 물에 살짝 데친다.
4. 4. 식용유(5g)를 두른 팬에 돼지고기를
한 번 먼저 굽는다.
5. 5. 꼬치에 방울토마토, 구운 돼지고기,
파인애플, 브로콜리 순서로 끼운 후
굽는다.
6. 6. 노릇하게 구운 꼬치구이에
배사과소스를 얹어 마무리한다.

## 영양 정보

- 칼로리: 255.4 kcal
- 탄수화물: 9.12g
- 단백질: 11.9g
- 지방: 6.14g
- 나트륨: 57mg
