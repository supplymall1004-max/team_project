# 버섯리조또

버섯리조또은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 양파(20g)
- 양송이(20g)
- 표고버섯(20g)
- 느타리버섯(20g)
- 버터(5g)
- 불린쌀(150g)
- 생크림(20g)
- 소금(0.05g)
- - 육수 : 건새우(10g)
- 건멸치(20g)
- 마늘(10g)
- 다시마(5g)
- 물(500ml)

## 조리 방법

1. 1. 양파와 양송이, 표고버섯, 느타리버섯은
작게 썰어 준비해 놓는다.
2. 2. 멸치와 새우, 마늘을 냄비에 담고,
물을 약 500㎖ 정도 붓고 끓여 육수를
만든다.
3. 3. 냄비에 버터를 넣고, 양파와 버섯을
넣어 볶는다.
4. 4. ③번의 냄비에 불린 쌀을 넣고 은근히
볶는다.
5. 5. 볶아지는 냄비에 만들어 놓은 육수를
조금 넣는다.
6. 6. 육수를 넣고 다시 은근히 볶으면서
뚜껑을 덮어 충분히 익힌 후, 마지막에
생크림과 소금을 넣고 완성한다.

## 영양 정보

- 칼로리: 213.7 kcal
- 탄수화물: 40.3g
- 단백질: 5.2g
- 지방: 3.5g
- 나트륨: 66.5mg
