# 간편콩국수

간편콩국수은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 메밀면 80g
- 백일송이 25g
- 마늘기름 15g
- 통깨 2g
- 볶은소금 1g
- 고명 : 애호박 50g
- 양파 50g
- 청고추 15g
- 홍고추 15g
- 국물 : 두유 200g
- 두부 60g

## 조리 방법

1. 1. 두유와 두부를 믹서에 생수 100g을 넣고
갈아준다.
2. 2. 애호박은 돌려깎기 해서 채 썰어주고 양파,
홍고추, 청고추도 가늘게 채 썰어준다.
3. 3. 백일송이는 가늘게 찢어 볶아준다.
4. 4. 애호박, 당근, 청고추, 홍고추를 각각 볶아준다.
5. 5. 메밀면을 삶아서 준비한다.
6. 6. 볼에 메밀면을 담고 고명을 올려준 후 믹서
에 갈은 두부두유를 뿌려 완성한다.

## 영양 정보

- 칼로리: 229.9 kcal
- 탄수화물: 26.2g
- 단백질: 12.3g
- 지방: 8.4g
- 나트륨: 183.4mg
