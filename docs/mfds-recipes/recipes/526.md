# 새우완자찜

새우완자찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 새우(100g)
- 소금(0.5g)
- 후춧가루(0.02g)
- 다진소고기(30g)
- 마늘(10g)
- 브로컬리(20g)
- 당근(20g)
- 양파(20g)
- 소금(0.5g)
- 달걀(30g)
- 녹말(10g)

## 조리 방법

1. 1. 준비한 새우의 반은 껍질을 벗겨
내장을 제거하고 곱게 다진다.
2. 2. 나머지 새우는 내장을 제거하고 껍질을
벗겨, 2/3 정도 반으로 갈라 소금, 후추로
밑간을 해 놓는다.
3. 3. 다진 새우는 소고기와 섞어, 마늘을
넣고 양념이 충분히 스며들도록 준비해
놓는다.
4. 4. 브로컬리는 데쳐 식힌 후 다지고, 당근과
양파도 다져 소금에 살짝 절여 물기를
제거해 놓는다.
5. 5. 양념된 재료(③)에 준비해 놓은 채소
(④)를 넣고, 녹말가루와 달걀을 풀어
골고루 섞는다.
6. 6. 반으로 갈라 준비한 새우에 녹말가루를
살짝 뿌리고, 섞어놓은 재료(⑤)를
완자를 빚어 새우에 올려 주고, 찜기에서
약 10분 정도 쪄서 완성한다.

## 영양 정보

- 칼로리: 121 kcal
- 탄수화물: 7g
- 단백질: 20.4g
- 지방: 1.3g
- 나트륨: 175.8mg
