# 포도호두스무디

포도호두스무디은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 얼린포도알 80g
- 오미자액 15g
- 호상요구르트 30g
- 깐호두 5g

## 조리 방법

1. 1. 포도를 씻는다.
2. 2. 포도 껍질을 까고, 포도알을 떼어 낸다.
3. 3. 포도알은 뚜껑이 있는 통에 담아 냉동고에 얼린다.
4. 4. 얼린 포도알에 오미자액과 호상요구르트를 넣고 믹서에 갈아 스무디를 만든다.
5. 5. 호두는 칼로 굵게 다진다.
6. 6. 다진 호두는 스무디 위에 뿌린다.

## 영양 정보

- 칼로리: 159.8 kcal
- 탄수화물: 24.1g
- 단백질: 4.5g
- 지방: 5.8g
- 나트륨: 26.5mg
