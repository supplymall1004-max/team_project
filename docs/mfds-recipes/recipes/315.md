# 연어롤샐러드

연어롤샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연어 60g
- 단호박 50g
- 방울토마토 4개
- 시금치 50g
- 영양부추 50g
- 소스 : 잣 5g
- 더덕 45g
- 마늘 5g
- 식초 15g
- 올리브오일 30g
- 다진홍고추 15g

## 조리 방법

1. 1. 영양부추, 시금치는 먹기 좋게 잘라 물에
담가준다.
2. 2. 단호박은 삶아서 체에 내려준다.
3. 3. 토마토는 곱게 다져주고 시금치도 데친 후
단호박, 연두부와 섞어준다.
4. 4. 연어를 포를 떠서 만들어진 단호박과 연두부를
연어에 돌돌 말아 냉동실에 살짝 얼려준다.
5. 5. 소스재료로 소스를 만든다.
6. 6. 접시에 준비된 채소를 깔고 연어를 올리고
소스를 올려 완성한다.

## 영양 정보

- 칼로리: 208.5 kcal
- 탄수화물: 1.1g
- 단백질: 19.5g
- 지방: 14g
- 나트륨: 51mg
