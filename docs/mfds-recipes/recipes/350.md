# 매생이찰범벅

매생이찰범벅은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 우유100g
- 매생이 20g
- 찹쌀 15g
- 단호박 50g
- 밤 30g
- 녹차소금 1g
- 멸치육수 150g

## 조리 방법

1. 1. 찹쌀은 미리 불려준다.
2. 2. 매생이는 물에 담가 짠기와 이물질을 빼준다.
3. 3. 단호박은 작은 사각형으로 잘라 멸치육수에
데쳐준다.
4. 4. 밤은 곱게 다져주어 고명으로 사용한다.
5. 5. 우유와 육수를 섞은 뒤 찹쌀을 끓여준다.
6. 6. 죽이 어느 정도 완성되면 데쳐낸 호박과
매생이를 넣어 범벅을 만들어준다.

## 영양 정보

- 칼로리: 122.3 kcal
- 탄수화물: 23.7g
- 단백질: 3.7g
- 지방: 1.4g
- 나트륨: 64.3mg
