# 가지말이샐러드

가지말이샐러드은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 가지 250g
- 감자 50g
- 미니파프리카 30g
- 양파 40g
- 마늘튀김 15g
- 소스 : 홀그레인머스터드 15g
- 올리브오일 30g
- 다진마늘 15g
- 설탕 5g
- 올리고당 15g
- 레드와인 30g
- 곁들이채소 : 어린채소 30g
- 고명 : 파마산치즈

## 조리 방법

1. 1. 가지는 길이로 포를 떠 준다. 이때 필러를
이용하면 간편하게 할 수가 있다.
2. 2. 감자는 채 썰어 냉수에 담가 전분기를 완전히
빼준다.
3. 3. 파프리카, 양파, 청고추는 감자와 같은 길이로
채 썰어준다.
4. 4. 소스를 만들어준다.
5. 5. 가지를 기름 없이 구워준다.
6. 6. 구운 가지를 펴 놓고 준비된 채소를 말아서
가지말이를 만들고, 마늘튀김을 올려 소스랑
완성한 뒤 파마산치즈를 다져서 먹기 직전에
뿌려준다.

## 영양 정보

- 칼로리: 89 kcal
- 탄수화물: 9.1g
- 단백질: 2.9g
- 지방: 4.6g
- 나트륨: 58mg
