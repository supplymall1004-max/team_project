# 고구마라떼

고구마라떼은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 고구마 90g
- 우유 100g
- 플레인 요거트 30g
- 다진 호두 10g
- 올리고당 7g
- 계피가루 0.25g

## 조리 방법

1. 1. 고구마를 전자렌지에서 익힌다.
2. 2. 전자레인지 마지막 1분이 남았을 때 컵에 우유를 따라 함께 데운다.
3. 3. 거품기로 살짝 저어 거품을 낸다.
4. 4. 고구마의 껍질을 벗겨 작게 잘라 우유, 요거트, 호두, 올리고당을 넣고 믹서에 간다.
5. 5. 거품기로 살짝 저어 거품을 낸다.
6. 6. 잔에 따르고 계피가루를 뿌린다.

## 영양 정보

- 칼로리: 291.7 kcal
- 탄수화물: 42.2g
- 단백질: 2.8g
- 지방: 14.7g
- 나트륨: 70mg
