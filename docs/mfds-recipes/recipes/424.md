# 참외깍두기

참외깍두기은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 참외 200g
- 양파 60g
- 청고추 17g
- 홍고추 17g
- 대파 15g
- 미나리 20g
- 양념장 : 저염간장 15g
- 생강청 10g
- 고춧가루 10g
- 통깨 5g
- 다진마늘 5g

## 조리 방법

1. 1. 참외는 껍질을 벗기고 반으로 잘라 속을
파 준 후 먹기 좋게 잘라준다.
2. 2. 청고추, 홍고추는 송송 잘라준다.
3. 3. 양파와 미나리는 참외크기로 자르고, 대파는
송송 자른다.
4. 4. 참외, 양파를 저염 간장에 살짝 절여준다.
5. 5. 절여진 채소에 고춧가루로 먼저 색을 입힌다.
6. 6. 나머지 재료를 넣고 김치를 만든다.

## 영양 정보

- 칼로리: 64.1 kcal
- 탄수화물: 13.9g
- 단백질: 1.5g
- 지방: 0.3g
- 나트륨: 464.1mg
