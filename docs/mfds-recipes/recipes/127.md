# 컵밥

컵밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 쌀밥 420g(2컵)
- 김 4g(1장)
- 배추김치 30g
- 달걀 200g(4개)
- 베이컨 30g(3장)
- 소시지 30g(1개)
- 시금치 70g
- 고춧가루 20g(1큰술)
- 참기름 6g(1작은술)
- 마늘 6g(1작은술)

## 조리 방법

1. 1. 베이컨과, 소시지를 다진 후 끓는 물에 데쳐 팬에 볶고 다진 김치는 물에 행군 후 고추가루를 넣어 볶는다.
2. 2. 시금치는 데친 후 다져 참기름과 다진 마늘을 넣어 무치고 달걀은 부쳐서 다진다.
3. 3. 김은 참기름을 발라 구워 길게 자르고 밥과 준비된 재료를 층층히 쌓아 완성한다.

## 영양 정보

- 칼로리: 218.2 kcal
- 탄수화물: 21.8g
- 단백질: 7.1g
- 지방: 11.4g
- 나트륨: 117.8mg
