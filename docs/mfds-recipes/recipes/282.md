# 버섯순두부찌개

버섯순두부찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수 재료 : 표고버섯(3g)
- 애호박(10g)
- 감자(10g)
- 양파(3g)
- 미나리(3g)
- 부추(1g)
- 대파(1g)
- 청양고추(1g)
- 순두부(100g)
- •육수 : 다시마(3g)
- 표고버섯 밑동(3g)
- 물(250g)
- •양념 : 저염된장(5g)
- 찹쌀가루(3g)
- 다진마늘(1g)
- 들깻가루(7g)
- 저염국간장(1g)

## 조리 방법

1. 1. 표고버섯은 머리와 밑동을 분리하고, 머리 부분은 잘게 썬다.
2. 2. 애호박, 감자, 양파는 한입 크기로 썰고, 미나리와 부추는 3cm 길이로 썰고, 대파와 청양고추는 송송 썬다.
3. 3. 물에 다시마, 표고버섯 밑동을 넣고 끓으면 건더기를 건져 육수를 만든다.
4. 4. 육수에 저염된장과 찹쌀가루를 풀어 한소끔 끓이고, 감자와 다진마늘을 넣어 끓인다.
5. 5. 애호박과 표고버섯을 넣고, 순두부, 양파, 들깻가루, 저염국간장을 넣어 한소끔 끓인다.
6. 6. 대파, 청양고추, 미나리, 부추를 듬뿍 올려 마무리한다.

## 영양 정보

- 칼로리: 85.7 kcal
- 탄수화물: 6.3g
- 단백질: 6.4g
- 지방: 3.9g
- 나트륨: 243.1mg
