# 망고자몽드레싱 해산물전체

망고자몽드레싱 해산물전체은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 관자(2개)
- 새우(4마리)
- 방울토마토(2개)
- 생크림(50g)
- 파슬리(5g)
- 당근(30g)
- 오이(50g)
- 단호박(1/4개)
- 브로콜리(30g)
- 감자(50g)
- 망고(1/2개)
- 레몬(1/4개)
- 자몽(1/2개)
- 설탕(10g)
- 소금(0.1g)

## 조리 방법

1. 1. 감자는 삶아 껍질을 벗긴다
2. 2. 새우와 관자는 끓는 물에 레몬즙을
넣고 삶아 건져 껍질을 벗긴다.
3. 3. 오이, 당근, 단호박은 껍질을 벗겨
슬라이스 썬다.
4. 4. 데친 관자는 토치로 한 번 더 익혀 썬다
5. 5. 믹서에 간 망고와 레몬에 설탕, 소금을
약간 넣고 소스를 만든다.
6. 6. 토마토는 먹기 좋은 크기로 자르고,
접시에 준비한 야채와 해산물을 담고,
소스는 먹기 직전에 올린다

## 영양 정보

- 칼로리: 163.5 kcal
- 탄수화물: 12.2g
- 단백질: 26.2g
- 지방: 1.1g
- 나트륨: 233.6mg
