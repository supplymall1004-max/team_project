# 돼지고기 숙주덮밥

돼지고기 숙주덮밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수 재료 : 가다랑어포(10g)
- 돼지고기(100g)
- 양파(30g)
- 표고버섯(1개)
- 팽이버섯(20g)
- 쪽파(10g)
- 달걀(1개)
- 쌀(90g)
- 숙주(50g)
- •밑간 : 소금(1g)
- 후춧가루(1g)
- •육수 :다시마(10g)
- 저염간장(5g)
- 물(150g)

## 조리 방법

1. 1. 다시마 우린 물에 가다랑어포를 넣고 10분간 담갔다가 체에 거르고, 쌀로 밥을 짓는다.
2. 2. 돼지고기는 소금과 후춧가루로 밑간을 한 뒤 굽는다.
3. 3. 양파, 표고버섯, 팽이버섯은 채 썰고, 쪽파는 버섯 길이로 썬다.
4. 4. 육수에 저염간장을 넣어 한소끔 끓이다가 양파와 버섯을 넣고 끓인다.
5. 5. 다시 한 번 끓어오르면 쪽파와 달걀을 고루 풀어 넣고 젓지 말고 그대로 끓인다.
6. 6. 그릇에 밥을 담고, 돼지고기와 숙주를 올린 뒤 뜨거운 육수를 부어 마무리한다.

## 영양 정보

- 칼로리: 182.1 kcal
- 탄수화물: 17.2g
- 단백질: 16.5g
- 지방: 5.3g
- 나트륨: 206.1mg
