# 바다완자탕

바다완자탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 새우(3마리)
- 바지락(50g)
- 무(20g)
- 다시마(10g)
- 대파(20g)
- 가쓰오부시(10g)
- 달걀(1개)
- 전분(10g)
- 홍고추(1/2개)
- 풋고추(1/2개)

## 조리 방법

1. 1. 대파, 무, 다시마를 물에 넣고 끓여
육수를 만든다.
2. 2. 육수에 가쓰오부시를 넣고 10분 정도
가라앉힌 뒤 거른다.
3. 3. 새우는 껍질을 벗겨 깨끗이 씻고
바지락은 주물러 씻는다.
4. 4. 손질한 새우와 바지락을 곱게 다진다.
5. 5. 다진 새우와 바지락에 달걀흰자와
전분을 넣고 골고루 섞어 완자를
만들어 후라이팬에 살짝 굴려 모양이
풀리지 않게 한다
6. 6. 준비된 육수에 완자를 넣고 완전히
익히고 그릇에 담은 뒤 어슷썬
홍고추와 풋고추를 위에 올린다.

## 영양 정보

- 칼로리: 146.9 kcal
- 탄수화물: 23g
- 단백질: 10.5g
- 지방: 1.5g
- 나트륨: 119.2mg
