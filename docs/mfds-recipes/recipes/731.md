# 홍시곶감화채

홍시곶감화채은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 얼린 홍시 70g
- 곶감 30g
- 호두 10g
- 꿀 20g
- 물 70g

## 조리 방법

1. 1. 홍시의 껍질을 벗긴다.
2. 2. 껍질 벗긴 홍시에 꿀과 물을 넣어 믹서에 간다.
3. 3. 곶감을 갈라 씨를 뺀다.
4. 4. 씨를 빼낸 곶감에 호두를 넣어 만다.
5. 5. 곶감말이를 0.5cm 두게로 썬다.
6. 6. 홍시 간 것에 곶감말이를 띄워 낸다.

## 영양 정보

- 칼로리: 173.2 kcal
- 탄수화물: 38.1g
- 단백질: 1.56g
- 지방: 3.44g
- 나트륨: 3.1mg
