# 유자샐러드라면

유자샐러드라면은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 라면(55g)
- 양상추(40g)
- 케일(5g)
- 파프리카(20g)
- 비타민(5g)
- 유자드레싱 식초(2g)
- 레몬즙(5g)
- 유자청(15g)
- 올리브유(1g)

## 조리 방법

1. 1. 끓는 물(600g)에 라면을 넣어
데친다.
2. 2. 면은 찬물에 헹궈 물기를 뺀다.
3. 3. 양상추, 케일, 파프리카는 얇게 채
썰고, 비타민은 먹기 좋은 크기로
자른다.
4. 4. 유자드레싱을 고루 섞은 뒤 라면,
채소를 버무려 마무리한다.

## 영양 정보

- 칼로리: 307.9 kcal
- 탄수화물: 26.6g
- 단백질: 3.14g
- 지방: 4.86g
- 나트륨: 145mg
