# 전복영양밥과 부추양념장

전복영양밥과 부추양념장은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 전복영양밥 : 쌀 90g
- 전복살 40g
- 양송이 버섯 13g
- 표고버섯 15g
- 당근 15g
- 건다시마 5g
- 물 400g - 부추양념장 : 다시마국물 5g
- 부추 5g
- 간장 5g
- 실파 2.5g
- 다진 마늘 2.5g
- 참기름 0.5g
- 통깨 0.5g

## 조리 방법

1. 1. 물 2컵에 건다시마를 넣어 다시마국물을 1컵 반 정도를 낸다.
2. 2. 우린 다시마와 전복살은 잘게 다진다.
3. 3. 표고버섯, 양송이버섯, 당근은 0.5~0.7cm 주사위모양으로 썬다.
4. 4. 쌀은 씻어 2와 3을 넣고 다시마국물 1컵 반을 부어 30분 정도 불린 후 밥을 짓는다.
5. 5. 부추와 실파는 다지고 참기름, 다진 마늘, 통깨, 다시마국물, 간장을 넣어 양념장을 만든다.
6. 6. 전복영양밥과에 부추양념장을 곁들인다.

## 영양 정보

- 칼로리: 387.2 kcal
- 탄수화물: 77.5g
- 단백질: 12.8g
- 지방: 1.9g
- 나트륨: 459.9mg
