# 밥크로켓

밥크로켓은(는) 밥 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 양파(20g)
- 대파(10g)
- 당근(20g)
- 마늘(10g)
- 식용유(10g)
- 메추리알(100g)
- 밥(100g)
- 소금(0.5g)
- 밀가루(20g)
- 녹말가루(10g)
- 달걀(50g)
- 빵가루(20g)
- 튀김기름(400g)
- - 소스 : 칠리소스(50g)

## 조리 방법

1. 1. 양파와 대파, 당근, 마늘은 입자있게
다져 준비한다.
2. 2. 팬에 기름을 두르고, 팬이 따끈해 지면
준비한 채소(①)를 넣어 재빠르게 볶아
식힌다.
3. 3. 메추리알은 찬물에 넣어 소금을 넣고
약 5분 정도 삶아 건져 찬물에 헹구어
껍질을 제거시켜 놓는다.
4. 4. 준비한 밥에 볶아 놓은 채소와 소금을
넣어 골고루 섞은 후 준비한 메추리알을
감싸 준다.
5. 5. 밥으로 감싼 메추리알을 밀가루와
녹말을 섞어 그 위에 굴리고, 다시
달걀물에 굴린 후 빵가루를 입혀준다.
6. 6. 튀김 기름이 약 170℃ 정도 되면,
만들어 놓은 메추리알을 넣어 골고루
색이 나도록 튀겨 완성하고, 칠리소소와
함께 제공한다.

## 영양 정보

- 칼로리: 731.8 kcal
- 탄수화물: 88.1g
- 단백질: 15.6g
- 지방: 35.2g
- 나트륨: 290.7mg
