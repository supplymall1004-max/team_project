# 간장소스를 곁들인 새우전복찜

간장소스를 곁들인 새우전복찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 전복(2개)
- 새우(3마리)
- 시금치(30g)
- 해초(20g)
- 저염간장(20g)
- 레몬(1/4개)
- 설탕(20g)
- 소금(0.1g)
- 마늘(10g)
- 녹말(10g)
- 식용꽃(60g)

## 조리 방법

1. 1. 전복은 수저로 떼어내어 소금으로 비벼
깨끗이 씻는다.
2. 2. 시금치는 데쳐 다지고, 새우는 껍질을
벗겨 전복, 해초와 함께 다진다.
3. 3. ?번을 잘 섞어 전복 껍질에 넣는다.
4. 4. 찜통에 김이 오르면 속을 채운 전복
껍질을 넣고 약 10분 정도 찐다.
5. 5. 저염간장, 레몬, 설탕, 다진마늘을
넣고 살짝 끓여 소스를 만든다.
6. 6. 간장소스에 전분을 풀어 농도를
맞추고, 전복찜에 여러번 바르고
식용꽃을 올린다.

## 영양 정보

- 칼로리: 119.1 kcal
- 탄수화물: 8.2g
- 단백질: 19.2g
- 지방: 1.1g
- 나트륨: 455.9mg
