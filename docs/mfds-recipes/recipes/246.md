# 삼계선

삼계선은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 재료 닭가슴살(200g)
- 두부(40g)
- 수삼(10g)
- 파프리카(20g)
- 대추(30g)
- 블루베리소스 블루베리잼(20g)
- 화이트와인(20g)
- 설탕(30g)

## 조리 방법

1. 1. 닭가슴살을 다져서 두부와 섞어준다.
2. 2. 수삼과 파프리카를 채 썰고, 대추는
돌려서 깎아준다.
3. 3. 닭가슴살두부반죽을 얇게 편 후
파프리카, 수삼, 대추를 올린 뒤
풀리지 않게 종이포일로 만다.
4. 4. 김 오른 찜기에 20분간 쪄준다.
5. 5. 블루베리잼과 화이트와인을 졸인 뒤
설탕을 넣고 블루베리소스를 만든다.
6. 6. 쪄낸 삼계선을 2㎝두께로 잘라 담고 블루베리소스를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 281.5 kcal
- 탄수화물: 8.03g
- 단백질: 16.1g
- 지방: 1.65g
- 나트륨: 83.5mg
