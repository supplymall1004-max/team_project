# 더덕비트물김치

더덕비트물김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 더덕 50g
- 양파 60g
- 청고추 15g
- 홍고추 15g
- 식초 10g
- 국물 : 비트 20g
- 함초소금 5g
- 생강청 30g
- 식초 15g

## 조리 방법

1. 1. 물과 식초가 포함된 물에 담가서 쓴맛을
빼준다.
2. 2. 물에 절여진 더덕은 키친타월을 이용해 물기를
완전히 빼준다.
3. 3. 양파, 청고추, 홍고추도 어슷하게 썰어준다.
4. 4. 비트를 편으로 썰어서 국물을 만들어준다.
5. 5. 그릇에 더덕부터 30분 정도 담가 물을 살짝
들여준다.
6. 6. 나머지 재료도 함께 담가 간을 배게 한 뒤 24
시간 후부터 먹을 수가 있다.

## 영양 정보

- 칼로리: 43.8 kcal
- 탄수화물: 8.9g
- 단백질: 1.8g
- 지방: 0.1g
- 나트륨: 107mg
