# 오렌지와 당근 만남주스

오렌지와 당근 만남주스은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 오렌지 당근펀치
- 당근 100g(1/2개)
- 오렌지 100g(1/2개)

## 조리 방법

1. 1. 오렌지는 깨끗이 씻어 껍질을 벗긴다.
2. 2. 당근은 깨끗이 씻어 작은 토막으로 썬다.
3. 3. 당근, 오렌지, 물(50ml)을 믹서에 곱게 간다.
4. 4. 믹서에 갈아낸 주스는 고운 체로 거른다.
5. 5. 주스를 살얼음이 생길 만큼 시원하게 냉동한다.
6. 6. 컵에 담아 마무리한다.

## 영양 정보

- 칼로리: 60 kcal
- 탄수화물: 14g
- 단백질: 1g
- 지방: 0g
- 나트륨: 76mg
