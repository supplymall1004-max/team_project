# 관자버터구이

관자버터구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 관자(2개)
- 시금치(30g)
- 비트(20g)
- 양파(30g)
- 오이피클(20g)
- 고추피클(20g)
- 마요네즈(20g)
- 꿀(20g)
- 버터(10g)

## 조리 방법

1. 1. 양파, 오이피클, 고추피클은 입자있게
다진다.
2. 2. 다진 양파와 피클에 마요네즈를 넣고
골고루 섞어 소스를 만든다.
3. 3. 마요네즈소스에 꿀을 넣고 섞는다
4. 4. 관자의 앞뒤에 버터를 바른다.
5. 5. 관자에는 버터를 발라 팬에 구워
접시에 담는다.
6. 6. 비트, 양파, 데친 시금치를 넣고 갈아서
소스를 만들고 관자에 마요네즈소스와
시금치소스를 올린다

## 영양 정보

- 칼로리: 202.3 kcal
- 탄수화물: 21.9g
- 단백질: 7.7g
- 지방: 9.3g
- 나트륨: 379.5mg
