# 오렌지마리네이드 돼지고기

오렌지마리네이드 돼지고기은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 돼지고기(150g)
- 시금치(30g)
- 양송이버섯(3개)
- 파프리카(1/4개)
- 마늘(10g)
- 오렌지(1/2개)
- 레몬(1/4개)
- 머스터드(20g)
- 꿀(10g)
- 생크림(20g)
- 우유(200g)
- 소금(0.2g)
- 후추(0.01g)
- 올리브오일(20g)

## 조리 방법

1. 1. 돼지고기는 오렌지와 레몬,
올리브오일로 마리네이드 한다.
2. 2. 파프리카, 양송이버섯은 썰어서 팬에
굽는다.
3. 3. 머스터드와 꿀을 섞어 소스를 만든다.
4. 4. 시금치는 끓는 물에 데친다.
5. 5. 마늘은 편으로 썰고 팬에 굽는다.
6. 6. 돼지고기를 팬에 구워 접시에 담고,
구운 마늘, 파프리카, 양송이버섯,
데친 시금치를 담고 그 위에
머스터드소스를 올린다.

## 영양 정보

- 칼로리: 301.4 kcal
- 탄수화물: 9.5g
- 단백질: 33.5g
- 지방: 14.4g
- 나트륨: 276.5mg
