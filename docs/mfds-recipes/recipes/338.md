# 단호박약식

단호박약식은(는) 밥 요리로, 찌기 조리법을 사용합니다.

## 재료

- 미니 단호박 250g
- 찹쌀 100g
- 밤 40g
- 대추 18g
- 크린베리 10g
- 양념장 : 맛간장 15g
- 참기름 10g
- 계피가루 5g

## 조리 방법

1. 1. 단 호박은 속을 파 준다.
2. 2. 찹쌀은 물에 불려준다.
3. 3. 밤은 먹기 좋게 잘라서 물에 담가준다.
4. 4. 대추는 씨를 제거하고 먹기 좋게 잘라준다.
5. 5. 양념장을 만들어 먼저 쌀과 밤을 재워준 후
찰밥을 만든다.
6. 6. 단호박 속에 찰밥을 넣어서 찜기에 쪄준다.

## 영양 정보

- 칼로리: 152.2 kcal
- 탄수화물: 33.1g
- 단백질: 3.1g
- 지방: 0.8g
- 나트륨: 64.7mg
