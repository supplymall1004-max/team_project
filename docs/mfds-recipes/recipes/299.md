# 생선베이컨말이

생선베이컨말이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 피망(5g)
- 파프리카(10g)
- 시금치(15g)
- 저염베이컨(60g)
- 동태포(30g)
- 저염버터(10g)
- 생크림(10g)
- 새싹채소(0.1g)

## 조리 방법

1. 1. 피망, 파프리카는 5㎝ 길이로 채
썰고, 시금치는 갈아둔다.
2. 2. 베이컨 위에 동태포를 올리고
그 위에 피망과 파프리카를 올려
돌돌 만다.
3. 3. 저염버터에 생크림을 넣어서 농도를
맞춘 후 갈아놓은 시금치를 넣어
시금치소스를 만든다.
4. 4. 달군 팬에 흰살생선베이컨말이를
올려 완전히 익힌다.
5. 5. 접시에 담아 시금치소스를
곁들인 뒤 새싹채소로 장식하여
마무리한다.

## 영양 정보

- 칼로리: 105.6 kcal
- 탄수화물: 1.53g
- 단백질: 18.71g
- 지방: 10.56g
- 나트륨: 146.5mg
