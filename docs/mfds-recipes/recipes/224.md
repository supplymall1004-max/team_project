# 봄주먹밥

봄주먹밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 봄동(10g)
- 깻잎(6g)
- 낙지 다리(6g)
- 참기름(3g)
- 두릅(8g)
- 올리브유(4g)
- 미나리(3g)
- 주먹밥 데친 꼬막살(14g)
- 올리브유(4g)
- 밥(100g)
- 참깨(3g)
- 두부강된장 참기름(10g)
- 다진 양파(14g)
- 다진 당근(14g)
- 다진 애호박(14g)
- 물(150g)
- 된장(14g)
- 고춧가루(4g)
- 다진 마늘(4g)
- 송송 썬 붉은 고추(4g)
- 송송 썬 청양고추(4g)
- 으깬 두부(76g)

## 조리 방법

1. 1. 봄동과 깻잎은 데친 뒤 체에 밭쳐
물기를 제거한다.
2. 2. 데친 꼬막살은 물기를 제거하고
다져 올리브유로 양념한 밥에
참깨와 함께 버무린 후 한입 크기로
주먹밥을 만든다.
3. 3. 낙지 다리는 데쳐 찬물에 담가 식힌
뒤 참기름(3g)에 양념하고, 두릅은
데쳐 물기를 제거한 뒤 올리브유로
양념한다.
4. 4. 봄동에 꼬막밥을 넣고 만 뒤
깻잎으로 싸고 데친 미나리로 묶어
봄주먹밥을 만든다.
5. 5. 냄비에 참기름(10g)을 두르고 양파,
당근, 애호박을 넣고 볶은 뒤 물,
된장, 고춧가루, 다진 마늘, 붉은
고추, 청양고추를 넣고 끓이다가
두부를 넣고 조려 두부강된장을
만든다.
6. 6. 봄주먹밥에 낙지와 두릅,
두부강된장을 곁들여 마무리한다.

## 영양 정보

- 탄수화물: 19.63g
- 단백질: 4.29g
- 지방: 2.12g
- 나트륨: 118.1mg
