# 야채찜을 곁들인 광어스테이크

야채찜을 곁들인 광어스테이크은(는) 일품 요리로, 찌기 조리법을 사용합니다.

## 재료

- 광어(1마리)
- 당근(30g)
- 단호박(30g)
- 가지(1/2개)
- 파프리카(50g)
- 콜리플라워(50g)
- 레몬(1/4개)
- 와사비(5g)
- 발사믹소스(50g)
- 저염간장(30g)
- 청주(15g)

## 조리 방법

1. 1. 광어는 비늘을 제거하고 포를 뜬다.
2. 2. 손질된 광어에 레몬과 청주를 뿌려
숙성 시킨다.
3. 3. 저염간장에 레몬즙, 와사비,
발사믹소스를 넣고 소스를 만든다.
4. 4. 당근, 애호박은 쿠퍼로 동글동글하게
깎는다.
5. 5. 당근, 애호박, 콜리플라워는 김이 오른
찜통에 5분 정도 쪄 낸다.
6. 6. 가지와 파프리카는 팬에 굽고,
숙성시킨 광어도 팬에 구워 접시에
담고 쪄 낸 야채를 담은 뒤 소스를
올린다.

## 영양 정보

- 칼로리: 259 kcal
- 탄수화물: 16g
- 단백질: 18.9g
- 지방: 13.2g
- 나트륨: 398.9mg
