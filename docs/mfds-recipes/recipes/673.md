# 찬밥이용닭죽

찬밥이용닭죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 찬밥 60g
- 닭다리살 80g
- 감자 30g
- 통마늘 15g
- 양파 10g
- 당근 8g
- 부추 5g
- 물 700g
- 소금 0.75g
- 후춧가루 0.3g

## 조리 방법

1. 1. 닭다리는 기름기를 제거하고 냄비에 물 3컵 반을 부어 통마늘과 삶아 육수를 만든다.
2. 2. 삶은 마늘을 체에 걸러 으깬 뒤 다시 육수에 넣는다.
3. 3. 닭 육수 2컵 반에 찬밥을 넣고 밥알이 무르도록 끓인다.
4. 4. 삶은 닭다리 살을 발라 잘게 찢어 넣는다.
5. 5. 감자, 양파, 당근, 부추는 잘게 다진다.
6. 6. 4에 5를 넣고 끓인 뒤 소금, 후춧가루로 간 한다.

## 영양 정보

- 칼로리: 234.5 kcal
- 탄수화물: 29.8g
- 단백질: 20.2g
- 지방: 3.3g
- 나트륨: 327.6mg
