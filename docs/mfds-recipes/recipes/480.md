# 가지토마토구이

가지토마토구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 가지(80g)
- 올리브오일(10g)
- 소금(0.5g)
- 토마토(40g)
- 모짜렐라치즈(30g)
- - 소스 : 토마토소스(20g)
- 토마토(40g)
- - 곁들임 : 팽이버섯(20g)
- 후춧가루(0.02g)

## 조리 방법

1. 1. 가지는 약 10cm 길이로 길고, 납작하게
썰어 준비한다.
2. 2. 팬에 올리브 오일을 살짝 둘러 썰어놓은
가지를 소금을 넣고 살짝 익힌다.
3. 3. 토마토는 반은 토마토 형태로 납작하게
썰어 놓는다.
4. 4. 토마토반은 입자있게 다져 준비한다.
5. 5. 다진 토마토를 준비한 토마토소스에
섞어 살짝 어우러지게 끓여 준비한다.
6. 6. 구운 가지위에 토마토와 모짜렐라치즈를
올려 팬에 굽고, 별도로 팽이버섯에
후춧가루를 뿌려 살짝 구워 접시에 담고,
끓여 놓은 토마토소스를 올려 완성한다.

## 영양 정보

- 칼로리: 164.8 kcal
- 탄수화물: 7.1g
- 단백질: 5.4g
- 지방: 12.7g
- 나트륨: 171.2mg
