# 콩나물부추볶음

콩나물부추볶음은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 콩나물(150g)
- 부추(50g)
- 양파(30g)
- 당근(20g)
- 파(10g)
- 감자(50g)
- 소금(0.3g)
- 식용유(20g)

## 조리 방법

1. 1. 감자는 채를 썰어 물에 담궈 전분을
제거하고, 양파와 당근 파는 채를 썬다.
2. 2. 콩나물은 머리와 꼬리를 손질한다
3. 3. 물이 끓으면 손질한 콩나물을 넣는다.
4. 4. 콩나물이 익으면 체에 건져 찬물에
헹군다.
5. 5. 부추는 약 6cm 길이로 썬다.
6. 6. 팬에 기름을 두르고 콩나물을 넣고
살짝 볶다가 썰어 놓은 야채를 넣고
볶으면서 소금 간을 하고 마지막에
부추를 넣고

## 영양 정보

- 칼로리: 133.8 kcal
- 탄수화물: 15g
- 단백질: 3.9g
- 지방: 6.4g
- 나트륨: 139.4mg
