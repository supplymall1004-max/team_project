# 발사믹소스를 곁들인 오리스테이크

발사믹소스를 곁들인 오리스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 오리고기(가슴살
- 120g)
- 당근(20g)
- 토마토(100g)
- 새송이버섯(1개)
- 표고버섯(1개)
- 감자(50g)
- 아스파라거스(2개)
- 소금(0.2g)
- 후추(0.01g)
- 올리브오일(10g)
- 발사믹소스(100g)
- 양파(20g)

## 조리 방법

1. 1. 감자는 껍질을 벗겨 슬라이스로 썰어
찬물에 담궈 전분을 빼고 건져 물기를
제거한다.
2. 2. 토마토, 양파, 새송이버섯. 표고버섯은
입자있게 다지고, 당근은 샤또
모양으로 썰어 준비한다.
3. 3. 오리 가슴살은 소금, 후춧가루로
밑간을 한다.
4. 4. 팬에 올리브오일을 두르고
마리네이드한 오리 가슴살을 넣고 구워
기름이 빠지면 접시에 담는다.
5. 5. 오리 가슴살을 구운 팬에
아스파라거스, 당근, 감자를 구워
접시에 담는다.
6. 6. 냄비 또는 팬에 다진 야채를 넣고
볶다가 발사믹소스를 넣고 졸여 소스를
만들고, 접시에 담아 놓은 오리
스테이크에 올린다.

## 영양 정보

- 칼로리: 307.5 kcal
- 탄수화물: 15.2g
- 단백질: 13.1g
- 지방: 21.6g
- 나트륨: 486.9mg
