# 등심채소롤

등심채소롤은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 돼지등심(100g)
- 소금(0.3g)
- 후춧가루(0.05g)
- 파프리카(30g)
- 시금치(30g)
- 파인애플(30g)
- 치즈(30g)
- 밀가루(20g)
- 달걀(50g)
- 빵가루(30g)
- 튀김기름(200g)
- - 소스 : 칠리소스(30g)
- 설탕(10g)
- 물녹말(녹말가루5g
- 물 5g)

## 조리 방법

1. 1. 등심은 넓게 펴서 소금과 후춧가루를
뿌려 준비한다.
2. 2. 파프리카와 파인애플은 속을 제거하고
약 7cm 길이로 썰어놓는다.
3. 3. 시금치는 끓는 물에 데쳐 찬물에 헹구어
준비한다.
4. 4. 밑간을 해 놓은 등심에 준비한 파프리카와
시금치 파인애플을 넣는다.
5. 5. 채소와 파인애플을 넣은 등심에 치즈를
넣고 돌돌말아 밀가루, 달걀물, 빵가루
순서로 입혀 튀김기름 170~180℃에서
약 3분 정도 튀겨 기름을 빼 놓는다.
6. 6. 냄비에 칠리소소와 물, 설탕을 넣고
바글바글 끓이다가 물녹말을 넣고
한 번 더 끓인 후, 튀겨놓은 등심채소
롤에 함께 뿌려 완성한다.

## 영양 정보

- 칼로리: 531.8 kcal
- 탄수화물: 52.5g
- 단백질: 26.5g
- 지방: 23.9g
- 나트륨: 522.4mg
