# 대구지리탕

대구지리탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 대구(150g)
- 미나리(15g)
- 청양고추(1g)
- 붉은 고추(1g)
- 콩나물(25g)
- 쑥갓(10g)
- 반죽 쑥가루(10g)
- 밀가루(20g)
- 육수 다시마(1g)
- 북어(15g)
- 무(30g)
- 물(500g)
- 양념 다진 마늘(10g)
- 다진 생강(3g)

## 조리 방법

1. 1. 대구는 비늘과 내장을 제거하고
흐르는 물에 씻어 비린 향을 없앤다.
2. 2. 미나리는 6cm 길이로 썰고,
청양고추, 붉은 고추는 어슷하게
썰고, 콩나물, 쑥갓을 씻어 준비한다.
3. 3. 쑥가루와 밀가루를 섞고 물(15g)을
조금씩 넣어가며 매끈해지도록
반죽한 뒤 30분~1시간 정도
숙성시킨다.
4. 4. 냄비에 육수 재료를 넣어 10분간
끓여 육수를 만든다.
5. 5. 다진 마늘과 생강을 넣고 향이
우러나면 대구를 넣은 뒤 대구가
어느 정도 익으면 콩나물을 넣고,
끓어오르면 수제비 반죽을 떼어
넣는다.
6. 6. 청양고추, 붉은 고추를 넣고 한소끔
더 끓이다가 미나리와 쑥갓을
얹은 뒤 바로 불을 꺼 마무리한다.

## 영양 정보

- 칼로리: 267.7 kcal
- 탄수화물: 3.87g
- 단백질: 7.19g
- 지방: 1.15g
- 나트륨: 277.3mg
