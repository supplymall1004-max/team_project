# 카레탄두리치킨과 닭가슴살냉채

카레탄두리치킨과 닭가슴살냉채은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 닭고기(1마리)
- 오곡(50g)
- 인삼(1뿌리)
- 대추(3알)
- 아스파라거스(3개)
- 당근(20g)
- 양파(30g)
- 오이(50g)
- 파프리카(50g)
- 카레가루(20g)
- 디종머스터드(30g)

## 조리 방법

1. 1. 닭은 살을 발라 소금, 후춧가루로
밑간을 한다.
2. 2. 오곡을 불려 밥을 짓는다.
3. 3. 오이, 아스파라거스, 당근, 양파는
길이대로 채썬다.
4. 4. 닭살에 지어 놓은 오곡밥을 넣는다
5. 5. 오곡밥을 넣은 닭살에 카레가루를
골고루 묻힌다.
6. 6. 팬에 기름을 두르고 5번을 올려 은근히
익혀 접시에 담고, 썰어 놓은 야채를
닭살위에 담은 뒤 디종머스터드를
올린다.

## 영양 정보

- 칼로리: 304.9 kcal
- 탄수화물: 26.7g
- 단백질: 16.7g
- 지방: 14.6g
- 나트륨: 427mg
