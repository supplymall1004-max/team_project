# 카레닭 룰라이드

카레닭 룰라이드은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기(120g)
- 카레가루(20g)
- 당근(20g)
- 양파(20g)
- 샐러리(1/2개)
- 아스파라거스(3개)
- 토마토(50g)
- 감자(50g)
- 생크림(30g)
- 소금(0.1g)
- 후춧가루(0.01g)

## 조리 방법

1. 1. 닭살에 소금, 후추로 밑간을 한다.
2. 2. 밑간한 닭살에 카레가루를 골고루
묻힌다.
3. 3. 양파, 당근, 토마토, 샐러리는 주사위
모양으로 자른다
4. 4. 냄비에 양파, 당근, 토마토, 샐러리를
볶다가 카레가루를 넣고 끓여
카레소스를 만든다.
5. 5. 팬에 기름을 두르고 닭살을 익힌다.
6. 6. 그릇에 익힌 닭살을 담고 카레소스를
올린다.

## 영양 정보

- 칼로리: 317.1 kcal
- 탄수화물: 18.2g
- 단백질: 24.6g
- 지방: 16.2g
- 나트륨: 978.4mg
