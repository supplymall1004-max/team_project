# 단호박생채

단호박생채은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 단호박(1/2개)
- 부추(30g)
- 양파(30g)
- 까나리액젓(20g)
- 건새우(20g)
- 다시마(10g)
- 매실액(20g)
- 마늘(10g)
- 배(1/4개)
- 고춧가루(20g)

## 조리 방법

1. 1. 다시마, 건새우, 물을 약 20분 정도
끓인 뒤 체에 걸러 육수를 만든다.
2. 2. 단호박은 껍질을 벗기고, 속을 깨끗이
파내 슬라이스 한다
3. 3. 슬라이스한 단호박을 까나리액젓에
20분 정도 절인 뒤 물로 씻는다.
4. 4. 부추는 5cm 길이로 썰고, 양파는
채썰고, 배는 껍질을 벗겨 슬라이스로
썰어둔다.
5. 5. 만들어 놓은 육수에 고춧가루, 매실액,
다진마늘을 섞어 양념장을 만든다
6. 6. 단호박, 배, 양파, 부추를 양념장에
버무린다.

## 영양 정보

- 칼로리: 167 kcal
- 탄수화물: 36.2g
- 단백질: 3.4g
- 지방: 1g
- 나트륨: 953.1mg
