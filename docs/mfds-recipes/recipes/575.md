# 음메꼬꼬돌돌

음메꼬꼬돌돌은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 소고기(120g)
- 달걀(1개)
- 닭고기(가슴살
- 100g)
- 미나리(50g)
- 찹쌀가루(50g)
- 홍고추(1개)
- 소금(0.3g)
- 후춧가루(0.01g)
- 통후추(5알)
- 저염간장(20g)
- 설탕(20g)
- 식초(10g)

## 조리 방법

1. 1. 소고기는 넓게 펴서 소금, 후춧가루로
밑간을 하고 굽는다.
2. 2. 닭가슴살은 소금, 후춧가루로 밑간을
하여 굽고 길이대로 썬다.
3. 3. 미나리는 끓는 물에 넣고 데쳐 찬물에
헹군다.
4. 4. 홍고추는 씨를 제거 하고 길게 썰어
준비한다.
5. 5. 달걀은 황백으로 나눠 지단을 부쳐
길게 썰어 놓는다
6. 6. 밑간한 소고기와 닭살 에 찹쌀가루를
뭍혀 홍고추와 달걀 황백지단을 넣어
돌돌 말아 미나리로 띠를 둘러 팬에
기름을 살짝 둘러 익힌다.

## 영양 정보

- 칼로리: 296.7 kcal
- 탄수화물: 0g
- 단백질: 41.5g
- 지방: 14.5g
- 나트륨: 275.8mg
