# 광어스테이크

광어스테이크은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 광어(150g)
- 소금(0.3g)
- 후춧가루(0.02g)
- 밀가루(20g)
- 달걀(50g)
- 아몬드(20g)
- 호두(20g)
- 올리브오일(20g)
- - 소스 : 플레인요거트(50g)
- 꿀(10g)
- 레몬즙(20g)

## 조리 방법

1. 1. 광어는 포를 길게 떠서 소금과
후춧가루에 밑간을 해 놓는다.
2. 2. 아몬드와 호두는 입자있게 다져
준비한다.
3. 3. 밑간이 되어 있는 광어포에 밀가루와
달걀물을 입힌다.
4. 4. 달걀물을 입힌 광어살에 다시 다져
놓은 견과류를 입혀 준다.
5. 5. 팬에 올리브 오일을 넣고, 팬이 따끈해
지면 광어 살을 조심스럽게 앞뒤로
익혀 접시에 담는다.
6. 6. 플레인요거트에 꿀과 레몬즙을 넣고
골고루 섞어 광어스테이크소스를
만들어 완성한다.

## 영양 정보

- 칼로리: 487.2 kcal
- 탄수화물: 19.8g
- 단백질: 32.2g
- 지방: 31g
- 나트륨: 192.9mg
