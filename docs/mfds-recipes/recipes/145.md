# 실치오이 초절임

실치오이 초절임은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 매실액 10g(2작은술)
- 실치 40g(1컵)
- 식초 20g(1⅓큰술)
- 오이 80g(1/2개)

## 조리 방법

1. 1. 실치는 끓는 물에 데친다.
2. 2. 오이는 반달 모양으로 슬라이스하고 식초에 절인 후 물기를 제거하고 매실액을 넣는다.
3. 3. 데쳐낸 실치와 오이를 함께 버무려 완성한다.

## 영양 정보

- 칼로리: 36.9 kcal
- 탄수화물: 6.1g
- 단백질: 3g
- 지방: 0g
- 나트륨: 59.1mg
