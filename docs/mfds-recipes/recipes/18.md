# 버섯 두유 소스 볶음

버섯 두유 소스 볶음은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 버섯볶음
- 새송이버섯 100g(3개)
- 올리브유 10g(2작은술)
- 두유 소스
- 두유 20g(1⅓큰술)
- 생크림 10g(2작은술)
- 청양고추 5g(1/2개)
- 후추 약간
- 곁들임
- 새싹채소 20g

## 조리 방법

1. 1. 새송이버섯은 굵게 채썬 후 달군 프라이팬에 기름을 두르고 노릇하게 볶는다.a
2. 2. 소스 팬에 두유, 생크림, 청양고추, 후추를 넣고 약한 불에 뭉근하게 졸인다.b
3. 3. 볶은 새송이버섯을 소스에 넣고 고루 섞어 가며 조금 더 볶아 접시에 담은 후 새싹채소를 함께 곁들인다.c

## 영양 정보

- 칼로리: 285 kcal
- 탄수화물: 17g
- 단백질: 9g
- 지방: 20g
- 나트륨: 53mg
