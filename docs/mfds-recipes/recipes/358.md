# 고등어카레탕수

고등어카레탕수은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 냉동고등어 300g
- 전분가루 50g
- 식용유 200g
- 우유 200g
- 카레가루 20g
- 감자 50g
- 애호박 25g
- 방울토마토 25g
- 양파 20g
- 사과 25g
- 꿀 15g

## 조리 방법

1. 1. 냉동고등어는 먹기 좋은 크기로 잘라 전분을
묻혀 기름에 두 번 튀겨준다.
2. 2. 애호박, 방울토마토, 사과는 적당한 크기로
잘라준다.
3. 3. 감자와 양파는 적당한 크기로 자른 뒤 물 1컵
에 감자를 먼저 넣고 끓여주다 양파를 넣고
끓여 단맛을 내준다.
4. 4. 우유에 카레가루를 섞어 불려준 뒤 끓고 있는
감자에 넣어준다.
5. 5. 튀겨진 고등어를 키친타월을 이용해 기름을
빼준다.
6. 6. 카레가 완성되면 사과, 방울토마토, 꿀을
넣어주고 튀겨 놓은 고등어에 뿌려 완성한다.

## 영양 정보

- 칼로리: 171.3 kcal
- 탄수화물: 15.5g
- 단백질: 7.5g
- 지방: 8.8g
- 나트륨: 254.7mg
