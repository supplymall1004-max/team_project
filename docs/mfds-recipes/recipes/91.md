# 버섯구이와 두부타르타르 소스

버섯구이와 두부타르타르 소스은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- ●버섯구이 : 새송이버섯 70g(7개)
- 올리브유 10g(2작은술)
- ●곁들임 : 치커리 10g(3줄기)
- ●두부타르타르 소스 : 연두부 30g(1/4모)
- 다진 양파 10g(2작은술)
- 다진 오이피클 10g(2작은술)
- 올리브유 2g(1/3작은술)
- 식초 5g(1작은술)
- 레몬즙 3g(2/3작은술)
- 머스터드 3g(2/3작은술)
- 꿀 2g(1/3작은술)
- 흰 후추 약간

## 조리 방법

1. 1. 곱게 다진 양파와 오이피클은 물기를 짠다.
2. 2. 연두부에 다진 양파, 다진 오이피클, 올리브유, 머스터드, 후추, 꿀, 식초, 레몬즙을 넣어 두부타르타르 소스를 만든다.
3. 3. 새송이버섯은 0.5cm 두께로 썰고 프라이팬에 노릇하게 구워준 후 접시에 새송이버섯과 치커리를 담고 소스를 곁들인다.

## 영양 정보

- 칼로리: 150 kcal
- 탄수화물: 18g
- 단백질: 8g
- 지방: 5g
- 나트륨: 133mg
