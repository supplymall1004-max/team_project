# 참깨흑임 자연두부샐러드

참깨흑임 자연두부샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연두부(100g)
- 참깨(20g)
- 흑임자(20g)
- 파프리카(50g)
- 무순(10g)
- 날치알(20g)
- 설탕(20g)
- 식초(10g)
- 마요네즈(30g)
- 우유(30g)

## 조리 방법

1. 1. 참깨와 흑임자를 섞어 곱게 간다.
2. 2. 갈아 놓은 참깨와 흑임자에 설탕과
식초를 넣고 섞는다.
3. 3. 2번에 우유와 마요네즈를 넣고 골고루
섞어 참깨 흑임자 드레싱을 만든다.
4. 4. 연두부는 동그랗게 자른다.
5. 5. 파프리카는 채 썬다.
6. 6. 접시에 드레싱, 연두부, 파프리카,
무순, 날치알 순으로 올린다.

## 영양 정보

- 칼로리: 309.2 kcal
- 탄수화물: 15.8g
- 단백질: 9.7g
- 지방: 23g
- 나트륨: 159.9mg
