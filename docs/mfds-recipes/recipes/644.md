# 검은콩국수

검은콩국수은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 검은콩(200g)
- 메밀면(120g)
- 오이(30g)
- 방울토마토(1개)
- 땅콩(20g)
- 잣(10g)
- 참깨(5g)

## 조리 방법

1. 1. 검은콩은 깨끗이 씻어 약 12시간 정도
충분히 불린다.
2. 2. 불린 검은콩은 20분 정도 삶아 깨끗이
씻는다.
3. 3. 삶은 검은콩은 믹서기에 곱게 갈아
차갑게 식힌다.
4. 4. 오이는 씨를 제거하고 깨끗이 씻어
채썬다.
5. 5. 물을 끓여 메밀면을 삶고, 찬물에
여러번 헹구어 건져 물기를 뺀다.
6. 6. 메밀면을 그릇에 담고 차가운 콩 국물,
땅콩, 잣, 오이, 방울토마토 및 통깨를
올린다.

## 영양 정보

- 칼로리: 343.6 kcal
- 탄수화물: 68.1g
- 단백질: 11.2g
- 지방: 3g
- 나트륨: 633.5mg
