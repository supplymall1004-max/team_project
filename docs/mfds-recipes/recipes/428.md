# 장어조림

장어조림은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 장어 150g
- 깻잎 10g
- 청고추17g
- 홍고추 17g
- 양파 60g
- 생강 15g
- 양념장 : 맛간장 25g
- 올리고당 15g
- 청주 15g
- 참기름 15g
- 흰후추 1g
- 통깨 5g
- 통마늘 20g
- 대파 10g

## 조리 방법

1. 1. 장어는 한입 크기로 잘라서 흰 후추와 청주를
뿌려 재워준다.
2. 2. 깻잎은 채를 썰어서 냉수에 담가 물기를
빼준다.
3. 3. 생강, 마늘은 편으로 썰어주고 양파, 청고추,
홍고추는 어슷하게 잘라준다.
4. 4. 참기름을 뺀 양념장 재료에 편마늘과 다진대파를
넣어서 양념장을 만들어준다.
5. 5. 만들어진 양념장 1/2에 장어를 넣고 졸여준다.
6. 6. 나머지 양념장에 준비된 채소를 넣어 볶아주다
졸인 장어를 넣어 졸여주고 깻잎을 깔고
완성한다.

## 영양 정보

- 칼로리: 190.1 kcal
- 탄수화물: 21g
- 단백질: 14.1g
- 지방: 505g
- 나트륨: 1236.7mg
