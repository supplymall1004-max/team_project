# 봄나물밥

봄나물밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수재료 : 현미(30g)
- 멥쌀(60g)
- 쑥갓(20g)
- 달래(10g)
- •두부양념장 : 고추장(5g)
- 된장(5g)
- 다시마 육수(60g)
- 두부(15g)

## 조리 방법

1. 1. 현미는 1시간 정도, 멥쌀은 30분 정도 각각 불린 뒤 섞는다.
2. 2. 쑥갓과 달래는 줄기부분과 잎으로 나누어 1㎝ 길이로 썬다.
3. 3. 쑥갓과 달래의 줄기부분은 불린 쌀과 함께 냄비에 넣고 밥물을 맞춰 센 불에서 끓어오르면 중약 불로 줄여 밥을 15분간 짓는다.
4. 4. 약한 불로 줄여 밥을 뜸 들인다.
5. 5. 다시마육수를 제외한 두부 양념장 재료를 믹서에 곱게 갈아 다시마 육수, 남겨둔 나물의 잎을 팬에 넣어 끓여 두부장을 만든다
6. 6. 나물밥에 두부장을 곁들여 마무리한다.

## 영양 정보

- 칼로리: 254.3 kcal
- 탄수화물: 54.4g
- 단백질: 6.6g
- 지방: 1.2g
- 나트륨: 138.3mg
