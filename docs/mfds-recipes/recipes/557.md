# 들깨죽

들깨죽은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 사골육수(800g)
- 들깨가루(200g)
- 마늘(5g)
- 부추(20g)
- 홍고추(1개)
- 양파(30g)
- 당근(15g)
- 찹쌀(50g)

## 조리 방법

1. 1. 찹쌀은 깨끗이 씻어 물에 약 30분 간
충분히 불린다.
2. 2. 부추, 홍고추, 양파 및 당근을 잘게
다진다.
3. 3. 마늘은 곱게 다진다.
4. 4. 냄비에 기름을 두르고 마늘을 볶다가
불린 찹쌀을 넣고 볶는다.
5. 5. 어느 정도 볶아지면 사골육수와
들깨가루를 넣고 은근히 끓인다.
6. 6. 찹쌀이 어느 정도 익으면 홍고추와
부추를 넣고 살짝 끓인다

## 영양 정보

- 칼로리: 338.7 kcal
- 탄수화물: 28.9g
- 단백질: 11.7g
- 지방: 19.6g
- 나트륨: 10.9mg
