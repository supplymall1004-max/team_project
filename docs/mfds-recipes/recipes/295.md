# 해물순두부된장찌개

해물순두부된장찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 바지락(5g)
- 주꾸미(60g)
- 마른 미역(1g)
- 마른 표고버섯(10g)
- 순두부(80g)
- 대파(5g)
- 육수 멸치(2g)
- 양념 된장(7g)
- 콩가루(10g)

## 조리 방법

1. 1. 물(250g)에 멸치를 넣고 육수를
낸다.
2. 2. 바지락은 해감해 준비하고 주꾸미는
손질한다.
3. 3. 마른 미역과 마른 표고버섯은 물에
불린다.
4. 4. 육수에 된장을 풀고, 바지락과
주꾸미를 넣어 끓인다.
5. 5. 해물이 어느 정도 익으면 미역,
표고버섯을 넣고 끓인다.
6. 6. 끓어오르면 순두부를 넣고 콩가루를
넣어 더 끓이다가 송송 썬 대파를
올려 마무리한다.

## 영양 정보

- 칼로리: 163.6 kcal
- 탄수화물: 4.63g
- 단백질: 4.87g
- 지방: 1.39g
- 나트륨: 677.9mg
