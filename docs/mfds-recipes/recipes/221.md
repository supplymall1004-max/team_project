# 구운주먹밥

구운주먹밥은(는) 밥 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 현미(100g)
- 톳(30g)
- 생김(5g)
- 달걀(50g)
- 밥 밑간 참기름(4g)
- 참깨(2g)
- 톳 무침양념 설탕(2g)
- 식초(2g)

## 조리 방법

1. 1. 현미로 밥을 지은 뒤 밥 밑간을 넣고
섞는다.
2. 2. 톳에 톳 무침양념을 넣고 버무린다.
3. 3. 밥에 톳을 넣고 버무린다.
4. 4. 밥을 먹기 좋은 크기로 둥글게 뭉친
뒤 부순 생김을 골고루 묻힌다.
5. 5. 주먹밥에 달걀물을 입힌 뒤 팬에
구워 마무리한다.

## 영양 정보

- 칼로리: 335.1 kcal
- 탄수화물: 35.43g
- 단백질: 5.79g
- 지방: 6.84g
- 나트륨: 79.4mg
