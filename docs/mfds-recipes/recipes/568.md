# 새우완자탕

새우완자탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 새우(5마리)
- 청경채(20g)
- 달걀(2개)
- 치킨육수(200g)
- 전분(30g)
- 후춧가루(0.01g)
- 요리술(20g)
- 소금(0.2g)

## 조리 방법

1. 1. 새우는 내장을 제거 하고 껍질을 벗겨
깨끗이 씻는다.
2. 2. 다진 새우에 전분, 후춧가루, 달걀흰자
및 요리술을 섞는다.
3. 3. 반죽새우를 잘 치대어 완자로 만든 후
치킨 육수에 넣고 끓인다
4. 4. 청경채는 깨끗이 씻어 반으로 길게
가른다.
5. 5. 달걀은 황백으로 나눠 지단을 부쳐
마름모꼴로 썬다.
6. 6. 완자가 익으면 물 녹말을 한 수저 넣고
청경채를 넣고 한소끔 끓인 뒤 그릇에
담고 황백지단을 올린다.

## 영양 정보

- 칼로리: 13.4 kcal
- 탄수화물: 13.2g
- 단백질: 12.2g
- 지방: 1.3g
- 나트륨: 141.6mg
