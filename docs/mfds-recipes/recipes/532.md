# 고구마바나나샐러드

고구마바나나샐러드은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 고구마(100g)
- 바나나(100g)
- 요거트(30g)
- 어린잎(20g)
- 잣(5g)
- 연유(20g)

## 조리 방법

1. 1. 고구마는 찜통에 약 20분 정도 쪄서
껍질을 벗겨 준비한다.
2. 2. 뜨거운 고구마가 한김 빠지면, 곱게
으깨 놓는다.
3. 3. 으깨진 고구마에 바나나를 썰어
요거트와 함께 넣는다.
4. 4. 3번에 다시 연유를 넣어 골고루 섞는다.
5. 5. 잣은 종이를 깔고 곱게 다져 준비한다.
6. 6. 어린잎은 물에 담그어, 건져 물기를
제거해 놓고, 으깬 고구마에 다진 잣을
넣어 샐러드를 완성한다.

## 영양 정보

- 칼로리: 348.8 kcal
- 탄수화물: 64.8g
- 단백질: 4.8g
- 지방: 7.8g
- 나트륨: 111.4mg
