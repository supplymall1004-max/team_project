# 크림소스치킨롤

크림소스치킨롤은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기(가슴살
- 150g)
- 새우(대하
- 3마리)
- 베이컨(20g)
- 마늘(20g)
- 바질(5g)
- 아스파라거스(3개)
- 새송이버섯(1개)
- 치즈(1장)
- 버터(10g)
- 소금(0.2g)
- 후춧가루(0.01g)

## 조리 방법

1. 1. 닭가슴살은 포를 떠 소금과 후춧가루로
밑간을 한다.
2. 2. 새우는 껍질을 벗겨 손질한다.
3. 3. 베이컨은 끓는 물에 데친다.
4. 4. 마늘과 바질은 다진 뒤 버터에 섞는다.
5. 5. 닭가슴살에 베이컨과 다진 버터를
올린다.
6. 6. 냄비에 마늘과 바질을 섞은 버터,
치즈를 넣고 끓이다가 ?를 놓고
졸인다.

## 영양 정보

- 칼로리: 401 kcal
- 탄수화물: 10.9g
- 단백질: 24.6g
- 지방: 28.8g
- 나트륨: 262.9mg
