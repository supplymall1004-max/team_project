# 채소어묵

채소어묵은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 생선살(100g)
- 맛술(10g)
- 생강(10g)
- 소금(0.2g)
- 밀가루(20g)
- 양파(20g)
- 당근(20g)
- 샐러리(10g)
- 마늘(10g)
- 옥수수(20g)
- 어린이치즈(20g)
- 튀김기름(400g)

## 조리 방법

1. 1. 생선살은 가시를 발라내고 곱게 으깨어
놓는다.
2. 2. 으깬 생선살에 맛술 과 생강즙, 소금을
넣어 골고루 치댄다.
3. 3. 잘 치댄 생선살에 밀가루를 넣어 다시
치대어 준다.
4. 4. 양파와 당근, 샐러리, 마늘은 곱게
다지고, 옥수수는 알맹이를 준비한다.
5. 5. 생선살에 다진 채소와 어린이 치즈를
다져 섞는다.
6. 6. 채소를 다져 섞은 생선살을 동글동글
하게 완자를 빚어 튀김기름에 약 150℃
온도에서 은근히 5분 정도 튀겨 완성
한다.

## 영양 정보

- 칼로리: 382.5 kcal
- 탄수화물: 39.89g
- 단백질: 26.7g
- 지방: 12.9g
- 나트륨: 320.1mg
