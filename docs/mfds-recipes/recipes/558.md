# 레몬크림콩다식

레몬크림콩다식은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 콩가루(100g)
- 꿀(30g)
- 레몬(1/2개)
- 생크림(30g)
- 설탕(20g)
- 레몬제스트(20g)

## 조리 방법

1. 1. 콩가루에 꿀을 넣고 반죽한다.
2. 2. 섞어 놓은 콩가루 반죽은 고운 천으로
감싸 숙성을 시킨다.
3. 3. 레몬은 즙을 짠다.
4. 4. 생크림에 설탕과 레몬즙을 넣고 휘핑해
레몬크림을 만든다
5. 5. 콩다식 판에 숙성된 콩반죽을 넣고
모양을 만든다.
6. 6. 콩다식에 레몬크림을 올리고
레몬제스트로 장식한다.

## 영양 정보

- 칼로리: 527.1 kcal
- 탄수화물: 83.5g
- 단백질: 18.4g
- 지방: 13.3g
- 나트륨: 206.4mg
