# 전복리조또

전복리조또은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 쌀(100g)
- 전복(1개)
- 당근(20g)
- 양파(30g)
- 양송이버섯(3개)
- 우유(200g)
- 올리브오일(10g)
- 소금(0.2g)
- 후춧가루(0.01g)
- 버터(10g)

## 조리 방법

1. 1. 쌀은 깨끗이 씻어 30분 정도 불린다
2. 2. 전복은 수저로 떼어 내어 소금으로
세척한다.
3. 3. 준비된 전복에 버터를 두르고 굽는다
4. 4. 당근. 양파 . 양송이는 입자 있게
다진다.
5. 5. 냄비에 올리브 오일을 넣고 불린 쌀을
볶는다.
6. 6. 쌀이 어느 정도 볶아지면 다져 놓은
야채를 넣어 볶다가 우유를 넣어
끓이고, 쌀이 퍼지면 구워 놓은 전복을
넣어 다시 끓인다.

## 영양 정보

- 칼로리: 282.9 kcal
- 탄수화물: 53.1g
- 단백질: 7.1g
- 지방: 4.7g
- 나트륨: 92.1mg
