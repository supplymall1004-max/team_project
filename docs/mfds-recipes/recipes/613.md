# 인절미 무스와 수정과 아이스크림

인절미 무스와 수정과 아이스크림은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 홍시(1개)
- 콩가루(20g)
- 계피(10g)
- 생강(20g)
- 호두(10g)
- 곶감(2개)
- 자몽(1/6개)
- 판젤라틴(1장)

## 조리 방법

1. 1. 곶감은 씨를 발라낸다.
2. 2. 생강과 계피를 넣고 끓여 수정과를
만든 뒤 냉동한다.
3. 3. 홍시는 믹서로 곱게 갈고 틀어 넣어
굳힌다.
4. 4. 생크림은 차갑게 휘핑한다.
5. 5. 불려 놓은 젤라틴을 굳혀 콩가루를
섞어 인절미를 만든다
6. 6. 자몽을 껍질을 제거 후 갈아서 젤리를
만들고, 접시에 굳혀놓은 홍시와
아이스 수정과를 담는다

## 영양 정보

- 칼로리: 126.1 kcal
- 탄수화물: 30.5g
- 단백질: 0.9g
- 지방: 0g
- 나트륨: 44.1mg
