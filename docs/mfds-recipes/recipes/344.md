# 크렌베리귀리밥

크렌베리귀리밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 백미 150g
- 크렌베리 15g
- 귀리 20g

## 조리 방법

1. 1. 백미와 귀리는 세척해 백미는 30분, 귀리는
1시간 정도 불려준다.
2. 2. 귀리를 먼저 물을 넣어 볶아 충분히 익혀준다.
3. 3. 귀리가 어느 정도 익을 때 쯤 쌀을 섞어 한 번
더 볶아준다.
4. 4. 귀뢰와 쌀이 익어갈 때 쌀의 물 1배를 넣고
밥을 해 준다.
5. 5. 불을 끄고 크린베리를 바로 넣어서 뚜껑을
덮어 5분 후에 밥을 완성한다.

## 영양 정보

- 칼로리: 494.2 kcal
- 탄수화물: 104.3g
- 단백질: 9.6g
- 지방: 4.3g
- 나트륨: 22.4mg
