# 호박죽

호박죽은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 단호박 200g
- 찹쌀가루 100g
- 브로콜리 50g
- 양파 60g
- 볶은 현미 25g
- 버섯마늘소금 2g
- 흰후추 1g
- 치즈가루 15g

## 조리 방법

1. 1. 단호박은 껍질을 까고 작게 잘라주고, 양파도
작게 잘라준다.
2. 2. 브로콜리는 뜨거운 물에 데쳐내 준 후 송송
잘라준다.
3. 3. 찹쌀가루는 물 100g에 풀어준다.
4. 4. 단호박과 양파는 물 300g에 삶아준다.
5. 5. 단호박과 양파가 익으면 믹서에 갈아서 다시
끓여준 뒤 찹쌀가루물을 넣어 끓여준다.
6. 6. 호박죽이 끓어오르면 치즈가루를 넣어준 후
볶은 현미랑 완성한다.

## 영양 정보

- 칼로리: 150.1 kcal
- 탄수화물: 26.8g
- 단백질: 5.6g
- 지방: 2.3g
- 나트륨: 101.4mg
