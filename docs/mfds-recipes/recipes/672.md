# 해물애호박 전병말이

해물애호박 전병말이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 새우(3마리)
- 오징어(50g)
- 애호박(1/2개)
- 당근(20g)
- 깻잎(3장)
- 대추(2알)
- 밀가루(100g)
- 찹쌀가루(50g)
- 저염간장(20g)
- 식초(10g)
- 설탕(10g)
- 소금(0.2g)

## 조리 방법

1. 1. 새우는 껍질을 벗겨 다진다.
2. 2. 오징어는 소금으로 껍질을 벗겨
다진다.
3. 3. 애호박, 깻잎, 대추는 곱게 채썬다.
4. 4. 채썬 애호박과 깻잎은 소금에 살짝
절인 뒤 물기를 짠다.
5. 5. 밀가루에 찹쌀가루, 애호박, 깻잎,
대추채와 다진 새우, 오징어를 넣고
팬에 기름을 둘러 전병을 만든 뒤 돌돌
말아 썰어 접시에 담는다.
6. 6. 저염간장과 식초, 설탕을 넣고 소스를
만든다.

## 영양 정보

- 칼로리: 408.4 kcal
- 탄수화물: 66.2g
- 단백질: 10.7g
- 지방: 11.2g
- 나트륨: 256.8mg
