# 닭고기스테이크

닭고기스테이크은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기살(120g)
- 소금(0.5g)
- 후춧가루(0.1g)
- 생강(5g)
- 정종(5g)
- 우유(50g)
- 오이(20g)
- 토마토(20g)
- 두부(80g)
- 흑임자(20g)
- 우유(50g)

## 조리 방법

1. 1. 닭고기살은 소금과 후춧가루, 생강즙과
정종을 뿌린 후 우유를 살짝 발라 숙성을
시킨다.
2. 2. 오이는 길게 껍질째 썰어 놓는다.
3. 3. 토마토는 반으로 잘라 편으로 썰어
준비한다.
4. 4. 두부는 소창으로 물기를 꼭 짜 놓는다.
5. 5. 물기를 제거 시킨 두부에 흑임자를
넣고 골고루 섞은 뒤 냄비에 넣고
우유와 함께 은근히 끓여 속을 만든다.
6. 6. 팬에 토마토를 먼저 구워 접시에 담고,
숙성된 닭고기를 구워 올린 뒤, 그 위에
오이로 원형기둥을 만들고 구운
닭고기를 다시 올려 완성한다.

## 영양 정보

- 칼로리: 222.4 kcal
- 탄수화물: 8g
- 단백질: 26.3g
- 지방: 9.5g
- 나트륨: 226.1mg
