# 수박껍질풋고추볶음

수박껍질풋고추볶음은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 수박 속껍질 50g
- 풋고추 15g
- 홍고추 5g
- 포도씨유 4g
- 소금 1g(절일 때 0.5g
- 간 맞출 때 0.5g)

## 조리 방법

1. 1. 수박껍질의 녹색부분을 제거하고 얄팍하게 채 썬다.
2. 2. 채 썬 수박껍질에 소금을 뿌려 살짝 절여 물기를 짠다.
3. 3. 고추는 길이로 2등분하여 씨를 뺀다.
4. 4. 씨를 뺀 고추는 굵게 채 썬다.
5. 5. 팬에 기름을 두르고 수박껍질과 고추를 넣어 살짝 볶는다.
6. 6. 소금으로 간을 맞추고 마무리한다.

## 영양 정보

- 칼로리: 41.2 kcal
- 탄수화물: 1.2g
- 단백질: 0.6g
- 지방: 4g
- 나트륨: 339.1mg
