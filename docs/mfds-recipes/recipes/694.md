# 미숫가루 부추전

미숫가루 부추전은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 부침가루 30g
- 미숫가루 10g
- 오징어 20g
- 부추 20g
- 감자 15g
- 당근 5g
- 양파 5g
- 현미유 4g
- 소금 0.5g
- 물 60g

## 조리 방법

1. 1. 오징어는 내장을 제거하고 깨끗하고 씻어낸다.
2. 2. 부추, 감자, 당근, 양파는 2cm 가량으로 곱게 채 썬다.
3. 3. 부침가루와 미숫가루를 3 : 1의 비율로 섞는다.
4. 4.  1, 2, 3에 물을 넣어 반죽하고 소금으로 간을 한다.
5. 5. 팬에 현미유를 두르고 3의 반죽을 둥글게 떠 넣어 노릇노릇하게 지진다.
6. 6. 그릇에 담아 완성한다.

## 영양 정보

- 칼로리: 212.8 kcal
- 탄수화물: 33g
- 단백질: 11.8g
- 지방: 4.9g
- 나트륨: 228.7mg
