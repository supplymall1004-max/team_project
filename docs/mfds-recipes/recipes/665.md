# 연근부각

연근부각은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 연근(60g)
- 식초(10g)
- 천일염(0.1g)
- 계피가루(20g)
- 식용유(200g)

## 조리 방법

1. 1. 연근은 껍질을 벗긴다.
2. 2. 껍질벗긴 연근은 얇게 썬다.
3. 3. 물 200g에 식초 10g을 섞는다
4. 4. 식촛물에 썬 연근을 담근다.
5. 5. 연근을 체에 건져 물기를 뺀다.
6. 6. 연근을 바삭하게 튀긴 뒤 천일염을
살짝 뿌리고, 계피가루를 묻힌다.

## 영양 정보

- 칼로리: 162.2 kcal
- 탄수화물: 25.3g
- 단백질: 2.6g
- 지방: 5.6g
- 나트륨: 67.7mg
