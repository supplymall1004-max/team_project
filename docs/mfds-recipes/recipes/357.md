# 삼합잡채

삼합잡채은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 주꾸미 50g
- 바지락살 25g
- 소고기 20g
- 닭가슴살 20g
- 양파 25g
- 청고추 15g
- 홍고추 15g
- 당근 20g
- 미나리 20g
- 양념장 : 맛간장 25g
- 마늘기름 15g
- 다진마늘 5g
- 통깨 1g
- 참기름 5g

## 조리 방법

1. 1. 주꾸미는 먹기 좋게 잘라주고, 바지락살은 물에
데쳐서 수분을 빼고 생강청으로 비린내를 제
거한 뒤 볶아준다.
2. 2. 소고기와 닭가슴살은 각각 양념해준다.
3. 3. 양파, 청고추, 홍고추, 당근은 채 썰어 함께
볶아준다.
4. 4. 볶아 놓은 채소와 고기, 해산물은 양념장에
버무린다.
5. 5. 미나리와 배를 곱게 채 썰어 올려서 완성한다.

## 영양 정보

- 칼로리: 160.4 kcal
- 탄수화물: 3.8g
- 단백질: 22.9g
- 지방: 5.9g
- 나트륨: 155.9mg
