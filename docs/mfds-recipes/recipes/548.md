# 소고기채소불고기

소고기채소불고기은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- •필수 재료 : 소불고기(120g)
- 배(20g)
- 정종(5g) 양배추(10g)
- 양파(10g)
- 깻잎(5g)
- 표고버섯(10g) 당근(10g)
- 대파(5g)
- 청고추(5g)
- 홍고추(5g)
- •곁들임 채소 : 대추(10g)
- 통깨(1g)
- •소스 : 저염간장(5g)
- 올리고당(5g)
- 참기름(2g) 다진마늘(5g)
- 후춧가루(0.1g)

## 조리 방법

1. 1. 배는 껍질을 벗겨 배즙을 갈아 준비한다.
2. 2. 소고기에 배즙과 정종을 넣어 버무려 숙성시킨다.
3. 3. 양배추와 양파, 깻잎, 표고버섯은 채 썰고 대파와 당근은 납작하게 5cm 길이로 썰고 청·홍고추는 어슷 썰어 준비한다.
4. 4. 간장과 올리고당, 참기름, 다진마늘, 후춧가루를 넣어 소스를 만든다.
5. 5. 만든 소스를 소불고기에 넣어 2차 숙성을 시킨다.
6. 6. 팬에 고기를 볶고 별도로 채소를 볶은 뒤 씨를 빼고 얇게 썬 대추와 통깨를 얹어 접시에 함께 담아 완성한다.

## 영양 정보

- 탄수화물: 8.7g
- 단백질: 34.1g
- 지방: 23.4g
- 나트륨: 196.8mg
