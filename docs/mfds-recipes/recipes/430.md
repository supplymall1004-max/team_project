# 저염된장 삼치구이

저염된장 삼치구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- •필수재료 : 삼치(140g)
- 치커리(35g)
- 전분가루(10g)
- •구이소스 : 일본된장(5g)
- 청주(5g)
- •곁들임소스 : 유자청(15g)
- 다진마늘(5g)
- 청고추(15g)
- 홍고추(20g)

## 조리 방법

1. 1. 삼치는 세척 후 전분가루로 옷을 입혀준다.
2. 2. 일본된장과 청주를 섞어 구이소스를 만들어 삼치를 재운다.
3. 3. 청고추와 홍고추는 곱게 다지고 치커리는 적당하게 잘라서 물에 담가준다.
4. 4. 유자청, 다진마늘, 다진 청고추와 홍고추를 섞어 곁들임소스를 만든다.
5. 5. 재워두었던 삼치를 굽는다.
6. 6. 접시에 구운 삼치와 치커리를 올리고 치커리를 올리고 곁들임소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 196.3 kcal
- 탄수화물: 5.8g
- 단백질: 18.9g
- 지방: 10.8g
- 나트륨: 168.8mg
