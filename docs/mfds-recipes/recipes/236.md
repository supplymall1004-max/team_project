# 닭가슴살카나페

닭가슴살카나페은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 고구마(100g)
- 우유(110g)
- 백김치(40g)
- 비트(40g)
- 닭가슴살(100g)
- 후춧가루(0.3g)
- 새싹(1g)
- 바나나식초(80g)

## 조리 방법

1. 1. 고구마를 삶아 껍질을 벗겨 으깬 뒤
우유와 섞어 고구마무스를 만든다.
2. 2. 백김치와 비트를 작게 썬 뒤
백김치에 물이 들도록 절여 놓는다.
3. 3. 닭가슴살을 한입 크기로 썬 후
칼등으로 두드린 뒤 후춧가루로
밑간한다.
4. 4. 식용유(15g)를 두른 팬에 닭가슴살을
올려 굽는다.
5. 5. 구운 닭가슴살 위에 고구마무스,
비트에 절인 백김치, 새싹을 얹는다.
6. 6. 그릇에 닭가슴살카나페를 넣은 후
바나나식초를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 281.4 kcal
- 탄수화물: 16.6g
- 단백질: 14.8g
- 지방: 1.96g
- 나트륨: 202.1mg
