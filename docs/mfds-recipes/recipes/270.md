# 해물토마토김치찌개

해물토마토김치찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 김치(40g)
- 토마토(30g)
- 양파(40g)
- 오징어(15g)
- 홍합(20g)
- 바지락(15g)
- 주꾸미(25g)
- 데친 미나리(2g)
- 쑥갓(5g)
- 느타리버섯(7g)
- 팽이버섯(7g)
- 어슷 썬 풋고추(1g)
- 김치주머니 소 새우(15g)
- 부추(3g)
- 불린 당면(7g)
- 불린 무말랭이(3g)
- 녹말가루(5g)
- 달걀(23g)

## 조리 방법

1. 1. 김치를 씻은 뒤 줄기부분은 한입
크기로 썰고, 잎은 따로 두고,
토마토, 양파도 비슷한 크기로 썬다.
2. 2. 오징어는 칼집을 넣고, 홍합,
바지락, 주꾸미와 함께 살짝 데친 뒤
식용유(10g) 두른 팬에 볶는다.
3. 3. 새우, 부추, 불린 당면과 무말랭이는
다진 뒤 녹말가루와 달걀을 넣어
반죽해 김치주머니 소를 만든다.
4. 4. 김치 잎에 소를 넣고 주머니
모양으로 감싸 데친 미나리로
묶는다.
5. 5. 김치와 채소를 볶아 냄비에 담은 뒤
해산물과 김치주머니를 넣고
끓이다가 끓어오르면 쑥갓,
느타리버섯, 팽이버섯, 고추를 올려
마무리한다.

## 영양 정보

- 칼로리: 88.2 kcal
- 탄수화물: 2g
- 단백질: 1g
- 지방: 0.5g
- 나트륨: 566.5mg
