# 삼겹가지볶음

삼겹가지볶음은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 삼겹살(100g)
- 생강(5g)
- 정종(10g)
- 가지(50g)
- 양파(20g)
- 당근(20g)
- 깻잎(10g)
- 느타리버섯(20g)
- 대파(10g)
- 식용유(5g)
- 참기름(5g)
- 잣(10g)
- 통깨(5g)
- - 양념장 : 고춧가루(20g)
- 다진마늘(20g)
- 저염간장(30g)
- 올리고당(20g)
- 후춧가루(0.1g)
- 설탕(10g)

## 조리 방법

1. 1. 썰어 놓은 삼겹살에 생갑즙과 정종을
넣고 약 2시간 정도 숙성을 시켜 준비
한다.
2. 2. 가지는 씨가 적은 것으로 골라 반으로
갈라 어슷썰어 놓는다.
3. 3. 양파와 깻잎은 채를 썰고, 당근은
직사각형으로 썰고, 대파는 송송 썰고,
느타리는 손으로 찢어 준비한다.
4. 4. 고춧가루과 다진마늘, 간장, 올리고당,
후춧가루, 설탕을 넣고 골고루 섞어
양념장을 만든다.
5. 5. 숙성시켜 놓은 삼겹살에 만들어 놓은
양념장을 넣고 골고루 무쳐 놓는다.
6. 6. 팬에 기름을 살짝 둘러 양념한 고기(⑤)
를 먼저 볶은 후 고기가 중간쯤 익으면,
가지(②)와 썰어 놓은 채소(③)를 넣어
볶다가 마지막에 참기름과 잣을 넣고
한 번 더 볶은 후 통깨를 올려 완성한다.

## 영양 정보

- 칼로리: 542.4 kcal
- 탄수화물: 15.9g
- 단백질: 19.5g
- 지방: 44.5g
- 나트륨: 398.8mg
