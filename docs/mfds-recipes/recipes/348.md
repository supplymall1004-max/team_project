# 초계샐러드

초계샐러드은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭가슴살 250g
- 달걀 60g
- 오이 25g
- 피망 25g
- 파프리카 25g
- 시금치 50g
- 소스 : 잣 20g
- 식초 15g
- 연겨자 15g
- 설탕 15g
- 참깨 15g
- 흰후추 1g
- 고명 : 대추 10g

## 조리 방법

1. 1. 닭가슴살은 데친 후 가늘게 찢어준다.
2. 2. 달걀은 황백지단을 부쳐서 다이아몬드 모양
으로 잘라준다.
3. 3. 오이, 피망, 파프리카도 마름모로 잘라준다.
4. 4. 시금치는 살짝 데쳐서 수분을 짜서 한입
크기로 잘라준다.
5. 5. 소스 재료를 믹서에 갈아 소스를 만들어준다.
6. 6. 대추는 돌려깎기 해서 잘라주고 준비된 모든
재료를 섞어 샐러드를 만들고 대추채를 올려
완성한다.

## 영양 정보

- 칼로리: 116.8 kcal
- 탄수화물: 5g
- 단백질: 14.2g
- 지방: 4.5g
- 나트륨: 63.9mg
