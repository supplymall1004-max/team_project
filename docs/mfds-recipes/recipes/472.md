# 맥적구이

맥적구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 돼지고기(120g)
- 생강(10g)
- 소금(0.2g)
- 후춧가루(0.02g)
- 저염간장(20g)
- 정종(10g)
- 설탕(20g)
- 대파(10g)
- 마늘(20g)
- 양파(20g)
- 참기름(5g)
- 통깨(3g)
- 새송이버섯(20g)
- 표고버섯(20g)
- 가지(20g)
- 홍피망(20g)
- - 된장소스 : 된장(10g)
- 마요네즈(20g)
- 생크림(10g)
- 다진 땅콩(20g)

## 조리 방법

1. 1. 돼지고기에 생강즙과 소금, 후춧가루를
넣어 숙성한다.
2. 2. 숙성된 돼지고기에 간장과 정종, 설탕,
파, 마늘, 양파, 참기름, 통깨 넣어
골고루 양념한다.
3. 3. 새송이, 가지, 표고, 홍피망을 편으로
썰어 준비한다.
4. 4. 된장에 마요네즈와 생크림, 다진
땅콩을 섞어놓는다.
5. 5. 새송이와 가지, 표고, 홍피망은 팬에서
굽는다.
6. 6. 팬이 따끈해지면 양념한 돼지고기를
구워 접시에 담고 그 위에 된장 소스를
올리고, 구워낸 채소를 다시 위에 올려
완성한다.

## 영양 정보

- 칼로리: 467.7 kcal
- 탄수화물: 17.6g
- 단백질: 30.7g
- 지방: 30.5g
- 나트륨: 551.7mg
