# 인삼갈비탕

인삼갈비탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 소고기(갈비
- 300g)
- 인삼(2뿌리)
- 당면(50g)
- 표고버섯(1개)
- 달걀(1개)
- 대추(3알)
- 양파(50g)
- 파(30g)
- 마늘(30g)
- 설탕(10g)
- 저염간장(20g)

## 조리 방법

1. 1. 갈비는 찬물에 담궈 핏물을 뺀다.
2. 2. 끓은 물에 갈비를 데친다.
3. 3. 찬물에 표고버섯, 파, 마늘, 양파 및
인삼을 넣고 데친 갈비를 넣고 약 2시간
정도 끓여 갈비탕을 만든다.
4. 4. 저염간장에 설탕을 섞어 고기소스를
만든다.
5. 5. 당면은 뜨거운 물에 불려 갈비탕을
먹기 직전에 넣는다.
6. 6. 달걀은 황백으로 나눠 지단을 부쳐
완성된 갈비탕 위에 대추와 함께 올려
완성한다.

## 영양 정보

- 칼로리: 361.2 kcal
- 탄수화물: 25.5g
- 단백질: 36.9g
- 지방: 12.4g
- 나트륨: 161.2mg
