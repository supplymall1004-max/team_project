# 곤약잡채

곤약잡채은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- •필수 재료 : 실곤약(50g)
- 당근(10g)
- 오이(10g)
- 양파(10g)
- 팽이버섯(5g)
- 청고추(10g)
- 홍고추(10g)
- 영양부추(10g)
- 달걀(30g)
- 식용유(3g)
- •소스 : 맛간장(5g)
- 흑설탕(5g)
- 다진마늘(2g)
- 다진대파(2g)
- 흰후추(0.5g)
- •양념 : 참기름(3g)
- 통깨(3g)
- 레몬즙(10g)

## 조리 방법

1. 1. 실곤약은 세척해 물기를 빼준 뒤 레몬즙을 살짝 뿌려준다.
2. 2. 당근, 양파, 홍고추, 청고추는 채 썰어 볶는다.
3. 3. 오이는 돌려깎기 해주고 팽이와 영양부추는 적당한 크기로 잘라 각각 살짝 볶아 건진다.
4. 4. 달걀은 황백지단으로 만들어 채 썰어준다.
5. 5. 소스를 팬에 만들어 준 뒤 곤약을 볶아준다.
6. 6. 볶은 곤약에 ②를 넣고 한 번 더 볶은 뒤 ③을 섞어 그릇에 담고, 지단채를 올리고 참기름, 통깨를 뿌려 마무리한다.

## 영양 정보

- 칼로리: 49.3 kcal
- 탄수화물: 4.4g
- 단백질: 1.6g
- 지방: 2.8g
- 나트륨: 75.2mg
