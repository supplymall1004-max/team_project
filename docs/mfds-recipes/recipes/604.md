# 아보카도 연어샐러드

아보카도 연어샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 연어(훈제연어
- 120g)
- 양파(1/2개)
- 아보카도(1/2개)
- 파프리카(1/2개)
- 오렌지(1/2개)
- 사워크림(20g)
- 어린잎(20g)

## 조리 방법

1. 1. 연어는 주사위 모양으로 썬다.
2. 2. 양파, 파프리카도 주사위 모양으로
썬다.
3. 3. 아보카도와 오렌지도 같은 형태로
썬다.
4. 4. 사워크림에 다진 양파를 섞어 소스를
만든다.
5. 5. 어린잎은 찬물에 담궈 싱싱하게 한다.
6. 6. 원형 몰드에 연어를 넣고, 옆으로
야채와 과일을 담고 그 위에
사워크림소스와 어린잎을 올린다.

## 영양 정보

- 칼로리: 239 kcal
- 탄수화물: 9.8g
- 단백질: 16.5g
- 지방: 14.9g
- 나트륨: 102.1mg
