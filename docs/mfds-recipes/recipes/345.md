# 블루베리냉스프

블루베리냉스프은(는) 일품 요리로, 기타 조리법을 사용합니다.

## 재료

- 검은콩두유 50g
- 꿀 15g
- 딸기 20g
- 블루베리 20g
- 저지방우유 50g
- 볶은현미 20g
- 바나나 50g
- 고명 : 브로콜리 15g
- 라이스페이퍼 1장
- 건조체리 15g

## 조리 방법

1. 1. 브로콜리는 먹기 좋게 잘라서 데쳐준다.
2. 2. 건조체리는 손질해 씨를 제거하고 송송 다져서
먹기 좋게 잘라준다.
3. 3. 라이스페이퍼를 뜨거운 물에 데쳐내 체리를
말아 체리 만두를 만든다.
4. 4. 만들어진 체리 만두를 팬에 살짝 구워서
라이스페이퍼를 고소하게 만든다.
5. 5. 블루베리, 딸기, 바나나, 검은콩두유와 우유를
한데 섞어 믹서에 갈아준다.
6. 6. 갈아둔 스프를 담고 브로콜리와 체리 만두를
올리고 먹기 전에 볶은 현미를 올려준다.

## 영양 정보

- 칼로리: 115.7 kcal
- 탄수화물: 20.6g
- 단백질: 2.2g
- 지방: 2.7g
- 나트륨: 32.8mg
