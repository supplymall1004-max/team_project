# 밥핫도그

밥핫도그은(는) 밥 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 소시지(100g)
- 나무막대기(5개)
- 밀가루(30g)
- 달걀(50g)
- 빵가루(30g)
- 튀김기름(200g)
- 파프리카(30g)
- 밥(50g)
- 감자(50g)
- 달걀흰자(10g)
- 소금(0.3g)

## 조리 방법

1. 1. 소시지는 먹기 좋은 크기로 썰어 나무
막대기에 꽂아놓는다.
2. 2. 파프리카는 작게 다져 준비한다.
3. 3. 으깬 감자를 준비된 밥과 함께 골고루
섞는다.
4. 4. 으깬 감자를 다시 준비된 밥과 함께
소금을 조금 넣고 골고루 섞는다.
5. 5. 밥과 감자가 잘 섞어지면, 다시 다져진
파프리카를 넣고 섞어준다.
6. 6. 막대기에 밀가루를 살짝 바르고, 소시
지에 반죽(⑤)을 감싸듯 손으로 둥글게
말아 밀가루와, 달걀물, 빵가루 순서로
입혀 튀김 기름 170℃의 온도에서
약 2분 정도 튀긴 후 기름을 제거하여
완성한다.

## 영양 정보

- 칼로리: 828.9 kcal
- 탄수화물: 74g
- 단백질: 22.1g
- 지방: 49.4g
- 나트륨: 774.5mg
