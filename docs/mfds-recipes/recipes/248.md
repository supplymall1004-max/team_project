# 떡갈비찜

떡갈비찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 재료 다진 돼지고기(50g)
- 다진 쇠고기(30g)
- 가지(10g)
- 가래떡(60g)
- 유자청(15g)
- 간장(5g)
- 연근(4g)
- 고기 밑간 다진 파(5g)
- 다진 마늘(5g)
- 다진 양파(10g)
- 소금(1g)
- 후춧가루(1g)
- 참기름(3g)

## 조리 방법

1. 1. 돼지고기와 쇠고기는 고기 밑간을
넣고 반죽을 만든다.
2. 2. 가지를 길고 얇게 썬다.
3. 3. 깨끗이 씻은 떡을 가지로 돌돌 말고
그 위에 고기를 말아 떡갈비를
만들고 팬에서 겉면이 노릇해질
때까지만 익힌 뒤 김 오른 찜기에
넣어 10분~15분 정도 찐다.
4. 4. 유자청과 간장을 섞고 팬에서
농도를 맞춰주며 살짝 졸여
유자소스를 만든다.
5. 5. 연근은 모양대로 얇게 썰어 기름에
튀겨 연근칩을 만든다.
6. 6. 떡갈비에 유자소스를 바르고
연근칩을 올려 마무리한다.

## 영양 정보

- 칼로리: 316.2 kcal
- 탄수화물: 29.2g
- 단백질: 10.2g
- 지방: 6.19g
- 나트륨: 264.9mg
