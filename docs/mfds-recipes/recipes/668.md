# 컬리플라워소스 광어스테이크

컬리플라워소스 광어스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 광어(1마리)
- 콜리플라워(50g)
- 감자(30g)
- 양파(20g)
- 시금치(20g)
- 가지(1/2개)
- 단호박(1/4개)
- 당근(30g)
- 우유(200g)
- 소금(0.2g)
- 후춧가루(0.01g)

## 조리 방법

1. 1. 광어는 비늘을 벗기고 포를 떠 소금,
후춧가루를 뿌린다.
2. 2. 콜리플라워는 팬에 볶는다.
3. 3. 감자와 양파를 썰어 콜리플라워와 함께
볶는다.
4. 4. ?번에 우유를 넣는다.
5. 5. ?를 믹서로 곱게 간 뒤, 끓인다.
6. 6. 가지, 당근. 단호박은 둥근 모양으로
만들어 팬에 시금치와 함께 익히고,
광어도 팬에 구워 접시에 담아
컬리플라워 소스를 올린다.

## 영양 정보

- 칼로리: 99.8 kcal
- 탄수화물: 10.8g
- 단백질: 10.6g
- 지방: 1.6g
- 나트륨: 167.7mg
