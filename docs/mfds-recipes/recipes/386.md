# 다이어트국수

다이어트국수은(는) 일품 요리로, 기타 조리법을 사용합니다.

## 재료

- 실곤약 200g
- 물김치국물 200g
- 오이 25g
- 달걀 1개(60g)
- 당근 20g
- 양파 50g
- 통깨 3g
- 레몬즙 15g
- 탄산수 50ml
- 어간장 5g

## 조리 방법

1. 1. 양파, 당근, 오이는 채 썰어준다.
2. 2. 열무 물김치와 탄산수, 어간장을 넣어 국수
국물을 만들어준다.
3. 3. 달걀은 삶아준다.
4. 4. 실곤약은 삶을 때 레몬즙을 넣어서 곤약의
특이한 냄새를 제거한다.
5. 5. 물기를 제거한 곤약에 어간장을 넣어 밑간을
해준다.
6. 6. 그릇에 담고 국물을 부어주고, 채소와 달걀을
곁들여 완성한다.

## 영양 정보

- 칼로리: 36.9 kcal
- 탄수화물: 6.1g
- 단백질: 0.1g
- 지방: 1.4g
- 나트륨: 490.7mg
