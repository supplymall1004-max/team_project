# 주꾸미머리순대

주꾸미머리순대은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 주꾸미 200g
- 미나리 20g
- 양파 25g
- 당근 20g
- 계란 60g
- 다진마늘 15g
- 생강청 15g
- 레몬즙 10g
- 소스 : 고추장 15g
- 레몬즙 10g
- 생강청 10g
- 통깨 3g

## 조리 방법

1. 1. 주꾸미를 몸과 다리를 분리 후 잘 씻어준다.
2. 2. 주꾸미 다리, 미나리, 양파, 당근을 다져준다.
3. 3. 다진 채소를 계란, 다진마늘, 생강청에 양념
한다.
4. 4. 양념한 3을 주꾸미 머리 속에 넣어주고
이쑤시개로 꽂아준다.
5. 5. 찜기에 찐다. 찔 때 레몬즙을 뿌리면 더욱
탱탱하다.
6. 6. 소스를 만든 뒤 주꾸미를 함께 완성한다.

## 영양 정보

- 칼로리: 60 kcal
- 탄수화물: 0.8g
- 단백질: 11.8g
- 지방: 1.1g
- 나트륨: 122mg
