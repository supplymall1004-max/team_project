# 조개크림파스타

조개크림파스타은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 스파게티(100g)
- 올리브오일(3g)
- 소금(0.3g)
- 양송이(20g)
- 새우(50g)
- 파슬리가루(0.3g)
- 버터(20g)
- 양파(20g)
- 마늘(20g)
- 밀가루(20g)
- 육수(100g)
- 우유(100g)
- 생크림(20g)
- - 육수 : 조개(30g)
- 물(100g)

## 조리 방법

1. 1. 스파게티는 끓는 물에 소금을 넣고 약
10분 정도 삶아 건져 올리브 오일에
살짝 발라 준비한다.
2. 2. 양송이는 껍질을 벗겨 편으로 썰어
놓는다.
3. 3. 마늘과 양파는 굵게 다져놓는다.
4. 4. 조개는 끓는 물에 데쳐 해감을 씻고,
삶은 물을 소창에 걸러 육수를 만들어
놓는다.
5. 5. 냄비에 버터를 넣고, 다진 마늘과
양파를 넣어 볶다가 밀가루를 넣고
은근히 볶는다.
6. 6. 볶고 있는 채소(⑤)에 육수(④)를 부어
한소끔 끓인 뒤, 우유와 생크림을 넣고
다시 끓으면 새우와 삶은 조개를 넣고
살짝 끓인 후, 삶아놓은 스파게티면
(①)을 넣고 한 번 더 끓이고 그 위에
파슬리 가루를 살짝 뿌려 완성한다.

## 영양 정보

- 칼로리: 224.2 kcal
- 탄수화물: 36.2g
- 단백질: 10.9g
- 지방: 4g
- 나트륨: 102.5mg
