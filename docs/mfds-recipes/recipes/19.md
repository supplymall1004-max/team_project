# 백일송이버섯 볶음

백일송이버섯 볶음은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 백일송이버섯볶음
- 백일송이버섯 65g(1컵)
- 노랑 파프리카 5g(5×1cm)
- 빨강 파프리카 5g(5×1cm)
- 카놀라유 3g(2/3작은술)

## 조리 방법

1. 1. 파프리카는 씨를 빼고 버섯 크기와 비슷하게 썬다.a
2. 2. 달구어진 팬에 카놀라유를 두르고 백일송이 버섯을 볶는다.b
3. 3. 버섯이 살짝 숨이 죽으면 파프리카를 넣어 함께 볶아 접시에 담는다.c

## 영양 정보

- 칼로리: 80 kcal
- 탄수화물: 10g
- 단백질: 3g
- 지방: 3g
- 나트륨: 11mg
