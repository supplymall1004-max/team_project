# 토마토맑은장국

토마토맑은장국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수 재료 : 방울토마토(100g)
- 시금치(30g)
- 알배추(25g)
- 팽이버섯(5g)
- •육수 : 닭육수(250g)
- •양념 : 일식된장(25g)

## 조리 방법

1. 1. 방울토마토를 뜨거운 물에 넣어 껍질을 벗겨주면 더욱 부드럽게 섭취할 수 있다.
2. 2. 시금치와 알배추는 한입 크기로 잘라준 뒤 뜨거운 물에 살짝 익혀준다.
3. 3. 닭육수에 미소된장을 풀어 체에 걸러준다.
4. 4. 체에 거른 된장국에 팽이버섯을 넣고 살짝 끓여준다.
5. 5. 그릇에 토마토, 시금치, 알배추를 담는다.
6. 6. 끓인 된장국을 붓고 완성한다.

## 영양 정보

- 칼로리: 25.4 kcal
- 탄수화물: 3g
- 단백질: 1.8g
- 지방: 0.7g
- 나트륨: 174.1mg
