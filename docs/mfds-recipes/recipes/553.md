# 골뱅이과일무침

골뱅이과일무침은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 골뱅이(50g)
- 통깨(2g)
- 당근(20g)
- 양파(20g)
- 양배추(20g)
- 깻잎(10g)
- 대파(10g)
- 홍고추(5g)
- 청고추(5g)
- 황도(20g)
- 배(20g)
- 사과(20g)
- 바나나(20g)
- 파인애플(20g)
- - 양념장 : 고추장(20g)
- 고춧가루(20g)
- 다진 마늘(10g)
- 케첩(20g)
- 설탕(20g)
- 식초(10g)
- 매실액(10g)
- 생강즙(5g)
- 참기름(5g)

## 조리 방법

1. 1. 골뱅이는 뜨거운 물에 살짝 데친 뒤
물기를 빼고 4등분으로 썬다.
2. 2. 양파와 깻잎은 채썰고, 당근은 양파랑
같은 길이로 얇게 저미고, 고추와
대파는 어슷썰고, 양배추는 1~2cm
두께로 썬다.
3. 3. 사과와 배는 채를 썰고 황도와 바나나,
파인애플은 작게 깍둑썰기 한다.
4. 4. 고추장에 고춧가루, 다진 마늘, 케첩,
설탕, 식초, 매실액, 생강즙, 참기름을
골고루 섞어 양념장을 만든다.
5. 5. 양념장에 준비한 골뱅이(①), 채소(②),
과일(③)을 넣고 고루 섞은 뒤 통깨를
얹어 완성한다.

## 영양 정보

- 칼로리: 225.3 kcal
- 탄수화물: 42.5g
- 단백질: 10.6g
- 지방: 1.4g
- 나트륨: 412.1mg
