# 묵계밥

묵계밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 도토리묵
- 올방개묵
- 검정깨묵 각각 25g
- 귀리밥 150g
- 계란 55g
- 콩나물 30g
- 국물 : 닭가슴살 70g
- 마늘 10g
- 대파 10g
- 양파 15g
- 맛간장 15g
- 흰후추 1g
- 버섯마늘소금 2g

## 조리 방법

1. 1. 닭가슴살을 이용해 국물을 만든다.
2. 2. 닭가슴살은 먹기 좋게 찢어 흰 후추, 버섯마늘
소금으로 밑간을 해준다.
3. 3. 콩나물은 머리를 떼어내고 데쳐준다.
4. 4. 계란은 황백지단으로 부친 후 채 썰어준다.
5. 5. 3종류의 묵은 먹기 좋은 크기로 썰어준다.
6. 6. 그릇에 밥을 담고 콩나물을 올리고 3종류의
묵을 올린 후 계란으로 고명해 완성한다.

## 영양 정보

- 칼로리: 227.9 kcal
- 탄수화물: 43.7g
- 단백질: 7g
- 지방: 2.8g
- 나트륨: 82.3mg
