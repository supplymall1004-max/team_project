# 모듬해물찜

모듬해물찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 전복 200g
- 자숙새우 60g
- 소라살 45g
- 알마늘 20g
- 대파 15g
- 청고추 15g
- 홍고추 15g
- 홍피망 15g
- 양파 50g
- 양념장 : 멕시코고추 2g
- 맛간장 30g
- 올리고당 15g
- 참기름 15g
- 흰후추 1g
- 물 200g

## 조리 방법

1. 1. 자숙새우와 소라살은 한번 데친 후 소라살은
먹기 좋게 잘라준다.
2. 2. 전복은 껍질을 벗기지 말고 세척해 준다.
3. 3. 양파, 홍피망, 대파는 먹기 좋게 잘라준다.
4. 4. 청고추, 홍고추도 먹기 좋게 잘라준다.
5. 5. 데친 물 200g에 양념장을 만들어 멕시코
고추를 담가 숙성시켜준 뒤 양념장에 알마늘을
넣어 끓여준다.
6. 6. 만들어진 양념장에 소라, 전복, 자숙새우,
준비된 채소 순으로 넣어 모듬해물찜을 완성
한다.

## 영양 정보

- 칼로리: 156.3 kcal
- 탄수화물: 12g
- 단백질: 21g
- 지방: 2.7g
- 나트륨: 1010.5mg
