# 조기까스

조기까스은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 조기 1마리(250g)
- 허브크러스트 100g
- 흰 후추 1g
- 화이트와인 15g
- 마늘오일 15g
- 전분 15g
- 소스 : 고추장 15g
- 저염간장 15g
- 화이트와인 15g
- 버터 5g
- 다진양파 15g
- 흰후추 1g
- 마늘오일 15g

## 조리 방법

1. 1. 다진마늘을 버터에 충분히 볶아준다.
2. 2. 화이트와인을 뿌려주고 소스재료로 소스를
만들어준다.
3. 3. 찜기에 살짝 져 준다. 이때 80% 정도까지만
익혀준다.
4. 4. 쪄준 조기는 와인과 후추로 밑간을 해준 후
전분을 발라준다.
5. 5. 전분를 바른 조기에 허브크러스트를 두툼하게
올려준다.
6. 6. 팬에 기름을 두르고 조기를 한쪽부터 구운 뒤,
뒤집어서 다시 구워주고 소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 135.8 kcal
- 탄수화물: 4.3g
- 단백질: 15.4g
- 지방: 6.3g
- 나트륨: 240.4mg
