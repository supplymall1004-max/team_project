# 함초소고기말이

함초소고기말이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 쇠고기구이용 150g
- 알마늘 20g
- 홍고추 15g
- 양파 60g
- 함초 30g
- 무순 2g
- 노란색파프리카 25g
- 소스 : 마늘기름 15g
- 맛간장 15g
- 허브식초 15g
- 연겨자 10g
- 버섯마늘소금 2g

## 조리 방법

1. 1. 파프라카, 양파, 홍고추는 가늘게 채 썰어준다.
2. 2. 함초는 세척 후 먹기 좋게 손질해 준다.
3. 3. 무순을 물 속에 담가준다.
4. 4. 편으로 썰어준 소고기에 채 썰어준 채소와
함초를 꼭 싸 말아준다.
5. 5. 마늘은 편으 로 썰 어 마늘기름에 한 번
구워준다.
6. 6. 구운 마늘을 깔고 말아준 고기를 굽는다.

## 영양 정보

- 칼로리: 360.9 kcal
- 탄수화물: 11.4g
- 단백질: 16.3g
- 지방: 27.8g
- 나트륨: 269.5mg
