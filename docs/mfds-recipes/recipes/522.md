# 단호박떡갈비

단호박떡갈비은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 단호박(150g)
- 육수(100g)
- 다진소고기(100g)
- 다진 파(10g)
- 다진 마늘(10g)
- 다진 양파(30g)
- 참기름(5g)
- 저염간장(10g)
- 후춧가루(0.2g)
- 매실액(10g)
- 찹쌀가루(20g)
- 설탕(10g)
- 잣(10g)

## 조리 방법

1. 1. 단호박은 칼로 잘라, 수저를 이용하여
호박씨를 제거한다.
2. 2. 손질이 된 단호박은 면보를 깔고,
김이 오른 찜통에서 약 15분 정도
쪄 준비한다.
3. 3. 다진 소고기는 파, 마늘, 양파, 참기름,
간장, 후춧가루, 매실액을 넣고 양념을
한다.
4. 4. 3번에 양념된 소고기에 찹쌀가루를
넣고 골고루 섞는다.
5. 5. 쪄 놓은 호박에 양념된 소고기를 넣고
모양을 잡아 준다.
6. 6. 육수가 한 소큼 끓으면 만들어 놓은
단호박떡갈비(⑤)를 넣어 약 20분 정도
은근히 쪄서 잣을 올려 완성한다.

## 영양 정보

- 칼로리: 274.1 kcal
- 탄수화물: 34.9g
- 단백질: 20.1g
- 지방: 6g
- 나트륨: 196mg
