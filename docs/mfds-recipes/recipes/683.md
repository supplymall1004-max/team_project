# 깻잎단호박튀김

깻잎단호박튀김은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 깻잎 5g
- 단호박 30g
- 호두 5g
- 꿀 3g
- 찹쌀가루 10g
- 물 15g
- 콩기름 4g

## 조리 방법

1. 1. 단호박을 반 갈라 씨를 빼고 8등분 한다.
2. 2. 김이 오른 찜통에 단호박을 찐다.
3. 3. 찐 단호박을 껍질을 벗겨 으깨고 호두는 잘게 다진다.
4. 4. 으깬 단호박에 다진 호두와 꿀 넣고 섞는다.
5. 5. 깻잎 뒷면에 4 를 한 숟가락 올리고 양끝을 오무려 돌돌 만다.
6. 6. 찹쌀가루에 물 한 큰술을 섞은 튀김옷을 골고루 입혀 바삭하게 튀겨낸다.

## 영양 정보

- 칼로리: 124.7 kcal
- 탄수화물: 13.4g
- 단백질: 2.1g
- 지방: 7.5g
- 나트륨: 2.2mg
