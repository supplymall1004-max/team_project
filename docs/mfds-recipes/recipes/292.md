# 산나물된장찌개

산나물된장찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 곤드레나물(25g)
- 고구마순(10g)
- 취나물(20g)
- 애호박(20g)
- 양파(10g)
- 청양고추(3g)
- 대파(1g)
- 순두부(40g)
- 저염재래된장 콩(백태
- 5g)
- 표고버섯(5g)
- 저염된장(10g)
- 날콩가루(10g)
- 육수 물(250g)
- 멸치(2g)
- 다시마(2g)
- 양념 다진 마늘(1.5g)
- 고춧가루(2g)
- 들깻가루(3g)

## 조리 방법

1. 1. 콩은 삶아서 표고버섯과 함께
물(30g)을 넣고 간다.
2. 2. 저염된장, 날콩가루, 갈아 놓은 콩과
표고버섯을 섞어 저염재래된장을
만든다.
3. 3. 냄비에 육수 재료를 끓여낸 뒤
건진다.
4. 4. 산나물과 애호박, 양파는 먹기 좋게
썰고, 청양고추와 대파는 어슷하게
썬다.
5. 5. 저염재래된장을 육수에 풀고
곤드레나물, 고구마순, 취나물을
넣고 끓여 반 이상 익으면 애호박,
양파를 넣는다.
6. 6. 순두부, 양념, 청양고추, 대파를 넣고
마무리한다.

## 영양 정보

- 칼로리: 156.2 kcal
- 탄수화물: 3.86g
- 단백질: 2.83g
- 지방: 1.13g
- 나트륨: 456.5mg
