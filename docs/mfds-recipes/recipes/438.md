# 미니밥버거

미니밥버거은(는) 밥 요리로, 굽기 조리법을 사용합니다.

## 재료

- 밥패티 : 찹쌀(100g)
- 현미쌀(50g)
- 올리브오일(10g)
- 버터(10g)
- - 소고기패티 : 다진소고기(50g)
- 양파(20g)
- 마늘(10g)
- 빵가루(10g)
- 생크림(10g)
- 으깬두부(20g)
- 소금(0.2g)
- 후춧가루(0.02g)
- - 소스 : 마요네즈(20g)
- 치즈(30g)
- 토마토(30g)

## 조리 방법

1. 1. 찹쌀과 현미는 깨끗이 씻어 30분 이상
충분히 불려 밥을 지어, 둥글둥글하게
패티를 만들어 놓는다.
2. 2. 마늘과 양파는 곱게 다져 볶아 준비
한다.
3. 3. 다진고기에 볶은 양파와 마늘, 빵가루,
생크림, 으깬두부, 소금, 후춧가루를
넣어 골고루 섞는다.
4. 4. 섞어진 소고기를 둥글게 패티를 만들어
놓는다.
5. 5. 토마토는 편으로 썰어 준비한다.
6. 6. 팬에 올리브 오일과 버터를 넣고 팬이
따끈하게 달궈지면, 밥패티를 먼저
굽고, 소고기 패티를 구운 후, 밥패티에
마요네즈를 살짝 바르고, 치즈와
토마토를 올리고, 구운 소고기패티를
올려 다시 밥패티를 덮어 완성한다.

## 영양 정보

- 칼로리: 530.9 kcal
- 탄수화물: 80g
- 단백질: 18g
- 지방: 15.4g
- 나트륨: 125.1mg
