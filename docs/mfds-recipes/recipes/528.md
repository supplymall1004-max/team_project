# 귀리죽

귀리죽은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 귀리(100g)
- 쌀(30g)
- 양파(20g)
- 부추(20g)
- 소고기(20g)
- 참기름(5g)
- 달걀(50g)
- 흑임자(2g)

## 조리 방법

1. 1. 귀리는 깨끗이 씻어 물에 불린다.
2. 2. 쌀도 씻어 따로 불려 놓는다.
3. 3. 양파와 부추는 송송 썰어 준비한다.
4. 4. 냄비에 참기름을 넣고, 소고기를 먼저
볶는다.
5. 5. 볶아지는 냄비에 다시 불린 귀리와
쌀을 넣어 볶아준다.
6. 6. 충분히 귀리가 볶아지면 물을 넣고
끓이다가, 썰어 놓은 채소를 넣고, 죽이
어우러지게 끓인 뒤 달걀을 풀어 넣고
다시 저어가면 끓여주고, 죽 위에
흑임자를 올려 완성한다.

## 영양 정보

- 칼로리: 317.9 kcal
- 탄수화물: 51.9g
- 단백질: 11.9g
- 지방: 7g
- 나트륨: 53.4mg
