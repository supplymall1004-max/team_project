# 해물들깨탕

해물들깨탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 주꾸미 80g
- 전복 100g
- 칵테일새우 50g
- 양파 560g
- 청고추 17g
- 홍고추 17g
- 계란 60g
- 청주 15g
- 국물 : 거피한 들깨가루 100g
- 저염간장 15g
- 다진마늘 15g
- 다진대파 7g

## 조리 방법

1. 1. 주꾸미, 전복, 칵테일새우는 각각 물에 데쳐서
준비한다.
2. 2. 이때 데치고 남은 물을 국물에 이용해주고
여기에 청주를 넣어준다.
3. 3. 양파, 청고추, 홍고추는 먹기좋게 썰어준다.
4. 4. 계란을 황백으로 부쳐서 먹기좋게 썰어준다.
5. 5. 만들어진 국물에 거피한 들깨가루와 국물재료를
넣어 끓여서 준비한다.
6. 6. 준비된 재료를 냄비에 넣고 국물을 부어 끓여
완성한다.

## 영양 정보

- 칼로리: 148.7 kcal
- 탄수화물: 7.4g
- 단백질: 21.1g
- 지방: 3.9g
- 나트륨: 295.8mg
