# 함초떡갈비

함초떡갈비은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 갈은 소고기 150g
- 갈은 돼지고기 150g
- 당근 25g
- 새송이버섯 20g
- 함초 30g
- 양파 25g
- 샐러리 10g
- 대파 10g
- 다진마늘 10g
- 참기름 15g
- 통깨 5g
- 함초소금 1g
- 소스 : 발사믹식초 30g
- 설탕 15g
- 흰후추 1g

## 조리 방법

1. 1. 함초는 송송 다져준다.
2. 2. 당근, 새송이, 샐러리, 양파를 송송 다져준다.
3. 3. 볼에 소고기, 돼지고기, 다져 놓은 채소와
다진마늘, 다진대파, 참기름을 넣고 반죽해준다.
4. 4. 여기에 함초 다진 것을 넣고 다시 반죽해 준다.
5. 5. 떡갈비 모양으로 만들어준 후 팬에 노릇하게
구워준다.
6. 6. 소스를 만들어 구워진 떡갈비를 졸여준다.

## 영양 정보

- 칼로리: 254.1 kcal
- 탄수화물: 5.7g
- 단백질: 19.8g
- 지방: 16.9g
- 나트륨: 210.5mg
