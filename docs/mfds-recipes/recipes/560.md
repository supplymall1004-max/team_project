# 매실소스를 곁들인 돼지고기만두

매실소스를 곁들인 돼지고기만두은(는) 반찬 요리로, 볶기 조리법을 사용합니다.

## 재료

- 돼지고기(안심
- 120g)
- 당면(50g)
- 애호박(1/2개)
- 당근(30g)
- 표고버섯(20g)
- 목이버섯(10g)
- 양파(50g)
- 마늘(20g)
- 토마토(100g)
- 매실액(50g)
- 후춧가루(0.2g)

## 조리 방법

1. 1. 돼지고기 안심을 요리망치로 얇게
두드려 펴고 후춧가루로 밑간한다.
2. 2. 목이버섯은 불려 잘게 썰고 양파와
마늘은 입자있게 다진다
3. 3. 당면은 따뜻한 물에 충분히 불려 썰고
애호박, 당근, 표고버섯은 잘게 썬다.
4. 4. 팬에 기름을 두르고 다진 양파와
마늘을 볶는다.
5. 5. 4번에 매실액과 잘게 자른 애호박,
당근, 표고버섯, 목이버섯, 당면을
넣고 볶아 만두소를 만든다.
6. 6. 돼지고기는 넓게 펴서 만두소를 넣고
주머니를 만들어 실로 묶고 토마토와
함께 오븐에 굽는다.

## 영양 정보

- 칼로리: 183.4 kcal
- 탄수화물: 21g
- 단백질: 15.1g
- 지방: 4.3g
- 나트륨: 235.5mg
