# 삼색소면

삼색소면은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 3가지색 소면(분홍
- 초록
- 흰색) 각 25g씩
- 쪽파 20g
- 청고추 20g
- 홍고추 15g
- 마늘 10g
- 국물 : 다시마물 400g
- 모시조개 200g
- 볶은소금 1g
- 청주 15g

## 조리 방법

1. 1. 모시조개는 볶은소금을 넣어서 해감시킨다.
2. 2. 준비된 다시마물에 준비된 채소의 국물재료를
넣고 국물을 만들어준다.
3. 3. 만들어진 국물에 모시조개를 넣고 청주를
부어서 국물을 만들어준다.
4. 4. 쪽파는 송송 다지고 마늘은 곱게 다져주고
청홍고추는 가늘게 채 썰어준다.
5. 5. 면을 각각 삶아준다.
6. 6. 삼색 면을 그릇에 담고 만들어진 국물을
부워서 완성한다.

## 영양 정보

- 칼로리: 168.1 kcal
- 탄수화물: 29.4g
- 단백질: 8.5g
- 지방: 1.8g
- 나트륨: 437.6mg
