# 버섯과 두부 비프웰링턴

버섯과 두부 비프웰링턴은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 소고기(안심
- 120g)
- 두부(50g)
- 양송이버섯(3개)
- 양파(30g)
- 당근(20g)
- 단호박(1/2개)
- 가지(1/2개)
- 감자(50g)
- 깻잎(3장)
- 마늘(20g)
- 버터(20g)
- 밀가루(100g)
- 달걀(1개)
- 소금(0.3g)

## 조리 방법

1. 1. 밀가루에 버터, 달걀을 넣고 골고루
반죽한 뒤 비닐봉투에 넣고
숙성시킨다.
2. 2. 안심과 두부는 잘게 썬다
3. 3. 단호박은 씨를 제거하고 삶아 다진다
4. 4. 양파, 당근, 마늘은 곱게 다져 안심,
두부와 섞고 소금을 약간 넣는다.
5. 5. 야채를 섞은 안심과 두부는 다진
단호박을 섞어 깻잎으로 먼저 감싼다.
6. 6. 숙성된 반죽을 밀대로 밀어 5번을 감싼
뒤 오븐에 넣고 25분 정도 굽는다.

## 영양 정보

- 칼로리: 451.6 kcal
- 탄수화물: 64.4g
- 단백질: 23.3g
- 지방: 11.2g
- 나트륨: 95.6mg
