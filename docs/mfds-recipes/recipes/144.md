# 겨자 오이고추소박이

겨자 오이고추소박이은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●주재료 : 오이고추 15g(1개)
- 당근 5g(5cm)
- 무 5g(5cm)
- 부추 5g(6cm)
- 양파 5g(5cm)
- ●소스 : 겨자분말 5g(1작은술)
- 식초 10g(2작은술)
- 설탕 10g(2작은술)

## 조리 방법

1. 1. 겨자분말은 물에 섞은 후, 뜨거운 물 3g에 중탕하여 발효시킨 후 설탕과 식초를 넣어 겨자 소스를 만들고 당근, 무, 양파는 0.1×5cm로 채 썬다.
2. 2. 부추는 3cm 길이로 썬다.
3. 3. 오이 고추는 길게 칼집을 넣어 씨를 제거한 다음 4cm로 썰고 당근, 무, 양파, 부추를 겨자냉채소스와 버무려 오이고추의 속을 채워 완성한다.

## 영양 정보

- 칼로리: 36.9 kcal
- 탄수화물: 7.9g
- 단백질: 1.2g
- 지방: 0.1g
- 나트륨: 43.4mg
