# 두가지묵샐러드

두가지묵샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 두가지 묵(올방개묵
- 검정깨묵) 각 25g씩
- 당귀잎 10g
- 양파 40g
- 노란 파프리카 25g
- 알배추 20g
- 소스 : 저염간장 15g
- 유자청 30g
- 다진청고추 7g
- 다진홍고추 7g
- 다진대파 7g

## 조리 방법

1. 1. 당귀는 잎만 떼어 먹기 좋게 잘라서 물에
담가준다.
2. 2. 두가지묵은 먹기 좋게 길이로 잘라준다.
3. 3. 양파, 파프리카, 알배추는 가늘게 채 썰어준다.
4. 4. 소스를 만들어준다.
5. 5. 모든 재료를 담고 소스에 통깨를 섞은 뒤
샐러드에 뿌려 완성한다.

## 영양 정보

- 칼로리: 53.6 kcal
- 탄수화물: 9.7g
- 단백질: 1g
- 지방: 1.2g
- 나트륨: 109.2mg
