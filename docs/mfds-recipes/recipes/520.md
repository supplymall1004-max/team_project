# 인삼닭살찜

인삼닭살찜은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭가슴살(200g)
- 소금(0.5g)
- 후춧가루(0.03g)
- 정종(10g)
- 대추(20g)
- 인삼(50g)
- 밤(20g)
- 표고버섯(30g)
- 새송이버섯(30g)
- 치킨육수(100g)

## 조리 방법

1. 1. 닭가슴살을 넓게 펴서 소금과
후춧가루, 정종으로 밑간을 한다.
2. 2. 대추는 돌려깎아 채를 썰고, 인삼은
세로 방향으로 반으로 가르고, 밤은
껍질을 벗겨 납작하게 썰고, 표고와
새송이 버섯은 뜨거운 물에 담그어
불린 후 채를 썰어 준비한다.
3. 3. 숙성된 닭가슴살(①)에 썰어놓은
재료(②)를 얹어 놓는다.
4. 4. 닭 가슴살을 돌돌 말아 놓는다.
5. 5. 냄비에 육수를 붓고, 육수가 끓으면
말아 놓은 닭가슴살을 넣는다.
6. 6. 뚜껑을 덮고, 약 15분 정도 은근히
찐 뒤, 닭가슴살이 익으면 불을 끄고
먹기 좋은 크기로 썰어서 접시에 담는다.

## 영양 정보

- 칼로리: 145.4 kcal
- 탄수화물: 19.4g
- 단백질: 14.1g
- 지방: 1.3g
- 나트륨: 253.4mg
