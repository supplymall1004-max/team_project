# 오이냉국을 곁들인 오색쌈밥

오이냉국을 곁들인 오색쌈밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 돼지고기(안심
- 100g)
- 쌀(200g)
- 도라지(30g)
- 애호박(1/2개)
- 양파(30g)
- 홍고추(1개)
- 미나리(20g)
- 오이(50g)
- 청포묵(20g)
- 쌈무(20g)
- 달걀(1개)
- 라이스페이퍼(5장)
- 고추장(20g)
- 소금(0.3g)
- 참기름(5g)
- 고춧가루(10g)

## 조리 방법

1. 1. 쌀은 깨끗이 씻어 30분 정도 불려 밥을
짓는다.
2. 2. 달걀을 풀어 소금을 넣고 지단을
만들어 채썬다.
3. 3. 도라지, 애호박, 양파, 홍고추는 채를
썰어 소금에 살짝 절여 볶고, 청포묵은
채를 썰어 참기름에 버무린다.
4. 4. 돼지 안심은 소금, 후춧가루로 양념 한
뒤 팬에 익힌다.
5. 5. 찬물에 식초, 설탕, 소금을 넣고 채 썬
오이를 넣어 냉국을 만든다.
6. 6. 고추장에 고춧가루를 섞어 쌈밥장을
만들고, 라이스페이퍼를 물에 담궈
건져 밥과 준비한 야채, 달걀지단,
쌈무, 쌈밥장을 올려 돌돌 말고 데친
미나리로 묶는다.

## 영양 정보

- 칼로리: 379.4 kcal
- 탄수화물: 77g
- 단백질: 9g
- 지방: 4g
- 나트륨: 522.8mg
