# 연어주먹밥튀김

연어주먹밥튀김은(는) 밥 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 재료 당근(30g)
- 피망(30g)
- 양파(30g)
- 밥(100g)
- 마요네즈(20g)
- 통조림 연어(1캔=130g)
- 식초(10g)
- 데리야키소스(15g)
- 달걀(40g)
- 빵가루(60g)
- 어린잎채소(50g)
- 오리엔탈소스 식초(10g)
- 레몬즙(15g)
- 간장(10g)
- 다진 마늘(10g)
- 꿀(10g)
- 올리브유(20g)
- 참깨(5g)

## 조리 방법

1. 1. 당근, 피망, 양파를 잘게 다진 뒤
기름에 살짝 볶아 밥에
마요네즈(10g)와 함께 넣어 섞는다.
2. 2. 통조림 연어는 기름기를 빼고,
식초를 넣은 끓는 물에서 30초 정도
데친다.
3. 3. 데리야키소스와 마요네즈(10g)에
연어를 넣고 버무려 양념한다.
4. 4. 밥을 동그랗게 빚은 뒤 가운데에
연어를 넣고 주먹밥을 만든다.
5. 5. 달걀물과 빵가루를 입혀 160℃로
예열한 기름에 바삭하게 튀긴다.
6. 6. 연어주먹밥튀김을 담고
어린잎채소를 곁들인 뒤
오리엔탈소스를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 660.2 kcal
- 탄수화물: 25.1g
- 단백질: 6.56g
- 지방: 6.75g
- 나트륨: 537.1mg
