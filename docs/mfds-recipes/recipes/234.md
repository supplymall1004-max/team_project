# 갈릭칩동태구이

갈릭칩동태구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 동태포(150g)
- 통마늘(10g)
- 양념 맛술(10g)
- 저염된장(10g)

## 조리 방법

1. 1. 동태는 맛술과 저염된장에 담가
재운다.
2. 2. 마늘은 얇게 편으로 썬 후
식용유(15g)에 노릇하게 튀긴다.
3. 3. 재운 동태는 흐르는 물에 살짝 헹궈
짠맛을 제거한 뒤 물기를 최대한
제거한다.
4. 4. 식용유(15g)를 두른 팬에 동태살을
얹어 부서지지 않게 굽는다.
5. 5. 갈릭칩을 올려 마무리한다.

## 영양 정보

- 칼로리: 141.8 kcal
- 탄수화물: 6.74g
- 단백질: 20.4g
- 지방: 3.82g
- 나트륨: 232.3mg
