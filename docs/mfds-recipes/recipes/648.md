# 단호박 된장매쉬와 해물굴림만두

단호박 된장매쉬와 해물굴림만두은(는) 일품 요리로, 찌기 조리법을 사용합니다.

## 재료

- 소고기(우둔살
- 50g)
- 새우(3마리)
- 흰살생선(100g)
- 단호박(1/2개)
- 고구마(50g)
- 녹말(10g)
- 달걀(1개)
- 된장(30g)
- 소금(0.1g)

## 조리 방법

1. 1. 새우와 흰살생선, 우둔살은 쪄낸 후 한
김 식힌다.
2. 2. 쪄낸 새우와 흰살 생선, 우둔살을 곱게
다진다.
3. 3. ?번에 녹말과 달걀물을 넣고 소금을
조금 넣고 골고루 섞어 숙성시킨다.
4. 4. 고구마와 단호박은 찜통에 찐다.
5. 5. 찐 고구마와 단호박을 곱게 갈아
된장을 섞는다.
6. 6. 숙성된 반죽으로 해물완자를 빚고,
된장을 섞은 단호박, 고구마를
끓이면서 해물 완자를 넣어준다

## 영양 정보

- 칼로리: 96.7 kcal
- 탄수화물: 10.2g
- 단백질: 11.3g
- 지방: 1.2g
- 나트륨: 414.6mg
