# 베이컨 가지 말이

베이컨 가지 말이은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- ●주재료 : 가지 150g(1/2개)
- 베이컨 80g(8장)
- 홍파프리카 50g(1/2개)
- 황파프리카 50g(1/2개)
- 청파프리카 50g(1/2개)
- 미나리 30g
- ●소스 : 고추장 5g(1작은술)
- 와사비 1g(1/3작은술)
- 고춧가루 2g(1/2작은술)
- 배 10g(2cm)
- 양파 2g(1cm)
- 마늘 1g(1/5개)
- 올리고당 5g(1작은술)
- 사과식초 5g(1/2작은술)

## 조리 방법

1. 1. 마늘, 양파, 배는 다지고 고추장, 와사비, 고춧가루, 올리고당, 사과식초와 함께 섞어 고추장 소스를 만든다.
2. 2. 가지는 길게 자르고, 미나리는 잎을 떼고, 파프리카는 채 썬다.
3. 3. 가지는 찌고, 미나리와 베이컨은 데친 후 베이컨, 가지, 파프리카 순으로 올려 말아준 뒤 미나리로 묶어 접시에 담고 소스를 뿌려 완성한다.

## 영양 정보

- 칼로리: 141.5 kcal
- 탄수화물: 7.5g
- 단백질: 4.7g
- 지방: 10.3g
- 나트륨: 150.1mg
