# 불고기미니볼

불고기미니볼은(는) 반찬 요리로, 튀기기 조리법을 사용합니다.

## 재료

- 다진소고기 40g
- 다진양파 15g 빵가루 30g
- 계란 60g
- 밀가루 15g
- 식용유 200g
- 단호박 25g
- 감자 25g
- 로즈마리 1g
- 양념장 : 참기름 5g
- 맛간장 15g
- 생강청 15g
- 흰후추 1g
- 다진마늘 1g
- 통깨 1g
- 소스 : 딸기잼 30g
- 오미자청 15g
- 물 30g

## 조리 방법

1. 1. 단호박, 감자는 가늘게 채 썰어 기름에 바싹
튀겨준다
2. 2. 불고기에 다진 양파를 섞어 양념장에 버무려
준다.
3. 3. 양념된 불고기는 완자로 만들어 밀가루, 계란,
빵가루 순으로 묻혀서 기름에 튀겨준다.
4. 4. 소스를 만들어준다
5. 5. 튀겨진 소고기 볼을 기름에 한 번 더 튀겨준다.
6. 6. 접시에 으깬 감자와 불고기볼 그리고 튀김을
올리고 소스와 바질을 올려 완성한다

## 영양 정보

- 칼로리: 346 kcal
- 탄수화물: 32.9g
- 단백질: 15g
- 지방: 17.2g
- 나트륨: 120.8mg
