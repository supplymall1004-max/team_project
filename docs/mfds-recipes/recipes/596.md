# 방울토마토를 곁들인 너비아니구이와 쌈밥

방울토마토를 곁들인 너비아니구이와 쌈밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 소고기(120g)
- 흑미(100g)
- 아스파라거스(3개)
- 감자(50g)
- 당근(20g)
- 사과(1/2개)
- 양송이버섯(5개)
- 방울토마토(5개)
- 깻잎(3장)
- 배(1/4개)
- 브로콜리(30g)
- 마늘(20g)
- 설탕(10g)
- 저염간장(10g)
- 참기름(2g)

## 조리 방법

1. 1. 감자는 삶아 껍질을 벗긴다.
2. 2. 아스파라거스는 2등분 하고,
방울토마토와 함께 끓는 물에 살짝
데친다.
3. 3. 닭고기에 소금, 후추, 배즙을 넣고 숙성
시킨다.
4. 4. 저염간장에 다진 마늘, 설탕, 후춧가루,
참기름을 넣고 양념장을 만들어 3번에
묻혀 굽는다
5. 5. 흑미는 충분히 불려 밥을 짓는다
6. 6. 깻잎은 간장, 식초, 설탕에 절여
흑미밥과 너비아니 구이를 넣고 돌돌
말아 쌈밥을 만든다

## 영양 정보

- 칼로리: 407.1 kcal
- 탄수화물: 25.2g
- 단백질: 49.6g
- 지방: 12g
- 나트륨: 349.6mg
