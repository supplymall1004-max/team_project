# 북어비빔밥

북어비빔밥은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 주재료
- 밥 210g(1컵)
- 북어 15g
- 당근 15g(5cm)
- 애호박 15g(5cm)
- 달걀 100g(2개)
- 고사리 15g
- 표고버섯 15g(2개)
- 우렁 25g(2큰술)
- 들기름 5g(1작은술) 장식
- 베이비채소 5g
- 식용꽃 5g

## 조리 방법

1. 1. 당근과 표고버섯, 고사리는 5cm로 채 썰고, 애호박도 돌려 깎아 5cm로
채 썬다.
2. 2. 우렁은 3등분으로 자른 후 1의 재료와 함께 들기름에 볶고 달걀은 황백으로 나누어 지단을 부쳐 채 썬다.
3. 3. 북어는 갈아 북어보푸라기를 만든다. 그릇에 밥을 담고 준비된 재료를 담고 베이비채소, 식용꽃, 북어보푸라기를 올려 완성한다.

## 영양 정보

- 칼로리: 157.4 kcal
- 탄수화물: 14.7g
- 단백질: 7.2g
- 지방: 7.8g
- 나트륨: 46.9mg
