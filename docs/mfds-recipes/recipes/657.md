# 봄날의 연못

봄날의 연못은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 문어(100g)
- 양파(30g)
- 오이(30g)
- 배(1/4개)
- 토마토(50g)
- 유자청(30g)
- 레몬(1/4개)
- 올리브오일(10g)
- 라임(1/2개)

## 조리 방법

1. 1. 끓는 물에 문어를 넣고 약 10분 정도
삶아 건진다.
2. 2. 라임은 얇게 썬다.
3. 3. 삶은 문어를 얇게 저미고 라임을 한 장
한 장 넣고 숙성 시킨다.
4. 4. 양파, 오이는 채썬다.
5. 5. 배와 토마토는 주사위 모양으로 썬다.
6. 6. 유자청, 레몬, 올리브오일, 소금으로
소스를 만들고, 접시에 문어와 라임을
담고 그 옆에 썰어 놓은 야채와 과일을
올려 소스를 올린다.

## 영양 정보

- 칼로리: 191.1 kcal
- 탄수화물: 26.5g
- 단백질: 20.3g
- 지방: 0.4g
- 나트륨: 318.5mg
