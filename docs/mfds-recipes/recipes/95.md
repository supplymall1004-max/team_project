# 치커리샐러드와 올리브 마늘 소스

치커리샐러드와 올리브 마늘 소스은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●치커리 샐러드 : 치커리 30g(10줄기)
- 적양배추 15g(5×3cm)
- 양파 10g(2×1cm)
- 당근 5g(3×1×1cm)
- ●올리브마늘 드레싱 : 올리브유 10g(2작은술)
- 식초 5g(1작은술)
- 설탕 5g(1작은술)
- 마늘 5g(1쪽)

## 조리 방법

1. 1. 올리브유, 식초, 설탕, 다진 마늘을 섞어 거품기로 충분히 저어주어 올리브마늘 드레싱을 만든다.
2. 2. 치커리는 싱싱하게 찬물에 담갔다가 물기를 뺀 후 한입 크기로 자르고, 적양배추, 양파, 당근은 곱게채를 썬다.
3. 3. 접시에 준비한 치커리, 적양배추, 양파, 당근을 담고 올리브마늘 드레싱을 뿌린다.

## 영양 정보

- 칼로리: 170 kcal
- 탄수화물: 13g
- 단백질: 3g
- 지방: 12g
- 나트륨: 74mg
