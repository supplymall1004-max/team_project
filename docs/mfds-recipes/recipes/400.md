# 카스텔라케이크와 해독주스

카스텔라케이크와 해독주스은(는) 후식 요리로, 기타 조리법을 사용합니다.

## 재료

- 카스텔라 100g
- 꿀 15g
- 견과류 15g
- 요거트 30g
- 체리 10g
- 해독주스 : 바나나 50g
- 브로콜리 25g
- 사과 50g
- 당근 25g
- 방울토마토 25g

## 조리 방법

1. 1. 카스텔라는 반으로 잘라준 후 꿀을 발라준다.
2. 2. 견과류는 먹기 좋게 잘라준다.
3. 3. 자른 카스텔라 위에 꿀을 발라준다.
4. 4. 바른 카스텔라 위에 자른 견과류를 뿌려준다.
5. 5. 브로콜리를 뺀 해독주스 재료를 끓이다 마지막에
브로콜리를 넣어 살짝 끓여준 뒤 믹서에 간다.
6. 6. 접시에 담고 요거트를 뿌리고 체리와 애플민트를
올려준다.

## 영양 정보

- 칼로리: 270.2 kcal
- 탄수화물: 32.8g
- 단백질: 5.3g
- 지방: 13.1g
- 나트륨: 91.6mg
