# 닭봉주먹밥

닭봉주먹밥은(는) 밥 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 닭봉(80g)
- 우유(20g)
- 단호박(10g)
- 새송이버섯(5g)
- 가지(15g)
- 주키니호박(15g)
- 당근(15g)
- 양파(10g)
- 시래기(20g)
- 치즈(20g)
- 밥(150g)
- 새싹채소(10g)
- 양념 후춧가루(3g)
- 참기름(20g)
- 간장(5g)
- 소스 레몬즙(5g)
- 마요네즈(12g)
- 꿀(5g)
- 부순 참깨(2g)

## 조리 방법

1. 1. 닭봉은 손질한 뒤 후춧가루를
섞은 우유에 담가두었다가 구워
준비한다.
2. 2. 단호박, 새송이버섯은 구워서
준비한다.
3. 3. 소스를 만든다.
4. 4. 가지, 주키니호박, 당근, 양파,
시래기는 잘게 다진 뒤 볶는다.
5. 5. 볶은 채소와 치즈를 참기름, 간장과
함께 밥에 넣고 섞는다.
6. 6. 구운 닭봉을 밥으로 감싼 뒤 구운
단호박, 새송이버섯, 새싹채소와
함께 담아 소스를 곁들여
마무리한다.

## 영양 정보

- 탄수화물: 20.5g
- 단백질: 7.12g
- 지방: 6.08g
- 나트륨: 454.1mg
