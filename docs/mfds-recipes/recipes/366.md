# 장어찜

장어찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 장어 250g
- 우유 100g
- 후추 1g
- 곁들이 샐러드 : 우엉 25g
- 인삼 20g
- 미나리 20g
- 팽이 15g
- 양파 40g
- 샐러드소스 : 청고추 17g
- 홍고추 15g
- 맛간장 7g

## 조리 방법

1. 1. 장어 살 쪽에 칼집을 내어 우유와 후추로
마리네이드 해서 비린내를 제거한다.
2. 2. 우엉, 인삼, 홍고추는 채 썰어 냉수에 담가둔다
3. 3. 미나리, 팽이는 먹기 좋게 썰어준다.
4. 4. 양파, 청고추는 가늘게 채 썰어준다.
5. 5. 장어는 찜기에 쪄준다.
6. 6. 모든 채소를 한데 섞어 샐러드소스로 버무려
샐러드를 만들어주고 장어와 함께 완성한다.

## 영양 정보

- 칼로리: 328.3 kcal
- 탄수화물: 2.1g
- 단백질: 20.4g
- 지방: 26.5g
- 나트륨: 74.7mg
