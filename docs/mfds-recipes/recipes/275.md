# 해물김치찌개

해물김치찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수 재료 : 주꾸미(40g)
- 김치(40g)
- 무(10g)
- 마늘(3g)
- 대파(3g)
- 양파(10g)
- 두부(10g)
- 참기름(3g)
- 배즙(10g)
- 팽이버섯(3g)
- •육수 재료 : 다시마(5g)
- 멸치(10g)
- 물(300g)

## 조리 방법

1. 1. 냄비에 육수 재료를 넣고 끓이다가 물이 끓어오르면 다시마를 건지고 조금 더 끓여 육수를 우려낸다.
2. 2. 주꾸미는 내장과 입, 눈을 제거하고 4cm 크기로 잘라 준비한다.
3. 3. 김치는 국물을 꽉 짜서 한입 크기로 썰고, 무는 편 썰고, 마늘은 다지고, 대파는 어슷 썰고, 양파는 굵게 채 썰고, 두부는 납작하게 썬다.
4. 4. 냄비에 참기름을 두르고, 김치, 무, 양파가 투명해질 때까지 볶다가 육수를 붓고 끓으면 주꾸미, 다진 마늘을 넣어 더 끓인다.
5. 5. 김칫국물과 배즙을 2:1로 섞은 뒤 찌개에 넣어 간을 맞춘다.
6. 6. 두부, 팽이버섯을 넣고 한소끔 끓여 마무리한다

## 영양 정보

- 칼로리: 53.9 kcal
- 탄수화물: 5.2g
- 단백질: 3g
- 지방: 2.3g
- 나트륨: 385.1mg
