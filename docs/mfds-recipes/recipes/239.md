# 차돌박이구이

차돌박이구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 연근(30g)
- 미나리(20g)
- 오렌지(50g)
- 오렌지주스(50g)
- 차돌박이(60g)
- 어린잎채소(10g)
- 홍초물 설탕(5g)
- 홍초(20g)
- 물(10g)

## 조리 방법

1. 1. 연근은 얇게 3㎜ 정도의 두께로 썰어
홍초물에 담가 절인다.
2. 2. 미나리는 연근과 비슷한 길이로
썬다.
3. 3. 오렌지는 껍질은 얇게 벗겨 채 썰고,
과육부분은 반달 모양으로 2조각
썬다.
4. 4. 과육을 오렌지주스와 섞은 뒤 갈아
오렌지소스를 만든다.
5. 5. 팬에 차돌박이를 펼친 뒤
오렌지소스를 조금씩 넣으며 익힌다.
6. 6. 그릇에 연근초절임, 미나리,
어린잎채소, 차돌박이 순서로
담은 뒤 오렌지 껍질과
오렌지소스를 얹어 마무리한다.

## 영양 정보

- 칼로리: 210.2 kcal
- 탄수화물: 9.64g
- 단백질: 8.88g
- 지방: 8.33g
- 나트륨: 32mg
