# 연어허브스테이크

연어허브스테이크은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 연어(150g)
- 로즈마리(3g)
- 레몬(10g)
- 올리브오일(30g)
- - 소스 : 당근(20g)
- 양파(20g)
- 토마토(30g)
- 마늘(10g)
- 버터(5g)
- 밀가루(10g)
- 토마토 소스(50g)
- 육수(150g)
- 우유(50g)
- 월계수잎(1장)

## 조리 방법

1. 1. 연어에 로즈마리를 위에 올려
마리네이드를 한다.
2. 2. 마리네이드를 해놓은 연어에 레몬즙을
뿌린다.
3. 3. 당근, 양파는 썰고 마늘과 토마토는
다져 준비한다.
4. 4. 냄비에 버터를 두르고, 채소(③)를
볶다가 밀가루를 넣고 볶으면서 다시
토마토소스를 넣어 볶다가 육수와
월계수잎을 넣어 끓인다.
5. 5. 소스가 자작 하게 끓고 있을 때, 우유를
넣고 한 소끔 끓여준다.
6. 6. 팬에 올리브 오일을 두르고, 연어
(①)를 올려 앞뒤로 구어 접시에
담고, 토마토소스를 올려 완성한다.

## 영양 정보

- 칼로리: 455.5 kcal
- 탄수화물: 3.5g
- 단백질: 47.3g
- 지방: 28.1g
- 나트륨: 256.4mg
