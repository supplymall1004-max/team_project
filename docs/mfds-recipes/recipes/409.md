# 주꾸미보양샐러드

주꾸미보양샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 주꾸미 50g
- 사과 60g
- 더덕 40g
- 양파 60g
- 청고추 15g
- 홍고추 15g
- 소스 : 마늘기름 30g
- 식초 15g
- 다진마늘 5g
- 다진대파 5g
- 레몬소금 1g
- 흰후추 1g
- 다진청고추 15g
- 레몬즙 15g

## 조리 방법

1. 1. 더덕은 손질해 어슷하게 잘라 소금물에 살짝
담가준다.
2. 2. 사과는 먹기 좋게 썰어서 레몬즙에 살짝
뿌려준다.
3. 3. 주꾸미는 레몬즙을 넣어 살짝 데쳐준다.
4. 4. 청고추, 홍고추는 씨를 제거하고 사과처럼
잘라주고. 양파도 잘라준다.
5. 5. 소스를 만들어준다.
6. 6. 모든 채소와 주꾸미를 섞어 소스랑 한데
버무려 완성한다.

## 영양 정보

- 칼로리: 132.7 kcal
- 탄수화물: 3.2g
- 단백질: 20.3g
- 지방: 4.3g
- 나트륨: 191.8mg
