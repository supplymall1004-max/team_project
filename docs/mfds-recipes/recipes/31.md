# 방울토마토 소박이

방울토마토 소박이은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●방울토마토 소박이 :
- 방울토마토 150g(5개)
- 양파 10g(3×1cm)
- 부추 10g(5줄기)
- ●양념장 :
- 고춧가루 4g(1작은술)
- 멸치액젓 3g(2/3작은술)
- 다진 마늘 2.5g(1/2쪽)
- 매실액 2g(1/3작은술)
- 설탕 2g(1/3작은술)
- 물 2ml(1/3작은술)
- 통깨 약간

## 조리 방법

1. 1. 물기를 빼고 2cm 정도의 크기로 썰은 부추와 양파를 양념장에 섞어 양념속을 만든다.
2. 2. 깨끗이 씻은 방울토마토는 꼭지를 떼고 윗부분에 칼로 십자모양으로 칼집을 낸다.
3. 3. 칼집을 낸 방울토마토에 양념속을 사이사이에 넣어 버무린다.

## 영양 정보

- 칼로리: 45 kcal
- 탄수화물: 9g
- 단백질: 2g
- 지방: 1g
- 나트륨: 277mg
