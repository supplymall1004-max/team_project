# 토란순두부찌개

토란순두부찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 토란(40g)
- 표고버섯(20g)
- 붉은 고추(1g)
- 청양고추(2g)
- 백합(150g)
- 들깻가루(40g)
- 순두부(150g)

## 조리 방법

1. 1. 토란은 2×2㎝로 썬다.
2. 2. 표고버섯은 채 썰고, 고추는
어슷하게 썬다.
3. 3. 냄비에 물(500ml)을 넣고 물이
끓으면 백합을 넣고 끓여 육수를
낸다.
4. 4. 조개가 입을 벌리면 토란과
들깻가루, 표고버섯을 넣고 끓인다.
5. 5. 육수가 어느 정도 걸쭉해지면
순두부를 넣고 끓인다.
6. 6. 붉은 고추와 청양고추를 올려
마무리한다.

## 영양 정보

- 칼로리: 312.7 kcal
- 탄수화물: 2.93g
- 단백질: 3.48g
- 지방: 2.96g
- 나트륨: 409.2mg
