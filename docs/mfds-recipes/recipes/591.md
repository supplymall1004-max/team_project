# 망고무스케이크와 마카롱

망고무스케이크와 마카롱은(는) 후식 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 달걀(1개)
- 아몬드(30g)
- 레몬(1/4개)
- 망고퓨레(20g)
- 버터(20g)
- 생크림(20g)
- 우유(50g)
- 물엿(50g)
- 소금(0.2g)
- 설탕(20g)
- 판젤라틴(1장)

## 조리 방법

1. 1. 아몬드는 곱게 갈아 체에 내려
준비한다.
2. 2. 달걀흰자로 머랭을 만들어 짤주머니로
짜준 후 20분 정도 말리고 오븐에
13분간 구워 마카롱을 만든다.
3. 3. 냄비에 설탕, 물엿, 소금을 넣고
끓인다.
4. 4. 달걀노른자, 설탕, 우유 및 생크림을
끓이다가 망고퓨레를 넣고 섞는다
5. 5. 판젤라틴은 물에 불려 중탕으로 녹여
4에 섞고, 차갑게 휘핑한 생크림을
넣는다.
6. 6. 5를 틀에 굳힌 후, 접시에 담고
마카롱으로 장식한다.

## 영양 정보

- 칼로리: 487.3 kcal
- 탄수화물: 56.7g
- 단백질: 7.7g
- 지방: 25.5g
- 나트륨: 133.2mg
