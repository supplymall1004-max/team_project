# 간장곤약볶음국수

간장곤약볶음국수은(는) 일품 요리로, 볶기 조리법을 사용합니다.

## 재료

- 곤약국수 200g
- 사과 60g
- 레몬즙 15g
- 볶음양념 : 맛간장 30g
- 다진생강 3g
- 다진양파 15g
- 버터 10g
- 소스 : 다진양파 10g
- 흑임자 7g
- 플레인요거트 30g
- 그라놀라 30g
- 꿀 10g

## 조리 방법

1. 1. 곤약은 세척 후 물기를 빼준다.
2. 2. 사과는 가늘게 채 썰어 레몬에 살짝 담가 주고,
다진양파도 레몬즙에 살짝 담가준다.
3. 3. 팬에 다진양파를 버터에 볶다가 나머지 볶음
재료를 넣고 볶음양념을 만들어준다.
4. 4. 볶음양념에 곤약 을 넣 어 고들고들하게
볶아준다.
5. 5. 사과를 물기를 완전히 제거하고 소스재료를
섞어 곁들이 소스를 만들어준다.
6. 6. 볶음곤약을 그릇에 담고 그라놀라를 뿌려서
완성한다.

## 영양 정보

- 칼로리: 133 kcal
- 탄수화물: 18.3g
- 단백질: 1.2g
- 지방: 6.1g
- 나트륨: 343.1mg
