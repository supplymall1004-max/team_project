# 배추관자카나페

배추관자카나페은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 관자(2개)
- 배춧잎(6장)
- 파프리카(1/2개)
- 레몬(1/2개)
- 고추장(20g)
- 매실액(10g)
- 설탕(20g)
- 소금(5g)
- 파슬리(5g)

## 조리 방법

1. 1. 배춧잎을 물에 담궈 싱싱하게 한다
2. 2. 레몬은 슬라이스로 썬다.
3. 3. 레몬, 고추장, 설탕 및 매실액을 넣고
양념장을 만든다.
4. 4. 파프리카는 작은 사각형 모양으로
자른다.
5. 5. 관자는 끓는 물에 소금을 넣고 데친다.
6. 6. 관자는 프라이팬에 구운 뒤 잘게 썰어
파프리카와 섞어 배춧잎에 올리고, 그
위에 파슬리와 만들어 놓은 양념장을
먹기 직전에 올린다.

## 영양 정보

- 칼로리: 104.7 kcal
- 탄수화물: 14.1g
- 단백질: 9.9g
- 지방: 0.9g
- 나트륨: 217.5mg
