# 완자김치찌개

완자김치찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 콩나물(30g)
- 김치(50g)
- 두부(40g)
- 붉은 고추(15g)
- 다진 돼지고기(등심
- 60g)
- 새우젓(5g)
- 육수 대파(10g)
- 마늘(5g)
- 쇠고기(50g)

## 조리 방법

1. 1. 대파, 마늘, 쇠고기를 물(300g)과
함께 끓인 뒤 면포에 걸러 육수를
만든다.
2. 2. 콩나물, 김치를 물에 씻어 손질한다.
3. 3. 두부는 물기를 빼고, 붉은 고추는
다진 뒤 돼지고기와 섞어 반죽해
동그랗게 빚어 완자를 만든다.
4. 4. 김 오른 찜기에 완자를 넣어 찐다.
5. 5. 육수를 끓이고 끓어오르면 김치,
콩나물, 새우젓을 넣는다.
6. 6. 한소끔 더 끓이다가 완자를 넣고
조금 더 끓여 마무리한다.

## 영양 정보

- 칼로리: 161.1 kcal
- 탄수화물: 1.34g
- 단백질: 6.78g
- 지방: 2.61g
- 나트륨: 355mg
