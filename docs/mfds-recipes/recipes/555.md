# 속채운대저토마토백김치

속채운대저토마토백김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 토마토(100g)
- 배춧잎(2장)
- 오이(50g)
- 무(50g)
- 양파(30g)
- 마늘(10g)
- 생강(5g)
- 소금(2g)

## 조리 방법

1. 1. 토마토는 윗부분을 자르고 속을
파낸다.
2. 2. 오이는 소금에 깨끗이 씻는다.
3. 3. 소금을 팬에 볶은 후 배추를 절인다.
4. 4. 절인 배추에 오이와 무를 넣고
돌돌만다.
5. 5. ?를 속을 파낸 토마토 안에 넣는다.
6. 6. ?번을 통에 담아 소금 국물을 붓고
양파와 마늘 생강을 넣는다.

## 영양 정보

- 칼로리: 76 kcal
- 탄수화물: 15.1g
- 단백질: 3.5g
- 지방: 0.2g
- 나트륨: 278.5mg
