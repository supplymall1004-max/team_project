# 미니함박스테이크

미니함박스테이크은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 소고기(100g)
- 돼지고기(100g)
- 대추(2알)
- 양파(50g)
- 마늘(10g)
- 새송이버섯(1개)
- 파프리카(50g)
- 배춧잎(1장)
- 단호박(1/4개)
- 당근(20g)
- 소금(0.3g)
- 후춧가루(0.02g)
- 배(1/4개)
- 저염간장(30g)
- 설탕(20g)

## 조리 방법

1. 1. 대추는 돌려 깎아 씨를 제거하고 곱게
다지고 팬에 볶는다.
2. 2. 소고기, 돼지고기, 양파 및 마늘을
다진다.
3. 3. 배춧잎, 파프리카, 새송이버섯, 당근은
입자있게 다진다.
4. 4. 다진 소고기, 돼지고기, 양파, 마늘,
대추, 배춧잎, 파프리카, 새송이버섯,
당근 및 양파에 소금, 후춧가루를 섞어
잘 치대어 스테이크 모양으로 만든다
5. 5. 팬에 기름을 살짝 두르고, 스테이크
고기를 충분히 익혀 접시에 담고,
단호박은 껍질을 벗겨 팬에 구워
담는다.
6. 6. 배는 갈아 저염간장과 설탕을 넣고 잘
저어 소스를 만들고 익혀 놓은
스테이크에 올려 완성한다.

## 영양 정보

- 칼로리: 210.3 kcal
- 탄수화물: 16.6g
- 단백질: 25.5g
- 지방: 4.7g
- 나트륨: 149.4mg
