# 수박즙돼지목심구이

수박즙돼지목심구이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- - 주재료 : 돼지고기 목심 70g
- 올리브유 2g
- 후추 0.25g - 소스 : 수박 15g
- 간장 3g
- 설탕 2g
- 물엿 3g
- 다진 마늘 2g
- 후추 0.25g

## 조리 방법

1. 1. 키친타월을 이용해 돼지고기의 핏물을 뺀다.
2. 2. 돼지고기에 칼집을 내준다.
3. 3. 칼집 낸 돼지고기에 올리브유, 후추를 뿌려 재워둔다.
4. 4. 수박의 씨를 빼고 살만 발라 믹서에 간다.
5. 5. 소스 재료를 모두 섞어 약불에서 살짝 졸여 식힌다.
6. 6. 재워 놓은 고기에 소스를 끼얹어 팬에 굽는다.

## 영양 정보

- 칼로리: 229 kcal
- 탄수화물: 6.8g
- 단백질: 197.5g
- 지방: 16.1g
- 나트륨: 199.8mg
