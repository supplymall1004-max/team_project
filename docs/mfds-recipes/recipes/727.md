# 더덕얼갈이겉절이

더덕얼갈이겉절이은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 얼갈이 배추 50g
- 더덕 20g
- 까나리액젓 3g
- 고춧가루 1.7g
- 깨소금 1g
- 다진 파 2.5g
- 다진 마늘 1.25g
- 다진 생강 0.65g
- 소금 0.5g
- 국간장 3g
- 물 10g

## 조리 방법

1. 1. 얼갈이는 깨끗이 씻어 한입 크기로 뜯는다.
2. 2. 더덕은 껍질을 벗긴다.
3. 3. 껍질 벗긴 더덕은 굵게 채 썰어 소금물에 담가둔다.
4. 4. 양념장 재료를 모두 섞어 양념장을 만든다.
5. 5. 더덕을 건져 물기를 제거한다.
6. 6. 더덕과 얼갈이 배추에 양념장을 넣고 가볍게 버무린다.

## 영양 정보

- 칼로리: 35.5 kcal
- 탄수화물: 4.6g
- 단백질: 2.6g
- 지방: 1.3g
- 나트륨: 448.4mg
