# 새우단배추된장국

새우단배추된장국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 마늘(0.3g)
- 양파(10g)
- 애호박(20g)
- 단배추(25g)
- 대파(5g)
- 두부(25g)
- 마른 새우(10g)
- 육수 무(15g)
- 멸치(2.5g)
- 다시마(2.5g)
- 물(300g)
- 양념 된장(10g)
- 표고버섯가루(3g)
- 날콩가루(10g)

## 조리 방법

1. 1. 냄비에 육수 재료를 넣고 10분간
끓여 건진다.
2. 2. 마늘은 다지고, 양파는 채 썰고,
애호박은 반달모양으로 납작하게
썬다.
3. 3. 단배추는 5cm로 썰고, 대파는
어슷 썰고, 두부는 0.5cm 두께로
썬다.
4. 4. 된장에 표고버섯가루, 날콩가루,
육수(50g)를 넣어 섞는다.
5. 5. 육수에 된장을 풀고 다진 마늘, 호박,
양파, 단배추 순으로 넣어가며
끓인다.
6. 6. 끓으면 마른 새우, 대파를 넣어 살짝
더 끓여 마무리한다.

## 영양 정보

- 칼로리: 129.2 kcal
- 탄수화물: 2.58g
- 단백질: 4.83g
- 지방: 0.88g
- 나트륨: 774.8mg
