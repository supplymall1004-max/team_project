# 망고샐러드

망고샐러드은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 증편 200g
- 망고 150g
- 양상추 30g
- 치커리 20g
- 방울토마토 60g
- 양파 60g
- 소스 : 올리브오일 30g
- 유자청 15g
- 식초 15g
- 볶은소금 2g
- 후추 1g
- 다진 파프리카(노랑) 15g
- 다진 청피망 15g
- 다진 홍피망 15g
- 통깨 5g

## 조리 방법

1. 1. 증편은 팬에 구워준다.
2. 2. 망고와 방울토마토는 먹기 좋게 잘라준다.
3. 3. 모든 채소는 먹기 좋게 잘라서 물에 담근다.
4. 4. 소스 재료를 섞어 소스를 만들고, 접시에 준비된
채소를 담고 소스를 뿌려준 후 구운 떡과 망고
를 올려 완성한다.

## 영양 정보

- 칼로리: 105.7 kcal
- 탄수화물: 15.5g
- 단백질: 1.9g
- 지방: 4g
- 나트륨: 50.3mg
