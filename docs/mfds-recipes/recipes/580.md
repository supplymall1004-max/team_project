# 콩비지포카치아 샌드위치

콩비지포카치아 샌드위치은(는) 일품 요리로, 기타 조리법을 사용합니다.

## 재료

- 포카치아빵(1개)
- 콩비지(50g)
- 무장아찌(20g)
- 토마토(50g)
- 생치즈(20g)
- 루꼴라(20g)
- 올리브오일(20g)

## 조리 방법

1. 1. 비지는 살짝 데쳐 물기를 뺀다.
2. 2. 무장아찌는 잘게 다져 비지에 골고루
섞어 소스를 만든다.
3. 3. 토마토는 슬라이스로 썰어 팬에 오일을
두르고 익힌다.
4. 4. 포카치아빵은 반으로 갈라 비지소스를
바른다.
5. 5. 생치즈는 동그랗게 원형으로 찍어,
비지를 바른 빵에 구운 토마토와 함께
올리고, 다시 그 위에 비지를 올려
접시에 담는다.
6. 6. 루꼴라는 찬물에 담궈 건져 포카치아빵
옆에 함께 놓는다

## 영양 정보

- 칼로리: 271.2 kcal
- 탄수화물: 36.8g
- 단백질: 12.9g
- 지방: 8g
- 나트륨: 459.2mg
