# 시금치브로컬리 파스타

시금치브로컬리 파스타은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 새우(대하
- 3마리)
- 스파게티(100g)
- 시금치(30g)
- 브로콜리(50g)
- 바질(10g)
- 생크림(50g)
- 올리브오일(10g)
- 소금 (0.2g)

## 조리 방법

1. 1. 시금치와 브로콜리는 끓는 물에 소금을
넣고 살짝 데친다.
2. 2. 데친 시금치, 브로콜리와 바질을
믹서에 갈아 준다.
3. 3. 새우는 껍질을 벗겨 손질한다
4. 4. 끓는 물에 스파게티를 10분 정도 삶아
건진다.
5. 5. 삶아진 스파게티는 팬에 올리브오일을
넣고 볶는다.
6. 6. 갈아 놓은 시금치소스에 생크림을 넣고
한소끔 끓이면서 볶은 파스타를 넣은
뒤 새우를 올린다

## 영양 정보

- 칼로리: 400.5 kcal
- 탄수화물: 68.4g
- 단백질: 15.3g
- 지방: 7.3g
- 나트륨: 77.2mg
