# 새우와스테이크

새우와스테이크은(는) 일품 요리로, 굽기 조리법을 사용합니다.

## 재료

- 새우(60g)
- 버터(10g)
- 소고기(120g)
- 로즈마리(2g)
- 소금(0.3g)
- 후춧가루(2g)
- - 소스 : 와인(50g)
- 설탕(20g)
- 당근(20g)
- 양파(20g)
- 토마토(30g)
- 마늘(20g)
- 대파(10g)
- 발사믹소스(20g)
- - 곁들임채소 : 아스파라거스(20g)
- 브로컬리(20g)
- 새송이(20g)
- 가지(30g)

## 조리 방법

1. 1. 새우는 꼬리를 자르고 내장을 제거
한다.
2. 2. 스테이크 소고기는 소금과 로즈마리를
뿌리고 후춧가루를 살짝 뿌려 놓는다.
3. 3. 아스파라거스는 팬에 기름 없이
굽는다.
4. 4. 새송이와 가지는 길게 썰어 브로컬리와
함께 기름 없이 팬에 살짝 구워 놓는다.
5. 5. 팬에 버터를 살짝 넣고 새우를 굽고,
스테이크를 구워 접시에 담는다.
6. 6. 팬에 와인과 설탕을 넣고 끓이다가,
당근, 양파와 토마토 마늘, 대파를 다져
넣고 볶듯이 끓이다가 발사믹소스를
넣고 한 번 더 졸인 소스를 스테이크
위에 얹고, 브로컬리와 새송이, 가지,
아스파라거스를 함께 놓아 완성한다.

## 영양 정보

- 칼로리: 508.4 kcal
- 탄수화물: 6.3g
- 단백질: 36.2g
- 지방: 37.6g
- 나트륨: 194.4mg
