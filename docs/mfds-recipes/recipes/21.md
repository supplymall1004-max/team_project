# 땅콩소스 버섯가지나물

땅콩소스 버섯가지나물은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- •필수 재료 : 가지(15g)
- 새송이버섯(15g)
- 표고버섯(10g)
- •땅콩 소스 : 땅콩가루(5g)
- 다진마늘(2g)
- 참기름(2g)
- 저염간장(1g)

## 조리 방법

1. 1. 가지는 꼭지를 떼어 편으로 썬다.
2. 2. 새송이버섯은 길게 편으로 썰고, 표고버섯은 기둥을 떼고 납작하게 썬다.
3. 3. 가지와 버섯은 끓는 물에 각각 데친 후 찬물에 식힌다.
4. 4. 땅콩소스 재료를 섞어 땅콩소스를 만든다.
5. 5. 데친 가지와 버섯은 물기를 제거해 그릇에 담고, 땅콩소스를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 64.2 kcal
- 탄수화물: 3.4g
- 단백질: 4.3g
- 지방: 3.7g
- 나트륨: 64.3mg
