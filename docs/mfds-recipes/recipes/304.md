# 황태포당면국수

황태포당면국수은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 감자(80g)
- 양파(15g)
- 당근(10g)
- 대파(5g)
- 청양고추(3g)
- 팽이버섯(20g)
- 달걀(30g)
- 당면(70g)
- 쑥갓(5g)
- 육수 멸치(2g)
- 황태포(10g)
- 양념 다진 마늘(1g)
- 저염간장(2g)
- 소금(0.6g)
- 후춧가루(0.5g)

## 조리 방법

1. 1. 감자는 한입 크기로 썰고, 양파와
당근, 대파는 길게 채 썰고,
청양고추는 다지고, 팽이버섯은
깨끗이 씻어 준비한다.
2. 2. 달걀은 노른자와 흰자를 분리해
지단을 부쳐 채 썬다.
3. 3. 당면은 찬물에 2시간 정도 불린다.
4. 4. 물(300g)에 멸치를 넣어 육수를
만든 후, 황태포를 넣어 다시
한 번 진하게 육수를 우려낸다.
5. 5. 육수에 감자, 당근, 양파, 당면을
순서대로 넣고 끓인 후 불을 끄고
팽이버섯을 얹어 잔열로 익힌다.
6. 6. 다진 마늘, 청양고추, 저염간장, 소금,
후춧가루를 넣어 간을 한 뒤 지단,
쑥갓, 대파를 얹어 마무리한다.

## 영양 정보

- 칼로리: 338.5 kcal
- 탄수화물: 17.26g
- 단백질: 0.95g
- 지방: 0.27g
- 나트륨: 72.4mg
