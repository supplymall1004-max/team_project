# 백김치콩비지찌개

백김치콩비지찌개은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 풋고추(10g)
- 백김치(100g)
- 콩비지(100g)
- 양념 참기름(8g)
- 후춧가루(0.2g)
- 다진 마늘(10g)
- 간장(2g)

## 조리 방법

1. 1. 풋고추는 잘게 다진다.
2. 2. 백김치는 먹기 좋은 크기로 썬다.
3. 3. 냄비에 백김치를 볶다가 김치가
익으면 참기름, 후춧가루,
다진 마늘을 넣고 볶는다.
4. 4. 물(200g)을 넣고 끓인다.
5. 5. 물이 끓으면 콩비지와 간장,
풋고추를 넣어 약한 불에서 15분
정도 끓여 마무리한다.

## 영양 정보

- 칼로리: 195.3 kcal
- 탄수화물: 3.13g
- 단백질: 2.99g
- 지방: 2.99g
- 나트륨: 447.2mg
