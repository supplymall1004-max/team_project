# 사과장아찌

사과장아찌은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- 풋사과 200g
- 양파 80g
- 청고추 20g
- 홍고추 20g
- 소스 : 맛간장 30g
- 식초 30g
- 설탕 10g
- 생강청 30g
- 매실청 15g
- 청주 30g

## 조리 방법

1. 1. 사과는 8조각으로 잘라 씨를 제거한다.
2. 2. 청고추, 홍고추는 어슷하게 잘라준다.
3. 3. 양파도 먹기 좋은 크기로 잘라준다.
4. 4. 사과는 식초 물에 살짝 절여준다.
5. 5. 양념장을 만든다.
6. 6. 양념장에 준비된 재료를 모두 섞어 완성한다.

## 영양 정보

- 칼로리: 291.2 kcal
- 탄수화물: 64.9g
- 단백질: 1.9g
- 지방: 2.7g
- 나트륨: 736.7mg
