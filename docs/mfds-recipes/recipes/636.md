# 우렁된장소스 배추롤

우렁된장소스 배추롤은(는) 일품 요리로, 찌기 조리법을 사용합니다.

## 재료

- 돼지고기(50g)
- 배춧잎(5장)
- 부추(30g)
- 쌀(100g)
- 당근(20g)
- 표고버섯(2개)
- 양파(50g)
- 애호박(1/2개)
- 우렁이(100g)
- 된장(30g)

## 조리 방법

1. 1. 끓는 물에 배춧잎을 데친다.
2. 2. 당근, 호박, 표고버섯, 양파는
입자있게 다진다.
3. 3. 돼지고기를 썰어 불린 쌀, 다진 야채와
함께 밥을 짓는다.
4. 4. 데친 배춧잎에 지어놓은 밥을 올려
돌돌만다.
5. 5. 말아 놓은 배춧잎을 부추로 묶는다.
6. 6. 찜통에 김이 오르면 ?번을 넣고 약 5분
정도 찐다.

## 영양 정보

- 칼로리: 326.3 kcal
- 탄수화물: 43g
- 단백질: 22.3g
- 지방: 7.2g
- 나트륨: 976mg
