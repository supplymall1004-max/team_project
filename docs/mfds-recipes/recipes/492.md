# 토마토덮밥

토마토덮밥은(는) 밥 요리로, 볶기 조리법을 사용합니다.

## 재료

- 쌀(100g)
- 가지(50g)
- 새우(20g)
- - 소스 : 양파(20g)
- 마늘(10g)
- 올리브오일(10g)
- 토마토소스(20g)
- 케첩(20g)
- 토마토(80g)
- 설탕(5g)
- - 곁들임채소 : 어린잎(5g)

## 조리 방법

1. 1. 쌀은 물에 불려 밥을 짓고, 가지는 길게
썰어 팬에 구워 놓는다.
2. 2. 양파와 마늘은 잘게 다져 준비한다.
3. 3. 토마토는 입자있게 다져 놓는다.
4. 4. 새우는 정종을 넣어 데쳐 껍질을 벗겨
준비한다.
5. 5. 팬에 올리브 오일을 넣고 양파와 마늘을
먼저 볶는다.
6. 6. 볶아지는 양파와 마늘에 토마토소스와
케첩을 넣어 볶다가 토마토와 설탕을
넣어 볶다가 졸이고, 밥을 그릇에 담고,
구운 가지와 새우를 올린 후 졸여 놓은
토마토소스를 위에 올린 후 어린잎을
살짝 올려 완성한다.

## 영양 정보

- 칼로리: 379.3 kcal
- 탄수화물: 71.1g
- 단백질: 15.6g
- 지방: 3.6g
- 나트륨: 181.2mg
