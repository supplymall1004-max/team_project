# 닭고기월남쌈

닭고기월남쌈은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 닭고기살(100g)
- 로즈마리(5g)
- 양파(20g)
- 소금(0.5g)
- 후춧가루(0.02g)
- 오이(30g)
- 파프리카(30g)
- 당근(30g)
- 라이스페이퍼(20g)
- 녹말가루(30g)
- 식용유(20g)
- - 소스 : 다진 마늘(10g)
- 고춧가루(10g)
- 설탕(20g)
- 식초(20g)
- 레몬즙(10g)

## 조리 방법

1. 1. 닭고기살에 로즈마리를 뿌려 약 3시간
이상 숙성을 시킨다.
2. 2. 냄비에 물을 1/3쯤 넣고, 양파채를 썰어
넣어 끓으면, 숙성된 닭고기살을 넣고
약 20분 정도 삶아 건진다.
3. 3. 삶아 건진 닭고기살을 한 김 빼고, 길게
손으로 찢어 소금과 후춧가루에 살짝
무치고, 오이와 파프리카는 씨를 제거
하고 채를 썰고, 당근도 오이채의 길이에
맞게 채를 썰어 준비해 놓는다.
4. 4. 마늘을 다지고, 고춧가루와 설탕, 식초,
레몬즙을 넣어 소스를 만든다.
5. 5. 뜨거운 물에 라이스페이퍼를 살짝 넣어
바로 건진다.
6. 6. 건져 놓은 라이스페이퍼에 준비한
닭고기살과 파프리카, 오이, 당근 넣고
돌돌 말아, 녹말가루를 살짝 입혀, 팬에
굴리듯 익혀 완성하고, 준비해 놓은
소스와 함께 제공한다.

## 영양 정보

- 칼로리: 353.7 kcal
- 탄수화물: 40.1g
- 단백질: 28.5g
- 지방: 8.8g
- 나트륨: 197mg
