# 유부완자탕

유부완자탕은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 유부(50g)
- 두부(30g)
- 새우(50g)
- 배추(30g)
- 표고버섯(30g)
- 양파(30g)
- 당근(30g)
- 마늘(10g)
- 저염소금(0.5g)
- 후춧가루(0.05g)
- 미나리(20g)
- 육수(200g)
- 저염간장(20g)

## 조리 방법

1. 1. 유부는 1/4 정도를 잘라 준비한다.
2. 2. 잘라진 유부는 끓는 물에 약 20초 정도
데쳐 물기를 짜 놓는다.
3. 3. 새우는 내장과 껍질을 제거 하고 곱게
으깨 준비한다.
4. 4. 배추와 미나리는 끓는 물에 데치고,
표고버섯은 뜨거운 물에 불려 준비
한다.
5. 5. 데친 배추와 표고버섯을 입자있게
다지고, 양파와 당근도 입자있게 다져
준비한다.
6. 6. 두부는 물기를 짜 놓은 후 새우(③)와
다진 야채(⑤), 마늘, 소금, 후춧가루
를 넣고 골고루 섞는다. 유부(②)에
속을 넣고 미나리로 묶은 후, 끓는
육수에 유부주머니를 넣고 간장으로
간을 하여 완성한다.

## 영양 정보

- 칼로리: 297.5 kcal
- 탄수화물: 10.1g
- 단백질: 24.8g
- 지방: 17.6g
- 나트륨: 325.3mg
