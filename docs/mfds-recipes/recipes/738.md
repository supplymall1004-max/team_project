# 찬밥김치달걀찜

찬밥김치달걀찜은(는) 반찬 요리로, 찌기 조리법을 사용합니다.

## 재료

- 달걀 50g
- 밥 30g(쌀 13g)
- 김장김치 20g
- 당근 5g
- 양파 5g
- 파 5g
- 새우젓국물 2g
- 물 100g

## 조리 방법

1. 1. 배추김치는 속을 털어낸다.
2. 2. 1의 김치를 다진 다음 물기를 짠다.
3. 3. 당근, 양파, 파는 잘게 다진다.
4. 4. 달걀은 흰자와 노른자를 체에 거른다.
5. 5. 달걀에 물 반컵을 넣고 잘 저은 후 1,2의 재료와 찬밥을 섞고 새우젓으로 간을 한다.
6. 6. 5를 찜통에 넣고 찐다.

## 영양 정보

- 칼로리: 134.2 kcal
- 탄수화물: 12.2g
- 단백질: 7.9g
- 지방: 5.7g
- 나트륨: 465.7mg
