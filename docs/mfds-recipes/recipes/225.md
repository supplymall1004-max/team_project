# 깻잎장아찌롤

깻잎장아찌롤은(는) 밥 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 쌀(200g)
- 깻잎장아찌(30g)
- 김(5g)
- 새싹(20g)
- 밥 밑간 참기름(5g)
- 참깨(3g)
- 허브타르타르드레싱 양파(20g)
- 피클(20g)
- 삶은 달걀(60g)
- 마요네즈(20g)
- 머스터드소스 설탕(3g)
- 물(5g)
- 머스터드소스(15g)

## 조리 방법

1. 1. 쌀을 씻어 밥을 지은 뒤 밥 밑간을
한다.
2. 2. 양파, 피클, 삶은 달걀은 다진 뒤
마요네즈에 섞어
허브타르타르드레싱을 만든다.
3. 3. 김 위에 양쪽으로 밥을 펴고
허브타르타르드레싱을 위에
올린 후 만다.
4. 4. 깻잎장아찌를 바닥에 깔고 말아놓은
롤을 올린 후 한 번 더 만다.
5. 5. 머스타드소스를 만든다.
6. 6. 한입 크기로 롤을 썬 뒤 새싹과
소스를 곁들여 마무리한다.

## 영양 정보

- 칼로리: 375.2 kcal
- 탄수화물: 18.4g
- 단백질: 2.6g
- 지방: 3.4g
- 나트륨: 750.7mg
