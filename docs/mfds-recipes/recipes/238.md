# 꽁치채소말이

꽁치채소말이은(는) 반찬 요리로, 굽기 조리법을 사용합니다.

## 재료

- 재료 꽁치살(200g)
- 마늘종(30g)
- 양파(40g)
- 파프리카(100g)
- 밀가루(30g)
- 깻잎(3g)
- 무순(8g)
- 새싹(8g)
- 오렌지소스 오렌지(200g)
- 레몬즙(15g)
- 설탕(0.3g)
- 고운 소금(0.3g)

## 조리 방법

1. 1. 꽁치는 머리와 꼬리를 떼어 내고
포 뜨듯이 길게 살만 발라내 잔뼈를
발라 쌀뜨물에 담갔다 건져 물기를
제거한다.
2. 2. 오렌지를 간 뒤 레몬즙과 설탕,
소금을 섞어 오렌지소스를 만든다.
3. 3. 마늘종은 꽁치 폭에 맞춰 썰고,
양파와 파프리카도 채 썬다.
4. 4. 수분을 제거한 꽁치에 밀가루를
묻혀 깻잎→마늘종→양파→
파프리카 순으로 올려 돌돌 말아
꼬치로 꽂아 고정한다.
5. 5. 밀가루를 묻히고 달군 팬에
앞뒤가 노릇노릇하게 익힌다.
6. 6. 소스를 팬에 넣고 끓이다가 꽁치
꼬치를 넣고 졸인 뒤 그릇에 무순,
새싹과 함께 담아 마무리한다.

## 영양 정보

- 칼로리: 523 kcal
- 탄수화물: 10.3g
- 단백질: 13.7g
- 지방: 12.6g
- 나트륨: 93.7mg
