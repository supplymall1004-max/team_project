# 토마토라면

토마토라면은(는) 일품 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 재료 애호박(30g)
- 양파(20g)
- 청양고추(5g)
- 대파(5g)
- 토마토(300g)
- 라면스프(3g)
- 라면(100g)
- 육수 다시마(2.5g)
- 멸치(2.5g)
- 무(10g)
- 물(250g)

## 조리 방법

1. 1. 냄비에 육수 재료를 넣고 10분간
끓여 건더기를 건진다.
2. 2. 애호박은 돌려 깎아 채 썰고, 양파도
채 썰고, 청양고추와 대파는
어슷 썬다.
3. 3. 토마토는 200g은 잘게 다지고,
100g은 두툼하게 썬다.
4. 4. 육수에 라면스프와 라면을 넣고
살짝 익힌 뒤 양파, 애호박, 다진
토마토를 넣고 끓인다.
5. 5. 면이 다 익으면 두툼하게 썬
토마토를 넣고 10초간 더 끓인다.
6. 6. 대파와 청양고추를 올려 마무리한다.

## 영양 정보

- 칼로리: 190.4 kcal
- 탄수화물: 4.9g
- 단백질: 1.1g
- 지방: 1g
- 나트륨: 423.2mg
