# 황태두부무국

황태두부무국은(는) 국&찌개 요리로, 끓이기 조리법을 사용합니다.

## 재료

- 황태 30g
- 두부 50g
- 함초 25g
- 무 100g
- 청고추 20g
- 홍고추 15g
- 쪽파 10g
- 흰후추 1g
- 어간장 7g
- 대파뿌리 7g

## 조리 방법

1. 1. 두부는 사방 1cm로 잘라주고, 무는 반은
나박썰기 하고 반은 국물을 낸다.
2. 2. 두부는 사방 1cm로 잘라주고, 무는 반은
나박썰기 하고 반은 국물을 낸다.
3. 3. 황태는 먹기 좋게 잘라준다.
4. 4. 황태, 대파뿌리, 함초, 무를 넣고 국물을 만든다.
5. 5. 만들어진 국물에 함초와 대파뿌리를 건져내고
어간장으로 간을 해준다. 국물을 만들고 남은
무는 나박썰기 해서 국에 넣어준다.
6. 6. 마지막에 두부를 넣어 끓여주고 먹기 직전에
청홍고추와 쪽파를 올려 완성한다.

## 영양 정보

- 칼로리: 53.5 kcal
- 탄수화물: 1g
- 단백질: 7.7g
- 지방: 2.1g
- 나트륨: 298.5mg
