# 가지나물냉국

가지나물냉국은(는) 반찬 요리로, 끓이기 조리법을 사용합니다.

## 재료

- - 주재료 : 가지 50g
- 다진 청양고추 2.5g
- 다진 홍고추 2.5g
- 다진 실파 2.5g
- 다진 마늘 1.25g
- 국간장 3g - 국물 : 다시마국물 100g(건다시마 5g)
- 사과식초 1.25g
- 감식초 1.25g
- 소금 0.5g
- 통깨 1g

## 조리 방법

1. 1. 가지는 반 갈라 4등분하여 찜통에 5분간 찐 후 한 김 식혀준다.
2. 2. 식은 가지를 새끼손가락 굵기로 찢는다.
3. 3. 다진 고추, 실파, 마늘, 국간장을 넣어 양념장을 만든다.
4. 4. 찐 가지에 양념장을 넣고 조물조물 무친다.
5. 5. 차게 한 다시마국물 반컵에 식초, 죽염을 넣어 간한다.
6. 6. 유리 그릇에 무친 가지를 담고 5를 부은 뒤 얼음을 띄우고 통깨를 뿌린다.

## 영양 정보

- 칼로리: 28.7 kcal
- 탄수화물: 4.9g
- 단백질: 1.5g
- 지방: 0.7g
- 나트륨: 540mg
