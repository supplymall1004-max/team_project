# 석류 보쌈김치

석류 보쌈김치은(는) 반찬 요리로, 기타 조리법을 사용합니다.

## 재료

- ●주재료 : 배추 70g(1/10개)
- 석류즙 10g(2작은술)
- 소금 5g(1/2작은술)
- ●양념 : 무 10g(3cm)
- 미나리 10g
- 쪽파 10g(1개)
- 다진 마늘 5g(1작은술)
- 다진 생강 5g(1작은술)
- 새우젓국 15g(1큰술)

## 조리 방법

1. 1. 배추를 소금물에 하루 동안 절인 후 물기를 빼고 석류는 즙을 낸다.
2. 2. 무는 채 썰고, 쪽파와 미나리는 4cm로 자른다.
3. 3. 무채, 미나리, 쪽파, 다진 마늘, 다진 생강, 새우젓국을 넣고 섞어 소를 만들고 절인 배추에 소를 넣고 오므려 싼다. 석류즙에 소를 채운 배추를 담가 완성한다.

## 영양 정보

- 칼로리: 24.5 kcal
- 탄수화물: 4.7g
- 단백질: 1.4g
- 지방: 0g
- 나트륨: 171.3mg
